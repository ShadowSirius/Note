# 日語生活情境會話

本章整理了影片中所涵蓋的五大常用生活與旅行情境（交通、機場、飯店、購物、餐飲），彙整了實用單字與核心對白演練。

> [!tip] 句子結構色彩：🟢主語/主題（#2ECC71） \| 🔵核心/謂語（#3498DB） \| 🟠修飾語/賓語（#E67E22） \| 🔴時間/場所/副詞（#E74C3C）

---

## 🚇 情境一：交通與車站 (電車・駅)

### 1. 核心單字
- <ruby>切符売り場<rt>きっぷうりば (kippuriba)</rt></ruby>：售票處
- <ruby>みどりの窓口<rt>みどりのまどぐち (midorinomadoguchi)</rt></ruby>：綠色窗口（JR 人工售票處）
- <ruby>券売機<rt>けんばいき (kenbaiki)</rt></ruby>：自動售票機
- <ruby>切符<rt>きっぷ (kippu)</rt></ruby>：車票
- <ruby>乗車券<rt>じょうしゃけん (joushaken)</rt></ruby>：普通車票
- <ruby>特急券<rt>とうきゅうけん (tokkyuuken)</rt></ruby>：特急車票
- <ruby>改札<rt>かいさつ (kaisatsu)</rt></ruby>：剪票口
- <ruby>出口<rt>でぐち (deguchi)</rt></ruby>：出口（東口、西口、北口、南口）
- １番ホーム / １番線：一號月台
- 新幹線 / 特急 / 快速 / 急行 / 普通

### 2. 實用問詢句
- <span style="color:#2ECC71">トイレは</span><span style="color:#3498DB">どこですか</span>。
  - 中文：廁所在哪裡？
- <span style="color:#2ECC71"><ruby>新宿行き<rt>しんじゅくゆき</rt></ruby>は</span><span style="color:#3498DB"><ruby>何番<rt>なんばん</rt></ruby>ホームですか</span>。
  - 中文：去新宿的在幾號月台？
- <span style="color:#E67E22"><ruby>切符<rt>きっぷ</rt></ruby>を</span><span style="color:#3498DB"><ruby>買<rt>か</rt></ruby>いたいんですが</span>、どうしたらいいですか。
  - 中文：我想買票，該怎麼做呢？
- <span style="color:#E67E22"><ruby>荷物<rt>にもつ</rt></ruby>を</span><span style="color:#E74C3C"><ruby>電車<rt>でんしゃ</rt></ruby>に</span><span style="color:#3498DB"><ruby>忘<rt>わす</rt></ruby>れてしまいました</span>。
  - 中文：我把行李忘在電車上了。
- <span style="color:#E67E22"><ruby>切符<rt>きっぷ</rt></ruby>を</span><span style="color:#3498DB">なくしてしまいました</span>。
  - 中文：我把車票弄丟了。

### 3. 對話演練（詢問搭乘列車）
- 旅客：<span style="color:#3498DB">すみません</span>、<span style="color:#2ECC71">この<ruby>電車<rt>でんしゃ</rt></ruby>は</span><span style="color:#E74C3C"><ruby>神戸<rt>こうべ</rt></ruby>に</span><span style="color:#3498DB"><ruby>停<rt>と</rt></ruby>まりますか</span>。
  （不好意思，這班電車會停靠神戶嗎？）
- 站員：<span style="color:#3498DB">はい</span>。でも、<span style="color:#2ECC71"><ruby>次<rt>つぎ</rt></ruby>の<ruby>快速<rt>かいそく</rt></ruby>のほうが</span><span style="color:#3498DB"><ruby>早<rt>はや</rt></ruby>いですよ</span>。
  （會的。不過下一班快速列車比較快喔。）
- 旅客：<span style="color:#3498DB">そうですか</span>。<span style="color:#2ECC71"><ruby>何番<rt>なんばん</rt></ruby>ホーム</span><span style="color:#3498DB">ですか</span>。
  （這樣啊。是在幾號月台呢？）
- 站員：<span style="color:#E74C3C">５番ホーム</span><span style="color:#3498DB">です</span>。<span style="color:#E74C3C">５分後に</span><span style="color:#3498DB"><ruby>発車<rt>はっしゃ</rt></ruby>します</span>。
  （５號月台。５分鐘後發車。）
- 旅客：<span style="color:#3498DB">ありがとうございます</span>。
  （謝謝您。）

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 試着（しちゃく）してみてもいいですか。
> （可以試穿看看嗎？）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 使（つか）って | **使（つか）って**みてもいいですか。 | 可以試用看看嗎？ |
| 食（た）べて | **食（た）べて**みてもいいですか。 | 可以試吃看看嗎？ |
| 飲（の）んで | **飲（の）んで**みてもいいですか。 | 可以試喝看看嗎？ |
| 履（は）いて | **履（は）いて**みてもいいですか。 | 可以試穿鞋子看看嗎？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 普通形（ふつうけい） | 試着（しちゃく）してみても**いい？** | 可以試穿看看嗎？（常體） |
| 許可否定（きょかひてい） | 試着（しちゃく）しては**いけませんか**。 | 不能試穿看看嗎？ |
| 過去語氣（かこごき） | 試着（しちゃく）してみても**よかったですか**。 | 剛才試穿看看也可以嗎？ |
| 請求允許（せいきゅういんきょ） | 試着（しちゃく）**させていただけませんか**。 | 能讓我試穿看看嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| これ、試着（しちゃく）してみてもいいですか。 | はい、いいですよ。どうぞ。 | 這個可以試穿看看嗎？ / 好的，可以啊。請。 |
| すみません、この靴（くつ）を履（は）いてみてもいいですか。 | はい、大丈夫（だいじょうぶ）ですよ。こちらでどうぞ。 | 不好意思，這雙鞋可以試穿看看嗎？ / 好的，沒問題。請在這邊試穿。 |
| このケーキ、食（た）べてみてもいいですか。 | はい、どうぞ。召（め）し上が（あ）ってください。 | 這個蛋糕可以試吃看看嗎？ / 好的，請用。請吃吧。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心（かくしん） | いいですか。 | 可以嗎？ |
| +助詞（じょし） | みてもいいですか。 | 看看可以嗎？ |
| +動作（どうさ） | 試着（しちゃく）してみてもいいですか。 | 試穿看看可以嗎？ |
| +丁寧（ていねい） | すみません、これを試着（しちゃく）してみてもいいですか。 | 不好意思，這個可以試穿看看嗎？ |
