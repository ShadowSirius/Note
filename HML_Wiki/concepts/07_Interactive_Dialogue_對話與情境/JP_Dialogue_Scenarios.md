# 日語生活情境會話

本章整理了影片中所涵蓋的五大常用生活與旅行情境（交通、機場、飯店、購物、餐飲），彙整了實用單字與核心對白演練。

---

## 🚇 情境一：交通與車站 (電車・駅)

### 1. 核心單字
- <ruby>切符売り場<rt>きっぷうりば (kippuriba)</rt></ruby>：售票處
- <ruby>みどりの窓口<rt>みどりのまどぐち (midorinomadoguchi)</rt></ruby>：綠色窗口（JR 人工售票處）
- <ruby>券売機<rt>けんばいき (kenbaiki)</rt></ruby>：自動售票機
- <ruby>切符<rt>きっぷ (kippu)</rt></ruby>：車票
- <ruby>乗車券<rt>じょうしゃけん (joushaken)</rt></ruby>：普通車票
- <ruby>特急券<rt>とっきゅうけん (tokkyuuken)</rt></ruby>：特急車票
- <ruby>改札<rt>かいさつ (kaisatsu)</rt></ruby>：剪票口
- <ruby>出口<rt>でぐち (deguchi)</rt></ruby>：出口（<ruby>東口<rt>ひがしぐち (higashiguchi)</rt></ruby>、<ruby>西口<rt>にしぐち (nishiguchi)</rt></ruby>、<ruby>北口<rt>きたぐち (kitaguchi)</rt></ruby>、<ruby>南口<rt>みなみぐち (minamiguchi)</rt></ruby>）
- <ruby>１番ホーム<rt>いちばんほーむ (ichiban ho-mu)</rt></ruby> / <ruby>１番線<rt>いちばんせん (ichibansen)</rt></ruby>：一號月台
- <ruby>新幹線<rt>しんかんせん (shinkansen)</rt></ruby>：新幹線
- <ruby>特急<rt>とっきゅう (tokkyuu)</rt></ruby>：特急列車
- <ruby>快速<rt>かいそく (kaisoku)</rt></ruby>：快速列車
- <ruby>急行<rt>きゅうこう (kyuukou)</rt></ruby>：急行列車
- <ruby>普通<rt>ふつう (futsuu)</rt></ruby>：普通車（各站皆停）

### 2. 實用問詢句
- トイレはどこですか。
  - 中文：廁所在哪裡？
- <ruby>新宿行き<rt>しんじゅくゆき (shinjukuyuki)</rt></ruby>は<ruby>何番ホーム<rt>なんばんほーむ (nanban ho-mu)</rt></ruby>ですか。
  - 中文：去新宿的在幾號月台？
- <ruby>切符<rt>きっぷ (kippu)</rt></ruby>を<ruby>買<rt>か (ka)</rt></ruby>いたいんですが、どうしたらいいですか。
  - 中文：我想買票，該怎麼做呢？
- <ruby>荷物<rt>にもつ (nimotsu)</rt></ruby>を<ruby>電車<rt>でんしゃ (densha)</rt></ruby>に<ruby>忘<rt>わす (wasu)</rt></ruby>れてしまいました。
  - 中文：我把行李忘在電車上了。
- <ruby>切符<rt>きっぷ (kippu)</rt></ruby>をなくしてしまいました。
  - 中文：我把車票弄丟了。

### 3. 對話演練（詢問搭乘列車）
- 旅客：すみません、この<ruby>電車<rt>でんしゃ (densha)</rt></ruby>は<ruby>神戸<rt>こうべ (koube)</rt></ruby>に<ruby>停<rt>と (to)</rt></ruby>まりますか。
  （不好意思，這班電車會停靠神戶嗎？）
- 站員：はい。でも、<ruby>次<rt>つぎ (tsugi)</rt></ruby>の<ruby>快速<rt>かいそく (kaisoku)</rt></ruby>のほうが<ruby>早<rt>はや (haya)</rt></ruby>いですよ。
  （會的。不過下一班快速列車比較快喔。）
- 旅客：そうですか。<ruby>何番ホーム<rt>なんばんほーむ (nanban ho-mu)</rt></ruby>ですか。
  （這樣啊。是在幾號月台呢？）
- 站員：<ruby>５番ホーム<rt>ごばんほーむ (goban ho-mu)</rt></ruby>です。<ruby>５分後<rt>ごふんご (gofungo)</rt></ruby>に<ruby>発車<rt>はっしゃ (hassha)</rt></ruby>します。
  （５號月台。５分鐘後發車。）
- 旅客：ありがとうございます。
  （謝謝您。）

### 2. 實用問詢句
- 毛布をもらえませんか。
  - 中文：可以給我一條毯子嗎？
- お<ruby>水<rt>みず (mizu)</rt></ruby>をもらえませんか。
  - 中文：可以給我一杯水嗎？
- イヤホンの<ruby>調子<rt>ちょうし (choushi)</rt></ruby>が<ruby>悪<rt>わる (waru)</rt></ruby>いんですが。
  - 中文：耳機的狀態好像不太好（壞了）。
- <ruby>入国カード<rt>にゅうこくかーど (nyuukoku ka-do)</rt></ruby>の<ruby>書き方<rt>かきかた (kakikata)</rt></ruby>を<ruby>教<rt>おしえ (oshie)</rt></ruby>てもらえませんか。
  - 中文：可以教我入境卡怎麼寫嗎？
- 私の<ruby>荷物<rt>にもつ (nimotsu)</rt></ruby>が<ruby>出<rt>で (de)</rt></ruby>てこないんですが。
  - 中文：我的行李沒有出來。

### 3. 對話演練（登機櫃台報到）
- 地勤：パスポートと<ruby>予約票<rt>よやくひょう (yoyakuhyou)</rt></ruby>、お<ruby>願<rt>ねが (nega)</rt></ruby>いします。
  （請給我您的護照與預約單。）
- 旅客：はい、どうぞ。
  （好的，請。）
- 地勤：お<ruby>預<rt>あず (azu)</rt></ruby>けの<ruby>荷物<rt>にもつ (nimotsu)</rt></ruby>案はありますか。
  （您有要託運的行李嗎？）
- 旅客：これとこれをお<ruby>願<rt>ねが (nega)</rt></ruby>いします。
  （這個跟這個，拜託了。）
- 地勤：スーツケースの<ruby>中<rt>なか (naka)</rt></ruby>にライター<ruby>等<rt>とう (tou)</rt></ruby>は<ruby>入<rt>はい (hai)</rt></ruby>っていませんか。
  （行李箱裡面沒有放打火機之類的東西吧？）
- 旅客：はい、<ruby>大丈夫<rt>だいじょうぶ (daijoubu)</rt></ruby>です。
  （是的，沒問題。）
- 地勤：<ruby>窓側<rt>まどがわ (madogawa)</rt></ruby>と<ruby>通路側<rt>つうろがわ (tsuurogawa)</rt></ruby>、どちらがよろしいですか。
  （靠窗還是靠走道，哪一個比較好呢？）
- 旅客：<ruby>窓側<rt>まどがわ (madogawa)</rt></ruby>をお<ruby>願<rt>ねが (nega)</rt></ruby>いします。
  （靠窗，拜託了。）
- 地勤：それでは、８<ruby>時<rt>じ (ji)</rt></ruby>までに１０<ruby>番搭乗口<rt>ばんとうじょうぐち (ban toujouguchi)</rt></ruby>へお<ruby>越<rt>こ (ko)</rt></ruby>しください。
  （那麼，請在８點前前往１０號登機門。）
- 旅客：わかりました。
  （好的，我明白了。）

- 旅客：ありがとうございます。
  （謝謝。）

##  Ramen ＆ Cafe 情境五：餐飲與餐廳 (レストラン・飲食店)

### 1. 核心單字
- レストラン：餐廳
- <ruby>居酒屋<rt>いざかや (izakaya)</rt></ruby>：居酒屋
- <ruby>喫茶店<rt>きっさてん (kissaten)</rt></ruby>：日式老咖啡廳
- カフェ：西式咖啡廳
- <ruby>食券<rt>しょっけん (shokken)</rt></ruby>：餐券
- <ruby>持ち帰り<rt>もちかえ (mochikae)</rt></ruby> / テイクアウト：外帶
- お<ruby>酒<rt>さけ (sake)</rt></ruby> / ビール / <ruby>日本酒<rt>にほんしゅ (nihonshu)</rt></ruby> / <ruby>梅酒<rt>うめしゅ (umeshu)</rt></ruby> / ワイン / ジュース
- <ruby>禁煙席<rt>きんえんせき (kinenseki)</rt></ruby> / <ruby>喫煙席<rt>きつえんせき (kitsuenseki)</rt></ruby>：禁菸席 / 吸菸席
- <ruby>相席<rt>あいせき (aiseki)</rt></ruby>：併桌
- オススメ的<ruby>料理<rt>りょうり (ryouri)</rt></ruby>：推薦料理
- ラストオーダー：最後點餐時間
- <ruby>醤油<rt>しょうゆ (shouyu)</rt></ruby> / <ruby>塩<rt>しお (shio)</rt></ruby> / わさび / <ruby>唐辛子<rt>とうがらし (tougarashi)</rt></ruby>：醬油 / 鹽 / 哇沙比 / 辣椒粉
- <ruby>取り皿<rt>とりざら (torizara)</rt></ruby>：小分裝碟
- おしぼり：濕紙巾 / 濕毛巾

### 2. 實用問詢句
- <ruby>注文<rt>ちゅうもん (chuumon)</rt></ruby>したいんですが。
  - 中文：我想點餐。
- お<ruby>水<rt>みず (mizu)</rt></ruby>のおかわりをください。
  - 中文：請再給我一杯水。
- ビールのおかわりをください。
  - 中文：請再給我一杯啤酒。
- テイクアウトはできますか。
  - 中文：可以外帶嗎？
- <ruby>席<rt>せき (seki)</rt></ruby>を替わってもいいですか。
  - 中文：可以換位子嗎？
- ここ<ruby>空<rt>あ (a)</rt></ruby>いていますか。
  - 中文：這裡有空位嗎（我可以坐這嗎）？
- <ruby>卵<rt>たまご (tamago)</rt></ruby>が<ruby>入<rt>はい (hai)</rt></ruby>っていますか。
  - 中文：這裡面有加蛋嗎？
- <ruby>卵<rt>たまご (tamago)</rt></ruby>を<ruby>入<rt>い (i)</rt></ruby>れないでください。
  - 中文：請不要加蛋。
- すみません、お<ruby>箸<rt>はし (hashi)</rt></ruby>を落としてしまったので、新しいのをください。
  - 中文：不好意思，我不小心把筷子掉了，請給我一雙新的。

### 3. 對話演練（點餐與推薦）
- 顧客：<ruby>注文<rt>ちゅうもん (chuumon)</rt></ruby>をお<ruby>願<rt>ねが (nega)</rt></ruby>いします。
  （麻煩幫我點餐。）
- 店員：はい。ご注文はお決まりですか。
  （好的。您決定好點什麼了嗎？）
- 顧客：これを一つと、これを一つください。あと、オススメ的<ruby>料理<rt>りょうり (ryouri)</rt></ruby>はありますか。
  （請給我這個一個，還有這個一個。另外，有推薦的料理嗎？）
- 店員：<ruby>今日<rt>きょう (kyou)</rt></ruby>はエビのパスタがおすすめです。
  （今天推薦鮮蝦義大利麵。）
- 顧客：じゃ、それを一つお<ruby>願<rt>ねが (nega)</rt></ruby>いします。
  （那，就給我來一份那個吧。）
- 店員：かしこまりました。少々お待ちください。
  （好的，我知道了。請稍候。）

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>試着<rt>しちゃく (shichaku)</rt></ruby>してみてもいいですか。
> 可以試穿看看嗎？

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| <ruby>使<rt>つか (tsuka)</rt></ruby>う | **<ruby>使<rt>つか (tsuka)</rt></ruby>って**みてもいいですか。 |
| <ruby>食<rt>た (ta)</rt></ruby>べる | **<ruby>食<rt>た (ta)</rt></ruby>べて**みてもいいですか。 |
| <ruby>飲<rt>の (no)</rt></ruby>む | **<ruby>飲<rt>の (no)</rt></ruby>んで**みてもいいですか。 |
| <ruby>履<rt>は (ha)</rt></ruby>く | **<ruby>履<rt>は (ha)</rt></ruby>いて**みてもいいですか。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 普通形 | 試着してみてもいい**？** |
| → 許可否定（禁止） | 試着しては**いけませんか**。 |
| → 過去形 | 試着してみてもよ**かったですか**。 |
| → 請求允許（使役受給） | 試着させて**いただけませんか**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| これ、試着してみてもいいですか。 | はい、いいですよ。どうぞ。 |
| すみません、この靴を履いてみてもいいですか。 | はい、大丈夫ですよ。こちらでどうぞ。 |
| このケーキ、食べてみてもいいですか。 | はい、どうぞ。召し上がってください。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | いいですか。 |
| +みても | みてもいいですか。 |
| +試着して | 試着してみてもいいですか。 |
| +すみません、これを | すみません、これを試着してみてもいいですか。 |
