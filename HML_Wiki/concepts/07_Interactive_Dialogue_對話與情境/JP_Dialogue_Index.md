# 模組索引：07 Interactive Dialogue 對話與情境

## 模組定位

本模組收錄「情境實戰對話」類卡片，聚焦於將已學過的文法、詞彙整合進真實生活場景的完整對話中，訓練學習者在旅行、飯店、餐廳、超市、學校、日常社交等具體情境下的即時應答能力。與 01～06 模組（文法點、詞彙）不同，本模組不逐條講解單一文法規則，而是以「場景 → 對白 → 句型解說 → FSI 口說訓練」的方式，讓學習者練習連續多輪的自然對話節奏與敬語應對。

## 學習建議路徑

1. 先完成 01～06 模組的基礎文法與詞彙卡，掌握「て形」「〜たいです」「〜てもらえますか」等句型。
2. 進入本模組後，先讀 [[JP_Dialogue_Travel_Hotel]] 建立情境對話的整體語感。
3. 再依實際需求選讀新增的三張情境卡，針對飯店、餐廳、超市三大服務業場景做針對性口說訓練。
4. 每張卡片底部的 🎯 FSI Pattern Drill 建議搭配朗讀練習，強化敬語與句型的即時反應速度。
