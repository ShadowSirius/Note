# 〜ましょうか

## 規則與接續
- `動詞ます形（去掉ます） + ましょうか`

## 用法說明
表示說話者主動向聽話者提議提供協助（我來幫忙做某事吧？），或用於探詢對方的意見。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 中文：我幫您拍照吧？

- 日文：<span style="color:#3498DB">お弁当を温めましょうか。</span>

- 日文：<span style="color:#3498DB">迎えに行きましょうか。</span>
- 中文：我順道去接您吧？
- 日文：<span style="color:#3498DB">掃除をしましょうか。</span>
- 中文：我來打掃吧？
- 日文：<span style="color:#3498DB">クーラーを消しましょうか。</span>

## 關係說明
- 與詢問對方意願的邀請句型 [[JP_Grammar_34_masen_ka|〜ませんか @sibling_of]] 互為姊妹文法，一為「邀請」，一為「主動提議協助」。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 荷物（にもつ）を持ち（もち）ましょうか。
> （我幫您提行李吧？）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 写真（しゃしん）を撮り（とり） | **写真（しゃしん）を撮り（と）りましょうか**。 | 我幫您拍照吧？ |
| 手伝い（てつだい） | **手伝（てつだ）いましょうか**。 | 我幫忙您吧？ |
| 案内し（あんないし） | **案内（あんない）しましょうか**。 | 我來為您帶路吧？ |
| 迎えに行き（むかえにいき） | **迎（むか）えに行（い）きましょうか**。 | 我順道去接您吧？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 指示請求（しじせいきゅう） | 荷物（にもつ）を**持（も）ってください**。 | 請提行李。 |
| 委婉請求（えいわんせいきゅう） | 荷物（にもつ）を**持（も）ってもらえませんか**。 | 能幫我提行李嗎？ |
| 恩恵提供（おんけいていきょう） | 荷物（にもつ）を**持（も）ってあげますよ**。 | 我幫你提行李吧。（對熟人/朋友） |
| 謙遜表達（けんそんひょうげん） | 荷物（にもつ）を**お持ち（もち）しましょうか**。 | 讓我為您提行李吧。（敬語） |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 荷物（にもつ）を持ち（もち）ましょうか。 | すみません、お願（ねが）いします。 | 我幫您提行李吧？ / 謝謝，麻煩您了。 |
| 手伝（てつだ）いましょうか。 | はい、お願（ねが）いします。 | 我幫忙您吧？ / 好的，麻煩您了。 |
| 迎え（むかえ）に行（い）きましょうか。 | いいえ、大丈夫（だいじょうぶ）です。 | 我順道去接您吧？ / 不用了，沒關係的。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心（かくしん） | ましょうか。 | 我來做吧？ |
| +動作（どうさ） | 持ち（もち）ましょうか。 | 我來提吧？ |
| +目的語（もくてきご） | 荷物（にもつ）を持ち（もち）ましょうか。 | 我幫您提行李吧？ |
| +主語（しゅご） | 私（わたし）が荷物（にもつ）を持ち（もち）ましょうか。 | 我來幫您提行李吧？ |
