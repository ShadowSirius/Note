# 〜ましょうか

## 規則與接續
- `動詞ます形（去掉ます） + ましょうか`

## 用法說明
表示說話者主動向聽話者提議提供協助（我來幫忙做某事吧？），或用於探詢對方的意見。

## 實用例句
- 日文：<ruby>写真<rt>しゃしん (shashin)</rt></ruby>を<ruby>撮<rt>と (to)</rt></ruby>りましょうか。
- 中文：我幫您拍照吧？
- 日文：<ruby>手伝<rt>てつだ (tetsuda)</rt></ruby>いましょうか。
- 中文：我幫忙您吧？
- 日文：<ruby>荷物<rt>にもつ (nimotsu)</rt></ruby>を<ruby>持<rt>も (mo)</rt></ruby>ちましょうか。
- 中文：我幫您提行李吧？
- 日文：<ruby>案内<rt>あんない (annai)</rt></ruby>しましょうか。
- 中文：我來為您帶路/導覽吧？
- 日文：お<ruby>弁当<rt>べんとう (bentou)</rt></ruby>を<ruby>温<rt>あたた (atata)</rt></ruby>めましょうか。
- 中文：我幫您加熱便當吧？
- 日文：<ruby>迎え<rt>むかえ (mukae)</rt></ruby>に<ruby>行<rt>い (i)</rt></ruby>きましょうか。
- 中文：我順道去接您吧？
- 日文：<ruby>掃除<rt>そうじ (souji)</rt></ruby>をしましょうか。
- 中文：我來打掃吧？
- 日文：クーラーを<ruby>消<rt>け (ke)</rt></ruby>しましょうか。
- 中文：我把冷氣關掉吧？
- 日文：<ruby>使<rt>つか (tsuka)</rt></ruby>い<ruby>方<rt>かた (kata)</rt></ruby>を<ruby>説明<rt>せつめい (setsumei)</rt></ruby>しましょうか。
- 中文：我向您說明使用方法吧？
- 日文：ラッピングしましょうか。
- 中文：我幫您包裝吧？
- 日文：スプーンをお<ruby>持<rt>も (mo)</rt></ruby>ちしましょうか。
- 中文：我幫您拿根湯匙過來吧？（更為禮貌謙遜的說法）

## 關係說明
- 與詢問對方意願的邀請句型 [[JP_Grammar_34_masen_ka|〜ませんか @sibling_of]] 互為姊妹文法，一為「邀請」，一為「主動提議協助」。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>荷物<rt>にもつ (nimotsu)</rt></ruby>を<ruby>持<rt>も (mo)</rt></ruby>ちましょうか。
> 我幫您提行李吧？

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| <ruby>写真<rt>しゃしん (shashin)</rt></ruby>を<ruby>撮<rt>と (to)</rt></ruby>る | **<ruby>写真<rt>しゃしん (shashin)</rt></ruby>を<ruby>撮<rt>と (to)</rt></ruby>り**ましょうか。 |
| <ruby>手伝<rt>てつだ (tetsuda)</rt></ruby>う | **<ruby>手伝<rt>てつだ (tetsuda)</rt></ruby>い**ましょうか。 |
| <ruby>案内<rt>あんない (annai)</rt></ruby>する | **<ruby>案内<rt>あんない (annai)</rt></ruby>し**ましょうか。 |
| お<ruby>弁当<rt>べんとう (bentou)</rt></ruby>を<ruby>温<rt>あたた (atata)</rt></ruby>める | **お<ruby>弁当<rt>べんとう (bentou)</rt></ruby>を<ruby>温<rt>あたた (atata)</rt></ruby>め**ましょうか。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 積極的な提案（〜ましょう） | 荷物を**持ちましょう**。 |
| → 謙譲語表現 | 荷物を**お持ちしましょうか**。 |
| → 疑問・許可（〜てもいいですか） | 荷物を**持ってもいいですか**。 |
| → 親しい表現（〜ようか） | 荷物を**持とうか**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 荷物を持ちましょうか。 | すみません、お願いします。 |
| 荷物を持ちましょうか。 | いいえ、大丈夫です。 |
| 荷物を持ちましょうか。 | ありがとう、助かります。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | ましょうか。 |
| +持ち | 持ちましょうか。 |
| +荷物を | 荷物を持ちましょうか。 |
| +重い | 重い荷物を持ちましょうか。 |
| +その | その重い荷物を持ちましょうか。 |
