## 關鍵句型解說

- **朝食も付けたいのですが／〜を追加できますか**：「〜たいのですが」用於委婉表達自己的請求或需求，語氣柔和不直接；「〜を追加できますか」則直接詢問是否可以追加某項服務，接續為「名詞＋を＋追加できますか」。皆屬中高禮貌度的服務業用語。
- **チェックアウトは何時ですか**：「名詞＋は＋疑問詞＋ですか」的基本問句結構，用於詢問時刻、地點等具體資訊，是最基礎且高頻的敬語問句形式。
- **エアコンが壊れているみたいです／故障しています**：「動詞て形＋いる」表示狀態持續，「壊れている」＝已損壞的狀態；「〜みたいです」為委婉推測語氣，比直接斷定更禮貌；「故障しています」則是較正式、常見於服務業場合的說法。
- **荷物を預かってもらえますか**：「動詞て形＋もらえますか」為請求他人為自己做某事的禮貌問法，語感比「〜てください」更委婉、更常用於對服務人員的請求。

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 正式報修語氣 | エアコンが**故障（こしょう）しています**。 | 空調故障了。（較正式） |
| → 過去形 | エアコンが**壊（こわ）れていました**。 | 空調（那時）壞掉了。 |
| → 請求動詞 | エアコンを**見（み）てもらえますか**。 | 可以請人來看一下空調嗎？ |
| → 更委婉請求 | エアコンを**見（み）ていただけますか**。 | 能否請您幫忙看一下空調呢？ |
