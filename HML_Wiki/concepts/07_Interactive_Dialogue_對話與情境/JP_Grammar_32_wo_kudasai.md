# 〜をください

## 規則與接續
- `名詞 + を + ください`

## 用法說明
表示請求對方給予某個「物品」（名詞），常用於點餐、購物或向他人要東西的場合。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#3498DB">これをください。</span>
- 中文：請給我這個。
- 日文：<span style="color:#F39C12">この</span><span style="color:#3498DB">パンを</span><span style="color:#FF477E"><ruby>一<rt>ひと (hito)</rt></ruby>つ</span><span style="color:#3498DB">ください。</span>
- 中文：請給我一個這個麵包。
- 日文：<span style="color:#3498DB"><ruby>切符<rt>きっぷ (kippu)</rt></ruby>を</span><span style="color:#FF477E"><ruby>１枚<rt>いちまい (ichimai)</rt></ruby></span><span style="color:#3498DB">ください。</span>
- 中文：請給我一張車票。
- 日文：<span style="color:#F39C12">この</span><span style="color:#3498DB">ケーキをください。</span>
- 中文：請給我這個蛋糕。
- 日文：<span style="color:#3498DB">チケットを</span><span style="color:#FF477E"><ruby>１枚<rt>いちまい (ichimai)</rt></ruby></span><span style="color:#3498DB">ください。</span>
- 中文：請給我一張票。

## 關係說明
- 與用於請求他人做某動作的 [[JP_Grammar_11_te_kudasai|〜てください @related_to]] 密切相關（後者接續動詞て形，前者接續名詞）。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> これをください。
> （請給我這個。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 水（みず） | **水（みず）**をください。 | 請給我水。 |
| コーヒー | **コーヒー**をください。 | 請給我咖啡。 |
| ビール | **ビール**をください。 | 請給我啤酒。 |
| お茶（おちゃ） | **お茶（おちゃ）**をください。 | 請給我茶。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| て形請求（てけいせいきゅう） | これを**見（み）せてください**。 | 請讓我看一下這個。 |
| 丁寧謙遜請求（ていねいけんそんせいきゅう） | これを**お願（ねが）いします**。 | 麻煩給我這個。 |
| 疑問請求（ぎもんせいきゅう） | これを**いただけますか**。 | 可以給我這個嗎？ |
| 否定請求（ひていせいきゅう） | これを**入（い）れないでください**。 | 請不要放這個。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| これをください。 | はい、どうぞ。 | 請給我這個。 / 好的，請。 |
| お水（みず）をください。 | はい、どうぞ。 | 請給我水。 / 好的，請。 |
| コーヒーをください。 | はい、どうぞ。 | 請給我咖啡。 / 好的，請。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心（かくしん） | ください。 | 請給。 |
| +目的語（もくてきご） | パンをください。 | 請給我麵包。 |
| +數量（すうりょう） | パンを一つ（ひとつ）ください。 | 請給我一個麵包。 |
| +修飾（しゅうしょく） | このパンを一つ（ひとつ）ください。 | 請給我一個這個麵包。 |
