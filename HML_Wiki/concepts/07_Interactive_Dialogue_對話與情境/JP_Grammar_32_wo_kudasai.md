# 〜をください

## 規則與接續
- `名詞 + を + ください`

## 用法說明
表示請求對方給予某個「物品」（名詞），常用於點餐、購物或向他人要東西的場合。

## 實用例句
- 日文：これをください。
- 中文：請給我這個。
- 日文：このパンを<ruby>一<rt>ひと (hito)</rt></ruby>つください。
- 中文：請給我一個這個麵包。
- 日文：<ruby>切符<rt>きっぷ (kippu)</rt></ruby>を<ruby>１枚<rt>いちまい (ichimai)</rt></ruby>ください。
- 中文：請給我一張車票。
- 日文：このケーキをください。
- 中文：請給我這個蛋糕。
- 日文：チケットを<ruby>１枚<rt>いちまい (ichimai)</rt></ruby>ください。
- 中文：請給我一張票。

## 關係說明
- 與用於請求他人做某動作的 [[JP_Grammar_11_te_kudasai|〜てください @related_to]] 密切相關（後者接續動詞て形，前者接續名詞）。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> このパンを<ruby>一<rt>ひと (hito)</rt></ruby>つください。
> 請給我一個這個麵包。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| このりんご / 二つ | **このりんごを二つ**ください。 |
| ビール / 一つ | **ビールを一つ**ください。 |
| コーヒー / 二つ | **コーヒーを二つ**ください。 |
| お<ruby>水<rt>みず (mizu)</rt></ruby> / 一つ | **お<ruby>水<rt>みず (mizu)</rt></ruby>を一つ**ください。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 丁寧な依頼（お願いします） | このパンを一つ**お願いします**。 |
| → 疑問形（〜もらえますか） | このパンを一つ**もらえますか**。 |
| → 丁寧な疑問形（〜いただけませんか） | このパンを一つ**いただけませんか**。 |
| → 否定（いりません） | このパンは**いりません**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| ご注文はお決まりですか。 | このパンを一つください。 |
| 何を差し上げましょうか。 | このパンを一つください。 |
| お飲み物はどうしますか。 | お水を一つください。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | ください。 |
| +パンを | パンをください。 |
| +一つ | パンを一つください。 |
| +この | このパンを一つください。 |
