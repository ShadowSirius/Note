| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| ペン | **ペン**を貸（か）してもらえますか。 | 可以借我筆記型電腦 / 筆嗎？ |
| 辞書（じしょ） | **辞書（じしょ）**を貸（か）してもらえますか。 | 可以借我字典嗎？ |
| 傘（かさ） | **傘（かさ）**を貸（か）してもらえますか. | 可以借我雨傘嗎？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 普通形（口語） | ノートを貸（か）して**もらえる**？ | 可以借我筆記嗎？（口語） |
| → 請求允許形式 | ノートを**貸（か）してください**。 | 請借我筆記。 |
| → 謙讓請託 | ノートを**貸（か）していただけますか**。 | 能否請您借我筆記呢？ |
| → 否定詢問 | ノートを貸（か）して**もらえませんか**。 | 不能借我筆記嗎？ |
