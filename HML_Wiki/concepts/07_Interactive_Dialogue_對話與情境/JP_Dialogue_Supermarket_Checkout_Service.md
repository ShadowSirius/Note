## 關鍵句型解說

- **〜はどこにありますか／あちらの棚にあります**：「場所を尋ねる」的基本句型，「名詞＋は＋どこにありますか」用於詢問無生命物品的位置；回答時用「場所＋に＋あります」，指示詞「あちら」比「あそこ」更禮貌、常見於服務業。
- **袋はご利用になりますか**：「ご＋動詞ます形＋になりますか」是尊敬語（尊敬表現）句型，店員用來禮貌地詢問客人的意願，比直接問「要不要」更正式客氣，常見於零售結帳場景。
- **ポイントカードはお持ちですか／持っていません**：「お＋持ち＋ですか」為「持っていますか」的尊敬語形式；回答「持っていません」使用「動詞て形＋いません」表示目前不具備某狀態，是簡潔自然的否定應答。
- **温めてもらえますか**：「動詞て形＋もらえますか」的請求句型，與飯店篇的「預かってもらえますか」同構，用於請求店員為自己執行動作，語氣禮貌但不過度生硬。
- **〜は承っておりません**：「承る」為「受ける／聞く」的自謙語（謙讓語），「〜ておりません」是「〜ていません」的謙讓形式；此句型用於店員婉拒客人請求時，既表達歉意又保持專業態度，是服務業常用的婉拒表現。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> お弁当（べんとう）を温（あたた）めてもらえますか。
> （便當可以幫我加熱嗎？）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| おにぎり | **おにぎり**を温（あたた）めてもらえますか。 | 飯糰可以幫我加熱嗎？ |
| このパン | **このパン**を温（あたた）めてもらえますか。 | 這個麵包可以幫我加熱嗎？ |
| カップラーメン | **カップラーメン**にお湯（ゆ）を入（い）れてもらえますか。 | 杯麵可以幫我加熱水嗎？ |
| 肉まん（にくまん） | **肉まん（にくまん）**を温（あたた）めてもらえますか。 | 肉包可以幫我加熱嗎？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 婉拒句型 | 申し訳ございません、温（あたた）めは**承（うけたまわ）っておりません**。 | 非常抱歉，本店不提供加熱服務。 |
| → 疑問形（店員） | 袋（ふくろ）は**ご利用になりますか**。 | 請問需要袋子嗎？ |
| → 尊敬語疑問 | ポイントカードは**お持ちですか**。 | 請問有集點卡嗎？ |
| → 否定應答 | いいえ、**持っていません**。 | 沒有，我沒有。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| お醤油（しょうゆ）はどこにありますか。 | あちらの棚（たな）にあります。 | 醬油在哪裡？ / 在那邊的架子上。 |
| 袋（ふくろ）はご利用になりますか。 | はい、一番（いちばん）小（ちい）さいサイズを一枚（いちまい）お願（ねが）いします。 | 需要袋子嗎？ / 好的，麻煩給我最小尺寸的一個。 |
| 宅配便（たくはいびん）の発送（はっそう）をお願（ねが）いしたいのですが。 | 申（もう）し訳（わけ）ございません、当店（とうてん）では承（うけたまわ）っておりません。 | 想請你們寄送宅配。 / 非常抱歉，本店不提供此服務。 |
