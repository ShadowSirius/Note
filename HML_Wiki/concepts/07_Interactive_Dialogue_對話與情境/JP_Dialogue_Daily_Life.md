## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 一緒（いっしょ）に写真（しゃしん）を撮（と）りませんか。
> （要不要一起拍張照片呢？）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 映画（えいが）を見（み）に | 一緒（いっしょ）に**映画（えいが）を見（み）に**行き（いき）ませんか。 | 要不要一起去看電影呢？ |
| ご飯（はん）を食（た）べに | 一緒（いっしょ）に**ご飯（はん）を食（た）べに**行き（いき）ませんか。 | 要不要一起去吃飯呢？ |
| お茶（ちゃ）を飲み（のみ）に | 一緒（いっしょ）に**お茶（ちゃ）を飲み（のみ）に**行き（いき）ませんか。 | 要不要一起去喝茶呢？ |
| 公園（こうえん）を散歩（さんぽ）しに | 一緒（いっしょ）に**公園（こうえん）を散歩（さんぽ）しに**行き（いき）ませんか。 | 要不要一起去公園散步呢？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 普通形（口語） | 一緒（いっしょ）に写真（しゃしん）を**撮（と）らない**？ | 要不要一起拍張照片？（口語） |
| → 意向形（提議） | 一緒（いっしょ）に写真（しゃしん）を**撮（と）りましょう**。 | 我們一起拍張照片吧。 |
| → 請求形式 | 一緒（いっしょ）に写真（しゃしん）を**撮（と）ってください**。 | 請一起拍張照片。 |
| → 過去推測 | 一緒（いっしょ）に写真（しゃしん）を**撮（と）りたかったですか**。 | 剛才想一起拍張照片嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 一緒（いっしょ）に写真（しゃしん）を撮（と）りませんか。 | いいですね。撮（と）りましょう。 | 要不要一起拍張照片呢？ / 好呀。拍吧。 |
| 一緒（いっしょ）に写真（しゃしん）を撮（と）りませんか。 | すみません、ちょっと…。 | 要不要一起拍張照片呢？ / 不好意思，有點不方便… |
| 今度（こんど）一緒（いっしょ）に映画（えいが）を見（み）に行（い）きませんか。 | はい、いいですよ。行き（いき）ましょう。 | 下次要不要一起去看電影呢？ / 好的，可以啊。去吧。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 撮（と）りませんか。 | 不拍嗎？ / 要不要拍？ |
| +目的語 | **写真（しゃしん）を**撮（と）りませんか。 | 要不要拍照片？ |
| +同伴 | **一緒（いっしょ）に**写真（しゃしん）を撮（と）りませんか。 | 要不要一起拍照片？ |
| +場所 | **公園（こうえん）で**一緒に写真を撮りませんか。 | 要不要一起在公園拍照片？ |
