# 〜ませんか

## 規則與接續
- `動詞ます形（去掉ます） + ませんか`

## 用法說明
表示委婉的邀請或提議（要不要一起做某事？）。相比於直接提議，這種說法會先詢問對方的意願，聽起來非常客氣有禮。通常與 `<ruby>一緒に<rt>いっしょに (isshoni)</rt></ruby>`（一起）搭配使用。

## 實用例句
- 日文：<ruby>一緒に<rt>いっしょに (isshoni)</rt></ruby><ruby>写真<rt>しゃしん (shashin)</rt></ruby>を<ruby>撮<rt>と (to)</rt></ruby>りませんか。
- 中文：要不要一起拍張照片呢？
- 日文：<ruby>一緒に<rt>いっしょに (isshoni)</rt></ruby>ご<ruby>飯<rt>はん (han)</rt></ruby>を<ruby>食<rt>た (ta)</rt></ruby>べませんか。
- 中文：要不要一起吃飯呢？
- 日文：<ruby>一緒に<rt>いっしょに (isshoni)</rt></ruby><ruby>京都<rt>きょうと (kyouto)</rt></ruby>へ<ruby>行<rt>い (i)</rt></ruby>きませんか。
- 中文：要不要一起去京都呢？
- 日文：<ruby>一緒に<rt>いっしょに (isshoni)</rt></ruby>タクシーに<ruby>乗<rt>の (no)</rt></ruby>りませんか。
- 中文：要不要一起搭計程車呢？
- 日文：lineを<ruby>交換<rt>こうかん (koukan)</rt></ruby>しませんか。
- 中文：要不要交換 Line 呢？
- 日文：<ruby>一緒に<rt>いっしょに (isshoni)</rt></ruby>お<ruby>茶<rt>ちゃ (cha)</rt></ruby>を<ruby>飲<rt>の (no)</rt></ruby>みませんか。

- 中文：要不要一起回去呢？
- 日文：ちょっと<ruby>休<rt>やす (yasu)</rt></ruby>みませんか。
- 中文：要不要稍作休息呢？
- 日文：<ruby>一緒に<rt>いっしょに (isshoni)</rt></ruby><ruby>富士山<rt>ふじさん (fujisan)</rt></ruby>を<ruby>見<rt>み (mi)</rt></ruby>に<ruby>行<rt>い (i)</rt></ruby>きませんか。

- 中文：要不要一起去玩呢？

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>一緒に<rt>いっしょに (isshoni)</rt></ruby>ご<ruby>飯<rt>はん (han)</rt></ruby>を<ruby>食<rt>た (ta)</rt></ruby>べませんか。

| Prompt | Expected Response |
|---|---|
| お<ruby>茶<rt>ちゃ (cha)</rt></ruby>を<ruby>飲<rt>の (no)</rt></ruby>む | **一緒にお<ruby>茶<rt>ちゃ (cha)</rt></ruby>を<ruby>飲<rt>の (no)</rt></ruby>み**ませんか。 |

| <ruby>遊<rt>あそ (aso)</rt></ruby>びに<ruby>行<rt>い (i)</rt></ruby>く | **一緒に<ruby>遊<rt>あそ (aso)</rt></ruby>びに<ruby>行<rt>い (i)</rt></ruby>**ませんか。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 積極的な提案（〜ましょう） | 一緒にご飯を**食べましょう**。 |
| → 親しい表現（〜ない？） | 一緒にご飯を**食べない？** |
| → 肯定・現在進行 | 一緒にご飯を**食べています**。 |
| → 助けを申し出る（〜ましょうか） | ご飯を**食べましょうか**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 一緒にご飯を食べませんか。 | いいですね。行きましょう。 |
| 一緒にご飯を食べませんか。 | すみません、今、時間がないので。 |
| 一緒にご飯を食べませんか。 | ええ、いいですよ。何を食べますか。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | 食べませんか。 |
| +ご飯を | ご飯を食べませんか。 |
| +一緒に | 一緒にご飯を食べませんか。 |
| +今夜、 | 今夜、一緒にご飯を食べませんか。 |
