# 〜ませんか

## 規則與接續
- `動詞ます形（去掉ます） + ませんか`

## 用法說明
表示委婉的邀請或提議（要不要一起做某事？）。相比於直接提議，這種說法會先詢問對方的意願，聽起來非常客氣有禮。通常與 `一緒に`（一起）搭配使用。

## 實用例句
- 日文：一緒に写真を撮りませんか。
- 中文：要不要一起拍張照片呢？
- 日文：一緒にご飯を食べませんか。
- 中文：要不要一起吃飯呢？
- 日文：一緒に京都へ行きませんか。
- 中文：要不要一起去京都呢？
- 日文：一緒にタクシーに乗りませんか。
- 中文：要不要一起搭計程車呢？
- 日文：lineを交換しませんか。
- 中文：要不要交換 Line 呢？
- 日文：一緒にお茶を飲みませんか。
- 中文：要不要一起喝茶呢？
- 日文：一緒に帰りませんか。
- 中文：要不要一起回去呢？
- 日文：ちょっと休みませんか。
- 中文：要不要稍作休息呢？
- 日文：一緒に富士山を見に行きませんか。
- 中文：要不要一起去看富士山呢？
- 日文：ちょっと話しませんか。
- 中文：要不要稍微聊聊呢？
- 日文：ちょっと喫茶店で話しませんか。
- 中文：要不要稍微在咖啡廳聊聊呢？
- 日文：一緒に遊びませんか。
- 中文：要不要一起去玩呢？

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 一緒に（いっしょに）ご飯（はん）を食べ（たべ）ませんか。
> （要不要一起吃飯呢？）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| お茶（ちゃ）を飲み（のみ） | 一緒（いっしょ）に**お茶（ちゃ）を飲み（の）みませんか**。 | 要不要一起喝茶呢？ |
| 京都（きょうと）へ行き（いき） | 一緒（いっしょ）に**京都（きょうと）へ行き（い）きませんか**。 | 要不要一起去京都呢？ |
| 写真（しゃしん）を撮り（とり） | 一緒（いっしょ）に**写真（しゃしん）を撮り（と）りませんか**。 | 要不要一起拍照呢？ |
| 遊び（あそび） | 一緒（いっしょ）に**遊び（あそび）ませんか**。 | 要不要一起去玩呢？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 主動協助（しゅどうきょうりょく） | 一緒（いっしょ）にご飯（はん）を**食べましょうか**。 | 我陪您一起吃飯吧？ |
| 普通邀請（ふつうようせい） | 一緒（いっしょ）にご飯（はん）を**食べよう**。 | 一起吃飯吧。（常體） |
| 丁寧進行（ていねいしんこう） | 一緒（いっしょ）にご飯（はん）を**食べます**。 | 一起吃飯。 |
| 否定常體（ひていじょうたい） | 一緒（いっしょ）にご飯（はん）を**食べない**？ | 不一起吃飯嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|

| 一緒（いっしょ）にお茶（ちゃ）を飲み（のみ）ませんか。 | いいですね、飲み（の）みましょう。 | 要不要一起喝茶呢？ / 好啊，一起喝吧。 |
