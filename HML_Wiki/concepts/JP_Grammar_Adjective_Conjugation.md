# 🇯🇵 日語形容詞變化指南（い形容詞・な形容詞）

> [!important] 句子結構色彩標記：
> 🟢 **主語/主題** (`#2ECC71`) \| 🔵 **核心/謂語** (`#3498DB`) \| 🟠 **修飾語/賓語** (`#E67E22`) \| 🔴 **時間/場所/副詞** (`#E74C3C`)

---

## 📂 第一部分：形容詞分類與判定

日語形容詞分為兩大類：
1.  **い形容詞**：原形字尾必然是日文字母的「い」（如：美味しい、寒い、高い）。
2.  **な形容詞**（形容動詞）：原形字尾通常沒有「い」，在修飾名詞時需要在詞尾加上「な」（如：親切（な）、靜か（な）、綺麗（な））。

> [!warning] 判定特例（字尾有「い」但屬於 な形容詞）：
> - **綺麗（きれい / 美麗、乾淨）**、**嫌い（きらい / 討厭）**、**幸い（さいわい / 幸運）**。這些單字漢字字尾雖然有「い」，但它們實際上是「な形容詞」，修飾名詞時需用「綺麗な花」、「嫌いな食べ物」。

## 📂 第二部分：形容詞時態與語氣變化對照表

| 時態與變化 | い形容詞 (以「美味しい / おいしい」為例) | な形容詞 (以「親切 / しんせつ」為例) | 語境與結構例句 |
| :--- | :--- | :--- | :--- |
| **① 現在肯定形** <br>(非過去肯定) | `美味しい` (普通體) <br>`美味しいです` (丁寧形) | `親切だ` (普通體) <br>`親切です` (丁寧形) | <span style="color:#2ECC71">このお茶は</span><span style="color:#3498DB">美味しいです。</span> <br>（這茶很好喝。） |
| **② 現在否定形** <br>(非過去否定) | 去掉字尾 **い** + `くない`。 <br> 普通體：`美味しくない` <br> 丁寧形：`美味しくないです` / `美味しくありません` | 普通體：`親切ではない` / `親切じゃない` <br> 丁寧形：`親切ではありません` / `親切じゃないです` | <span style="color:#2ECC71">彼は</span><span style="color:#3498DB">親切じゃない。</span> <br>（他不親切。） |
| **③ 過去肯定形** | 去掉字尾 **い** + `かった`。 <br> 普通體：`美味しかった` <br> 丁寧形：`美味しかったです` | 普通體：`親切だった` <br> 丁寧形：`親切でした` | <span style="color:#E74C3C">昨日の料理は</span><span style="color:#3498DB">美味しかったです。</span> <br>（昨天的料理很好吃。） |
| **④ 過去否定形** | 去掉字尾 **い** + `くなかった`。 <br> 普通體：`美味しくなかった` <br> 丁寧形：`美味しくなかったです` / `美味しくありませんでした` | 普通體：`親切ではなかった` / `親切じゃなかった` <br> 丁寧形：`親切ではありませんでした` / `親切じゃなかったです` | <span style="color:#E74C3C">以前の彼は</span><span style="color:#3498DB">親切ではなかった。</span> <br>（以前的他並不親切。） |
| **⑤ 修飾名詞** <br>(作連體修飾) | 直接修飾名詞。 <br> 例：`美味しいお茶` | 加上 **な** 修飾名詞。 <br> 例：`親切な人` | <span style="color:#2ECC71">あそこは</span><span style="color:#E67E22">綺麗な</span><span style="color:#3498DB">場所です。</span> <br>（那是個美麗的地方。） |
| **⑥ 連用形** <br>(修飾動詞/副詞化) | 去掉字尾 **い** + `く`。 <br> 例：`美味しく` | 加上 **に**。 <br> 例：`親切に` | <span style="color:#E74C3C">美味しく</span><span style="color:#3498DB">なりました。</span> <br>（變得好吃了。） |

## 📂 第三部分：特殊形容詞變形規則

> [!danger] **「いい / よい（好）」的特殊變形規則**
> 日常口語中「好」常說成「いい」，但在所有文法變形時，必須回到原字根「**よい**」進行變化：
> - 肯定形：**いい** / **よい**
> - 否定形：**よくない**（而非 いくない）
>   - 丁寧形否定：**よくないです** / **よくありません**
> - 過去肯定：**よかった**（而非 いかった）
> - 過去否定：**よくなかった**
> - 連用形 (修飾動詞)：**よく**
>   - 例句：<span style="color:#E74C3C">よく</span><span style="color:#3498DB">できました！</span>（做得很好！）
