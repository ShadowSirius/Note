# 〜たほうがいいです

## 規則與接續
- `動詞た形 + ほうがいいです`

## 用法說明
用於提出明確建議或勸告（最好... / 還是...比較好）。

## 實用例句
- 日文：<ruby>早<rt>はや (haya)</rt></ruby>く<ruby>寢<rt>ね (ne)</rt></ruby>たほうがいいですよ。
- 中文：最好早點睡喔。
- 日文：<ruby>病院<rt>びょういん (byouin)</rt></ruby>に<ruby>行<rt>い (i)</rt></ruby>ったほうがいいです。
- 中文：還是去醫院比較好。

## 關係說明
- 給予正面建議，否定建議（最好不要）常與 [[JP_Grammar_20_nai_de_kudasai|〜ないでください @related_to]] 混用。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>病院<rt>びょういん</rt></ruby>に<ruby>行<rt>い</rt></ruby>ったほうがいいです。
> 還是去醫院比較好。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 薬を飲みます | **薬を飲んだ**ほうがいいです。 |
| 早く寝ます | **早く寝た**ほうがいいです。 |
| 先生に聞きます | **先生に聞いた**ほうがいいです。 |
| 少し休みます | **少し休んだ**ほうがいいです。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定建議 (〜ないほうがいいです) | 病院に**行かない**ほうがいいです。 |
| → 普通形建議 (〜たほうがいいよ) | 病院に行ったほうが**いいよ**。 |
| → 過去疑問 (〜たほうがよかったですか) | 病院に行ったほうが**よかったですか**。 |
| → 口語否定建議 (〜ないほうがいい) | 病院に行かないほうが**いいよ**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 体の調子が悪いですが、どうすればいいですか。 | 病院に行ったほうがいいです。 |
| 熱があります。病院に行ったほうがいいですか。 | はい、病院に行ったほうがいいです。 |
| 薬はいつ飲んだほうがいいですか。 | 早く飲んだほうがいいです。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | いいです。 |
| +ほうがいいです | ほうがいいです。 |
| +行ったほうが | 行ったほうがいいです。 |
| +病院に | 病院に行ったほうがいいです。 |
