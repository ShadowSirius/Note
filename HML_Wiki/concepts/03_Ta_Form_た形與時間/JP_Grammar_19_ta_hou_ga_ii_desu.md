# 〜たほうがいいです

## 規則與接續
- `動詞た形 + ほうがいいです`

## 用法說明
用於提出明確建議或勸告（最好... / 還是...比較好）。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E"><ruby>早<rt>はや (haya)</rt></ruby>く</span><span style="color:#3498DB"><ruby>寢<rt>ね (ne)</rt></ruby>たほうがいいですよ。</span>
- 中文：最好早點睡喔。
- 日文：<span style="color:#FF477E"><ruby>病院<rt>びょういん (byouin)</rt></ruby>に</span><span style="color:#3498DB"><ruby>行<rt>い (i)</rt></ruby>ったほうがいいです。</span>
- 中文：還是去醫院比較好。

## 關係說明
- 給予正面建議，否定建議（最好不要）常與 [[JP_Grammar_20_nai_de_kudasai|〜ないでください @related_to]] 混用。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 病院（びょういん）に行（い）ったほうがいいです。
> （還是去醫院比較好。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 薬（くすり） | **薬（くすり）を飲（の）んだ**ほうがいいです。 | 還是吃藥比較好。 |
| 早く（はや）寝る（ね） | **早く（はや）寝（ね）た**ほうがいいです。 | 還是早點睡比較好。 |
| お風呂（ふろ） | **お風呂（ふろ）に入（はい）った**ほうがいいです。 | 還是洗個熱水澡比較好。 |
| 休む（やす） | **休（やす）んだ**ほうがいいです。 | 還是休息一下比較好。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 否定建議 | 病院（びょういん）に**行（い）かない**ほうがいいです。 | 最好不要去醫院。 |
| → 普通形建議 | 病院（びょういん）に行（い）ったほうが**いいよ**。 | 還是去醫院比較好喔。 |
| → 過去肯定評價 | 病院（びょういん）に行（い）ったほうが**よかったです**。 | 那時還是去醫院比較好。 |
| → 疑問建議 | 病院（びょういん）に行（い）ったほうがいいです**か**。 | 去醫院比較好嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 熱（ねつ）があります。どうすればいいですか。 | 病院（びょういん）に行（い）ったほうがいいです。 | 還是去醫院比較好。 |
| 薬（くすり）を飲（の）んだほうがいいですか。 | はい、薬（くすり）を飲（の）んだほうがいいです。 | 是的，還是吃藥比較好。 |
| 明日（あした）も学校（がっこう）に行（い）ったほうがいいですか。 | いいえ、無理（むり）をせずに休（やす）んだほうがいいです。 | 不，不要勉強，還是休息比較好。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | いいです。 | 比較好。 |
| +句型 | **ほうが**いいです。 | 還是...比較好。 |
| +動詞 | **行（い）った**ほうがいいです。 | 還是去比較好。 |
| +目的地 | **病院（びょういん）に**行（い）ったほうがいいです。 | 還是去醫院比較好。 |
