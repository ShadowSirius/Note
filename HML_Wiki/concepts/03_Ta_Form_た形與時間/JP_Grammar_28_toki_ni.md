# 〜ときに

## 規則與接續
- `動詞普通形/名詞の/形容詞 + とき(に)`

## 用法說明
表示動作進行的時間點（當...的時候）。

## 實用例句
- 日文：<ruby>子供<rt>こども (kodomo)</rt></ruby>のとき、よく<ruby>川<rt>かわ (kawa)</rt></ruby>で<ruby>泳<rt>およ (oyo)</rt></ruby>ぎました。
- 中文：小時候，常在河裡游泳。
- 日文：<ruby>日本<rt>にほん (nihon)</rt></ruby>へ<ruby>行<rt>い (i)</rt></ruby>くときに、カメラを<ruby>買<rt>か (ka)</rt></ruby>いました。
- 中文：去日本的時候（出發前），買了相機。

## 關係說明
- 標記特定動作發生的時間點。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>日本<rt>にほん</rt></ruby>へ<ruby>行<rt>い</rt></ruby>くときに、カメラを<ruby>買<rt>か</rt></ruby>いました。
> 去日本的時候，買了相機。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 携帯 (けいたい) | 日本へ行くときに、**携帯**を買いました。 |
| お土産 (おみやげ) | 日本へ行くときに、**お土産**を買いました。 |
| 切符 (きっぷ) | 日本へ行くときに、**切符**を買いました。 |
| ガイドブック | 日本へ行くときに、**ガイドブック**を買いました。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 過去完成 (〜たときに) | 日本へ**行った**ときに、カメラを買いました。 |
| → 否定結果 (〜買いませんでした) | 日本へ行くときに、カメラを**買いませんでした**。 |
| → 普通形 (〜買った) | 日本へ行くときに、カメラを**買った**。 |
| → 正在進行 (〜ているときに) | 日本へ**行っている**ときに、カメラを買いました。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 日本へ行くときに、何を買いましたか。 | 日本へ行くときに、カメラを買いました。 |
| カメラはいつ買いましたか。 | 日本へ行くときに買いました。 |
| 日本へ行ったときに、何か買いましたか。 | はい、日本へ行ったときに、お土産を買いました。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | 買いました。 |
| +カメラを | カメラを買いました。 |
| +行くときに | 行くときに、カメラを買いました。 |
| +日本へ | 日本へ行くときに、カメラを買いました。 |
