# 〜ときに

## 規則與接續
- `動詞普通形/名詞の/形容詞 + とき(に)`

## 用法說明
表示動作進行的時間點（當...的時候）。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#F39C12"><ruby>子供<rt>こども (kodomo)</rt></ruby>の</span><span style="color:#FF477E">とき、</span><span style="color:#FF477E">よく</span><span style="color:#FF477E"><ruby>川<rt>かわ (kawa)</rt></ruby>で</span><span style="color:#3498DB"><ruby>泳<rt>およ (oyo)</rt></ruby>ぎました。</span>
- 中文：小時候，常在河裡游泳。
- 日文：<span style="color:#FF477E"><ruby>日本<rt>にほん (nihon)</rt></ruby>へ</span><span style="color:#3498DB"><ruby>行<rt>い (i)</rt></ruby>く</span><span style="color:#FF477E">ときに、</span><span style="color:#3498DB">カメラを<ruby>買<rt>か (ka)</rt></ruby>いました。</span>
- 中文：去日本的時候（出發前），買了相機。

## 關係說明
- 標記特定動作發生的時間點。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 日本（にほん）へ行（い）くときに、カメラを買（か）いました。
> （去日本的時候（出發前），買了相機。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| アメリカ / パスポート | **アメリカへ行（い）くときに、パスポートを作（つく）りました**。 | 去美國的時候，辦了護照。 |
| 京都（きょうと） / お土産（みやげ） | **京都（きょうと）へ行（い）くときに、お土産（みやげ）を買（か）いました**。 | 去京都的時候，買了特產。 |
| 東京（とうきょう） / 切符（きっぷ） | **東京（とうきょう）へ行（い）くときに、切符（きっぷ）を買（か）いました**。 | 去東京的時候，買了車票。 |
| イギリス / スーツケース | **イギリスへ行（い）くときに、スーツケースを買（か）いました**。 | 去英國的時候，買了行李箱。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去動詞形（去之後） | 日本（にほん）へ**行（い）ったときに**、カメラを買（か）いました。 | 去了日本之後，買了相機。 |
| → 名詞接續形（小時候） | **子供（こども）のときに**、カメラを買（か）いました。 | 小時候買了相機。 |
| → 否定接續形 | **わからないときに**、先生（せんせい）に聞（き）きます。 | 不懂的時候，會問老師。 |
| → 現在時結尾 | 日本（にほん）へ行（い）くときに、カメラを**買（か）います**。 | 去日本的時候，會買相機。 |
