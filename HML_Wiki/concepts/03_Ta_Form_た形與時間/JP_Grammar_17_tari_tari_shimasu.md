# 〜たり〜たりします

## 規則與接續
- `動詞た形 + り + 動詞た形 + りします`

## 用法說明
列舉複數的代表性動作（一會兒...一會兒... / 又做...又做...）。

## 實用例句
- 日文：<ruby>週末<rt>しゅうまつ (shuumatsu)</rt></ruby>は<ruby>映畫<rt>えいが (eiga)</rt></ruby>を<ruby>見<rt>み (mi)</rt></ruby>たり、<ruby>本<rt>ほん (hon)</rt></ruby>を<ruby>讀<rt>よ (yo)</rt></ruby>んだりします。
- 中文：週末有時看電影，有時看書。
- 日文：<ruby>歌<rt>うた (uta)</rt></ruby>columnったり<ruby>踊<rt>おど (odo)</rt></ruby>ったりしました。
- 中文：又唱歌又跳舞了。

## 關係說明
- 使用動詞た形，與表示過去經驗的 [[JP_Grammar_18_ta_koto_ga_arimasu|〜たことがあります @sibling_of]] 屬同類變化。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>週末<rt>しゅうまつ</rt></ruby>は<ruby>映画<rt>えいが</rt></ruby>を<ruby>見<rt>み</rt></ruby>たり、<ruby>本<rt>ほん</rt></ruby>を<ruby>読<rt>よ</rt></ruby>んだりします。
> 週末有時看電影，有時看書。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 音楽を聞く (おんがくをきく) / 掃除する (そうじする) | 週末は**音楽を聞いたり、掃除したり**します。 |
| 友達と会う (ともだちとあう) / 買い物する (かいものする) | 週末は**友達と会ったり、買い物したり**します。 |
| テニスをする / 泳ぐ (およぐ) | 週末は**テニスをしたり、泳いだり**します。 |
| テレビを見る / ゲームをする | 週末は**テレビを見たり、ゲームをしたり**します。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 過去形 (〜ました) | 週末は映画を見たり、本を読んだり**しました**。 |
| → 否定形 (〜しません) | 週末は映画を見たり、本を読んだり**しません**。 |
| → 普通形 (〜する) | 週末は映画を見たり、本を読んだり**する**。 |
| → 普通形過去 (〜した) | 週末は映画を見たり、本を読んだり**した**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 週末は映画を見たり、本を読んだりしますか。 | はい、映画を見たり、本を読んだりします。 |
| 週末は何をしますか。 | 映画を見たり、本を読んだりします。 |
| 先週末は何をしましたか。 | 映画を見たり、本を読んだりしました。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | 読んだりします。 |
| +本を | 本を読んだりします。 |
| +映画を見たり | 映画を見たり、本を読んだりします。 |
| +週末は | 週末は映画を見たり、本を読んだりします。 |
