# 〜まえに

## 規則與接續
- `動詞原形/名詞の + まえに`

## 用法說明
表示在某動作或時間點之前（在...之前）。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#3498DB"><ruby>寢<rt>ね (ne)</rt></ruby>る</span><span style="color:#FF477E"><ruby>前<rt>まえ (mae)</rt></ruby>に、</span><span style="color:#3498DB"><ruby>本<rt>ほん (hon)</rt></ruby>を<ruby>讀<rt>よ (yo)</rt></ruby>みます。</span>
- 中文：睡覺之前會看書。
- 日文：<span style="color:#F39C12"><ruby>食事<rt>しょくじ (shokuji)</rt></ruby>の</span><span style="color:#FF477E"><ruby>前<rt>まえ (mae)</rt></ruby>に、</span><span style="color:#3498DB"><ruby>手<rt>て (te)</rt></ruby>を<ruby>洗<rt>あら (ara)</rt></ruby>います。</span>
- 中文：吃飯前要洗手。

## 關係說明
- 表示時間先後順序，與表示之後的 [[JP_Grammar_30_ato_de|〜あとで @contradicts]] 意思相反。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 寝（ね）る前（まえ）に、本（ほん）を読（よ）みます。
> （睡覺之前會看書。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| お風呂（ふろ） | **お風呂（ふろ）に入（はい）る**前（まえ）に、本（ほん）を読（よ）みます。 | 洗澡前會看書。 |
| 歯（は） | **歯（は）を磨（みが）く**前（まえ）に、本（ほん）を読（よ）みます。 | 刷牙前會看書。 |
| 薬（くすり） | **薬（くすり）を飲（の）む**前（まえ）に、本（ほん）を読（よ）みます。 | 吃藥前會看書。 |
| 電気（でんき） | **電気（でんき）を消（け）す**前（まえ）に、本（ほん）を読（よ）みます。 | 關燈前會看書。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 寝（ね）る前（まえ）に、本（ほん）を**読（よ）みました**。 | 睡覺之前看書了。 |
| → 否定形 | 寝（ね）る前（まえ）に、本（ほん）を**読（よ）みません**。 | 睡覺之前不看書。 |
| → 名詞接續形 | **食事（しょくじ）の**前（まえ）に、本（ほん）を読（よ）みます。 | 吃飯前會看書。 |
| → 動作與目的語變更 | 食事（しょくじ）の前（まえ）に、**手（て）を洗（あら）います**。 | 吃飯前要洗手。 |
