# 〜まえに

## 規則與接續
- `動詞原形/名詞の + まえに`

## 用法說明
表示在某動作或時間點之前（在...之前）。

## 實用例句
- 日文：<ruby>寢<rt>ね (ne)</rt></ruby>る<ruby>前<rt>まえ (mae)</rt></ruby>に、<ruby>本<rt>ほん (hon)</rt></ruby>を<ruby>讀<rt>よ (yo)</rt></ruby>みます。
- 中文：睡覺之前會看書。
- 日文：<ruby>食事<rt>しょくじ (shokuji)</rt></ruby>の<ruby>前<rt>まえ (mae)</rt></ruby>に、<ruby>手<rt>て (te)</rt></ruby>を<ruby>洗<rt>あら (ara)</rt></ruby>います。
- 中文：吃飯前要洗手。

## 關係說明
- 表示時間先後順序，與表示之後的 [[JP_Grammar_30_ato_de|〜あとで @contradicts]] 意思相反。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>寝<rt>ね</rt></ruby>る<ruby>前<rt>まえ</rt></ruby>に、<ruby>本<rt>ほん</rt></ruby>を<ruby>読<rt>よ</rt></ruby>みます。
> 睡覺之前會看書。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 薬を飲みます | 寝る前に、**薬を飲みます**。 |
| シャワーを浴びます | 寝る前に、**シャワーを浴びます**。 |
| 日記を書きます | 寝る前に、**日記を書きます**。 |
| 歯を磨 (みが) きます | 寝る前に、**歯を磨きます**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 過去形 (〜ました) | 寝る前に、本を**読みました**。 |
| → 否定 (〜ません) | 寝る前に、本を**読みません**。 |
| → 後 (〜あとで) | 本を**読んだあとで**、寝ます。 |
| → 普通形 (〜む) | 寝る前に、本を**読む**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 寝る前に、何をしますか。 | 寝る前に、本を読みます。 |
| 本はいつ読みますか。 | 寝る前に読みます。 |
| 食事の前に、手を洗いますか。 | はい、食事の前に、手を洗います。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | 読みます。 |
| +本を | 本を読みます。 |
| +前に | 前に、本を読みます。 |
| +寝る | 寝る前に、本を読みます。 |
