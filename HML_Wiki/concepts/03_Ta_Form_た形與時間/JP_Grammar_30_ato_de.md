# 〜あとで

## 規則與接續
- `動詞た形/名詞の + あとで`

## 用法說明
表示在某動作或事件之後（在...之後）。

## 實用例句
- 日文：<ruby>仕事<rt>しごと (shigoto)</rt></ruby>が<ruby>終<rt>お (o)</rt></ruby>わったあとで、<ruby>飲<rt>の (no)</rt></ruby>みに<ruby>行<rt>い (i)</rt></ruby>きます。
- 中文：工作結束後去喝一杯。
- 日文：<ruby>授業<rt>じゅぎょう (jugyou)</rt></ruby>のあとで、<ruby>質問<rt>しつもん (shitsumon)</rt></ruby>します。
- 中文：下課後要提問。

## 關係說明
- 表示先後順序，與表示之前的 [[JP_Grammar_29_mae_ni|〜まえに @contradicts]] 意思相反。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>仕事<rt>しごと</rt></ruby>が<ruby>終<rt>お</rt></ruby>わったあとで、<ruby>飲<rt>の</rt></ruby>みに<ruby>行<rt>い</rt></ruby>きます。
> 工作結束後去喝一杯。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 帰ります | 仕事が終わったあとで、**帰ります**。 |
| 買い物します | 仕事が終わったあとで、**買い物します**。 |
| 映画を見ます | 仕事が終わったあとで、**映画を見ます**。 |
| 友達と会います | 仕事が終わったあとで、**友達と会います**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 過去形 (〜ました) | 仕事が終わったあとで、飲みに**行きました**。 |
| → 否定 (〜ません) | 仕事が終わったあとで、飲みに**行きません**。 |
| → 普通形 (〜く) | 仕事が終わったあとで、飲みに**行く**。 |
| → 前 (〜前に) | 飲みに行く**前に**、仕事を終わらせます。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 仕事が終わったあとで、何をしますか。 | 仕事が終わったあとで、飲みに行きます。 |
| いつ飲みに行きますか。 | 仕事が終わったあとで、飲みに行きます。 |
| 授業のあとで、質問しますか。 | はい、授業のあとで、質問します。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | 行きます。 |
| +飲みに | 飲みに行きます。 |
| +あとで | あとで、飲みに行きます。 |
| +終わった | 終わったあとで、飲みに行きます。 |
| +仕事が | 仕事が終わったあとで、飲みに行きます。 |
