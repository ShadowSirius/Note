# 〜あとで

## 規則與接續
- `動詞た形/名詞の + あとで`

## 用法說明
表示在某動作或事件之後（在...之後）。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71"><ruby>仕事<rt>しごと (shigoto)</rt></ruby>が</span><span style="color:#3498DB"><ruby>終<rt>お (o)</rt></ruby>わった</span><span style="color:#FF477E">あとで、</span><span style="color:#3498DB"><ruby>飲<rt>の (no)</rt></ruby>みに<ruby>行<rt>い (i)</rt></ruby>きます。</span>
- 中文：工作結束後去喝一杯。
- 日文：<span style="color:#F39C12"><ruby>授業<rt>じゅぎょう (jugyou)</rt></ruby>の</span><span style="color:#FF477E">あとで、</span><span style="color:#3498DB"><ruby>質問<rt>しつもん (shitsumon)</rt></ruby>します。</span>
- 中文：下課後要提問。

## 關係說明
- 表示先後順序，與表示之前的 [[JP_Grammar_29_mae_ni|〜まえに @contradicts]] 意思相反。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 仕事（しごと）が終わ（お）わったあとで、飲（の）みに行（い）きます。
> （工作結束後去喝一杯。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| ご飯 | 仕事（しごと）が終わ（お）わったあとで、**ごはんを食べ（たべ）ます**。 | 工作結束後去吃飯。 |
| 家（うち） | 仕事（しごと）が終わ（お）わったあとで、**家（うち）へ帰り（かえ）ます**。 | 工作結束後回家。 |
| 映画（えいが） | 仕事（しごと）が終わ（お）わったあとで、**映画（えいが）を見（み）ます**。 | 工作結束後看電影。 |
| スポーツジム | 仕事（しごと）が終わ（お）わったあとで、**スポーツジムへ行（い）きます**。 | 工作結束後去健身房。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 仕事（しごと）が終わ（お）わったあとで、飲（の）みに**行（い）きました**。 | 工作結束後去喝了一杯。 |
| → 名詞接續形 | **仕事（しごと）の**あとで、飲（の）みに行（い）きます。 | 下班後去喝一杯。 |
| → 否定形 | 仕事（しごと）が終わ（お）わったあとで、飲（の）みに**行（い）きません**。 | 工作結束後不去喝一杯。 |
| → 句型與動作變更 | **授業（じゅぎょう）の**あとで、**質問（しつもん）します**。 | 下課後要提問。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| いつ飲（の）みに行（い）きますか。 | 仕事（しごと）が終わ（お）わったあとで、飲（の）みに行（い）きます。 | 工作結束後去喝一杯。 |
| 授業（じゅぎょう）のあとで、質問（しつもん）をしましたか。 | はい、授業（じゅぎょう）のあとで、先生（せんせい）に質問（しつもん）しました。 | 是的，下課後向老師提問了。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 行（い）きます。 | 去。 |
| +目的 | **飲（の）みに**行（い）きます。 | 去喝一杯。 |
| +時間結尾 | **終わ（お）わったあとで、**飲（の）みに行（い）きます。 | 在...之後，去喝一杯。 |
| +完整時空 | **仕事（しごと）が終わ（お）わった**あとで、飲（の）みに行（い）きます。 | 工作結束後，去喝一杯。 |
