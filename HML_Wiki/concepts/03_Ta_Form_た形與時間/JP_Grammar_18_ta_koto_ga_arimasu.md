# 〜たことがあります

## 規則與接續
- `動詞た形 + ことがあります`

## 用法說明
表示過去的經驗（曾經做過某事）。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71"><ruby>私<rt>わたし (watashi)</rt></ruby>は</span><span style="color:#FF477E"><ruby>富士山<rt>ふじさん (fujisan)</rt></ruby>に</span><span style="color:#3498DB"><ruby>登<rt>のぼ (nobo)</rt></ruby>ったことがあります。</span>
- 中文：我曾經登過富士山。
- 日文：<span style="color:#3498DB"><ruby>壽司<rt>すし (sushi)</rt></ruby>を<ruby>食<rt>た (ta)</rt></ruby>べたことがありますか。</span>
- 中文：你吃過壽司嗎？

## 關係說明
- 同樣基於過去助動詞「た」，與給予建議的 [[JP_Grammar_19_ta_hou_ga_ii_desu|〜たほうがいいです @sibling_of]] 屬於同根句型。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 私（わたし）は富士山（ふじさん）に登（のぼ）ったことがあります。
> （我曾經登過富士山。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 日本（にほん） | 私（わたし）は**日本（にほん）に行（い）った**ことがあります。 | 我曾去過日本。 |
| お寿司（すし） | 私（わたし）は**お寿司（すし）を食べ（たべ）た**ことがあります。 | 我曾吃過壽司。 |
| 歌舞伎（かぶき） | 私（わたし）は**歌舞伎（かぶき）を見（み）た**ことがあります。 | 我曾看過歌舞伎。 |
| 新幹線（しんかんせん） | 私（わたし）は**新幹線（しんかんせん）に乗（の）った**ことがあります。 | 我曾坐過新幹線。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 否定形 | 私（わたし）は富士山（ふじさん）に登（のぼ）ったことは**ありません**。 | 我沒登過富士山。 |
| → 疑問形 | あなたは富士山（ふじさん）に登（のぼ）ったことがあります**か**。 | 你登過富士山嗎？ |
| → 強調語氣形 | 私（わたし）は富士山（ふじさん）に登（のぼ）ったことが**一度（いちど）**あります。 | 我登過一次富士山。 |
| → 普通體形式 | 私（わたし）は富士山（ふじさん）に登（のぼ）ったことが**ある**。 | 我登過富士山。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 富士山（ふじさん）に登（のぼ）ったことがありますか。 | はい、富士山（ふじさん）に登（のぼ）ったことがあります。 | 是的，我曾登過富士山。 |
| お寿司（すし）を食べ（たべ）たことがありますか。 | いいえ、お寿司（すし）を食べ（たべ）たことはありません。 | 不，我沒吃過壽司。 |
| 日本（にほん）に行（い）ったことがありますか。 | はい、二回（にかい）行（い）ったことがあります。 | 是的，我去過兩次。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | あります。 | 有。 |
| +經驗 | **登（のぼ）ったこと**があります。 | 曾登過。 |
| +場所 | **富士山（ふじさん）に**登（のぼ）ったことがあります。 | 曾登過富士山。 |
| +主語 | **私（わたし）は**富士山（ふじさん）に登（のぼ）ったことがあります。 | 我曾登過富士山。 |
