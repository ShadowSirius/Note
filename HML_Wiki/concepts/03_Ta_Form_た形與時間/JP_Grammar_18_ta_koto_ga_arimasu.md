# 〜たことがあります

## 規則與接續
- `動詞た形 + ことがあります`

## 用法說明
表示過去的經驗（曾經做過某事）。

## 實用例句
- 日文：<ruby>私<rt>わたし (watashi)</rt></ruby>は<ruby>富士山<rt>ふじさん (fujisan)</rt></ruby>に<ruby>登<rt>のぼ (nobo)</rt></ruby>ったことがあります。
- 中文：我曾經登過富士山。
- 日文：<ruby>壽司<rt>すし (sushi)</rt></ruby>を<ruby>食<rt>た (ta)</rt></ruby>べたことがありますか。
- 中文：你吃過壽司嗎？

## 關係說明
- 同樣基於過去助動詞「た」，與給予建議的 [[JP_Grammar_19_ta_hou_ga_ii_desu|〜たほうがいいです @sibling_of]] 屬於同根句型。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>私<rt>わたし</rt></ruby>は<ruby>富士山<rt>ふじさん</rt></ruby>に<ruby>登<rt>のぼ</rt></ruby>ったことがあります。
> 我曾經登過富士山。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 日本に行きます | 私は**日本に行った**ことがあります。 |
| 寿司を食べます | 私は**寿司を食べた**ことがあります。 |
| 歌舞伎 (かぶき) を見ます | 私は**歌舞伎を見た**ことがあります。 |
| 日本語を話します | 私は**日本語を話した**ことがあります。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定 (〜ありません) | 私は富士山に登ったことが**ありません**。 |
| → 疑問 (〜ありますか) | 富士山に登ったことが**ありますか**。 |
| → 普通形 (〜ある) | 私は富士山に登ったことが**ある**。 |
| → 普通形否定 (〜ない) | 私は富士山に登ったことが**ない**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 富士山に登ったことがありますか。 | はい、登ったことがあります。 |
| 寿司を食べたことがありますか。 | いいえ、食べたことがありません。 |
| 日本に行ったことがありますか。 | はい、一度だけ行ったことがあります。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | あります。 |
| +登ったこと | 登ったことがあります。 |
| +富士山に | 富士山に登ったことがあります。 |
| +私は | 私は富士山に登ったことがあります。 |
