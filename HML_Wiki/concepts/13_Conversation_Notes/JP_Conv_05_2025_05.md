### 項目 1：口答え（くちごたえ / kuchigotae）
* **地道中文釋義**：頂嘴、反駁
* **用語意思解釋**：指對上級、長輩或老師的指示與批評進行不禮貌的回擊。
* **來源與典故**：由「口（嘴）」與「答え（回答）」結合而成，暗示不服從、不尊重，帶有強烈的負面感情色彩。
* **使用時機與社交場合**：通常專指「以下犯上」的反駁。同輩之間的爭論不叫口答え。常用於長輩對晚輩、上司對部下的訓斥。
* **完整生活對話與例句對照**：
  | 對照句子 | 類型/語氣 | 中文翻譯 |
|  | :--- | :--- | :--- |
|  | 私に<span style="color: #e74c3c;">**口答えするな**</span>！ | <span style="color: #e74c3c;">**強勢命令 / 粗獷**</span> | 不准跟我頂嘴！ |
|  | <span style="color: #2ecc71;">**口答えしないで**</span>ください。 | <span style="color: #2ecc71;">**溫柔規勸 / 禮貌**</span> | 請不要頂嘴。 |

### 項目 5：滅相もない（めっそうもない / messoumonai）
* **地道中文釋義**：荒唐透頂、哪裡的話（不敢當）
* **用語意思解釋**：在否定句中，表示「荒謬、絕對不行、不能原諒」；在社交謙遜場合，表示「萬萬不敢當、您太過獎了」。
* **來源與典故**：源自佛教八相中的「滅相」（萬物消滅、破壞的相狀）。「滅相もない」字面意思是「連消滅的可能性都沒有」，引申為事情荒謬至極、不合常理。
* **使用時機與社交場合**：用於強烈否定不合理的批評或謠言；或者在商務場合中，謙虛地推辭長輩、客戶的讚美。
* **完整生活對話與例句對照**：
  | 對照句子 | 類型/語氣 | 中文翻譯 |
|  | :--- | :--- | :--- |
|  | そんなことをするなんて、<span style="color: #3498db;">**滅相もない**</span>ことです。 | <span style="color: #3498db;">**客觀自謙/強烈否定**</span> | 做出那種事，簡直是萬萬不可能（荒唐至極）。 |
|  | A: 英語がお上手ですね。<br>B: <span style="color: #2ecc71;">**滅相もない**</span>、まだまだです。 | <span style="color: #2ecc71;">**溫和自謙 / 社交敬語**</span> | A: 您英文真好！<br>B: **哪裡的話，不敢當**，還差得遠呢。 |

### 項目 6：ストレス（すとれす / sutoresu）

* **來源與典故**：源自英語 "stress"。日本社會非常注重心理健康與身心症狀的關聯，疲勞、失眠通常會被歸因於此。
* **使用時機與社交場合**：抱怨工作繁重、傾訴心理壓抑、關心他人身心狀況時。

### 項目 9：丸呑み（まるのみ / marunomi）

* **使用時機與社交場合**：商業談判被迫妥協、或在不對等的人際關係中無奈服從時。

|  | <span style="color: #e74c3c;">**人見知りしちゃって**</span>、パーティーで誰とも話せなかった。 | <span style="color: #e74c3c;">**口語遺憾體 (懊惱)**</span> | 因為不小心太怕生，在派對上我居然跟誰都沒說到話。 |

### 項目 30：顔見知り（かおみしり / kaomishiri）
* **地道中文釋義**：點頭之交、臉熟的人
* **用語意思解釋**：指見過幾次面、互相認得，但完全談不上熟悉、沒有深交的人。是社會關係中最小的熟人單位。
* **來源與典故**：由「顔（臉）」和「知り（知道、認得）」結合而成，意為僅僅是認得這張臉。
* **使用時機與社交場合**：形容社區的鄰居、常去便利商店的店員、或是跨部門見過幾次的同事，強調維持目前的距離即可。
* **完整生活對話與例句對照**：
  | 對照句子 | 類型/語氣 | 中文翻譯 |
|  | :--- | :--- | :--- |
|  | 彼とはただの<span style="color: #3498db;">**顔見知りだ**</span>から、連絡先は知らない。 | <span style="color: #3498db;">**基本客觀陳述**</span> | 我跟他只是點頭之交（臉熟），所以沒有他的聯絡方式。 |
|  | 近所の<span style="color: #2ecc71;">**顔見知りの人**</span>に、笑顔で挨拶する。 | <span style="color: #2ecc71;">**溫和良好行為**</span> | 面對鄰居那些面熟的點頭之交，我會微笑著打招呼。 |
