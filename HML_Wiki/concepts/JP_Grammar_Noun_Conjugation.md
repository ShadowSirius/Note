> [!important] 句子結構色彩標記：
> 🟢 **主語/主題** (`#2ECC71`) \| 🔵 **核心/謂語** (`#3498DB`) \| 🟠 **修飾語/賓語** (`#E67E22`) \| 🔴 **時間/場所/副詞** (`#E74C3C`)

## 📂 第一部分：名詞時態與語氣變化表

日語名詞與 **な形容詞** 的語法變化幾乎完全一致，但在修飾名詞或接續時有著微妙的不同。

| 時態與變化種類 | 普通體 (Casual) | 丁寧形 (Polite) | 結構例句 |
| :--- | :--- | :--- | :--- |
| **① 現在肯定形** | 名詞 + `だ`。 <br> 例：`学生だ` | 名詞 + `です`。 <br> 例：`学生です` | <span style="color:#2ECC71">彼は</span><span style="color:#3498DB">学生です。</span> <br>（他是學生。） |
| **② 現在否定形** | 名詞 + `ではない` / `じゃない`。 <br> 例：`学生じゃない` | 名詞 + `ではありません` / `じゃないです`。 <br> 例：`学生ではありません` | <span style="color:#2ECC71">私は</span><span style="color:#3498DB">先生じゃない。</span> <br>（我不是老師。） |
| **③ 過去肯定形** | 名詞 + `だった`。 <br> 例：`学生だった` | 名詞 + `でした`。 <br> 例：`学生でした` | <span style="color:#E74C3C">去年まで</span><span style="color:#2ECC71">彼は</span><span style="color:#3498DB">学生だった。</span> <br>（到去年為止他是學生。） |
| **④ 過去否定形** | 名詞 + `ではなかった` / `じゃなかった`。 <br> 例：`学生じゃなかった` | 名詞 + `ではありませんでした` / `じゃなかったです`。 <br> 例：`学生ではありませんでした` | <span style="color:#E74C3C">昨日は</span><span style="color:#3498DB">雨ではありませんでした。</span> <br>（昨天沒有下雨。） |

### 2. 語氣延伸：勧誘・推測・確認

| 語氣 | 動詞 | 名詞／な形／い形 |
| :--- | :--- | :--- |
| **勧誘**（邀約） | 〜ましょう | （無此形式） |
| **推測**（下降調） | （無此形式，用 だろう／でしょう 接動詞普通形） | 〜でしょう ↘（語調下降，表示說話者自己不確定的推測，如「明日は雨でしょう。」） |
| **確認**（上揚調） | （同上） | 〜でしょう ↗（語調上揚，接近「〜ですよね」，表示向對方尋求同意或確認，如「これでいいでしょう？」） |

> [!warning] 易混淆
> でしょう 這一形式**書寫上完全相同**，差別只在**語調**：句尾語調下降＝單純推測（不確定，講給自己聽的猜測）；句尾語調上揚＝向對方確認、尋求附和（近似疑問句「對吧？」）。詳細例句與 かもしれない 對照見 [[JP_Grammar_50_C33_conjecture_deshou_kamoshirenai|〜でしょう・〜かもしれない：推量 @related_to]]。

## 📂 第二部分：名詞的接續規則

### 1. 名詞與名詞的修飾接續
當一個名詞修飾另一個名詞時，中間必須使用助詞 **の** 連接：
*   **規則**：`名詞 A` + **の** + `名詞 B`
*   例：`日本語` + `本` → **日本語の本** (日文書)
*   例：`先生` + `車` → **先生の車** (老師的車)

### 2. 名詞的中頓連接（並列或因果）
在句子中間連接兩個名詞時，使用 **で** 作為中頓形式：
*   **規則**：`名詞 A` + **で**，`名詞 B` ...
*   例：<span style="color:#2ECC71">彼は</span><span style="color:#3498DB">医者で、</span><span style="color:#2ECC71">親切な</span><span style="color:#3498DB">人です。</span> (他是醫生，而且是個親切的人。)
*   例：<span style="color:#E74C3C">台風で、</span><span style="color:#3498DB">学校が休みになりました。</span> (因為颱風，學校放假了。)
