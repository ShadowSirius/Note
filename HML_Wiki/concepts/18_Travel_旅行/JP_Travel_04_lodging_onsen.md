# 旅遊詞彙 04：旅館與溫泉

## 🧠 記憶鉤
- **「湯」字家族一網打盡**：核心字「湯(ゆ)」＝熱水／泡湯，一次串起一整組複合詞——**銭湯**(せんとう，公共澡堂)、**足湯**(あしゆ，泡腳池)、**男湯／女湯**(おとこゆ／おんなゆ，男湯女湯)、**湯船**(ゆぶね，浴缸/浴池)、**湯殿**(ゆどの，湯屋，較古雅的說法)。記住「湯」單獨唸音読ト**ウ**（例：熱湯）、訓読み獨用或結尾時唸**ゆ**，複合詞常見連濁（如 銭湯 せんとう 不濁，但足湯 あしゆ 不濁——連濁與否需逐詞記，不是規則）。
- **素泊まり(すどまり)拆字記法**：素(す)＝樸素／不附加 ＋ 泊まり(とまり)＝過夜住宿 ＝「不含餐的純住宿方案」。對比「一泊二食(いっぱくにしょく)」＝一泊含兩餐（晚餐+早餐），素泊まり是它的反義planned方案。
- **貸切(かしきり)＝貸(貸出/借出) ＋ 切(切斷/整個)＝「整個借斷」＝包場**。常見搭配：貸切風呂(かしきりぶろ，包場浴池，情侶/家庭湯屋首選)。
- **風呂(ふろ)音読み小知識**：「風呂」本身是唯讀訓読み固定詞，但構詞時常見「〜風呂」保留清音（水風呂 みずぶろ 連濁成 ぶろ），記口訣「單獨風呂唸フロ，接在後面就變濁ブロ」。
- **食堂(しょくどう)＝食(しょく，吃) ＋ 堂(どう，大廳/場所)**，與「食堂車」「学生食堂」同一構詞邏輯，旅館食堂＝提供餐食的公共用餐空間，與私人房內用餐（部屋食）相對。

## ⚠️ 易混淆對比
| 易混組合 | 差異 |
|---|---|
| 風呂(ふろ) vs 浴室(よくしつ) vs バスルーム | 風呂＝和式泡澡文化整體概念（含浴缸泡澡行為本身，如「お風呂に入る」）；浴室＝房間格局用詞，單純指「有浴缸/淋浴的房間」；バスルーム＝西式外來語，多見於飯店／西式住宅語境，三者常可指同一空間但語感由和到洋遞進 |
| 温泉(おんせん) vs 銭湯(せんとう) | 温泉＝天然湧出的溫泉水，多在旅館或溫泉地，強調泉質與療效；銭湯＝人工燒水的**公共澡堂**，設於一般市區街坊供居民付費洗澡，兩者水源本質不同，但外觀都可能有「大浴場」 |
| 旅館(りょかん) vs ホテル | 旅館＝日式傳統住宿，多為榻榻米房＋一泊二食＋浴衣＋大浴場文化；ホテル＝西式飯店，多為床鋪房＋自理餐食，兩者服務型態與空間設計截然不同 |
| 垢すり(あかすり) vs タオル | **垢すり＝搓垢巾（或指搓澡服務），用來搓掉皮膚汚垢的專用粗布/浴巾**；毛巾／擦拭用巾是「タオル」。原稿誤將垢すり標為「毛巾」，需訂正：垢すり是搓澡動作/工具，タオル才是一般毛巾 |
| 日帰り(ひがえり) vs 一泊 | **日帰り讀作「ひがえり」（原稿誤標ひかえり，已訂正）**，指「當天來回、不過夜」，常見於「日帰り温泉」（只泡湯不住宿的溫泉設施）；一泊(いっぱく)則指過夜住宿一晚 |
| 広間(ひろま) vs ロビー | **広間＝旅館內的宴會/集會大廳（榻榻米大房間，常用於宴席、卡拉OK、休憩），原稿標註「lobby?」並不準確**；西式飯店的「大廳」正確說法是外來語**ロビー**(robii)，両者用途與空間性質不同，不可互換 |
| 部屋食(へやしょく) vs 食堂 | 部屋食＝餐點送到自己房間內用餐（高級旅館常見）；食堂＝公共用餐空間，需前往指定場所用餐 |

| 門限(もんげん)はありますか。 | 有門禁嗎? |
| Wi-Fiのパスワードを教(おし)えてください。 | 請告訴我 Wi-Fi 密碼。 |

> [!tip] 樓層讀音陷阱:1階(いっかい)、3階(さんがい,連濁)、何階(なんがい)——1・3・何 後面的「階」變音,其餘唸かい。「〜となっております」「〜でございます」是櫃檯敬語定型句,聽懂即可,不必會說。
