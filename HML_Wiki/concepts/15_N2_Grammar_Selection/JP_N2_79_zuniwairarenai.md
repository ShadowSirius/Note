# 〜ずにはいられない

## 規則與接續
- `動詞ない形（去ない） + ずにはいられない`（する→せずにはいられない）

## 用法說明
表示「禁不住...、不由自主地...」，指情感或衝動強烈到無法抑制，非做不可。與 [[JP_N2_73_naidewairarenai|〜ないではいられない @related_to]] 意思相同、可互換，但「ずにはいられない」語感較書面。「ず」為文語否定，後接「には＋いられない」構成雙重否定的強調。另與 [[JP_Grammar_50_te_wa_irarenai|〜てはいられない @related_to]] 形近而義異：後者表「沒空、不能再持續某狀態」，非情感抑制。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> その知（し）らせを聞（き）いて、涙（なみだ）を流（なが）さずにはいられなかった。
> （聽到那個消息，忍不住流下了眼淚。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 笑（わら）う | **笑わずにはいられなかった**。 | 忍不住笑了出來。 |
| 文句（もんく）を言（い）う | **文句を言わずにはいられなかった**。 | 忍不住抱怨了。 |
| 心配（しんぱい）する | **心配せずにはいられなかった**。 | 忍不住擔心了起來。 |
| 質問（しつもん）する | **質問せずにはいられなかった**。 | 忍不住提問了。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 口語形（ないでは） | 涙を流（なが）**さないではいられなかった**。 | 忍不住流下了眼淚（口語）。 |
| → 現在形 | 涙を流（なが）**さずにはいられない**。 | 忍不住要流淚。 |
| → 疑問形 | 涙を流（なが）さずにはいられなかった**か**。 | 是不是忍不住流淚了？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうして泣（な）いたんですか。 | あの知らせを聞いて、泣かずにはいられなかったんです。 | 因為聽到那個消息，忍不住哭了。 |
| その小説（しょうせつ）、面白（おもしろ）いですか。 | ええ、読み始めたら最後まで読まずにはいられませんよ。 | 是啊，一開始讀就忍不住要讀完。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 流（なが）さずにはいられなかった。 | 忍不住流了。 |
| +對象 | **涙（なみだ）を**流さずにはいられなかった。 | 忍不住流下了眼淚。 |
| +契機 | **その知（し）らせを聞（き）いて、**涙を流さずにはいられなかった。 | 聽到那個消息，忍不住流下了眼淚。 |
