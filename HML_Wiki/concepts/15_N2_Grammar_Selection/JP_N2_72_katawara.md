# 〜かたわら

## 規則與接續
- `動詞辞書形 + かたわら`
- `名詞 + の + かたわら`

## 用法說明
表示「一邊...一邊（從事另一件事）」，指以前項為本業或主要活動，同時長期進行後項的另一項活動（多為兼職、副業、興趣），強調兩者並行持續一段時間的狀態，語氣書面、正式，常見於介紹人物經歷。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

  中文：他一邊在公司上班，一邊寫小說。
- 日文：<span style="color:#FF9F1C"><ruby>育児<rt>いくじ</rt></ruby>のかたわら、</span><span style="color:#3498DB"><ruby>大学院<rt>だいがくいん</rt></ruby>で<ruby>研究<rt>けんきゅう</rt></ruby>を<ruby>続<rt>つづ</rt></ruby>けている。</span>
  中文：一邊育兒，一邊在研究所持續做研究。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）は会社（かいしゃ）勤（づと）めのかたわら、小説（しょうせつ）を書（か）いている。
> （他一邊在公司上班，一邊寫小說。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 主婦業（しゅふぎょう）・翻訳（ほんやく）の仕事をしている | 彼女は**主婦業のかたわら**、翻訳の仕事をしている。 | 她一邊當家庭主婦，一邊做翻譯工作。 |
| 農業（のうぎょう）・民宿（みんしゅく）を経営（けいえい）している | 彼は**農業のかたわら**、民宿を経営している。 | 他一邊務農，一邊經營民宿。 |
| 教師（きょうし）を務（つと）める・研究（けんきゅう）を進（すす）める | 彼は**教師を務めるかたわら**、研究を進めている。 | 他一邊擔任教師，一邊進行研究。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 動詞辞書形 | 彼は音楽（おんがく）を**教えるかたわら**、作曲（さっきょく）もしている。 | 他一邊教音樂，一邊也從事作曲。 |
| → 近義比較（がてらとの違い） | かたわらは長期的（ちょうきてき）な兼業（けんぎょう）、がてらは一回限（いっかいかぎ）りの動作に使う。 | 「かたわら」用於長期兼業，「がてら」用於單次動作。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 彼はいつ小説を書いているんですか。 | **会社勤めのかたわら**、書いているそうです。 | 聽說他是在上班之餘寫的。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 小説を書いている。 | 在寫小說。 |
| +本業 | 会社勤めの**かたわら**、小説を書いている。 | 一邊在公司上班，一邊寫小說。 |
| +補述 | 彼は昼間（ひるま）の会社勤めの**かたわら**、夜（よる）は毎日欠（か）かさず小説を書いている。 | 他白天在公司上班，晚上則每天不間斷地寫小說。 |
