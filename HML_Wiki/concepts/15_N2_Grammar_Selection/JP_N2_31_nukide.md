# 〜ぬきで

## 規則與接續
- `名詞 + ぬきで／ぬきに`
- `名詞 + ぬきにして／ぬきにしろ`
- `名詞 + ぬきの + 名詞`

## 用法說明
表示「省去、撇開、不包含」原本應該有或習慣上會有的事物，語氣帶有「刻意排除」的主動感。常見於商務或正式場合，如「冗談はぬきにして（不開玩笑了）」。若接「〜ぬきでは／〜ぬきには」＋否定，則強調該項目是不可或缺的前提條件。

> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

  中文：撇開玩笑不談，我們認真討論吧。
- 日文：<span style="color:#2ECC71"><ruby>田中<rt>たなか</rt></ruby>さん</span><span style="color:#FF477E">ぬきでは、</span><span style="color:#3498DB">このプロジェクトは<ruby>成功<rt>せいこう</rt></ruby>しない。</span>
  中文：如果沒有田中先生，這個專案就不會成功。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 挨拶（あいさつ）はぬきにして、さっそく会議（かいぎ）を始（はじ）めましょう。
> （省略寒暄，我們馬上開始會議吧。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| お世辞（せじ） | お世辞は**ぬきにして**、さっそく会議を始めましょう。 | 撇開客套話不談，我們馬上開始會議吧。 |
| 前置（まえお）き | 前置きは**ぬきにして**、さっそく会議を始めましょう。 | 撇開開場白不談，我們馬上開始會議吧。 |
| 遠慮（えんりょ） | 遠慮は**ぬきにして**、さっそく会議を始めましょう。 | 不要客氣，我們馬上開始會議吧。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → ぬきの＋名詞 | わさびを入（い）れないお寿司（すし） → わさび**ぬきの**お寿司 | 不加芥末的壽司 |
| → ぬきでは＋否定 | 社長（しゃちょう）がいないと進（すす）められない → 社長**ぬきでは**進められない | 沒有社長就無法推進 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| コーヒーには砂糖（さとう）を入れますか。 | いいえ、砂糖**ぬきで**お願いします。 | 不，麻煩不要加糖。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 仕事（しごと）の話はぬきにしましょう。 | 撇開工作的話題吧。 |
| +場合 | 今夜（こんや）は、仕事の話は**ぬきにして**パーティーをしましょう。 | 今晚就撇開工作話題辦派對吧。 |
| +心情 | 今夜は、仕事の話は**ぬきにして**、楽（たの）しくパーティーをしましょう。 | 今晚就撇開工作話題，開心地辦派對吧。 |
