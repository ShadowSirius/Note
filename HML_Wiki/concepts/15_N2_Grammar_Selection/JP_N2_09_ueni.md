# 〜うえに

## 規則與接續
- `用言普通形 + うえに`
- `な形容詞語幹 / 名詞 + なうえに`

## 用法說明
表示「不僅...而且...」，用來疊加兩件性質相同（同為正面或同為負面）的事項，語氣比たり／し更書面正式。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71">この<ruby>部屋<rt>へや</rt></ruby>は</span><span style="color:#FF477E"><ruby>狭<rt>せま</rt></ruby>いうえに、</span><span style="color:#3498DB"><ruby>家賃<rt>やちん</rt></ruby>も<ruby>高<rt>たか</rt></ruby>い。</span>
  中文：這房間不僅狹小，房租還很貴。
- 日文：<span style="color:#2ECC71"><ruby>彼<rt>かれ</rt></ruby>は</span><span style="color:#FF477E"><ruby>頭<rt>あたま</rt></ruby>がいいうえに、</span><span style="color:#3498DB"><ruby>努力家<rt>どりょくか</rt></ruby>でもある。</span>
  中文：他不僅頭腦好，還是個努力的人。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> この部屋（へや）は狭（せま）いうえに、家賃（やちん）も高（たか）い。
> （這房間不僅狹小，房租還很貴。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 便利だ（べんりだ） | ここは**便利なうえに**、静かだ。 | 這裡不僅方便，還很安靜。 |
| 雨が降った | **雨が降ったうえに**、風も強かった。 | 不僅下雨，風也很大。 |
| 学生だ | 彼は**学生であるうえに**、アルバイトもしている。 | 他不僅是學生，還在打工。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 狭かった**うえに**、暗かった。 | 不僅狹小，還很暗。 |
| → 否定形 | 便利では**ないうえに**、危険だ。 | 不僅不方便，還很危險。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| この部屋はどうですか。 | 狭いうえに、家賃も高いです。 | 不僅狹小，房租還很貴。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | うえに、高い。 | 不僅如此，還很貴。 |
| +前項 | **狭いうえに**、高い。 | 不僅狹小，還很貴。 |
| +主語 | **この部屋は**狭いうえに、家賃も高い。 | 這房間不僅狹小，房租還很貴。 |
