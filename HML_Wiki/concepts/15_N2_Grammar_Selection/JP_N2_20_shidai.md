# 〜次第

## 規則與接續
- `動詞ます形 + 次第`（一...就立刻...）
- `名詞 + 次第`（全憑...、視...而定）

## 用法說明
兩種主要用法：①「動詞ます形＋次第」表示前項一旦完成，就立刻進行後項動作；②「名詞＋次第」表示結果完全取決於該事物，如「結果次第」「天気次第」。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 資料（しりょう）が届（とど）き次第（しだい）、ご連絡（れんらく）いたします。
> （資料一送到，就立刻與您聯絡。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 準備ができる（じゅんびができる） | **準備ができ次第**、出発します。 | 一準備好，就立刻出發。 |
| 決まる（きまる） | 日程が**決まり次第**、お知らせします。 | 行程一決定，就通知您。 |
| 天気（てんき） | 遠足に行くかどうかは**天気次第だ**。 | 是否去遠足全憑天氣。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 丁寧形 | 連絡いたし**ます**。 | （謙讓禮貌形） |
| → 對比たら | 届いた**ら**、連絡します。 | 到了的話就聯絡。（たら較口語） |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| いつ連絡してもらえますか。 | 資料が届き次第、ご連絡いたします。 | 資料一到就立刻聯絡您。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 次第、連絡します。 | 一...就聯絡。 |
| +動作 | **資料が届き**次第、連絡します。 | 資料一到就聯絡。 |
| +禮貌 | 資料が届き次第、**ご連絡いたします**。 | 資料一到，就與您聯絡。 |
