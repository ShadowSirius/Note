# 〜ぶり

## 規則與接續
- `時間名詞 + ぶり`（表示「闊別...、相隔...」）
- `動詞ます形 + ぶり`（表示「...的樣子、方式」，如 話し方＝話しぶり）

## 用法說明
有兩種常見用法：①接時間名詞，表示「闊別...、相隔...」，強調距離上次發生已經過了一段時間才再度發生；②接動詞ます形，表示「...的樣子、方式」（話しぶり＝說話的樣子、飲みっぷり＝喝酒的豪爽程度），描述做某動作時所展現的風格或態度。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 十年（じゅうねん）ぶりに故郷（こきょう）に帰（かえ）った。
> （闊別十年回到了故鄉。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 三年（さんねん）・友達（ともだち）に会（あ）った | **三年ぶりに**友達に会った。 | 睽違三年見到了朋友。 |
| 一週間（いっしゅうかん）・雨（あめ）が止（や）んだ | **一週間ぶりに**雨が止んだ。 | 睽違一週雨終於停了。 |
| 久（ひさ）しぶり・運動（うんどう）した | **久しぶりに**運動した。 | 隔了好久終於運動了。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 動詞ます形＋ぶり | 彼の**飲みっぷり**は本当に豪快（ごうかい）だ。 | 他喝酒的豪爽勁真是了不起。 |
| → 名詞修飾 | 十年ぶりの**故郷は変わっていた**。 | 闊別十年的故鄉已經改變了。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| いつ故郷に帰ったんですか。 | **十年ぶりに**帰りました。 | 是闊別十年才回去的。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 故郷に帰った。 | 回到了故鄉。 |
| +期間 | **十年ぶりに**故郷に帰った。 | 闊別十年回到了故鄉。 |
| +補述 | 仕事の都合（つごう）でずっと帰れなかったが、ようやく**十年ぶりに**故郷に帰った。 | 因為工作的緣故一直沒能回去，終於在闊別十年後回到了故鄉。 |
