# 〜だらけ

## 規則與接續
- `名詞 + だらけ`

## 用法說明
表示「滿是...」，指某個空間或事物表面到處都是（多為負面、不理想的東西），常接抽象名詞（間違い、借金）或具體污漬（泥、傷）。與「まみれ」意思相近，但「まみれ」偏向液狀、黏著物附著於身體或物體表面（如泥まみれ、汗まみれ），使用範圍較窄；「だらけ」的適用範圍更廣，也可用於抽象事物（間違いだらけ、傷だらけ）。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> この作文（さくぶん）は間違（まちが）いだらけだ。
> （這篇作文錯誤百出。）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 借金（しゃっきん） | 彼の生活（せいかつ）は**借金だらけだ**。 | 他的生活滿是債務。 |
| 傷（きず） | この机（つくえ）は**傷だらけだ**。 | 這張桌子滿是刮痕。 |
| ほこり | 部屋（へや）は**ほこりだらけだ**。 | 房間裡滿是灰塵。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 修飾名詞 | 間違いだらけの**作文を提出した**。 | 交出了一篇錯誤百出的作文。 |
| → になる | 子どもは泥**だらけになって**帰ってきた。 | 孩子弄得滿身泥巴地回來了。 |

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 間違いだらけだ。 | 滿是錯誤。 |
| +主題 | この作文は**間違いだらけだ**。 | 這篇作文錯誤百出。 |
| +補述 | 急いで書いたせいか、この作文は**間違いだらけで**、先生に叱られた。 | 也許是趕著寫的緣故，這篇作文錯誤百出，被老師罵了一頓。 |
