# 〜げ

## 規則與接續
- `い形容詞語幹／な形容詞語幹 + げ`（+ な + 名詞、+ に + 動詞、+ だ 結句）
- `動詞ます形 + たげ`（如 言いたげ＝欲言又止的樣子）

## 用法說明
接在形容詞語幹後，表示「看起來...的樣子」，是說話者從外部觀察到的印象，帶有一種微妙、含蓄的樣態描寫，多見於文學性描寫（悲しげ、意味ありげ、自信ありげ）。與「そうだ（樣態）」相近，但「げ」語感更古典、文雅，且僅限少數固定搭配。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼女（かのじょ）は悲（かな）しげな顔（かお）をしてうつむいた。
> （她露出悲傷的神情低下了頭。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 不安（ふあん） | 彼女は**不安げな顔**をしてうつむいた。 | 她露出不安的神情低下了頭。 |
| 寂（さび）しい | 彼女は**寂しげな顔**をしてうつむいた。 | 她露出寂寞的神情低下了頭。 |
| 満足（まんぞく） | 彼女は**満足げな顔**をしてうなずいた。 | 她露出滿足的神情點了點頭。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → に＋動詞 | 彼は自信**ありげに**手を挙げた。 | 他一副胸有成竹的樣子舉起了手。 |
| → 言いたげ | 彼女は何か**言いたげな顔**でこちらを見た。 | 她一副欲言又止的樣子看著這邊。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 彼女はどんな様子（ようす）でしたか。 | **悲しげな顔**をしてうつむいていました。 | 她露出悲傷的神情低著頭。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 悲しげな顔をした。 | 露出了悲傷的神情。 |
| +主語 | 彼女は**悲しげな顔**をした。 | 她露出了悲傷的神情。 |
| +補述 | 彼女は何も言わずに**悲しげな顔**をしてうつむいてしまった。 | 她什麼也沒說，就露出悲傷的神情低下了頭。 |
