# 〜とみえる

## 規則與接續
- `動詞・形容詞普通形 + とみえる`
- `な形容詞語幹／名詞 + だとみえる`

## 用法說明
表示「看來是...」，依據觀察到的具體線索或客觀狀況所做出的推測，語氣較客觀、留有餘地，並非說話者主觀臆測。常可與「らしい」「ようだ」互換，但「とみえる」更強調「從眼前的證據看來」的推論過程。

> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

  中文：他看起來很累，跟他說話也心不在焉。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 道（みち）が濡（ぬ）れている。夜（よる）のうちに雨（あめ）が降（ふ）ったとみえる。
> （路是濕的。看來是晚上下過雨了。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 電気（でんき）が消（き）えている・出（で）かけた | 電気が消えている。**出かけたとみえる**。 | 燈是關的。看來是出門了。 |
| 傘（かさ）が濡（ぬ）れている・雨（あめ）に降（ふ）られた | 傘が濡れている。**雨に降られたとみえる**。 | 傘是濕的。看來是被雨淋到了。 |
| 部屋（へや）が静（しず）かだ・もう寝（ね）た | 部屋が静かだ。**もう寝たとみえる**。 | 房間很安靜。看來已經睡了。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → な形容詞 | 表情（ひょうじょう）から見て、不満（ふまん）**だとみえる**。 | 從表情看來，似乎是不滿意。 |
| → て形＋とみえて | 彼は疲れている**とみえて**、話しかけても上の空だった。 | 他看起來很累，跟他說話也心不在焉。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうして道が濡れているんですか。 | 夜のうちに**雨が降ったとみえます**。 | 看來是晚上下過雨了。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 雨が降ったとみえる。 | 看來是下過雨了。 |
| +時間 | 夜のうちに**雨が降ったとみえる**。 | 看來是晚上下過雨了。 |
| +根拠 | 道が濡れているところを見ると、夜のうちに**雨が降ったとみえる**。 | 從路面濕的樣子看來，晚上似乎下過雨了。 |
