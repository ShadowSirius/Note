# 〜限りだ

## 規則與接續
- `い形容詞普通形（現在形） + 限りだ`
- `な形容詞語幹 + な + 限りだ`

## 用法說明
表示「真是非常...」，用來表達說話者當下極端強烈的個人感受，多接感情形容詞（うれしい、心細い、羨ましい、残念だ等），帶有主觀誇張的語氣，僅限第一人稱視角使用，不可用於描述他人的感受。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 一人（ひとり）で海外（かいがい）に住（す）むのは、心細（こころぼそ）い限（かぎ）りだ。
> （一個人住在國外，真是非常不安。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 羨（うらや）ましい | 若（わか）くて才能（さいのう）がある彼が**羨ましい限りだ**。 | 他既年輕又有才華，真是令人羨慕。 |
| 心強（こころづよ）い | 皆（みな）が助（たす）けてくれて、**心強い限りだ**。 | 大家都來幫忙，真是非常安心可靠。 |
| 残念（ざんねん）な | 試合（しあい）に負（ま）けて、**残念な限りだ**。 | 比賽輸了，真是非常遺憾。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → な形容詞 | 皆に祝（いわ）ってもらって、幸（しあわ）せ**な限りだ**。 | 得到大家的祝福，真是太幸福了。 |
| → 理由節＋限りだ | 息子が第一志望に合格して、**うれしい限りだ**。 | 兒子考上了第一志願，真是太高興了。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 一人暮（ひとりぐ）らしはどうですか。 | 正直（しょうじき）、**心細い限りです**。 | 老實說，真是非常不安。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 心細い限りだ。 | 真是非常不安。 |
| +状況 | 一人で海外に住むのは、**心細い限りだ**。 | 一個人住在國外，真是非常不安。 |
| +補述 | 言葉（ことば）も通（つう）じない一人で海外に住むのは、**心細い限りだ**。 | 語言又不通，一個人住在國外，真是非常不安。 |
