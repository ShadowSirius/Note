# 〜せいか

## 規則與接續
- `用言普通形 + せいか`
- `な形容詞語幹 / 名詞 + の / な + せいか`

## 用法說明
表示「可能是因為...」，用來推測某個不確定的原因所導致的結果，語氣比 [[JP_N2_11_seide|〜せいで @related_to]] 更委婉不確定。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E"><ruby>年<rt>とし</rt></ruby>のせいか、</span><span style="color:#3498DB">すぐ<ruby>疲<rt>つか</rt></ruby>れるようになった。</span>
  中文：可能是因為年紀大了，變得很容易累。
- 日文：<span style="color:#FF477E"><ruby>寝不足<rt>ねぶそく</rt></ruby>のせいか、</span><span style="color:#3498DB"><ruby>頭<rt>あたま</rt></ruby>が<ruby>痛<rt>いた</rt></ruby>い。</span>
  中文：可能是睡眠不足的緣故，頭很痛。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 年（とし）のせいか、すぐ疲（つか）れるようになった。
> （可能是因為年紀大了，變得很容易累。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 疲れている（つかれている） | **疲れているせいか**、集中できない。 | 可能是因為累了，無法集中精神。 |
| 天気が悪い（てんきがわるい） | **天気が悪いせいか**、客が少ない。 | 可能是因為天氣不好，客人很少。 |
| 静かだ | ここは**静かなせいか**、よく眠れる。 | 可能是因為這裡安靜，睡得很好。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 對比せいで | 寝不足の**せいで**、頭が痛い。 | 都怪睡眠不足，頭很痛。（更確定的原因） |
| → 過去形 | 疲れていた**せいか**、集中できなかった。 | 可能是因為累了，沒能集中精神。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうしてすぐ疲れるようになったんですか。 | 年のせいか、最近すぐ疲れます。 | 可能是年紀的關係，最近很容易累。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | せいか、疲れる。 | 可能是因為如此，容易累。 |
| +原因 | **年の**せいか、疲れる。 | 可能是年紀的關係，容易累。 |
| +程度 | 年のせいか、**すぐ**疲れるようになった。 | 可能是年紀的關係，變得很容易累。 |
