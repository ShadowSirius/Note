# 〜に相違ない

## 規則與接續
- `動詞・形容詞普通形 + に相違ない`
- `な形容詞語幹／名詞 + に相違ない`

## 用法說明
表示「一定是...、必定是...」，說話者依據客觀根據做出強烈的斷定推測。語氣比「に違いない」更書面、正式，常見於新聞報導、鑑定報告、推理小說等場合，帶有不容置疑的確信感。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

  中文：從他的說話方式來看，這個人肯定是關西出身。

  中文：從現場留下的指紋來看，犯人肯定進過這個房間。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）の話（はな）し方（かた）から見（み）て、この人（ひと）は関西（かんさい）出身（しゅっしん）に相違（そうい）ない。
> （從他的說話方式來看，這個人肯定是關西出身。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 東京（とうきょう）出身（しゅっしん） | この人は**東京出身に相違ない**。 | 這個人肯定是東京出身。 |
| 経験者（けいけんしゃ） | この人は**経験者に相違ない**。 | 這個人肯定是有經驗的人。 |
| プロの選手（せんしゅ） | この人は**プロの選手に相違ない**。 | 這個人肯定是職業選手。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 動詞普通形 | 証拠から見て、彼が**嘘をついたに相違ない**。 | 從證據來看，他肯定說了謊。 |
| → 過去形 | 足跡から見て、犯人はここを**通ったに相違ない**。 | 從腳印來看，犯人肯定經過了這裡。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| この人はどこの出身だと思いますか。 | 話し方から見て、**関西出身に相違ありません**。 | 從說話方式來看，肯定是關西出身。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 関西出身に相違ない。 | 肯定是關西出身。 |
| +根拠 | 話し方から見て、**関西出身に相違ない**。 | 從說話方式來看，肯定是關西出身。 |
| +補述 | 彼の話し方や態度から見て、この人は間違いなく**関西出身に相違ない**。 | 從他的說話方式和態度來看，這個人無疑肯定是關西出身。 |
