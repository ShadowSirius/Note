# 〜せいで

## 規則與接續
- `用言普通形 + せいで`
- `な形容詞語幹 / 名詞 + の / な + せいで`

## 用法說明
表示「都怪...、因為...的緣故」，用來說明負面結果的原因，帶有埋怨語氣。與表正面結果的 [[JP_N2_10_okagede|〜おかげで @related_to]] 相對。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E"><ruby>渋滞<rt>じゅうたい</rt></ruby>のせいで、</span><span style="color:#3498DB"><ruby>約束<rt>やくそく</rt></ruby>の<ruby>時間<rt>じかん</rt></ruby>に<ruby>遅<rt>おく</rt></ruby>れた。</span>
  中文：都怪塞車，赴約遲到了。
- 日文：<span style="color:#2ECC71"><ruby>彼<rt>かれ</rt></ruby>が</span><span style="color:#FF477E"><ruby>怠<rt>なま</rt></ruby>けたせいで、</span><span style="color:#3498DB">プロジェクトが<ruby>遅<rt>おく</rt></ruby>れた。</span>
  中文：都怪他偷懶，專案延遲了。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 渋滞（じゅうたい）のせいで、約束（やくそく）の時間（じかん）に遅（おく）れた。
> （都怪塞車，赴約遲到了。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 台風（たいふう） | **台風のせいで**、旅行が中止になった。 | 都怪颱風，旅行取消了。 |
| 寝坊した（ねぼうした） | **寝坊したせいで**、電車に乗り遅れた。 | 都怪睡過頭，錯過電車了。 |
| 不便だ | このアプリは**不便なせいで**、あまり使われない。 | 都怪這個App不方便，很少人用。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 反義おかげで | 早起きした**おかげで**、間に合った。 | 多虧早起，趕上了。（對比） |
| → 過去形 | 渋滞の**せいで**、遅れて**しまった**。 | 都怪塞車，結果遲到了。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうして遅れたんですか。 | 渋滞のせいで、遅れてしまいました。 | 都怪塞車，才遲到的。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | せいで、遅れた。 | 都怪如此，遲到了。 |
| +原因 | **渋滞の**せいで、遅れた。 | 都怪塞車，遲到了。 |
| +結果詳述 | 渋滞のせいで、**約束の時間に**遅れた。 | 都怪塞車，赴約時間遲到了。 |
