# 〜なしに（は）

## 規則與接續
- `名詞 + なしに（は）`

## 用法說明
表示「沒有...就（不能）...」，直接接在名詞後，指缺少前項這個事物或條件，後項就無法成立或不應該進行。語感比「ことなしに」（接動詞辞書形＋こと）更簡潔、口語化，比書面語「なくして（は）」語氣稍輕。三者為同義家族的變體：「なしに」直接接名詞、「ことなしに」接動詞、「なくして（は）」語氣最書面正式。

  中文：沒有許可，不能進入這個房間。
- 日文：<span style="color:#FF9F1C"><ruby>努力<rt>どりょく</rt></ruby>なしには、</span><span style="color:#3498DB">いい<ruby>結果<rt>けっか</rt></ruby>は<ruby>得<rt>え</rt></ruby>られない。</span>
  中文：沒有努力，就得不到好結果。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 許可（きょか）なしに、この部屋（へや）に入（はい）ってはいけない。
> （沒有許可，不能進入這個房間。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 予約（よやく）・入店（にゅうてん）できない | **予約なしに**、入店できない。 | 沒有預約就不能進店。 |
| 連絡（れんらく）・訪問（ほうもん）しないでください | **連絡なしに**、訪問しないでください。 | 沒有聯絡就請不要來訪。 |
| 説明（せつめい）・判断（はんだん）できない | **説明なしに**、判断できない。 | 沒有說明就無法判斷。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → なしには（強調） | 努力**なしには**、いい結果は得られない。 | 沒有努力，就得不到好結果。 |
| → ことなしに（近義） | 努力することなしに、いい結果は得られない。 | 不努力，就得不到好結果（語感稍書面）。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| この部屋に勝手（かって）に入ってもいいですか。 | いいえ、**許可なしに**入ってはいけません。 | 不行，沒有許可不能進入。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 入ってはいけない。 | 不能進入。 |
| +条件 | **許可なしに**、この部屋に入ってはいけない。 | 沒有許可，不能進入這個房間。 |
| +補述 | 担当者（たんとうしゃ）の**許可なしに**、この部屋に誰も入ってはいけないことになっている。 | 規定沒有負責人的許可，任何人都不能進入這個房間。 |
