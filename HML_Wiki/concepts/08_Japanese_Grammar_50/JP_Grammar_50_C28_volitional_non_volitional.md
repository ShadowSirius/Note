# C28 - 意志動詞與非意志動詞

## 分類
- 條件・意志・推量

## 句型與接續
- `意志動詞 / 無意志動詞`

## 核心用法
意志動詞可受命令、請求、意圖控制；非意志動詞描述自然發生或狀態變化。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#3498DB">ドアを開けます</span>。
  中文：打開門。
- 日文：<span style="color:#2ECC71">ドアが</span><span style="color:#3498DB">開きます</span>。
  中文：門開了。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> ドアを開（あ）けます。
> （打開門。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 窓（まど）（窗戶） | **窓（まど）**を開（あ）けます。 | 打開窗戶。 |
| 電気（でんき）（燈） | **電気（でんき）**をつけます。 | 開燈。 |
| テレビ（電視） | **テレビ**をつけます。 | 開電視。 |
| 鍵（かぎ）（鎖） | **鍵（かぎ）**をかけます。 | 鎖門。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 非意志自動詞形 | ドアが**開（あ）きます**。 | 門開了。 |
| → ている進行形 | ドアを**開（あ）けています**。 | 正開著門。 |
| → 過去形 | ドアを**開（あ）けました**。 | 開了門。 |
| → 命令形 | ドアを**開（あ）けなさい**。 | 把門打開。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| ドアを開（あ）けましたか。 | はい、開（あ）けました。 | 是的，開了。 |
| 誰がドアを開（あ）けましたか。 | 私（わたし）が開（あ）けました。 | 我開的。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 開（あ）けます。 | 打開。 |
| +目的 | **ドアを**開（あ）けます。 | 打開門。 |
| +修飾 | **部屋（へや）の**ドアを開（あ）けます。 | 打開房間的門。 |
| +副詞 | **早（はや）く**部屋（へや）のドアを開（あ）けます。 | 快點打開房間的門。 |
