# C35 - 〜てもらえませんか：委婉請求

## 分類
- て形・ない形・た形

## 句型與接續
- `Vて + もらえませんか`

## 核心用法
比 〜てください 更客氣的請求表達，詢問對方能否為我方做某事。

## 實用例句
- 日文：もう一度説明してもらえませんか。
  中文：可以請你再說明一次嗎？

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> もう一度（いちど）説明（せつめい）してもらえませんか。
> （可以請你再說明一次嗎？）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| ゆっくり話（はな）す（慢點說） | **ゆっくり話（はな）してもらえませんか**。 | 可以請你說慢一點嗎？ |
| 英語（えいご）で言（い）う（用英文說） | **英語（えいご）で言（い）ってもらえませんか**。 | 可以請你用英文說嗎？ |
| 漢字（かんじ）を書（か）く（寫漢字） | **漢字（かんじ）を書（か）ってもらえませんか**。 | 可以請你寫漢字嗎？ |
| ここで待（ま）つ（在這裡等） | **ここで待（ま）ってもらえませんか**。 | 可以請你在這裡等一下嗎？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 丁寧形（てくださいませんか） | もう一度（いちど）説明（せつめい）**してくださいませんか**。 | 請你再說明一次好嗎？ |
| → 普通體（口語） | もう一度（いちど）説明（せつめい）**してくれない？** | 可以再說明一次嗎？ |
| → 否定過去形 | 説明（せつめい）**してもらえませんでした**。 | 沒有得到對方的說明。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| もう一度（いちど）説明（せつめい）してもらえませんか。 | はい、いいですよ。 | 好的，沒問題喔。 |
| ゆっくり話（はな）してもらえませんか。 | はい、わかりました。 | 好的，我知道了。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | もらえませんか。 | 可以請你...嗎？ |
| +動詞修飾 | **説明（せつめい）して**もらえませんか。 | 可以請你說明嗎？ |
| +修飾 | **もう一度（いちど）**説明（せつめい）してもらえませんか。 | 可以請你再說明一次嗎？ |
| +副詞 | **すみませんが**、もう一度（いちど）説明（せつめい）してもらえませんか。 | 不好意思，可以請你再說明一次嗎？ |
