# C38 - 被動形：直接被動

## 分類
- 被動形・使役形・使役被動

## 句型與接續
- `AがBに + V受身形`

## 核心用法
表示「A 被 B 做某事」。動詞的受体當主語 A，施体用 助詞 に 標示。

## 實用例句
- 日文：私は先生にほめられました。
  中文：我被老師稱讚了。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 私（わたし）は先生（せんせい）にほめられました。
> （我被老師稱讚了。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 叱（しか）る（責備） | 私（わたし）は先生（せんせい）に**叱（しか）られました**。 | 我被老師責備了。 |
| 頼（たの）む（拜託） | 私（わたし）は先生（せんせい）に**頼（たの）まれました**。 | 我被老師拜託了。 |
| 呼（よ）ぶ（叫） | 私（わたし）は先生（せんせい）に**呼（よ）ばれました**。 | 我被老師叫了。 |
| 質問（しつもん）する（提問） | 私（わたし）は先生（せんせい）に**質問（しつもん）されました**。 | 我被老師提問了。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 主動形 | 先生（せんせい）は私（わたし）を**ほめました**。 | 老師稱讚了我。 |
| → 否定形 | 私（わたし）は先生（せんせい）にほめられ**ませんでした**。 | 我沒有被老師稱讚。 |
| → 疑問形 | あなたは先生（せんせい）にほめられました**か**。 | 你被老師稱讚了嗎？ |
