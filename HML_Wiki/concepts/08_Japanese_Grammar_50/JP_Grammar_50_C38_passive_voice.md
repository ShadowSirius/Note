# C38 - 受身形：被動

## 分類
- 被動・使役・敬語

## 句型與接續
- `AはBにV受身`

## 核心用法
表示主語受到他人動作影響；可中性描述，也可帶受害感。

## 實用例句
- 日文：私は先生にほめられました。
  中文：我被老師稱讚了。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> 私は先生にほめられました。
> 我被老師稱讚了（受害者/受益者的被動語態）。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 叱る | 私は先生に**叱られました**。 |
| 頼む | 私は先生に**頼まれました**。 |
| 招待する | 私は先生に**招待されました**。 |
| 質問する | 私は先生に**質問されました**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 主動形 (先生稱讚我) | 先生が私を**ほめました**。 |
| → 否定被動 | 私は先生にほめられ**ませんでした**。 |
| → 普通形口語 | 私は先生にほめられ**た**。 |
| → 現在被動 | 私は先生にほめられ**ます**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 誰にほめられましたか。 | 先生にほめられました。 |
| 先生に叱られましたか。 | いいえ、叱られませんでした。ほめられました。 |
| パーティーに招待されましたか。 | はい、友達に招待されました。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | ほめられました。 |
| +行為者 | **先生に**ほめられました。 |
| +主語 | **私は**先生にほめられました。 |
| +原因 | **テストで百点を取って**、私は先生にほめられました。 |
