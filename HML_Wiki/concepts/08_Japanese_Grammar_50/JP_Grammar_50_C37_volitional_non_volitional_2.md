# C37 - 意志・非意志的句型限制

## 分類
- 條件・意志・推量

## 句型與接續
- `意志動詞 + 意向/命令/依賴`

## 核心用法
許多表示意圖、命令、請求的句型只能接意志動詞；非意志動詞需改成狀態描述。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E">雨を降らせるとは言わず、</span><span style="color:#2ECC71">雨が</span><span style="color:#3498DB">降ると言います</span>。
  中文：不說讓雨下，而說雨下了。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 日本語（にほんご）を勉強（べんきょう）しようと思（おも）います。
> （我想學習日文。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 英語（えいご）（英文） | **英語（えいご）**を勉強（べんきょう）しようと思（おも）います。 | 我想學習英文。 |
| 本（ほん）を読（よ）む（看書） | **本（ほん）を読（よ）もう**と思（おモ）います。 | 我想看書。 |
| 映画（えいが）を見（み）る（看電影） | **映画（えいが）を見よう**と思（おも）います。 | 我想看電影。 |
| 手紙（てがみ）を書（か）く（寫信） | **手紙（てがみ）を書（か）こう**と思（おも）います。 | 我想寫信。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 否定打算（つもり） | 日本語（にほんご）を**勉強（べんきょう）しないつもりです**。 | 我不打算學習日文。 |
| → 意向形（普通體） | 日本語（にほんご）を**勉強（べんきょう）しよう**。 | 一起學習日文吧。 |
| → 疑問形 | 日本語（にほんご）を勉強（べんきょう）しようと**思いますか**。 | 你想學習日文嗎？ |
| → 過去形 | 日本語（にほんご）を勉強（べんきょう）しようと**思いました**。 | 那時我想學習日文。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 日本語（にほんご）を勉強（べんきょう）しようと思いますか。 | はい、勉強（べんきょう）しようと思います。 | 是的，我想學習。 |
| 何（なに）を勉強（べんきょう）しようと思いますか。 | 日本語（にほんご）を勉強（べんきょう）しようと思います。 | 我想學習日文。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 思（おも）います。 | 想。 |
| +動詞修飾 | **勉強（べんきょう）しよう**と思います。 | 想學習。 |
| +目的 | **日本語（にほんご）を**勉強（べんきょう）しようと思います。 | 我想學習日文。 |
| +頻率 | **毎日（まいにち）**日本語（にほんご）を勉強（べんきょう）しようと思います。 | 我想每天學習日文。 |
