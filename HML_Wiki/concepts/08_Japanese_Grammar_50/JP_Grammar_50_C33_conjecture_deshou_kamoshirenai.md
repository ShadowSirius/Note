# C33 - 〜でしょう・〜かもしれない：推量

## 分類
- 條件・意志・推量

## 句型與接續
- `普通形 + でしょう / かもしれません`

## 核心用法
でしょう 表示較有把握的推測；かもしれない 表示可能性較低或保留。

## 實用例句
- 日文：明日は雨でしょう。
  中文：明天大概會下雨吧。
- 日文：彼は来ないかもしれません。
  中文：他也許不會來。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 明日（あした）は雨（あめ）でしょう。
> （明天大概會下雨吧。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 晴（は）れ（晴天） | 明日（あした）は**晴（は）れ**でしょう。 | 明天大概是晴天吧。 |
| 曇（くも）り（陰天） | 明日（あした）は**曇（くも）り**でしょう。 | 明天大概是陰天吧。 |
| 雪（ゆき）（下雪） | 明日（あした）は**雪（ゆき）**でしょう。 | 明天大概會下雪吧。 |
| 寒（さむ）い（冷） | 明日（あした）は**寒（さむ）い**でしょう。 | 明天大概會很冷吧。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 可能性低推測（かもしれない） | 明日（あした）は雨（あめ）**かもしれません**。 | 明天也許會下雨。 |
| → 否定推測 | 明日（あした）は雨（あめ）**ではないでしょう**。 | 明天大概不會下雨吧。 |
| → 過去推測 | 昨日（きのう）は雨（あめ）**だったでしょう**。 | 昨天大概下雨了吧。 |
| → 丁寧非推量形 | 明日（あした）は雨（あめ）**です**。 | 明天會下雨。 |

|---|---|---|
