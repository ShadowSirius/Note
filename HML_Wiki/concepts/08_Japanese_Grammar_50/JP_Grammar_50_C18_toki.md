# C18 - 〜とき：時間點

## 分類
- て形・ない形・た形

## 句型與接續
- `名詞の/形容詞/動詞普通形 + とき`

## 核心用法
表示在某狀態或動作發生的時候；前後時態會影響先後關係。

## 實用例句
- 日文：日本へ行くとき、パスポートを持っていきます。
  中文：去日本時會帶護照。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>日本<rt>にほん</rt></ruby>へ<ruby>行<rt>い</rt></ruby>くとき、パスポートを<ruby>持<rt>も</rt></ruby>っていきます。
> 去日本時會帶護照。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 旅行する | **<ruby>旅行<rt>りょこう</rt></ruby>する**とき、パスポートを持っていきます。 |
| 飛行機に乗る | **<ruby>飛行機<rt>ひこうき</rt></ruby>に<ruby>乗<rt>の</rt></ruby>る**とき、パスポートを持っていきます。 |
| ホテルにチェックインする | **ホテルにチェックインする**とき、パスポートを持っていきます。 |
| 外国へ行く | **<ruby>外国<rt>がいこく</rt></ruby>へ<ruby>行<rt>い</rt></ruby>く**とき、パスポートを持っていきます。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 過去動作 (た形 + とき) | <ruby>日本<rt>にほん</rt></ruby>へ**<ruby>行<rt>い</rt></ruby>った**とき、パスポートを持っていきました。 |
| → 否定形 (ない形 + とき) | お金が**ない**とき、旅行しません。 |
| → 形容詞接續 (い形容詞 + とき) | **<ruby>若<rt>わか</rt></ruby>い**とき、よく勉強しました。 |
| → 名詞接續 (名詞 + のとき) | **<ruby>子供<rt>こども</rt></ruby>の**とき、よく遊びました。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| いつパスポートを持っていきますか。 | <ruby>日本<rt>にほん</rt></ruby>へ行くときに持っていきます。 |
| 体の調子が悪いとき、どうしますか。 | <ruby>薬<rt>くすり</rt></ruby>を飲んで寝ます。 |
| 道がわからないとき、誰に聞きますか。 | <ruby>警察<rt>けいさつ</rt></ruby>に聞きます。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>持<rt>も</rt></ruby>っていきます。 |
| +目的語 | **パスポートを**持っていきます。 |
| +時間條件 | **<ruby>日本<rt>にほん</rt></ruby>へ行くと**き、パスポートを持っていきます。 |
| +對象確認 | 日本へ行くとき、**必ず**パスポートを持っていきます。 |
