# C21 - 授受補助動詞：〜てあげる・〜てもらう・〜てくれる

## 分類
- て形・ない形・た形

## 句型與接續
- `Vて + あげる/もらう/くれる`

## 核心用法
描述誰為誰做動作，重點是恩惠方向與內外關係。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E">友達に</span><span style="color:#3498DB">手伝ってあげます</span>。
  中文：我幫朋友忙。
- 日文：<span style="color:#FF477E">先生に</span><span style="color:#3498DB">教えてもらいました</span>。
  中文：請老師教了我。
- 日文：<span style="color:#2ECC71">友達が</span><span style="color:#3498DB">手伝ってくれました</span>。
  中文：朋友幫了我。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 友達（ともだち）に手伝（てつだ）ってあげます。
> （我幫朋友忙。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 本を貸して | 友達（ともだち）に**本（ほん）を貸（か）して**あげます。 | 借書給朋友。 |
| 日本語を教えて | 友達（ともだち）に**日本語（にほんご）を教（おし）え特（て）**あげます。 | 教朋友日文。 |
| 荷物を持って | 友達（ともだち）に**荷物（にもつ）を持（も）って**あげます。 | 幫朋友拿行李。 |
| 道を案内して | 友達（ともだち）に**道（みち）を案内（あんない）して**あげます。 | 幫朋友指路。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → くれる形 | 友達（ともだち）が私（わたし）を**手伝（てつだ）ってくれます**。 | 朋友幫我的忙。 |
| → もらう形 | 友達（ともだち）に**手伝（てつだ）ってもらいます**。 | 我請朋友幫忙。 |
| → 過去形 | 友達（ともだち）に手伝（てつだ）って**あげました**。 | 我以前幫了朋友的忙。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 誰（だれ）が手伝（てつだ）ってくれましたか。 | 友達（ともだち）が手伝（てつだ）ってくれました。 | 朋友幫了我的忙。 |
| 友達（ともだち）に手伝（てつだ）ってもらいましたか。 | はい、手伝（てつだ）ってもらいました。 | 是的，請他幫了忙。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | あげます。 | 給予。 |
| +動作 | **手伝（てつだ）って**あげます。 | 幫忙。 |
| +對象 | **友達（ともだち）に**手伝（てつだ）ってあげます。 | 幫朋友忙。 |
| +時間 | **明日（あした）**友達（ともだち）に手伝（てつだ）ってあげます。 | 明天會幫朋友忙。 |
