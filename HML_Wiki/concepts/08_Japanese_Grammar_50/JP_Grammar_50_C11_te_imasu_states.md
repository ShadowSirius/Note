# C11 - 〜ています：進行與狀態

## 分類
- て形・ない形・た形

## 句型與接續
- `Vて + います`

## 核心用法
持續動詞表示正在做；瞬間動詞常表示結果狀態。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71">雨が</span><span style="color:#3498DB">降っています</span>。
  中文：正在下雨。
- 日文：<span style="color:#2ECC71">ドアが</span><span style="color:#3498DB">開いています</span>。
  中文：門開著。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 雨（あめ）が降（ふ）っています。
> （正在下雨。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 雪（ゆき） | **雪（ゆき）**が降（ふ）っています。 | 正在下雪。 |
| 風（かぜ） | **風（かぜ）**が吹（ふ）いています。 | 正在刮風。 |
| 桜（さくら） | **桜（さくら）**が散（ち）っています。 | 櫻花正在飄落。 |
| 星（ほし） | **星（ほし）**が光（ひか）っています。 | 星星正閃爍著。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 雨（あめ）が**降（ふ）っていました**。 | 剛才在下雨。 |
| → 否定形 | 雨（あめ）は**降（ふ）っていません**。 | 沒在下雨。 |
| → 疑問形 | 雨（あめ）が降（ふ）っています**か**。 | 在下雨嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 外（そと）は雨（あめ）が降（ふ）っていますか。 | はい、雨（あめ）が降（ふ）っています。 | 是的，正在下雨。 |
| 傘（かさ）を持（も）っていますか。 | いいえ、持（も）っていません。 | 不，我沒帶傘。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 降（ふ）っています。 | 在下。 |
| +主語 | **雨（あめ）が**降（ふ）っています。 | 正在下雨。 |
| +副詞 | **たくさん**雨（あめ）が降（ふ）っています。 | 雨下得很大。 |
| +時間 | **朝（あさ）から**たくさん雨（あめ）が降（ふ）っています。 | 從早上開始雨就下得很大。 |
