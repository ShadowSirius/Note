# C19 - 名詞修飾節

## 分類
- て形・ない形・た形

## 句型與接續
- `普通形 + 名詞`

## 核心用法
用句子直接修飾名詞，修飾節內不用 の 連接動詞句。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#F39C12">昨日買った</span><span style="color:#3498DB">本を読みます</span>。
  中文：讀昨天買的書。
- 日文：<span style="color:#F39C12">駅にいる</span><span style="color:#2ECC71">人は</span><span style="color:#3498DB">先生です</span>。
  中文：在車站的人是老師。

## 學習提示
- 先判算此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 昨日（きのう）買（か）った本（ほん）を読（よ）みます。
> （讀昨天買的書。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 雑誌 | 昨日（きのう）買（か）った**雑誌（ざっし）**を読（よ）みます。 | 讀昨天買的雜誌。 |
| 新聞 | 昨日（きのう）買（か）った**新聞（しんぶん）**を読（よ）みます。 | 讀昨天買的報紙。 |
| 手紙 | 昨日（きのう）買（か）った**手紙（てがみ）**を読（よ）みます。 | 讀昨天收到的信（或買的信紙/卡片）。 |
| 小説 | 昨日（きのう）買（か）った**小説（しょうせつ）**を読（よ）みます。 | 讀昨天買的小說。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 昨日（きのう）買（か）った本（ほん）を**読（よ）みました**。 | 讀了昨天買的書。 |
| → 進行形 | 昨日（きのう）買（か）った本（ほん）を**読（よ）んでいます**。 | 正在讀昨天買的書。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| その本（ほん）はいつ買いましたか。 | 昨日（きのう）買いました。 | 昨天買的。 |
| 昨日（きのう）買（か）った本（ほん）は面白（おもしろ）いですか。 | はい、とても面白（おもしろ）いです。 | 是的，非常有趣。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 読（よ）みます。 | 讀。 |
| +對象 | **本（ほん）を**読（よ）みます。 | 讀書。 |
| +修飾 | **買（か）った**本（ほん）を読（よ）みます。 | 讀買來的書。 |
| +時間 | **昨日（きのう）**買（か）った本（ほん）を読（よ）みます。 | 讀昨天買的書。 |
