# C25 - 可能動詞與可能句

## 分類
- 條件・意志・推量

## 句型與接續
- `V可能形 + ます`

## 核心用法
從能力、條件、許可三個角度說明「能不能做」。注意可能句的主體常用 が。

## 實用例句
- 日文：漢字が読めます。
  中文：看得懂漢字。
- 日文：ここで写真が撮れます。
  中文：這裡可以拍照。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>漢字<rt>かんじ</rt></ruby>が<ruby>読<rt>よ</rt></ruby>めます。
> 看得懂漢字（能力可能）。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 日本語を話す | <ruby>日本語<rt>にほんご</rt></ruby>が**<ruby>話<rt>はな</rt></ruby>せます**。 |
| 刺身を食べる | お<ruby>刺身<rt>さしみ</rt></ruby>が**<ruby>食<rt>た</rt></ruby>べられます**。 |
| 日本語の新聞を読む | 日本語の<ruby>新聞<rt>しんぶん</rt></ruby>が**<ruby>読<rt>よ</rt></ruby>めます**。 |
| 一人で来る | <ruby>一人<rt>ひとり</rt></ruby>で**<ruby>来<rt>こ</rt></ruby>られます**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定形 | <ruby>漢字<rt>かんじ</rt></ruby>が**<ruby>読<rt>よ</rt></ruby>めません**。 |
| → 過去肯定 | <ruby>漢字<rt>かんじ</rt></ruby>が**<ruby>読<rt>よ</rt></ruby>めました**。 |
| → 過去否定 | <ruby>漢字<rt>かんじ</rt></ruby>が**<ruby>読<rt>よ</rt></ruby>めませんでした**。 |
| → 狀況可能 (撮れます) | ここで<ruby>写真<rt>しゃしん</rt></ruby>が**<ruby>撮<rt>と</rt></ruby>れます**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 漢字が読めますか。 | はい、簡単な漢字なら読めます。 |
| 納豆が食べられますか。 | いいえ、どうしても食べられません。 |
| ここで自転車が借りられますか。 | はい、あそこで借りられます。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>読<rt>よ</rt></ruby>めます。 |
| +對象 | **<ruby>漢字<rt>かんじ</rt></ruby>が**読めます。 |
| +範圍修飾 | **日本の新聞の**漢字が読めます。 |
| +程度 | 日本の新聞の漢字が**少し**読めます。 |
