# C08 - 〜から：原因理由

## 分類
- 基礎句型與格助詞

## 句型與接續
- `普通形/丁寧形 + から`

## 核心用法
表示原因、理由，語氣直接，常用於解釋判斷、請求或結果。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E">雨が降っていますから、</span><span style="color:#3498DB">出かけません</span>。
  中文：因為正在下雨，所以不出門。
- 日文：<span style="color:#FF477E">危ないから、</span><span style="color:#3498DB">走らないでください</span>。
  中文：因為危險，請不要跑。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 雨（あめ）が降（ふ）っていますから、出（で）かけません。
> （因為正在下雨，所以不出門。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 風（かぜ）が強（つよ）い | **風（かぜ）が強（つよ）い**ですから、出（で）かけません。 | 因為風很大，所以不出門。 |
| 雪（ゆき）が降（ふ）っている | **雪（ゆき）が降（ふ）っている**ですから、出（で）かけません。 | 因為正在下雪，所以不出門。 |
| 体調（たいちょう）が良（よ）くない | **体調（たいちょう）が良（よ）くない**ですから、出（で）かけません。 | 因為身體狀況不好，所以不出門。 |
| 忙（いそが）しい | **忙（いそが）しい**ですから、出（で）かけません。 | 因為很忙，所以不出門。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 雨（あめ）が降（ふ）っていましたから、**出（で）かけませんでした**。 | 因為那時正在下雨，所以沒出門。 |
| → 疑問形 | 雨（あめ）が降（ふ）っていますから、出（で）かけません**か**。 | 因為正在下雨，所以不出門嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうして出（で）かけませんか。 | 雨（あめ）が降（ふ）っていますから。 | 因為正在下雨。 |
| 雨（あめ）が降（ふ）っていますから、タクシーで行（い）きますか。 | はい、そうしましょう。 | 是的，我們就坐計程車去吧。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 出（で）かけません。 | 不出門。 |
| +理由 | **雨（あめ）が降（ふ）っていますから、**出（で）かけません。 | 因為正在下雨，所以不出門。 |
| +伴隨 | 雨（あめ）が降（ふ）っていますから、**一人（ひとり）で**出（ない）かけません。 | 因為正在下雨，所以我不會一個人出門。 |
| +時間 | 雨（あめ）が降（ふ）っていますから、**今日（きょう）は**一人（ひとり）で出（で）かけません。 | 因為正在下雨，所以今天我不會一個人出門。 |
