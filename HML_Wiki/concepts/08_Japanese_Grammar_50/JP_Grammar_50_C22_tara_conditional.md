# C22 - 〜たら：完成後條件

## 分類
- 條件・意志・推量

## 句型與接續
- `Vた/い形かった/な形だった/名詞だった + ら`

## 核心用法
表示如果、當...之後；可用於一次性條件、假設與時間先後。

## 實用例句
- 日文：雨が降ったら、行きません。
  中文：如果下雨，就不去。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 雨（あめ）が降（ふ）ったら、行（い）きません。
> （如果下雨，就不去。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 風が強かったら | **風（かぜ）が強（つよ）かったら**、行（い）きません。 | 如果風很大，就不去。 |
| 雪が降ったら | **雪（ゆき）が降（ふ）ったら**、行（い）きません。 | 如果下雪，就不去。 |
| 忙しかったら | **忙（いそが）しかったら**、行（い）きません。 | 如果很忙，就不去。 |
| 寒かったら | **寒（さむ）かったら**、行（い）きません。 | 如果很冷，就不去。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 雨（あめ）が降（ふ）ったら、**行（い）きませんでした**。 | 當時如果下雨，就沒去。 |
| → 肯定形 | 晴（は）れたら、**行（い）きます**。 | 如果放晴，就去。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 雨（あめ）が降（ふ）ったら、どうしますか。 | 行（い）きません。 | 不去。 |
| 時間（じかん）があったら、行（い）きますか。 | はい、行（い）きます。 | 是的，會去。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 行（い）きません。 | 不去。 |
| +條件 | **降（ふ）ったら、**行（い）きません。 | 下（雨）的話，就不去。 |
| +主語 | **雨（あめ）が**降（ふ）ったら、行（い）きません。 | 如果下雨，就不去。 |
| +目的地 | 雨（あめ）が降（ふ）ったら、**京都（きょうと）へ**行（い）きません。 | 如果下雨，就不去京都。 |
