# C45 - 〜はずです・〜べきです：推論與義務

## 分類
- 條件・意志・推量

## 句型與接續
- `普通形 + はずです` / `V辞書形 + べきです`

## 核心用法
はずです 表示「根據根據理應...」的強烈推測；べきです 表示「道義上、常理上應該...」的義務。

## 實用例句
- 日文：彼は来るはずです。
  中文：他照理說應該會來。
- 日文：約束を守るべきです。
  中文：應該遵守約定。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）は来（く）るはずです。
> （他照理說應該會來。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 知（し）っている（知道） | 彼（かれ）は**知（し）っているはずです**。 | 他照理說應該知道。 |
| 帰（かえ）る（回去） | 彼（かれ）は**帰（かえ）るはずです**。 | 他照理說應該會回去。 |
| 買（か）う（買） | 彼（かれ）は**買（か）うはずです**。 | 他照理說應該會買。 |
| 勉強（べんきょう）する（學習） | 彼（かれ）は**勉強（べんきょう）するはずです**。 | 他照理說應該會學習。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 否定推測（ないはず） | 彼（かれ）は**来（こ）ないはずです**。 | 他照理說應該不會來。 |
| → 過去推測 | 彼（かれ）は来（く）るはず**でした**。 | 他那時照理說應該會來的。 |
| → 義務表達（べき） | 彼（かれ）は**来（く）るべきです**。 | 他應該要來。 |
| → 疑問形 | 彼（かれ）は来（く）るはずです**か**。 | 他照理說應該會來嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 彼（かれ）は来（く）ますか。 | はい、来（く）るはずです。 | 是的，照理說應該會來。 |
| 今日（きょう）は来（こ）ないはずですか。 | いいえ、来（く）るはずです。 | 不，照理說應該會來。 |
