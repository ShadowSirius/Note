# C45 - 〜はずです・〜べきです

## 分類
- 條件・意志・推量

## 句型與接續
- `普通形 + はずです` / `V辞書形 + べきです`

## 核心用法
はずです 表示按資訊推定理應如此；べきです 表示責任、義務或道德上應該。

## 實用例句
- 日文：彼は来るはずです。
  中文：他照理應該會來。
- 日文：約束を守るべきです。
  中文：應該遵守約定。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> 彼は来るはずです。
> 他照理應該會來（根據客觀事實的推理）。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 忙しい | 彼は**<ruby>忙<rt>いそが</rt></ruby>しいはずです**。 |
| 休みだ | 今日は**<ruby>休み<rt>やす</rt></ruby>のはずです**。 |
| 知っている | 彼はそれを**<ruby>知<rt>し</rt></ruby>っているはずです**。 |
| 義務表現 (約束を守る / べき) | <ruby>約束<rt>やくそく</rt></ruby>を**<ruby>守<rt>まも</rt></ruby>るべきです**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定推斷 (不應該來) | 彼は**来ないはずです**。 |
| → 強烈否定推斷 (不可能來) | 彼は**来るはずがありません**。 |
| → 過去推斷 | 彼は**来たはずです**。 |
| → 普通形（口語） | 彼は来る**はずだ**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 田中さんは今日来ますか。 | はい、来るはずです。昨日確認しましたから。 |
| 約束を破ってもいいですか。 | いいえ、約束は守るべきです。 |
| この本は図書館にあるはずですか。 | はい、あるはずですが、確認してみましょう。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | はずです。 |
| +動詞推斷 | **来る**はずです。 |
| +主語 | **彼は**来るはずです。 |
| +時間 | 彼は**もうすぐ**来るはずです。 |
