# C13 - 〜て、〜て：動作連接

## 分類
- て形・ない形・た形

## 句型與接續
- `Vて、Vて、Vます`

## 核心用法
用て形串接動作或狀態，可表示時間順序、並列或方式。

## 實用例句
- 日文：朝ご飯を食べて、学校へ行きます。
  中文：吃早餐後去學校。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 朝（あさ）ご飯（はん）を食（た）べて、学校（がっこう）へ行（い）きます。
> （吃早餐後去學校。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 新聞を読んで | **新聞（しんぶん）を読（よ）んで**、学校（がっこう）へ行（い）きます。 | 讀報紙後去學校。 |
| 顔を洗って | **顔（かお）を洗（あら）って**、学校（がっこう）へ行（い）きます。 | 洗臉後去學校。 |
| 服を着て | **服（ふく）を着（き）て**、学校（がっこう）へ行（い）きます。 | 穿衣服後去學校。 |
| 歯を磨いて | **歯（は）を磨（みが）いて**、学校（がっこう）へ行（い）きます。 | 刷牙後去學校。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 朝（あさ）ご飯（はん）を食（た）べて、学校（がっこう）へ**行（い）きました**。 | 吃了早餐後去了學校。 |
| → 否定形 | 朝（あさ）ご飯（はん）を食（た）べて、学校（がっこう）へ**行（い）きません**。 | 吃早餐後不去學校。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 朝（あさ）ご飯（はん）を食（た）べて、どこへ行（い）きますか。 | 学校（がっこう）へ行（い）きます。 | 去學校。 |
| 学校（がっこう）へ行く前（まえ）に、朝（あさ）ご飯（はん）を食（た）べますか。 | はい、食（た）べてから行（い）きます。 | 是的，吃過後再去。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 学校（がっこう）へ行（い）きます。 | 去學校。 |
| +前動 | **食（た）べて、**学校（がっこう）へ行（い）きます。 | 吃完後去學校。 |
| +目的 | **朝（あさ）ご飯（はん）を**食べて、学校（がっこう）へ行（い）きます。 | 吃早餐後去學校。 |
| +伴隨 | 朝（あさ）ご飯（はん）を食べて、**友達（ともだち）と**学校（がっこう）へ行（い）きます。 | 吃早餐後和朋友去學校。 |
