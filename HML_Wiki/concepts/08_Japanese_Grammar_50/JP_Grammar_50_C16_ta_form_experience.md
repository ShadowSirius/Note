# C16 - 〜たことがあります：經驗

## 分類
- て形・ない形・た形

## 句型與接續
- `Vた + ことがあります`

## 核心用法
表示曾經有過某經驗，重點是人生經驗或過去經歷是否存在。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E">日本へ</span><span style="color:#2ECC71">行ったことが</span><span style="color:#3498DB">あります</span>。
  中文：曾經去過日本。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 日本（にほん）へ行（い）ったことがあります。
> （曾經去過日本。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 北海道 | **北海道（ほっかいどう）**へ行（い）ったことがあります。 | 曾經去過北海道。 |
| 京都 | **京都（きょうと）**へ行（い）ったことがあります。 | 曾經去過京都。 |
| 富士山 | **富士山（ふじさん）**へ行（い）ったことがあります。 | 曾經去過富士山。 |
| 沖縄 | **沖縄（おきなわ）**へ行（い）ったことがあります。 | 曾經去過沖繩。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 否定形 | 日本（にほん）へ行（い）ったことは**ありません**。 | 沒去過日本。 |
| → 疑問形 | 日本（にほん）へ行（い）ったことがあります**か**。 | 去過日本嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 日本（にほん）へ行（い）ったことがありますか。 | はい、二回（にかい）行ったことがあります。 | 是的，去過兩次。 |
| 富士山（ふじさん）に登（のぼ）ったことがありますか。 | いいえ、一度（いちど）もありません。 | 不，一次也沒有。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | あります。 | 有。 |
| +経験 | **行（い）ったことが**あります。 | 有去過的經歷。 |
| +目的地 | **日本（にほん）へ**行（い）ったことがあります。 | 曾經去過日本。 |
| +頻率 | **何度も（なんども）**日本（にほん）へ行（い）ったことがあります。 | 曾去過日本很多次。 |
