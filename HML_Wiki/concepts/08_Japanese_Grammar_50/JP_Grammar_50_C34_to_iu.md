# C34 - 〜という：引用與命名

## 分類
- 條件・意志・推量

## 句型與接續
- `句子 + という + 名詞`

## 核心用法
用來引用內容、說明名稱，或把一段話變成名詞修飾。

## 實用例句
- 日文：田中さんという人を知っていますか。
  中文：你認識一個叫田中的人嗎？

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>田中<rt>たなか</rt></ruby>さんという<ruby>人<rt>ひと</rt></ruby>を<ruby>知<rt>し</rt></ruby>っていますか。
> 你認識一個叫田中的人嗎？

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 木村 | **<ruby>木村<rt>きむら</rt></ruby>**さんという人をしっていますか。 |
| サクラ酒店 | **サクラホテル**というホテルを知っていますか。 |
| さくらんぼ (櫻桃) / 果物 | **さくらんぼ**という**果物**を知っていますか。 |
| 歌 | 日本の**「ふるさと」**という**歌**を知っていますか。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定回答 | 田中さんという人は**知りません**。 |
| → 過去稱呼 | 田中さんという人**でした**。 |
| → 丁寧形詢問 (何と) | **何という**人を知っていますか。 |
| → 普通形口語 (って) | 田中さん**って**人知ってる？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| <ruby>田中<rt>たなか</rt></ruby>さんという人を知っていますか。 | はい、知っています。私の友達です。 |
| 「たこ焼き」という料理を食べたことがありますか。 | はい、大阪で食べたことがあります。 |
| どんな犬を飼っていますか。 | 「ハチ」という名前の犬を飼っています。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>知<rt>し</rt></ruby>っていますか。 |
| +被修飾名詞 | **<ruby>人<rt>ひと</rt></ruby>を**知っていますか。 |
| +稱呼關係 | **田中さんという**人を知っていますか。 |
| +對象修飾 | **あなたの会社の**田中さんという人を知っていますか。 |
