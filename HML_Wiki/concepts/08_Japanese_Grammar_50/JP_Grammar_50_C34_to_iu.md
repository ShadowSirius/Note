# C34 - 〜という：引用與命名

## 分類
- 條件・意志・推量

## 句型與接續
- `句子 + という + 名詞`

## 核心用法
用來引用內容、說明名稱，或把一段話變成名詞修飾。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#F39C12">田中さんという</span><span style="color:#3498DB">人を知っていますか</span>。
  中文：你認識一個叫田中的人嗎？

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 田中（たなか）さんという人（ひと）を知（し）っていますか。
> （你認識一個叫田中的人嗎？）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 佐藤（さとう）さん（佐藤先生） | **佐藤（さとう）さん**という人（ひと）を知（し）っていますか。 | 你認識一個叫佐藤的人嗎？ |
| 鈴木（すずき）さん（鈴木先生） | **鈴木（すずき）さん**という人（ひと）を知（し）っていますか。 | 你認識一個叫鈴木的人嗎？ |
| 木村（きむら）さん（木村先生） | **木村（きむら）さん**という人（ひと）を知（し）っていますか。 | 你認識一個叫木村的人嗎？ |
| 山田（やまだ）さん（山田先生） | **山田（やまだ）さん**という人（ひと）を知（し）っていますか。 | 你認識一個叫山田的人嗎？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 丁寧非引用形 | 田中（たなか）さんを**知（し）っていますか**。 | 你認識田中先生嗎？ |
| → 過去否定形 | 田中（たなか）さんという人（ひと）を**知（し）りませんでした**。 | 那時不認識一個叫田中的人。 |
| → 普通體（口語） | 田中（たなか）さんっていう人（ひと）を**知（し）ってる？** | 你認識一個叫田中的人嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 田中（たなか）さんという人（ひと）を知（し）っていますか。 | はい、知（し）っています。 | 是的，我認識。 |
| 鈴木（すずき）さんという人（ひと）を知（し）っていますか。 | いいえ、知（し）りません。 | 不，不認識。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 知（し）っていますか。 | 知道嗎/認識嗎？ |
| +目標 | **人（ひと）を**知（し）っていますか。 | 認識人嗎？ |
| +名稱引用 | **田中（たなか）さんという**人（ひと）を知（し）っていますか。 | 你認識一個叫田中的人嗎？ |
| +修飾 | **あの**田中（たなか）さんという人（ひと）を知（し）っていますか。 | 你認識那個叫田中的人嗎？ |
