# C36 - 〜ば：假定形

## 分類
- 條件・意志・推量

## 句型與接續
- `Vば / Aければ / Naなら / Nなら`

## 核心用法
表示假定條件，即「如果前項成立，後項就會成立」。後項一般不能接意志、命令、請求（除非前項是狀態或非意志）。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E">時間があれば、</span><span style="color:#3498DB">行きます</span>。
  中文：如果有時間就去。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 時間（じかん）があれば、行（い）きます。
> （如果有時間就去。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| お金（かね）がある（有錢） | **お金（かね）があれば**、行（い）きます。 | 如果有錢就去。 |
| 用事（ようじ）がない（沒事） | **用事（ようじ）がなければ**、行（い）きます。 | 如果沒事就去。 |
| 天気（てんき）がいい（天氣好） | **天気（てんき）がよければ**、行（い）きます。 | 如果天氣好就去。 |
| 誘（さそ）いがある（有邀約） | **誘（さそ）いがあれば**、行（い）きます。 | 如果有邀約就去。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 否定條件形（なければ） | 時間（じかん）が**なければ**、行（い）**きません**。 | 如果沒有時間就不去。 |
| → 〜たら形 | 時間（じかん）が**あったら**、行（い）きます。 | 如果有時間就去。 |
| → 過去事實形 | 時間（じかん）があれば、行（い）**きました**。 | 那時如果有時間就去了。 |
| → 疑問形 | 時間（じかん）があれば、行（い）きます**か**。 | 如果有時間去嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 時間（じかん）があれば、行（い）きますか。 | はい、時間（じかん）があれば、行（い）きます。 | 是的，如果有時間就去。 |
| 時間（じかん）がなければ、行（い）きませんか。 | はい、行（い）きません。 | 是的，不去。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 行（い）きます。 | 去。 |
| +條件 | **あれば**、行（い）きます。 | 如果有的話就去。 |
| +主語 | **時間（じかん）が**あれば、行（い）きます。 | 如果有時間就去。 |
| +時間 | **明日（あした）**時間（じかん）があれば、行（い）きます。 | 明天如果有時間就去。 |
