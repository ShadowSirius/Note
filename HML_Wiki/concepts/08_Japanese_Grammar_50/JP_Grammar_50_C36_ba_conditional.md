# C36 - 〜ば：條件形

## 分類
- 條件・意志・推量

## 句型與接續
- `Vば / い形ければ / な形なら / 名詞なら`

## 核心用法
表示條件成立時的結果，常用於一般條件、建議與必要條件。

## 實用例句
- 日文：時間があれば、行きます。
  中文：如果有時間就去。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>時間<rt>じかん</rt></ruby>があれば、<ruby>行<rt>い</rt></ruby>きます。
> 如果有時間就去（假定條件ば形）。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| お金がある | **お<ruby>金<rt>かね</rt></ruby>があれば**、行きます。 |
| 安い | **<ruby>安<rt>やす</rt></ruby>ければ**、買います。 |
| 天気がいい | **<ruby>天気<rt>てんき</rt></ruby>がよければ**、行きます。 |
| 暇だ (な形容詞) | **<ruby>暇<rt>ひま</rt></ruby>なら**、行きます。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定條件 (なければ) | <ruby>時間<rt>じかん</rt></ruby>が**なければ**、行きません。 |
| → 過去事實結果 (たら) | <ruby>時間<rt>じかん</rt></ruby>が**あったら**、行きました。 |
| → 讓步條件 (ても) | <ruby>時間<rt>じかん</rt></ruby>が**あっても**、行きません。 |
| → 普通形（口語） | <ruby>時間<rt>じかん</rt></ruby>があれば、**<ruby>行<rt>い</rt></ruby>く**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 時間があれば、行きますか。 | はい、時間があれば、喜んで行きます。 |
| どうすれば日本語が上手になりますか。 | 毎日練習すれば、上手になりますよ。 |
| 安ければ買いますか。 | はい、安ければ買いたいです。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>行<rt>い</rt></ruby>きます。 |
| +條件句 | **<ruby>時間<rt>じかん</rt></ruby>があれば**、行きます。 |
| +目的地 | 時間があれば、**日本へ**行きます。 |
| +伴隨 | 時間があれば、**家族と**日本へ行きます。 |
