# C26 - 〜ながら：同時進行

## 分類
- 條件・意志・推量

## 句型與接續
- `Vます語幹 + ながら`

## 核心用法
表示同一主語同時做兩個動作，後項通常是主要動作。

## 實用例句
- 日文：音楽を聞きながら、勉強します。
  中文：一邊聽音樂一邊念書。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>音楽<rt>おんがく</rt></ruby>を<ruby>聞<rt>き</rt></ruby>きながら、<ruby>勉強<rt>べんきょう</rt></ruby>します。
> 一邊聽音樂一邊念書（同時進行二動作，後者為主）。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| テレビを見る / ご飯を食べる | **テレビを<ruby>見<rt>み</rt></ruby>ながら**、ご飯を<ruby>食<rt>た</rt></ruby>べます。 |
| 歌を歌う / お風呂に入る | **<ruby>歌<rt>うた</rt></ruby>を<ruby>歌<rt>うた</rt></ruby>いながら**、お風呂に<ruby>入<rt>はい</rt></ruby>ります。 |
| コーヒーを飲む / 喋る | **コーヒーを<ruby>飲<rt>の</rt></ruby>みながら**、話します。 |
| 歩く / 考える | **<ruby>歩<rt>ある</rt></ruby>きながら**、考えます。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 過去形 | <ruby>音楽<rt>おんがく</rt></ruby>を<ruby>聞<rt>き</rt></ruby>きながら、勉強**しました**。 |
| → 否定形 | <ruby>音楽<rt>おんがく</rt></ruby>を<ruby>聞<rt>き</rt></ruby>きながら、勉強**しません**。 |
| → 進行形 | <ruby>音楽<rt>おんがく</rt></ruby>を<ruby>聞<rt>き</rt></ruby>きながら、勉強**しています**。 |
| → 普通形（口語） | <ruby>音楽<rt>おんがく</rt></ruby>を<ruby>聞<rt>き</rt></ruby>きながら、勉強**する**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 勉強するとき、いつも音楽を聞きますか。 | はい、いつも音楽を聞きながら勉強します。 |
| 歩きながらスマホを使ってもいいですか。 | いいえ、危ないですから、歩きながら使ってはいけません。 |
| お茶を飲みながら少し話しませんか。 | はい、いいですね。話しましょう。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>勉強<rt>べんきょう</rt></ruby>します。 |
| +附隨動作 | **<ruby>音楽<rt>おんがく</rt></ruby>を<ruby>聞<rt>き</rt></ruby>きながら**、勉強します。 |
