# C24 - 〜んです：說明與背景

## 分類
- 條件・意志・推量

## 句型與接續
- `普通形 + んです`

## 核心用法
用來補充背景、解釋理由、確認情況；比單純陳述更帶說明感。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#3498DB">どうしたんですか</span>。
  中文：你怎麼了？
- 日文：<span style="color:#2ECC71">頭が</span><span style="color:#3498DB">痛いんです</span>。
  中文：我頭痛。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 頭（あたま）が痛（いた）いんです。
> （我頭痛。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| お腹が痛い | **お腹（なか）が痛（いた）い**んです。 | 我肚子痛。 |
| 熱がある | **熱（ねつ）がある**んです。 | 我發燒。 |
| 風邪をひいた | **風邪（かぜ）をひいた**んです。 | 我感冒了。 |
| 気分が悪い | **気分（きぶん）が悪い（わるい）**んです。 | 我身體不舒服。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 疑問形 | 頭（あたま）が痛（いた）いんです**か**。 | 你是頭痛嗎？ |
| → 過去形 | 頭（あたま）が**痛（いた）かった**んです。 | 我剛才頭痛。 |
| → 否定形 | 頭（あたま）が**痛（いた）くない**んです。 | 我頭不痛。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうしたんですか。 | 少し頭（あたま）が痛（いた）いんです。 | 我有點頭痛。 |
| 病院（びょういん）へ行（い）くんですか。 | はい、これから行（い）くんです。 | 是的，待會要去。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 痛（いた）いんです。 | 是痛的。 |
| +主語 | **頭（あたま）が**痛（いた）いんです。 | 我頭痛。 |
| +程度 | 頭（あたま）が**とても**痛（いた）いんです。 | 我頭非常痛。 |
| +時間 | **朝（あさ）から**頭（あたま）がとても痛（いた）いんです。 | 我從早上開始頭就很痛。 |
