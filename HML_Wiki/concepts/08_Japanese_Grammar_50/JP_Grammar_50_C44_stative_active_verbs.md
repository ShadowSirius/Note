# C44 - 狀態動詞與動態動詞

## 分類
- て形・ない形・た形

## 句型與接續
- `Vて + いる`

## 核心用法
區分動詞屬性：動作動詞（如 書く）加 ている 表示進行，狀態動詞（如 知る、結婚する）加 ている 表示狀態的持續。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E">今、</span><span style="color:#3498DB">論文を書いています</span>。
  中文：現在正在寫論文。
- 日文：<span style="color:#3498DB">結婚しています</span>。
  中文：已婚。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 結婚（けっこん）しています。
> （已婚。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 大阪（おおさか）に住（す）む（住在大阪） | **大阪（おおさか）に住（す）んでいます**。 | 住在大阪。 |
| 車（くるま）を持（も）つ（有車） | **車（くるま）を持（も）っています**。 | 有車。 |
| 田中（たなか）さんを知（し）っている（認識田中） | **田中（たなか）さんを知（し）っています**。 | 認識田中先生。 |
| 眼鏡（めがね）をかける（戴著眼鏡） | **眼鏡（めがね）をかけています**。 | 戴著眼鏡。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 否定形 | 結婚（けっこん）して**いません**。 | 沒結婚/未婚。 |
| → 過去形 | 結婚（けっこん）して**いました**。 | 以前結過婚/那時結著婚。 |
| → 過去否定形 | 結婚（けっこん）して**いませんでした**。 | 那時沒結婚。 |
| → 疑問形 | 結婚（けっこん）しています**か**。 | 結婚了嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 結婚（けっこん）していますか。 | はい、結婚（けっこん）しています。 | 是的，已婚。 |
| 田中（たなか）さんを知（し）っていますか。 | はい、知（し）っています。 | 是的，我認識。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | います。 | 在/著。 |
| +動詞修飾 | **しています**。 | 做著/處於...狀態。 |
| +狀態 | **結婚（けっこん）**しています。 | 已婚。 |
| +主語 | **姉（あね）は**結婚（けっこん）しています。 | 姐姐已婚。 |
