# C43 - 〜そうです：樣態與看起來

## 分類
- 條件・意志・推量

## 句型與接續
- `Vます語幹 / A語幹 / Na語幹 + そうです`

## 核心用法
表示根據視覺或直覺進行推測，即「看起來好像...」、「快要...」。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71">雨が</span><span style="color:#3498DB">降りそうです</span>。
  中文：看起來快下雨了。
- 日文：<span style="color:#F39C12">この</span><span style="color:#2ECC71">ケーキは</span><span style="color:#3498DB">おいしそうです</span>。
  中文：這個蛋糕看起來很好吃。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 雨（あめ）が降（ふ）りそうです。
> （看起來快下雨了。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 雪（ゆき）（雪） | **雪（ゆき）**が降（ふ）りそうです。 | 看起來快下雪了。 |
| 風（かぜ）が吹（ふ）く（刮風） | **風（かぜ）が吹（ふ）きそうです**。 | 看起來快刮風了。 |
| ビルが倒（たお）れる（大樓倒塌） | **ビルが倒（たお）れそうです**。 | 大樓看起來快倒塌了。 |
| ボタンが取（と）れる（鈕扣掉落） | **ボタンが取（と）れそうです**。 | 鈕扣看起來快掉落了。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 否定推測 | 雨（あめ）は**降（ふ）りそうにありません**。 | 看起來不像會下雨。 |
| → 過去形 | 雨（あめ）が降（ふ）り**そうでした**。 | 那時看起來快下雨了。 |
| → 形容詞樣態形 | 美味（おい）しい**そうです**。 | 看起來很好吃。 |
| → 疑問形 | 雨（あめ）が降（ふ）りそうです**か**。 | 看起來快下雨了嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 雨（あめ）が降（ふ）りますか。 | 雨（あめ）が降（ふ）りそうです。 | 看起來快下雨了。 |
| 雨（あめ）が降（ふ）りそうですか。 | はい、今（いま）にも降（ふ）りそうです。 | 是的，看起來隨時都會下雨。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | そうです。 | 看起來是。 |
| +動詞修飾 | **降（ふ）り**そうです。 | 看起來快下了。 |
| +主語 | **雨（あめ）が**降（ふ）りそうです。 | 看起來快下雨了。 |
| +副詞 | **今（いま）にも**雨（あめ）が降（ふ）りそうです。 | 看起來隨時都會下雨。 |
