# C43 - 〜そうです：樣態推測

## 分類
- 條件・意志・推量

## 句型與接續
- `Vます語幹/い形語幹/な形語幹 + そうです`

## 核心用法
根據眼前樣子做直覺判斷，表示看起來快要或似乎如此。

## 實用例句
- 日文：雨が降りそうです。
  中文：看起來快下雨了。
- 日文：このケーキはおいしそうです。
  中文：這個蛋糕看起來很好吃。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>雨<rt>あめ</rt></ruby>が<ruby>降<rt>ふ</rt></ruby>りそうです。
> 看起來快下雨了（樣態/推測）。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 美味しい | このケーキは**おいしそうです**。 |
| 忙しい | 彼は**<ruby>忙<rt>いそが</rt></ruby>しそうです**。 |
| 落ちる | カバンから財布が**<ruby>落<rt>お</rt></ruby>ちそうです**。 |
| 寒そうだ | 今日は**<ruby>寒<rt>さむ</rt></ruby>そうです**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定樣態 | <ruby>雨<rt>あめ</rt></ruby>は**<ruby>降<rt>ふ</rt></ruby>りそうにありません**。 |
| → 過去樣態 | <ruby>雨<rt>あめ</rt></ruby>が**<ruby>降<rt>ふ</rt></ruby>りそうでした**。 |
| → 修飾名詞 (そうな名詞) | **おいしそうな**ケーキですね。 |
| → 傳聞 sou desu (原形) | <ruby>雨<rt>あめ</rt></ruby>が**<ruby>降<rt>ふ</rt></ruby>るそうです**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 空が暗いですね。雨が降りますか。 | ええ、雨が降りそうですね。 |
| このケーキはどうですか。 | とてもおいしそうですね。食べたいです。 |
| 木村さんは元気ですか。 | いいえ、忙しそうで、疲れているようです。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | そうです。 |
| +動詞連用形推測 | **<ruby>降<rt>ふ</rt></ruby>り**そうです。 |
| +主體 | **<ruby>雨<rt>あめ</rt></ruby>が**降りそうです。 |
| +時間 | **今にも**雨が降りそうです。 |
