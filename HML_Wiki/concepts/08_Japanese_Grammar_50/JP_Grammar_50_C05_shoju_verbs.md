# C05 - 授受動詞：あげる・もらう・くれる

## 分類
- 基礎句型與格助詞

## 句型與接續
- `AがBにNをあげる / AがBにNをもらう / Aが私にNをくれる`

## 核心用法
授受句型要同時看物品移動方向與心理內外：あげる向外，もらう向內取得，くれる是他人給我方。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E">友達に</span><span style="color:#3498DB">お土産をあげました</span>。
  中文：送伴手禮給朋友。
- 日文：<span style="color:#FF477E">友達に</span><span style="color:#3498DB">手紙をもらいました</span>。
  中文：從朋友那裡收到信。
- 日文：<span style="color:#2ECC71">友達が</span><span style="color:#3498DB">チョコをくれました</span>。
  中文：朋友給了我巧克力。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 友達（ともだち）にお土産（みやげ）をあげました。
> （送伴手禮給朋友。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 本（ほん） | 友達（ともだち）に**本（ほん）**をあげました。 | 送書給朋友。 |
| 花（はな） | 友達（ともだち）に**花（はな）**をあげました。 | 送花給朋友。 |
| 手紙（てがみ） | 友達（ともだち）に**手紙（てがみ）**をあげました。 | 給朋友寫信/寄信。 |
| 時計（とけい） | 友達（ともだち）に**時計（とけい）**をあげました。 | 送手錶給朋友。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → くれる形 | 友達（ともだち）が私（わたし）にお土産（みやげ）を**くれました**。 | 朋友送我伴手禮。 |
| → もらう形 | 友達（ともだち）にお土産（みやげ）を**もらいました**。 | 從朋友那裡收到伴手禮。 |
| → 否定形 | 友達（ともだち）にお土産（みやげ）を**あげませんでした**。 | 沒有送伴手禮給朋友。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 誰（だれ）にお土産（みやげ）をあげましたか。 | 友達（ともだち）にあげました。 | 送給朋友了。 |
| 友達（ともだち）は喜（よろこ）びましたか。 | はい、とても喜（よろこ）びました。 | 是的，他非常高興。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | あげました。 | 給了。 |
| +對象 | **友達（ともだち）に**あげました。 | 給了朋友。 |
| +目的 | 友達（ともだち）に**お土産（みやげ）を**あげました。 | 送伴手禮給朋友。 |
| +時間 | **昨日（きのう）**友達（ともだち）にお土産（みやげ）をあげました。 | 昨天送伴手禮給朋友。 |
