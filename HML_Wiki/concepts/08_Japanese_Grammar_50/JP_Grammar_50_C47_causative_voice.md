# C47 - 使役形：讓・使

## 分類
- 被動・使役・敬語

## 句型與接續
- `AはBに/をV使役`

## 核心用法
表示命令、指示、允許、放任，或使某人產生情緒。自動詞常用 を 標示被使役者，他動詞常用 に。

## 實用例句
- 日文：先生は学生に宿題をさせました。
  中文：老師讓學生做作業。
- 日文：父は子供を笑わせました。
  中文：爸爸逗小孩笑。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> 先生は学生に宿題をさせました。
> 老師讓學生做作業（使役語態：強迫/允許）。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 本を読ませる | 先生は学生に**本を読ませました**。 |
| 掃除をさせる | 先生は学生に**掃除をさせました**。 |
| 漢字を書かせる | 先生は学生に**漢字を書かせました**。 |
| 自動詞使役 (子供を笑わせる) | 父は子供を**笑わせました**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 使役被動 (被逼做) | 学生は先生に宿題を**させられました**。 |
| → 否定使役 | 先生は学生に宿題を**させませんでした**。 |
| → 允許請求 (〜させてください) | 宿題を**させてください**。 |
| → 普通形（口語） | 先生は学生に宿題を**させた**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 誰が学生に宿題をさせましたか。 | 先生がさせました。 |
| お父さんは子供に何をさせましたか。 | 部屋の掃除をさせました。 |
| ここで少し待たせてもらえませんか。 | はい、いいですよ。どうぞ。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | させました。 |
| +使役動作 | **宿題を**させました。 |
| +被使役者 | **学生に**宿題をさせました。 |
| +使役主體 | **先生は**学生に宿題をさせました。 |
