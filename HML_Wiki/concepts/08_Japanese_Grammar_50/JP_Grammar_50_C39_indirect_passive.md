# C39 - 間接受身：受害被動

## 分類
- 被動・使役・敬語

## 句型與接續
- `AはBにNをV受身`

## 核心用法
主語不是直接受詞，但受到別人的動作影響，常帶困擾或受害語感。

## 實用例句
- 日文：私は弟にケーキを食べられました。
  中文：我的蛋糕被弟弟吃掉了。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> 私は弟にケーキを食べられました。
> 我的蛋糕被弟弟吃掉了（間接被動/受害被動）。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 妹 / パソコンを壊す | 私は妹にパソコンを**壊されました**。 |
| 泥棒 / 財布を盗む | 私は泥棒に財布を**盗まれました**。 |
| 友達 / 秘密を言う | 私は友達に秘密を**言われました**。 |
| 子供 / 服を汚す | 私は子供に服を**汚されました**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 主動形 (弟弟吃蛋糕) | 弟が私のケーキを**食べました**。 |
| → 否定被動 | 私は弟にケーキを食べられ**ませんでした**。 |
| → 普通形口語 | 私は弟にケーキ食べられ**ちゃった**。 |
| → 自動詞受害被動 (雨が降る) | 私は雨に**降られました**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| どうしたんですか。悲しそうですね。 | 弟にケーキを食べられたんです。 |
| 財布はどこですか。 | 泥棒に盗まれました。 |
| どうして服が濡れているのですか。 | 帰る途中で雨に降られたんです。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | 食べられました。 |
| +受害物 | **ケーキを**食べられました。 |
| +行為者 | **弟に**ケーキを食べられました。 |
| +主格 | **私は**弟にケーキを食べられました。 |
