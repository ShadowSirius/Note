# C41 - 〜て：原因理由

## 分類
- 基礎句型與格助詞

## 句型與接續
- `Vて / Aくて / Naで`

## 核心用法
表示前項是引起後項（感情、驚訝、狀態變化、無意志）的原因，後項不能接意志、命令或請求。

## 實用例句
- 日文：ニュースを聞いて、驚きました。
  中文：聽到新聞後嚇了一跳。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> ニュースを聞（き）いて、驚（おどろ）きました。
> （聽到新聞後嚇了一跳。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 手紙（てがみ）を読（よ）む（讀信） | **手紙（てがみ）を読（よ）んで**、驚（おどろ）きました。 | 讀了信後嚇了一跳。 |
| 話（はなし）を聞（き）く（聽說） | **話（はなし）を聞（き）いて**、驚（おどろ）きました。 | 聽說了這件事後嚇了一跳。 |
| 写真（しゃしん）を見（み）る（看照片） | **写真（しゃしん）を見（み）て**、驚（おどろ）きました。 | 看了照片後嚇了一跳。 |
| 結果（けっか）を見（み）る（看結果） | **結果（けっか）を見（み）て**、驚（おどろ）きました。 | 看了結果後嚇了一跳。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 安心（て） | ニュースを聞（き）いて、**安心（あんしん）しました**。 | 聽到新聞後鬆了一口氣。 |
| → から理由形 | ニュースを聞（き）きました**から**、驚（おどろ）きました。 | 因為聽到了新聞，所以嚇了一跳。 |
| → 疑問形 | ニュースを聞（き）いて、驚きました**か**。 | 聽到新聞後嚇了一跳嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| ニュースを聞（き）いて、驚（おどろ）きましたか。 | はい、驚（おどろ）きました。 | 是的，嚇了一跳。 |
| どうして驚（おどろ）きましたか。 | ニュースを聞（き）いて、驚（おどろ）きました。 | 因為聽了新聞，所以嚇了一跳。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 驚（おどろ）きました。 | 嚇了一跳。 |
| +動詞修飾 | **聞（き）いて**、驚（おどろ）きました。 | 聽到後嚇了一跳。 |

| +時間 | **朝（あさ）**ニュースを聞（き）いて、驚（おどろ）きました。 | 早上聽到新聞後嚇了一跳。 |
