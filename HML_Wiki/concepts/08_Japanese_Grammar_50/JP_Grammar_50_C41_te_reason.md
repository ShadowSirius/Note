# C41 - 〜て：原因理由

## 分類
- 理由・目的

## 句型與接續
- `Vて / い形くて / な形で / 名詞で`

## 核心用法
用て形表示自然導致的原因，常用於感情、狀態或結果。

## 實用例句
- 日文：ニュースを聞いて、驚きました。
  中文：聽到新聞後嚇了一跳。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> ニュースを<ruby>聞<rt>き</rt></ruby>いて、<ruby>驚<rt>おどろ</rt></ruby>きました。
> 聽到新聞後嚇了一跳（原因由て形表示，後半多為感情、狀態等非意志動詞）。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 家族に会う / 嬉しい | 家族に**<ruby>会<rt>あ</rt></ruby>えて**、**<ruby>嬉<rt>うれ</rt></ruby>しいです**。 |
| テストに落ちる / 悲しい | テストに**<ruby>落<rt>お</rt></ruby>ちて**、**<ruby>悲<rt>かな</rt></ruby>しいです**。 |
| 宿題が多い / 大変だ | 宿題が**<ruby>多<rt>おお</rt></ruby>くて**、**<ruby>大変<rt>たいへん</rt></ruby>です**。 |
| 電車が遅れる / 遅刻する | 電車が**<ruby>遅<rt>おく</rt></ruby>れて**、**<ruby>遅刻<rt>ちこく</rt></ruby>しました**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定形原因 (なくて) | ニュースが**<ruby>聞<rt>き</rt></ruby>けなくて**、驚きました。 |
| → 比較 node (原因) | ニュースを**<ruby>聞<rt>き</rt></ruby>いたので**、驚きました。 |
| → 現在形感情 | ニュースを聞いて、**<ruby>驚<rt>おどろ</rt></ruby>きます**。 |
| → 普通形（口語） | ニュースを聞いて、**<ruby>驚<rt>おどろ</rt></ruby>いた**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| どうして驚いたのですか。 | そのニュースを聞いて、驚きました。 |
| なんで遅れたのですか。 | 電車が遅れて、遅刻してしまいました。 |
| 家族に会えてどうですか。 | 会えて、とても嬉しいです。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>驚<rt>おどろ</rt></ruby>きました。 |
| +感情原因 (て) | **<ruby>聞<rt>き</rt></ruby>いて**、驚きました。 |

| +時間 | **今朝の**ニュースを聞いて、驚きました。 |
