# C49 - 謙讓語

## 分類
- 被動・使役・敬語

## 句型與接續
- `お/ご + Vます語幹 + する` / `特殊謙讓語`

## 核心用法
描述我方動作，降低我方以向聽話者或動作對象表敬意。

## 實用例句
- 日文：資料をお送りします。
  中文：我寄資料給您。
- 日文：先生に伺います。
  中文：我去請教老師/拜訪老師。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> 資料をお送りします。
> 我寄資料給您（謙遜語：お+連用形+します/いたします）。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 手伝う | 荷物を**お手伝いします**。 |
| 案内する (サ變) | 駅まで**ご案内します**。 |
| 紹介する (サ變) | 友達を**ご紹介します**. |
| 特殊謙遜 (聞く / 伺う) | 先生に**伺います**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 更自謙 (〜いたします) | 資料を**お送りいたします**。 |
| → 尊敬語 (對方寄) | 先生が資料を**お送りになります**。 |
| → 特殊謙遜 (食べる・もらう / いただく) | お土産を**いただきました**。 |
| → 普通口語 | 資料**送るよ**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| その資料は誰が送りますか。 | 私がお送りします。 |
| 明日、何時に伺えばいいですか。 | 三時にいらしてください。 |
| 重そうな荷物ですね。お手伝いしましょうか。 | はい、ありがとうございます。お願いします。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | します。 |
| +謙遜動作 | **お送り**します。 |
| +目的語 | **資料を**お送りします。 |
| +時間目的 | **明日、メールで**資料をお送りします。 |
