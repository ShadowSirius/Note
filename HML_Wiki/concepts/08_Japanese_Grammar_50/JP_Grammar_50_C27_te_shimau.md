# C27 - 〜てしまう：完成與遺憾

## 分類
- て形・ない形・た形

## 句型與接續
- `Vて + しまう`

## 核心用法
可表示徹底完成，也可表示不小心造成的遺憾、後悔。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#3498DB">宿題を</span><span style="color:#FF477E">全部</span><span style="color:#3498DB">してしまいました</span>。
  中文：把作業全部做完了。
- 日文：<span style="color:#3498DB">財布を忘れてしまいました</span>。
  中文：不小心忘了錢包。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 財布（さいふ）を忘（わす）れてしまいました。
> （不小心忘了錢包。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 鍵（かぎ）をなくす（丟了鑰匙） | **鍵（かぎ）をなくしてしまいました**。 | 不小心丟了鑰匙。 |
| 電車（でんしゃ）を間違える（搭錯電車） | **電車（でんしゃ）を間違えてしまいました**。 | 不小心搭錯電車。 |
| パスポートを落とす（掉了護照） | **パスポートを落としてしまいました**。 | 不小心掉了護照。 |
| 傘（かさ）を忘（わす）れる（忘了帶傘） | **傘（かさ）を忘（わす）れてしまいました**。 | 不小心忘了帶傘。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 口語形（ちゃう） | 財布（さいふ）を**忘（わす）れちゃいました**。 | 不小心忘了錢包。 |
| → 丁寧形（無遺憾） | 財布（さいふ）を**忘（わす）れました**。 | 忘了錢包。 |
| → 疑問過去形 | 財布（さいふ）を忘（わす）れてしまいました**か**。 | 你不小心忘了錢包嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうしたんですか。 | 財布（さいふ）を忘（わす）れてしまいました。 | 我不小心忘了錢包。 |
| 鍵（かぎ）をなくしてしまいましたか。 | はい、なくしてしまいました。 | 是的，不小心丟了。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | しまいました。 | 糟了/完了。 |
| +動詞修飾 | **忘（わす）れました**。 | 不小心忘了。 |
| +目的 | **財布（さいふ）を**忘（わす）れてしまいました。 | 不小心忘了錢包。 |
| +地點 | **電車（でんしゃ）に**財布（さいふ）を忘（わす）れてしまいました。 | 不小心把錢包忘在電車上了。 |
