# C50 - 使役受身形：被迫

## 分類
- 被動・使役・敬語

## 句型與接續
- `AはBにV使役受身`

## 核心用法
表示被要求、被迫做不想做的事，通常帶困擾、不滿或受害感；也可表示某情感被引發。

## 實用例句
- 日文：母に買い物に行かされました。
  中文：被媽媽叫去買東西。
- 日文：その写真を見て考えさせられました。
  中文：看了那張照片後，不禁思考起來。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> 母に買い物に行かされました。
> 被媽媽叫去買東西（使役被動：無奈、被迫）。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 宿題をする | 先生に宿題を**させられました**。 |
| 歌を歌う | 友達に歌を**歌わされました**。 |
| ピーマンを食べる | 母親にピーマンを**食べさせられました**。 |
| 部屋を掃除する | 親に部屋を**掃除させられました**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 使役形 (主動使役) | 母が私を買い物に**行かせました**。 |
| → 被動形 (一般被動) | 母に**頼まれました**。 |
| → 思考感應 (感情/被動發) | その写真を見て**考えさせられました**。 |
| → 普通口語 (〜させられた) | 母に買い物に**行かされた**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| どうして買い物に行ったのですか。 | 母に行かされたんです。 |
| カラオケで歌を歌いましたか。 | はい、友達に無理矢理歌わされました。 |
| その映画はどうでしたか。 | 人生について考えさせられました。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | 行かされました。 |
| +目的 | **買い物に**行かされました。 |
| +行為者 | **母に**買い物に行かされました。 |
| +時間 | **昨日の夕方**、母に買い物に行かされました。 |
