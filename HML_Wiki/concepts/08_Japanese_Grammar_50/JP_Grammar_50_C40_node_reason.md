# C40 - 〜ので：客觀原因理由

## 分類
- 基礎句型與格助詞

## 句型與接續
- `普通形（Na/N + な） + ので`

## 核心用法
客觀、委婉地表示原因、理由，常顯得禮貌，常用於請假、說明客觀事實或請求許可。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E">雨が降っているので、</span><span style="color:#3498DB">出かけません</span>。
  中文：因為正在下雨，所以不出門。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 雨（あめ）が降（ふ）っているので、出（で）かけません。
> （因為正在下雨，所以不出門。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 風邪（かぜ）をひいている（感冒了） | **風邪（かぜ）をひいているので**、出（で）かけません。 | 因為感冒了，所以不出門。 |
| 用事（ようじ）がある（有事） | **用事（ようじ）があるので**、出（で）かけません。 | 因為有事，所以不出門。 |
| 宿題（しゅくだい）が多（おお）い（作業多） | **宿題（しゅくだい）が多（おお）いので**、出（で）かけません。 | 因為作業多，所以不出門。 |
| お金（かね）がない（沒有錢） | **お金（かね）がないので**、出（で）かけません。 | 因為沒有錢，所以不出門。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → から理由形 | 雨（あめ）が降（ふ）っています**から**、出（で）かけません。 | 因為正在下雨，所以不出門。 |
| → 過去形 | 雨（あめ）が降（ふ）っていた**ので**、出（で）かけ**ませんでした**。 | 因為那時在下雨，所以沒有出門。 |
| → 疑問形 | 雨（あめ）が降（ふ）っているので、出（で）かけません**か**。 | 因為在下雨，所以不出門嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうして出（で）かけませんか。 | 雨（あめ）が降（ふ）っているので、出（で）かけません。 | 因為正在下雨，所以不出門。 |
| 雨（あめ）が降（ふ）っているので、出（で）かけませんか。 | はい、出（で）かけません。 | 是的，不出門。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 出（で）かけません。 | 不出門。 |
| +時間 | **今日（きょう）は**出（で）かけません。 | 今天不出門。 |
| +原因接續 | **ので**、今日（きょう）は出（で）かけません。 | 因為...今天不出門。 |
| +核心原因 | **雨（あめ）が降（ふ）っているので**、今日（きょう）は出（で）かけません。 | 因為正在下雨，今天不出門。 |
