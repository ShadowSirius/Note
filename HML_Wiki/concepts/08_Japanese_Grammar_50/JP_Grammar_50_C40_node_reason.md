# C40 - 〜ので：原因理由

## 分類
- 理由・目的

## 句型與接續
- `普通形 + ので`

## 核心用法
表示原因理由，語氣比 から 客觀、柔和，常用於禮貌說明。

## 實用例句
- 日文：雨が降っているので、出かけません。
  中文：因為正在下雨，所以不出門。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>雨<rt>あめ</rt></ruby>が<ruby>降<rt>ふ</rt></ruby>っているので、<ruby>出<rt>で</rt></ruby>かけません。
> 因為正在下雨，所以不出門（客觀原因 node）。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 頭が痛い | **<ruby>頭<rt>あたま</rt></ruby>が<ruby>痛<rt>いた</rt></ruby>いので**、出かけません。 |
| 用事がある | **<ruby>用事<rt>ようじ</rt></ruby>があるので**、出かけません。 |
| 忙しい | **<ruby>忙<rt>いそが</rt></ruby>しいので**、出かけません。 |
| 嫌だ (な形容詞) | **嫌な<ruby>ので<rt></rt></ruby>**、出かけません。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 比較 kara (主觀原因) | <ruby>雨<rt>あめ</rt></ruby>が降っている**から**、出かけません。 |
| → 丁寧形原因 (ますので) | <ruby>雨<rt>あめ</rt></ruby>が降って**いますので**、出かけません。 |
| → 過去形原因 | <ruby>雨<rt>あめ</rt></ruby>が**降ったので**、出かけませんでした。 |
| → 口語普通形 | <ruby>雨<rt>あめ</rt></ruby>が降っているので、**出かけない**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| どうして出かけないのですか。 | 雨が降っているので。 |
| 遅れてすみません。道が混んでいました。 | 道が混んでいたので、仕方がないですね。 |
| なぜ日本語を勉強しているのですか。 | 日本のアニメが好きなので、勉強しています。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>出<rt>で</rt></ruby>かけません。 |
| +原因句 (node) | **<ruby>雨<rt>あめ</rt></ruby>なので**、出かけません。 |
| +動詞原因 | **<ruby>雨<rt>あめ</rt></ruby>が降っているので**、出かけません。 |
| +丁寧結尾 | 雨が降っているので、**<ruby>出<rt>で</rt></ruby>かけられません**。 |
