# C46 - 〜そうです：傳聞與聽說

## 分類
- 條件・意志・推量

## 句型與接續
- `普通形 + そうです`

## 核心用法
表示傳聞，即「聽說...」。常與 `〜によると`（根據...）搭配使用，標示資訊來源。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E">ニュースによると、</span><span style="color:#2ECC71">台風が</span><span style="color:#3498DB">来るそうです</span>。
  中文：根據新聞，聽說颱風要來。
- 日文：<span style="color:#2ECC71">王さんは</span><span style="color:#FF477E">台北へ</span><span style="color:#3498DB">行くと言っています</span>。
  中文：王先生說他要去台北。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> ニュースによると、台風（たいふう）が来（く）るそうです。
> （根據新聞，聽說颱風要來。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 天気予報（てんきよほう）／雨（あめ）が降（ふ）る（天氣預報/下雨） | **天気予報（てんきよほう）によると**、**雨（あめ）が降（ふ）る**そうです。 | 根據天氣預報，聽說會下雨。 |
| 噂（うわさ）／二人は結婚（けっこん）する（傳聞/兩人結婚） | **噂（うわさ）によると**、二人は**結婚（けっこん）する**そうです。 | 根據傳聞，聽說兩人要結婚。 |
| 友達（ともだち）の話（はなし）／明日（あした）は休み（やすみ）（朋友的話/明天休息） | **友達（ともだち）の話（はなし）によると**、明日（あした）は**休み（やすみ）だ**そうです。 | 根據朋友的話，聽說明天休息。 |
| 先生（せんせい）の話（はなし）／試験（しけん）がある（老師的話/有考試） | **先生（せんせい）の話（はなし）によると**、**試験（しけん）がある**そうです。 | 根據老師的話，聽說有考試。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 樣態推測（來そうです） | 台風（たいふう）が**来（き）そうです**。 | 颱風看起來快要來了。 |
| → 過去傳聞 | 台風（たいふう）が**来（き）たそうです**。 | 聽說颱風已經來了。 |
| → 疑問形 | 台風（たいふう）が来（く）るそうです**か**。 | 聽說颱風要來嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 台風（たいふう）が来（く）ますか。 | ニュースによると、台風（たいふう）が来（く）るそうです。 | 根據新聞，聽說颱風要來。 |
| 明日（あした）は休み（やすみ）ですか。 | 友達（ともだち）の話（はなし）によると、休み（やすみ）だそうです。 | 根據朋友的話，聽說明天休息。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | そうです。 | 聽說是。 |
| +動詞修飾 | **来（く）る**そうです。 | 聽說要來。 |
| +主語 | **台風（たいふう）が**来（く）るそうです。 | 聽說颱風要來。 |
| +來源 | **ニュースによると**、台風（たいふう）が来（く）るそうです。 | 根據新聞，聽說颱風要來。 |
