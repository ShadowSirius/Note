# C46 - 傳聞：〜そうです・〜と言っています

## 分類
- 引用・傳聞

## 句型與接續
- `普通形 + そうです` / `普通形 + と言っています`

## 核心用法
傳聞そうです表示聽說的資訊；と言っています 引用某人明確說過的內容。

## 實用例句
- 日文：ニュースによると、台風が来るそうです。
  中文：根據新聞，聽說颱風要來。
- 日文：王さんは台北へ行くと言っています。
  中文：王先生說他要去台北。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> ニュースによると、台風が来るそうです。
> 根據新聞，聽說颱風要來（傳聞）。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 明日は雨だ | 明日は**雨だそうです**。 |
| 日本は寒い | 日本は**寒いそうです**。 |
| 彼は行かない | 彼は**行かないそうです**。 |
| 田中さんの話によると / 結婚する | 田中さんの話によると、木村さんは**結婚するそうです**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 間接引用 (と言っています) | 王さんは台北へ**行くと言っています**。 |
| → 過去引用 | 王さんは台北へ行くと**言いました**。 |
| → 口語傳聞 (〜って) | 台風が**来るって**。 |
| → 樣態推測 (sou desu) | 台風が**来そうです**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 台風は来ますか。 | ニュースによると、来るそうです。 |
| 王さんは何と言っていましたか。 | 来週台北へ行くと言っていました。 |
| 明日は寒いですか。 | ええ、とても寒いそうですよ。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | そうです。 |
| +傳聞內容 | **台風が来る**そうです。 |
| +傳聞來源 | **ニュースによると**、台風が来るそうです。 |
| +時間 | ニュースによると、**明日**台風が来るそうです。 |
