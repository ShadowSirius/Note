# C15 - 可能形：能夠做

## 分類
- て形・ない形・た形

## 句型與接續
- `V可能形` / `V辞書形 + ことができます`

## 核心用法
表示能力或條件允許。可能形中，原本的 を 常可變成 が。

## 實用例句
- 日文：日本語が話せます。
  中文：會說日文。
- 日文：泳ぐことができます。
  中文：會游泳。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>日本語<rt>にほんご</rt></ruby>が<ruby>話<rt>はな</rt></ruby>せます。
> 會說日文。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 英語 | **<ruby>英語<rt>えいご</rt></ruby>**が<ruby>話<rt>はな</rt></ruby>せます。 |
| 泳ぐ | <ruby>泳<rt>およ</rt></ruby>**げます**。 |
| 運転する | <ruby>運転<rt>うんてん</rt></ruby>**できます**。 |
| 漢字を書く | <ruby>漢字<rt>かんじ</rt></ruby>が**<ruby>書<rt>か</rt></ruby>けます**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定形 | <ruby>日本語<rt>にほんご</rt></ruby>が**<ruby>話<rt>はな</rt></ruby>せません**。 |
| → 過去肯定形 | <ruby>日本語<rt>にほんご</rt></ruby>が**<ruby>話<rt>はな</rt></ruby>せました**。 |
| → 過去否定形 | <ruby>日本語<rt>にほんご</rt></ruby>が**<ruby>話<rt>はな</rt></ruby>せませんでした**。 |
| → 字典形+能力 (ことができます) | <ruby>日本語<rt>にほんご</rt></ruby>を**<ruby>話<rt>はな</rt></ruby>すことができます**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| <ruby>日本語<rt>にほんご</rt></ruby>が<ruby>話<rt>はな</rt></ruby>せますか。 | はい、少し<ruby>話<rt>はな</rt></ruby>せます。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>話<rt>はな</rt></ruby>せます。 |
| +對象 | **<ruby>日本語<rt>にほんご</rt></ruby>が**<ruby>話<rt>はな</rt></ruby>せます。 |

| +對比 | 英語は話せませんが、**日本語は**上手に話せます。 |
