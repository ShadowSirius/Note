# C12 - 〜てから：先後順序

## 分類
- て形・ない形・た形

## 句型與接續
- `Vて + から`

## 核心用法
表示做完前項之後才進行後項，前後順序明確。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E">宿題をしてから、</span><span style="color:#3498DB">テレビを見ます</span>。
  中文：做完作業之後看電視。

## 學習提示

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 宿題（しゅくだい）をしてから、テレビを見（み）ます。
> （做完作業之後看電視。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| ご飯（はん）を食べて | **ご飯（はん）を食べて**から、テレビを見（み）ます。 | 吃過飯之後看電視。 |
| お風呂に入って | **お風呂（ふろ）に入（はい）って**から、テレビを見（み）ます。 | 洗過澡之後看電視。 |
| 仕事を終えて | **仕事（しごと）を終（お）えて**から、テレビを見（み）ます。 | 工作結束之後看電視。 |
| 掃除をして | **掃除（そうじ）をして**から、テレビを見（み）ます。 | 打掃完之後看電視。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 宿題（しゅくだい）をしてから、テレビを**見（み）ました**。 | 做完作業之後看了電視。 |
| → 否定形 | 宿題（しゅくだい）をしてから、テレビを**見（み）ません**。 | 做完作業之後不看電視。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 宿題（しゅくだい）をしてから何をしますか。 | テレビを見（み）ます。 | 看電視。 |
| いつテレビを見（み）ますか。 | 宿題（しゅくだい）をしてから見（み）ます。 | 做完作業之後看。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | テレビを見（み）ます。 | 看電視。 |
| +前動 | **してから、**テレビを見（み）ます。 | 做完……之後看電視。 |
| +作業 | **宿題（しゅくだい）を**してから、テレビを見（み）ます。 | 做完作業之後看電視。 |
| +時間 | **毎日（まいにち）**宿題（しゅくだい）をしてから、テレビを見（み）ます。 | 每天做完作業之後看電視。 |
