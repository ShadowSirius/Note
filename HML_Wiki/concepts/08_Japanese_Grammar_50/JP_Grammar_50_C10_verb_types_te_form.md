# C10 - 動詞分類與て形

## 分類
- て形・ない形・た形

## 句型與接續
- `Vて`

## 核心用法
て形是連接多數助動詞與句型的核心形式；先判斷 I/II/III 類動詞，再套用變化規則。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#3498DB">書く</span>→<span style="color:#3498DB">書いて</span>
  中文：書く 的て形是 書いて。
- 日文：<span style="color:#3498DB">読む</span>→<span style="color:#3498DB">読んで</span>
  中文：読む 的て形是 読んで。
- 日文：<span style="color:#3498DB">食べる</span>→<span style="color:#3498DB">食べて</span>
  中文：食べる 的て形是 食べて。
- 日文：<span style="color:#3498DB">する</span>→<span style="color:#3498DB">して</span>
  中文：する 的て形是 して。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 手紙（てがみ）を書（か）いて送（おく）ります。
> （寫信並寄出。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 本（ほん）を読（よ）んで | **本（ほん）を読（よ）んで**送（おく）ります。 | 讀完書後寄出。 |
| 資料（しりょう）を作（つく）って | **資料（しりょう）を作（つく）って**送（おく）ります。 | 做完資料後寄出。 |
| 写真（しゃしん）を撮（と）って | **写真（しゃしん）を撮（と）って**送（おく）ります。 | 拍完照片後寄出。 |
| 荷物（にもつ）をまとめて | **荷物（にもつ）をまとめて**送（おく）ります。 | 整理好行李後寄出。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 手紙（てがみ）を書（か）いて**送（おく）りました**。 | 寫信並寄出了。 |
| → 否定形 | 手紙（てがみ）を書（か）いて**送（おく）りません**。 | 寫信但不寄出。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 手紙（てがみ）を書（か）いて送（おく）りましたか。 | はい、手紙（てがみ）を書（か）いて送（おく）りました。 | 是的，寫信並寄出了。 |
| 誰（だれ）に手紙（てがみ）を送（おく）りますか。 | 先生（せんせい）に送（おく）ります。 | 寄給老師。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 送（おく）ります。 | 寄。 |
| +手紙 | **手紙（てがみ）を**送（おく）ります。 | 寄信。 |
| +動作連鎖 | **書（か）いて**手紙（てがみ）を送（おく）ります。 | 寫信並寄出。 |
| +對象 | **友達（ともだち）に**書いて手紙（てがみ）を送（おく）ります。 | 寫信寄給朋友。 |
