# C23 - 〜ても：逆接條件

## 分類
- 條件・意志・推量

## 句型與接續
- `Vて/い形くて/な形で/名詞で + も`

## 核心用法
表示即使前項成立，後項仍不受影響。

## 實用例句
- 日文：雨が降っても、行きます。
  中文：即使下雨也去。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>雨<rt>あめ</rt></ruby>が<ruby>降<rt>ふ</rt></ruby>っても、<ruby>行<rt>い</rt></ruby>きます。
> 即使下雨也去（讓步條件）。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 寒い | **<ruby>寒<rt>さむ</rt></ruby>くても**、行きます。 |
| 忙しい | **<ruby>忙<rt>いそが</rt></ruby>しくても**、行きます。 |
| 高い | **<ruby>高<rt>たか</rt></ruby>くても**、買います。 |
| 嫌だ | **<ruby>嫌<rt>いや</rt></ruby>でも**、やります。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 假定條件 (たら) | <ruby>雨<rt>あめ</rt></ruby>が**<ruby>降<rt>ふ</rt></ruby>ったら**、行きます。 |
| → 過去肯定 | <ruby>雨<rt>あめ</rt></ruby>が降っても、**<ruby>行<rt>い</rt></ruby>きました**。 |
| → 否定形 | <ruby>雨<rt>あめ</rt></ruby>が降っても、**<ruby>行<rt>い</rt></ruby>きません**。 |
| → 普通形（口語） | <ruby>雨<rt>あめ</rt></ruby>が降っても、**<ruby>行<rt>い</rt></ruby>く**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 雨が降っても行きますか。 | はい、雨が降っても行きます。 |
| 高くても買いますか。 | はい、どうしても欲しいですから、高くても買います。 |
| 忙しくても日本語を勉強しますか。 | はい、毎日少しでも勉強します。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>行<rt>い</rt></ruby>きます。 |
| +讓步條件 | **<ruby>雨<rt>あめ</rt></ruby>が降っても**、行きます。 |
| +目的 | 雨が降っても、**学校へ**行きます。 |
| +時間 | **明日**、雨が降っても、学校へ行きます。 |
