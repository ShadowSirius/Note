# C23 - 〜ても：逆接條件

## 分類
- 條件・意志・推量

## 句型與接續
- `Vて/い形くて/な形で/名詞で + も`

## 核心用法
表示即使前項成立，後項仍不受影響。

## 實用例句
- 日文：雨が降っても、行きます。
  中文：即使下雨也去。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 雨（あめ）が降（ふ）っても、行（い）きます。
> （即使下雨也去。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 風が強くても | **風（かぜ）が強（つよ）くても**、行（い）きます。 | 即使風大也去。 |
| 雪が降っても | **雪（ゆき）が降（ふ）っても**、行（い）きます。 | 即使下雪也去。 |
| 寒くても | **寒（さむ）くても**、行（い）きます。 | 即使冷也去。 |
| 遅くなっても | **遅（おそ）くなっても**、行（い）きます。 | 即使變晚也去。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 雨（あめ）が降（ふ）っても、**行（い）きました**。 | 即使當時下雨也去了。 |
| → 否定形 | 雨（あめ）が降（ふ）っても、**行（い）きません**。 | 即使下雨也不去。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 雨（あめ）が降（ふ）っても、行（い）きますか。 | はい、降（ふ）っても行（い）きます。 | 是的，即使下雨也去。 |
| 安（やす）くても買（か）いませんか。 | いいえ、安（やす）ければ買（か）います。 | 不，如果便宜的話會買。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 行（い）きます。 | 去。 |
| +逆接 | **降（ふ）っても、**行（い）きます。 | 即使下（雨）也去。 |
| +主語 | **雨（あめ）が**降（ふ）っても、行（い）きます。 | 即使下雨也去。 |
| +伴隨 | 雨（あめ）が降（ふ）っても、**一人（ひとり）で**行（い）きます。 | 即使下雨也一個人去。 |
