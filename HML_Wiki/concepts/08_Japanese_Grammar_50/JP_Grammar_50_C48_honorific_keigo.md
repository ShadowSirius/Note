# C48 - 尊敬語

## 分類
- 被動・使役・敬語

## 句型與接續
- `お/ご + Vます語幹 + になる` / `V受身` / `特殊尊敬語`

## 核心用法
描述對方或上位者的動作，提升動作主以表敬意。

## 實用例句
- 日文：先生はもうお帰りになりました。
  中文：老師已經回去了。
- 日文：先生がいらっしゃいます。
  中文：老師在/老師來了。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> 先生はもうお帰りになりました。
> 老師已經回去了（尊敬語：お+連用形+になります）。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 本を読む | 先生はもう本を**お読みになりました**。 |
| 写真を撮る | 先生はもう写真を**お撮りになりました**。 |
| 帰る / いらっしゃる (特殊) | 先生はもう**いらっしゃいました**。 |
| 食べる / 召し上がる (特殊) | 先生はもう晩ご飯を**召し上がりました**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 尊敬語命令/請求 (お〜ください) | どうぞ**お入りください**。 |
| → 謙遜語自謙 (謙遜) | 私はもう**帰りました / お帰りしました**。 (後續特殊：失礼します) |
| → 普通形 (普通尊稱) | 先生はもう**帰りました**。 |
| → 普通口語 | 先生もう**帰ったよ**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 先生はもうお帰りになりましたか。 | はい、もうお帰りになりました。 |
| 何を召し上がりますか。 | お寿司を召し上がります。(修正: お寿司をいただきます) |
| 先生はどこにいらっしゃいますか。 | 会議室にいらっしゃいます。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | なりました。 |
| +尊敬表現 | **お帰り物に**なりました。 (修正: **お帰り**になりました) |
| +主體 | **先生は**お帰りになりました。 |
| +時間修飾 | 先生は**もう**お帰りになりました。 |
