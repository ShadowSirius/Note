# C48 - 敬語：尊他語

## 分類
- 敬語

## 句型與接續
- `お + Vます語幹 + になる` / `特殊尊敬語`

## 核心用法
用於抬高對方（長輩、上司、顧客）的動作、狀態，表示敬意。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71">先生は</span><span style="color:#FF477E">もう</span><span style="color:#3498DB">お帰りになりました</span>。
  中文：老師已經回去了。
- 日文：<span style="color:#2ECC71">先生が</span><span style="color:#3498DB">いらっしゃいます</span>。
  中文：老師在/老師來了。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 先生（せんせい）はもうお帰（かえ）りになりました。
> （老師已經回去了。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| お読（よ）みになる（讀） | 先生（せんせい）はもう**お読（よ）みになりました**。 | 老師已經讀過了。 |
| お書（か）きになる（寫） | 先生（せんせい）はもう**お書（か）きになりました**。 | 老師已經寫了。 |
| お聞（き）きになる（聽/問） | 先生（せんせい）はもう**お聞（き）きになりました**。 | 老師已經聽了/問了。 |
| お会（あ）いになる（見面） | 先生（せんせい）はもう**お会（あ）いになりました**。 | 老師已經跟人見面了。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 丁寧非敬語形 | 先生（せんせい）はもう**帰（かえ）りました**。 | 老師已經回去了。 |
| → 特殊敬語形（いらっしゃる） | 先生（せんせい）はもう**いらっしゃいました**。 | 老師已經回去了。 |
| → 疑問形 | 先生（せんせい）はもうお帰（かえ）りになりました**か**。 | 老師已經回去了嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 先生（せんせい）はもうお帰（かえ）りになりましたか。 | はい、もうお帰（かえ）りになりました。 | 是的，已經回去了。 |
| 先生（せんせい）は本（ほん）をお読（よ）みになりましたか。 | はい、お読（よ）みになりました。 | 是的，已經讀了。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | なりました。 | 成了。 |
| +敬語動作 | **お帰（かえ）りに**なりました。 | 回去了/尊他。 |
| +主語 | **先生（せんせい）は**お帰（かえ）りになりました。 | 老師回去了。 |
| +時間 | 先生は**もう**お帰（かえ）りになりました。 | 老師已經回去了。 |
