# C14 - ない形與義務

## 分類
- て形・ない形・た形

## 句型與接續
- `Vない` / `Vなければなりません`

## 核心用法
ない形是禁止、義務、許可否定等句型的基礎；なければなりません 表示不得不做。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#3498DB">薬を飲まなければなりません</span>。
  中文：必須吃藥。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 薬（くすり）を飲（の）まなければなりません。
> （必須吃藥。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 学校に行く | 学校に**行（い）かなければなりません**。 | 必須去學校。 |
| 宿題をする | 宿題を**しなければなりません**。 | 必須做作業。 |
| 早く寝る | 早く**寝（ね）なければなりません**。 | 必須早睡。 |
| 日本語を勉強する | 日本語を**勉強（べんきょう）しなければなりません**。 | 必須學日文。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 薬（くすり）を飲（意為：の）まなければ**なりませんでした**。 | 以前必須吃藥。 |
| → 疑問形 | 薬（くすり）を飲（の）まなければなりません**か**。 | 必須吃藥嗎？ |
| → 緩和說法 | 薬（くすり）を飲（の）ま**ないと**。 | 得吃藥了。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 毎日（まいにち）薬（くすり）を飲（の）まなければなりませんか。 | はい、飲（の）まなければなりません。 | 是的，必須吃。 |
| いつ宿題（しゅくだい）をしなければなりませんか。 | 今日中（きょうじゅう）にしなければなりません。 | 必須在今天之內做。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | なりません。 | 不行。 |
| +條件 | **飲（の）まなければ**なりません。 | 不吃的話不行。 |
| +目的 | **薬（くすり）を**飲（の）まなければなりません。 | 必須吃藥。 |
| +副詞 | **毎日（まいにち）**薬（くすり）を飲（の）まなければなりません。 | 每天都必須吃藥。 |
