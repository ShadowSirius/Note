# C17 - 〜たり〜たりします：列舉

## 分類
- て形・ない形・た形

## 句型與接續
- `Vた + り、Vた + りします`

## 核心用法
列舉代表性動作，不表示全部動作都列完。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71">休みの日は</span><span style="color:#3498DB">映画を見たり、本を読んだりします</span>。
  中文：休假日會看電影、看書等等。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 休み（やすみ）の日（ひ）は映画（えいが）を見（み）たり、本（ほん）を読（よ）んだりします。
> （休假日會看電影、看書等等。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 音楽を聞いたり、ゲームをしたり | 休み（やすみ）の日（ひ）は**音楽（おんがく）を聞（き）いたり、ゲームをしたり**します。 | 休假日會聽音樂、打遊戲等。 |
| 買い物をしたり、散歩をしたり | 休み（やすみ）の日（ひ）は**買い物（かいもの）をしたり、散歩（さんぽ）をしたり**します。 | 休假日會買東西、散步等。 |
| 友達と会ったり、食事をしたり | 休み（やすみ）の日（ひ）は**友達（ともだち）と會（あ）ったり、食事（しょくじ）をしたり**します。 | 休假日會和朋友見面、吃飯等。 |
| 掃除をしたり、洗濯をしたり | 休み（やすみ）の日（ひ）は**掃除（そうじ）をしたり、洗濯（せんたく）をしたり**します。 | 休假日會打掃、洗衣服等。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 休み（やすみ）の日（ひ）は映画（えいが）を見（み）たり、本（ほん）を読（よ）んだり**しました**。 | 之前的休假日看了電影、看了書等。 |
| → 疑問形 | 休み（やすみ）の日（ひ）は映画（えいが）を見（み）たり、本（ほん）を読（よ）んだりします**か**。 | 休假日會看電影、看書等嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 日曜日（にちようび）は何をしますか。 | 本（ほん）を読（よ）んだり、音楽（おんがく）を聞（き）いたりします。 | 會看書、聽音樂等。 |
| 昨日（きのう）は買い物（かいもの）をしたりしましたか。 | はい、買い物（かいもの）をしたり、映画（えいが）を見（み）たりしました。 | 是的，去買了東西，也看了電影。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 読（よ）んだりします。 | 做讀書等事情。 |
| +前項 | **見（み）たり、**本（ほん）を読（よ）んだりします。 | 看（電影）或看書等。 |
| +對象 | **映画（えいが）を**見（み）たり、本（ほん）を読（よ）んだりします。 | 看電影或看書等。 |
| +時間 | **休み（やすみ）の日（ひ）は**映画（えいが）を見（み）たり、本（ほん）を読（よ）んだりします。 | 休假日會看電影、看書等。 |
