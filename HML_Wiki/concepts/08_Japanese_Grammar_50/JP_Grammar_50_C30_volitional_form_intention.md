# C30 - 意向形：〜よう

## 分類
- 條件・意志・推量

## 句型與接續
- `V意向形`

## 核心用法
表示說話者意志、提議或打算，是 ましょう 的普通形基礎。

## 實用例句
- 日文：一緒に帰ろう。
  中文：一起回去吧。
- 日文：明日勉強しよう。
  中文：明天來念書吧。

## 學習提示
- 先判斷此句型處理的是「格助詞、動詞變化、條件推量、理由目的、敬語」哪一類，再回到例句確認助詞與動詞形。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>一緒<rt>いっしょ</rt></ruby>に<ruby>帰<rt>かえ</rt></ruby>ろう。
> 一起回去吧（意向形口語提議）。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 休む | <ruby>一緒<rt>いっしょ</rt></ruby>に**<ruby>休<rt>やす</rt></ruby>もう**。 |
| 食べる | <ruby>一緒<rt>いっしょ</rt></ruby>に**<ruby>食<rt>た</rt></ruby>べよう**。 |
| 勉強する | <ruby>一緒<rt>いっしょ</rt></ruby>に**<ruby>勉強<rt>べんきょう</rt></ruby>しよう**。 |
| 映画を見る | <ruby>一緒<rt>いっしょ</rt></ruby>に**<ruby>映画<rt>えいが</rt></ruby>を<ruby>見<rt>み</rt></ruby>よう**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 禮貌提案 (ましょう) | <ruby>一緒<rt>いっしょ</rt></ruby>に**<ruby>帰<rt>かえ</rt></ruby>りましょう**。 |
| → 意志表現 (〜と思っています) | <ruby>一緒<rt>いっしょ</rt></ruby>に帰ろうと**<ruby>思<rt>おも</rt></ruby>っています**。 |
| → 疑問提議 (口語) | <ruby>一緒<rt>いっしょ</rt></ruby>に帰ろう**か**。 |
| → 普通否定 | <ruby>一緒<rt>いっしょ</rt></ruby>に**帰らない**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 一緒に帰ろう。 | うん、帰ろう。 |
| 明日、何をしようか。 | 図書館で勉強しよう。 |
| お腹がすいたね。何か食べようか。 | そうだね。ラーメンを食べよう。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>帰<rt>かえ</rt></ruby>ろう。 |
| +伴隨 | **<ruby>一緒<rt>いっしょ</rt></ruby>に**帰ろう。 |
| +方向目的 | 一緒に**家へ**帰ろう。 |
| +時間 | **もう遅いから**、一緒に家へ帰ろう。 |
