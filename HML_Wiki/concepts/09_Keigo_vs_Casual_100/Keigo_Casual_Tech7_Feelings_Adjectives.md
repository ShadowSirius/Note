# 技法七：感受、形容詞與狀態感嘆

在日常對話中，表達自己的生理感受（如累、痛、睏、熱、冷）或對事物的看法與感嘆（如好吃、可愛、漂亮、厲害）時，丁寧句型通常會以形容詞搭配 `です`、`ですね` 或 `ですよ` 結尾。在口語中，最核心的規律是**直接去掉 `です`**，或將丁寧的語氣助詞做對應的口語變換。

## 💡 語法變形規律

1. **表達感受或看法 (去掉 `です` / `ですよ` ➡️ `だよ`)**：
   - 丁寧體：`[形容詞/名詞] + です / ですよ` ➡️ 口語：`[形容詞/名詞]`（直接去 `です`）或 `[名詞/な形容詞] + だ / だよ`
   - 例：`嬉しいです` ➡️ `嬉しい`
   - 例：`大丈夫ですよ` ➡️ `大丈夫だよ` (な形容詞 `大丈夫` 後面加上 `だよ` 緩和語氣)
2. **共感或確認感嘆 (省略 `です` ➡️ `ね` / `だね`)**：
   - 丁寧體：`[形容詞/名詞] + ですね` ➡️ 口語：`[い形容詞] + ね` / `[名詞/な形容詞] + だね`
   - 例：`美味しいですね` ➡️ `おいしい` / `おいしいね`
   - 例：`そうですね` ➡️ `そうだね` (名詞性用法後面加 `だね`)
   - 例：`綺麗ですね` ➡️ `綺麗` / `綺麗だね` (な形容詞後面加 `だね`)
3. **生理感受的疑問句 (省略 `ですか` ➡️ 詞幹上揚)**：
   - 丁寧體：`[形容詞] + ですか？` ➡️ 口語：`[形容詞]？`
   - 例：`眠いですか？` ➡️ `眠い？`
   - 例：`痛いですか？` ➡️ `痛い？`

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 嬉（うれ）しいですね ➡️ 嬉（うれ）しい
> （真高興 / 好開心）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 面白（おもしろ）い | **面白（おもしろ）い**！ | 真有趣！ |
| 美味（おい）しい | **美味（おい）しい**！ | 真好吃！ |
| 可愛（かわい）い | **可愛（かわい）い**！ | 好可愛！ |
| 難（むずか）しい | **難（むずか）しい**。 | 真難。 |
| すごい | **すごい**！ | 好厲害！ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 大丈夫（だいじょうぶ）ですよ ➡️ | 大丈夫（だいじょうぶ）だよ。 | 沒關係的。 |
| 綺麗（きれい）ですね ➡️ | 綺麗（きれい）だね。 | 好漂亮喔。 |
| 眠（ねむ）いですか？ ➡️ | 眠（ねむ）い？ | 想睡覺嗎？ |
| 痛（いた）いですか？ ➡️ | 痛（いた）い？ | 痛嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 眠（ねむ）い？ | うん、ちょっと眠（ねむ）い。 | 嗯，有點想睡。 |
| 痛（いた）い？ | ううん、痛（いた）くないよ。 | 不，不痛喔。 |
| 大丈夫（だいじょうぶ）だよ。 | ありがとう、安心（あんしん）した。 | 謝謝，我放心了。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 嬉（うれ）しい。 | 開心。 |
| +程度 | **すごく**嬉（うれ）しい。 | 非常開心。 |
| +原因 | **会（あ）えて**すごく嬉（うれ）しい。 | 能見到面非常開心。 |
| +對象 | **みんなに会（あ）えて**すごく嬉（うれ）しい。 | 能見到大家非常開心。 |
