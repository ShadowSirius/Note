# 技法七：感受、形容詞與狀態感嘆

在日常對話中，表達自己的生理感受（如累、痛、睏、熱、冷）或對事物的看法與感嘆（如好吃、可愛、漂亮、厲害）時，丁寧句型通常會以形容詞搭配 `です`、`ですね` 或 `ですよ` 結尾。在口語中，最核心的規律是**直接去掉 `です`**，或將丁寧的語氣助詞做對應的口語變換。

## 💡 語法變形規律

1. **表達感受或看法 (去掉 `です` / `ですよ` ➡️ `だよ`)**：
   - 丁寧體：`[形容詞/名詞] + です / ですよ` ➡️ 口語：`[形容詞/名詞]`（直接去 `です`）或 `[名詞/な形容詞] + だ / だよ`
   - 例：`嬉しいです` ➡️ `嬉しい`
   - 例：`大丈夫ですよ` ➡️ `大丈夫だよ` (な形容詞 `大丈夫` 後面加上 `だよ` 緩和語氣)
2. **共感或確認感嘆 (省略 `です` ➡️ `ね` / `だね`)**：
   - 丁寧體：`[形容詞/名詞] + ですね` ➡️ 口語：`[い形容詞] + ね` / `[名詞/な形容詞] + だね`
   - 例：`美味しいですね` ➡️ `おいしい` / `おいしいね`
   - 例：`そうですね` ➡️ `そうだね` (名詞性用法後面加 `だね`)
   - 例：`綺麗ですね` ➡️ `綺麗` / `綺麗だね` (な形容詞後面加 `だね`)
3. **生理感受的疑問句 (省略 `ですか` ➡️ 詞幹上揚)**：
   - 丁寧體：`[形容詞] + ですか？` ➡️ 口語：`[形容詞]？`
   - 例：`眠いですか？` ➡️ `眠い？`
   - 例：`痛いですか？` ➡️ `痛い？`

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> 嬉しいですね ➡️ <ruby>嬉しい<rt>うれしい</rt></ruby>
> 真高興 / 好開心

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 有趣 | <ruby>面白い<rt>おもしろい</rt></ruby> |
| 好吃 | おいしい |
| 可愛 | <ruby>可愛い<rt>かわいい</rt></ruby> |
| 厲害 | すごい |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| そうですね → | そうだね |
| 大丈夫ですよ → | <ruby>大丈夫<rt>だいじょうぶ</rt></ruby>だよ |
| 綺麗ですね → | <ruby>綺麗<rt>きれい</rt></ruby>だね |
| 眠いですか？ → | <ruby>眠い<rt>ねむい</rt></ruby>？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| そうだね。 | うん、そうだね。 |
| <ruby>大丈夫<rt>だいじょうぶ</rt></ruby>だよ。 | ありがとう、安心した。 |
| <ruby>眠い<rt>ねむい</rt></ruby>？ | うん、ちょっと眠い。 |
| <ruby>痛い<rt>いたい</rt></ruby>？ | ううん、痛くないよ。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>嬉しい<rt>うれしい</rt></ruby>。 |
| +程度 | すごく嬉しい。 |
| +原因 | 会えてすごく嬉しい。 |
