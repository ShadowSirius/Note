# 技法六：日常社交、寒暄與祝福的縮簡

日常社交中的禮儀用語（如感謝、致歉、打招呼、給予祝福等），在丁寧體中通常有著非常完整且公式化的長句。然而，在朋友或家人之間，使用如此繁複的句型會讓人感到彆扭。因此，口語會進行**省略敬語詞尾**或**直接替換為更親暱的日常代詞**。

## 💡 語法變形規律

1. **省略 `〜ございます`**：
   - 丁寧體以 `〜ございます` 結尾的祝福或感謝，在口語中一律直接砍掉。
   - 例：`ありがとうございます` ➡️ `ありがとう`
   - 例：`おめでとうございます` ➡️ `おめでとう`
2. **省略結尾動詞**：
   - 例：`よろしくお願いします` ➡️ `よろしく`
3. **道歉詞彙的親暱化**：
   - 丁寧體的 `すみません`（抱歉/對不起，也可用作請教）或較為自謙的 `失礼しました`（失禮了），在口語中會替換為 `ごめん`（抱歉）或 `ごめんね`（抱歉喔）。
4. **生活狀態的日常確認 (回家/洗澡/起床)**：
   - 丁寧體以過去疑問式 `〜ましたか？` 來關心對方的狀態。在口語中，直接轉為 `動詞た形 + ？`。
   - 例：`もう帰りますか？` ➡️ `もう帰るの？`（`帰る` 是辭書形，加上 `の` 表疑問，常用於詢問當下）
   - 例：`お風呂に入りましたか？` ➡️ `お風呂入った？`（省略助詞 `に`，轉為 `動詞た形` 加問號）

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> ありがとうございます ➡️ ありがとう
> （謝謝）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| おめでとう | **おめでとう**！ | 恭喜！ |
| よろしく | **よろしく**。 | 請多指教。 |
| ごめん | **ごめん**。 | 抱歉。 |
| 頑張（がんば）って | **頑張（がんば）って**！ | 加油！ |
| 気（き）をつけて | **気（き）をつけて**。 | 小心 / 保重。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| ありがとうございます ➡️ | ありがとう。 | 謝謝。 |
| おめでとうございます ➡️ | おめでとう。 | 恭喜。 |
| よろしくお願（ねが）いします ➡️ | よろしく。 | 請多指教。 |
| すみません ➡️ | ごめん。 | 抱歉。 |
| 頑張（がんば）ってください ➡️ | 頑張（がんば）って。 | 加油。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 誕生日（たんじょうび）おめでとう！ | ありがとう！ | 謝謝！ |
| よろしくね。 | うん、よろしく。 | 嗯，請多指教。 |
| ごめんね。 | 大丈夫（だいじょうぶ）だよ。 | 沒關係啦。 |
| もう起（お）きた？ | うん、もう起（お）きたよ。 | 嗯，已經起床了喔。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | ありがとう。 | 謝謝。 |
| +副詞 | **いつも**ありがとう。 | 一直以來謝謝你。 |
| +動作 | **手伝（てつだ）ってくれて**いつもありがとう。 | 一直很謝謝你幫我忙。 |
| +對象 | **田中（たなか）さん**、手伝（てつだ）ってくれていつもありがとう。 | 田中先生，一直很謝謝你幫我忙。 |
