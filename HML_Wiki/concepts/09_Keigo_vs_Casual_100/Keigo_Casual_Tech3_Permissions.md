# 技法三：請求許可與後續約定

當我們想向對方請求准許做某事（例如拍照、坐下、借用物品等），在丁寧體中常用 `〜てもいいですか`（可以做某事嗎？）。在日常口語中，日本人會省略助詞 `も` 以及結尾的 `ですか`，將其縮簡為 `〜ていい？`，並以**上升語調**發問。

此外，在後續約定聯絡時，請求句 `〜てください` 在口語中常透過句尾助詞 `ね` 轉化為 `〜てね`，以營造溫柔、期待再次互動的親暱語氣。

## 💡 語法變形規律

1. **請求許可句型**：
   - 丁寧體：`[動詞て形] + てもいいですか？` ➡️ 口語：`[動詞て形] + ていい？`（省略 `も` 和 `ですか`，句尾上揚）
   - 例：`入ってもいいですか？` ➡️ `入っていい？`
2. **約定聯絡句型**：
   - 丁寧體：`また + [動詞て形] + てください` ➡️ 口語：`また + [動詞て形] + てね`（將 `てください` 替換為 `てね`）
   - 例：`また連絡してください` ➡️ `また連絡してね`

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 写真（しゃしん）を撮（と）ってもいいですか？ ➡️ 写真（しゃしん）撮（と）っていい？
> （可以拍照嗎？）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 使（つか）う | **使（つか）っていい**？ | 可以使用嗎？ |
| 開（あ）ける | **開（あ）けていい**？ | 可以打開嗎？ |
| 入（はい）る | **入（はい）っていい**？ | 可以進去嗎？ |
| 座（すわ）る | **座（すわ）っていい**？ | 可以坐嗎？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 開（あ）けてもいいですか？ ➡️ | 開（あ）けていい？ | 可以打開嗎？ |
| 一緒（いっしょ）に行（い）ってもいいですか？ ➡️ | 一緒（いっしょ）に行（い）っていい？ | 可以一起去嗎？ |
| 電話（でんわ）してもいいですか？ ➡️ | 電話（でんわ）していい？ | 可以打電話嗎？ |
| また連絡（れんらく）してください ➡️ | また連絡（れんらく）してね。 | 要再聯絡喔。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 写真（しゃしん）撮（と）っていい？ | うん、いいよ。撮（と）って。 | 嗯，可以啊。拍吧。 |
| 入（はい）っていい？ | うん、いいよ。入（はい）って。 | 嗯，可以啊。進去吧。 |
| 電話（でんわ）していい？ | ごめん、今（いま）はちょっと忙（いそが）しい。 | 抱歉，現在有點忙。 |
| また連絡（れんらく）してね。 | うん、また連絡（れんらく）するね。 | 嗯，會再聯絡的。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 撮（と）っていい？ | 可以拍嗎？ |
| +名詞 | **写真（しゃしん）**撮（と）っていい？ | 可以拍照嗎？ |
| +地點 | **ここで**写真（しゃしん）撮（と）っていい？ | 可以在這裡拍照嗎？ |
| +對象 | **一緒（いっしょ）に**ここで写真（しゃしん）撮（と）っていい？ | 可以一起在這裡拍照嗎？ |
