# 技法三：請求許可與後續約定

當我們想向對方請求准許做某事（例如拍照、坐下、借用物品等），在丁寧體中常用 `〜てもいいですか`（可以做某事嗎？）。在日常口語中，日本人會省略助詞 `も` 以及結尾的 `ですか`，將其縮簡為 `〜ていい？`，並以**上升語調**發問。

此外，在後續約定聯絡時，請求句 `〜てください` 在口語中常透過句尾助詞 `ね` 轉化為 `〜てね`，以營造溫柔、期待再次互動的親暱語氣。

## 💡 語法變形規律

1. **請求許可句型**：
   - 丁寧體：`[動詞て形] + てもいいですか？` ➡️ 口語：`[動詞て形] + ていい？`（省略 `も` 和 `ですか`，句尾上揚）
   - 例：`入ってもいいですか？` ➡️ `入っていい？`
2. **約定聯絡句型**：
   - 丁寧體：`また + [動詞て形] + てください` ➡️ 口語：`また + [動詞て形] + てね`（將 `てください` 替換為 `てね`）
   - 例：`また連絡してください` ➡️ `また連絡してね`

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> 写真を撮ってもいいですか？ ➡️ <ruby>写真撮っていい<rt>しゃしんとっていい</rt></ruby>？
> 可以拍照嗎？

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 使って | <ruby>使っていい<rt>つかっていい</rt></ruby>？ |
| 開けて | <ruby>開けていい<rt>あけていい</rt></ruby>？ |
| 入って | <ruby>入っていい<rt>はいっていい</rt></ruby>？ |
| 座って | <ruby>座っていい<rt>すわっていい</rt></ruby>？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| 開けてもいいですか？ → | <ruby>開けていい<rt>あけていい</rt></ruby>？ |
| 一緒に行ってもいいですか？ → | <ruby>一緒に行っていい<rt>いっしょにいっていい</rt></ruby>？ |
| 電話してもいいですか？ → | <ruby>電話していい<rt>でんわしていい</rt></ruby>？ |
| また連絡してください → | また<ruby>連絡してね<rt>れんらくしてね</rt></ruby> |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| <ruby>写真撮っていい<rt>しゃしんとっていい</rt></ruby>？ | うん、いいよ。どうぞ。 |
| <ruby>入っていい<rt>はいっていい</rt></ruby>？ | うん、入って。 |
| <ruby>電話していい<rt>でんわしていい</rt></ruby>？ | ごめん、今はちょっと忙しい。 |
| また<ruby>連絡してね<rt>れんらくしてね</rt></ruby>。 | うん、また連絡するね。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>撮っていい<rt>とっていい</rt></ruby>？ |
| +名詞 | <ruby>写真撮っていい<rt>しゃしんとっていい</rt></ruby>？ |
| +地點 | ここで<ruby>写真撮っていい<rt>しゃしんとっていい</rt></ruby>？ |
