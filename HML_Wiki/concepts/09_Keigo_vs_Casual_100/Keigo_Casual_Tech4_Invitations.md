# 技法四：邀請、提議與意向對比

在日語社交中，向他人發出邀請（要不要一起做某事？）或主動提議提供協助（要我幫忙做某事嗎？）是極為高頻的對話。這些表達在丁寧體和日常口語中有非常鮮明的語法規則對應。

## 💡 語法變形規律

1. **委婉邀請 (要不要...？)**：
   - 丁寧體：`[動詞ます形] + ませんか？`（不...嗎？） ➡️ 口語：`[動詞ない形] + ない？`（省略 `か`，句尾上揚）
   - 例：`行きませんか？` ➡️ `行かない？`
2. **主動提議協助 (要我...嗎？/ 讓我...吧？)**：
   - 丁寧體：`[動詞ます形] + ましょうか？` ➡️ 口語：`[動詞意向形] + おうか / ようか？`
   - 例：`手伝いましょうか？` ➡️ `手伝おうか？` (五段動詞變 `お` 段音 + `う`)
3. **詢問對方意圖 (要[動詞]嗎？)**：
   - 丁寧體：`[動詞ます形] + ますか？` ➡️ 口語：`[動詞辭書形]？`（用現在式普通形，句尾上揚）
   - 例：`飲みますか？` ➡️ `飲む？`
4. **共同提議 (一起...吧)**：
   - 丁寧體：`[動詞ます形] + ましょう` ➡️ 口語：`[動詞意向形] + おう / よう`（常配合句尾助詞 `ね` 緩和語氣）
   - 例：`また会いましょう` ➡️ `また会おうね`

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 一緒（いっしょ）に行（い）きませんか？ ➡️ 一緒（いっしょ）に行（い）かない？
> （要不要一起去？）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 食（た）べる | 一緒（いっしょ）に**食（た）べない**？ | 要不要一起吃？ |
| 飲（の）む | 一緒（いっしょ）に**飲（の）まない**？ | 要不要一起喝？ |
| 来（く）る | 一緒（いっしょ）に**来（こ）ない**？ | 要不要一起來？ |
| 遊（あそ）ぶ | 一緒（いっしょ）に**遊（あそ）ばない**？ | 要不要一起玩？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 行（い）きますか？ ➡️ | 行（い）く？ | 去嗎？ |
| 手伝（てつだ）いましょうか？ ➡️ | 手伝（てつだ）おうか？ | 要我幫忙嗎？ |
| また会（あ）いましょう ➡️ | また会（あ）おうね。 | 下次再見喔。 |
| 食（た）べますか？ ➡️ | 食（た）べる？ | 要吃嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 一緒（いっしょ）に行（い）かない？ | うん、行（い）く！ | 嗯，去！ |
| 手伝（てつだ）おうか？ | ありがとう、お願（ねが）い。 | 謝謝，拜託了。 |
| また会（あ）おうね。 | うん、またね。 | 嗯，再見喔。 |
| 食（た）べる？ | ううん、いらない。ありがとう。 | 不，不用了。謝謝。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 行（い）かない？ | 不去嗎？ |
| +副詞 | **一緒（いっしょ）に**行（い）かない？ | 要不要一起去？ |
| +對象 | **田中（たなか）さんも**一緒（いっしょ）に行（い）かない？ | 田中先生也要不要一起去？ |
| +時間地點 | **明日（あした）京都（きょうと）に**田中（たなか）さんも一緒（いっしょ）に行（い）かない？ | 明天田中先生也要不要一起去京都？ |
