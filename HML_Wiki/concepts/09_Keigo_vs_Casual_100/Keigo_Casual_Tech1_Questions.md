# 技法一：疑問與確認的轉換

在正式日語中，疑問句通常以助詞 `か` 結尾（如 `〜ですか`、`〜ますか`）。然而在日常口語中，句尾的 `か` 被認為過於生硬或刻板，因此日本人會直接**省略 `か`**，並以**句尾音調上揚**來表達疑問。

## 💡 語法變形規律

1. **名詞 / 形容動詞疑問句**：
   - 丁寧體：`[名詞/形容動詞] + ですか？` ➡️ 口語：`[名詞/形容動詞]？`（省略 `ですか`，句尾上揚）
   - 例：`お元気ですか？` ➡️ `元気？`
2. **動詞疑問句**：
   - 丁寧體：`[動詞ます形] + ますか？` ➡️ 口語：`[動詞普通形]？`（省略 `ますか`，句尾上揚，或使用 `〜の？`）
   - 例：`忙しいですか？` ➡️ `忙しい？`
   - 例：`行きますか？` ➡️ `行くの？` / `行く？`
3. **形容詞疑問句**：
   - 丁寧體：`[形容詞] + ですか？` ➡️ 口語：`[形容詞]？`（直接去 `です` 即可）
   - 例：`痛いですか？` ➡️ `痛い？`

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> 今忙しいですか？ ➡️ <ruby>今忙しい<rt>いまいそがしい</rt></ruby>？
> 現在忙嗎？

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 明日 | <ruby>明日忙しい<rt>あしたいそがしい</rt></ruby>？ |
| 大丈夫 | <ruby>今大丈夫<rt>いまだいじょうぶ</rt></ruby>？ |
| 暇 | <ruby>今暇<rt>いまひま</rt></ruby>？ |
| 今日 | <ruby>今日忙しい<rt>きょういそがしい</rt></ruby>？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| お元気ですか？ → | <ruby>元気<rt>げんき</rt></ruby>？ |
| どこですか？ → | どこ？ |
| 本当ですか？ → | <ruby>本当に<rt>ほんとうに</rt></ruby>？ |
| 行きますか？ → | <ruby>行く<rt>いく</rt></ruby>？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| <ruby>今大丈夫<rt>いまだいじょうぶ</rt></ruby>？ | うん、<ruby>大丈夫<rt>だいじょうぶ</rt></ruby>だよ。 |
| お<ruby>腹空<rt>はらす</rt></ruby>いた？ | うん、お<ruby>腹空<rt>はらす</rt></ruby>いた。 |
| <ruby>美味<rt>おい</rt></ruby>しかった？ | うん、<ruby>美味<rt>おい</rt></ruby>しかったよ。 |
| <ruby>明日暇<rt>あしたひま</rt></ruby>？ | ううん、<ruby>明日<rt>あした</rt></ruby>は<ruby>忙しい<rt>いそがしい</rt></ruby>。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>忙しい<rt>いそがしい</rt></ruby>？ |
| +時間 | <ruby>今忙しい<rt>いまいそがしい</rt></ruby>？ |
| +對象 | <ruby>田中<rt>たなか</rt></ruby>さんは<ruby>今忙しい<rt>いまいそがしい</rt></ruby>？ |
