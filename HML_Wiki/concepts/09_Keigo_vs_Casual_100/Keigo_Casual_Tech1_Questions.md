# 技法一：疑問與確認的轉換

在正式日語中，疑問句通常以助詞 `か` 結尾（如 `〜ですか`、`〜ますか`）。然而在日常口語中，句尾的 `か` 被認為過於生硬或刻板，因此日本人會直接**省略 `か`**，並以**句尾音調上揚**來表達疑問。

## 💡 語法變形規律

1. **名詞 / 形容動詞疑問句**：
   - 丁寧體：`[名詞/形容動詞] + ですか？` ➡️ 口語：`[名詞/形容動詞]？`（省略 `ですか`，句尾上揚）
   - 例：`お元気ですか？` ➡️ `元気？`
2. **動詞疑問句**：
   - 丁寧體：`[動詞ます形] + ますか？` ➡️ 口語：`[動詞普通形]？`（省略 `ますか`，句尾上揚，或使用 `〜の？`）
   - 例：`忙しいですか？` ➡️ `忙しい？`
   - 例：`行きますか？` ➡️ `行くの？` / `行く？`
3. **形容詞疑問句**：
   - 丁寧體：`[形容詞] + ですか？` ➡️ 口語：`[形容詞]？`（直接去 `です` 即可）
   - 例：`痛いですか？` ➡️ `痛い？`

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 今（いま）忙（いそが）しいですか？ ➡️ 今（いま）忙（いそが）しい？
> （現在忙嗎？）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 暇（ひま） | 今（いま）**暇（ひま）**？ | 現在有空嗎？ |
| 大丈夫（だいじょうぶ） | 今（いま）**大丈夫（だいじょうぶ）**？ | 現在方便嗎？ |
| 元気（げんき） | 今（いま）**元気（げんき）**？ | 現在好嗎？ |
| 眠（ねむ）い | 今（いま）**眠（ねむ）い**？ | 現在想睡嗎？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 忙（いそが）しいですか？ ➡️ | 忙（いそが）しい？ | 忙嗎？ |
| 本当（ほんとう）ですか？ ➡️ | 本当（ほんとう）に？ | 真的嗎？ |
| 行（い）きますか？ ➡️ | 行（い）くの？ | 要去嗎？ |
| 美味（おい）しかったですか？ ➡️ | 美味（おい）しかった？ | 好吃嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 今（いま）忙（いそが）しい？ | うん、ちょっと忙（いそが）しい。 | 嗯，有點忙。 |
| 今（いま）大丈夫（だいじょうぶ）？ | うん、大丈夫（だいじょうぶ）だよ。 | 嗯，沒問題喔。 |
| 美味（おい）しかった？ | うん、すごく美味（おい）しかった！ | 嗯，非常好吃！ |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 忙（いそが）しい？ | 忙嗎？ |
| +時間 | **今（いま）**忙（いそが）しい？ | 現在忙嗎？ |
| +主語 | **田中（たなか）さん**、今（いま）忙（いそが）しい？ | 田中先生，現在忙嗎？ |
| +原因 | **仕事（しごと）で**田中（たなか）さん、今（いま）忙（いそが）しい？ | 田中先生因為工作現在忙嗎？ |
