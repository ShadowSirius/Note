# 技法二：請求與指示的簡化

正式日語中，向對方提出請求、拜託或發出行動指示時，常用 `〜てください`（請做某事）或 `〜ないでください`（請不要做某事）。在口語中，為了使語氣顯得隨和且不呆板，會**省略 `ください`**，直接使用動詞的 **て形（連接形）** 或 **ないで形（否定連接形）**。

## 💡 語法變形規律

1. **肯定請求**：
   - 丁寧體：`[動詞て形] + てください` ➡️ 口語：`[動詞て形]`（直接去掉 `ください`）
   - 例：`貸してください` ➡️ `貸して`
   - *口語疊詞擴充*：有時會重複兩次以增添活潑或撒嬌的感覺，如 `見てください` ➡️ `見てみて`（看嘛看嘛）、`聞いてください` ➡️ `聞いて聞いて`（聽我說聽我說）。
2. **否定請求**：
   - 丁寧體：`[動詞ない形] + ないでください` ➡️ 口語：`[動詞ない形] + ないで`（直接去掉 `ください`）
   - 例：`心配しないでください` ➡️ `心配しないで`
3. **超敬語與口語的跨度**：
   - 部分職場高階敬語（如 `少々お待ちください`）在私下會直接被簡化為最基本的口語 `ちょっと待って`。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> ちょっと待（ま）ってください ➡️ ちょっと待（ま）って
> （請等一下）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 見（み）せて | ちょっと**見（み）せて**。 | 讓我看一下。 |
| 教（おし）えて | ちょっと**教（おし）えて**。 | 告訴我一下。 |
| 貸（か）して | ちょっと**貸（か）して**。 | 借我一下。 |
| 聞（き）いて | ちょっと**聞（き）いて**。 | 聽我說一下。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 見（み）てください ➡️ | 見（み）て！ | 看！ |
| 心配（しんぱい）しないでください ➡️ | 心配（しんぱい）しないで。 | 不要擔心。 |
| 日本語（にほんご）で言（い）ってください ➡️ | 日本語（にほんご）で言（い）って。 | 用日語說。 |
| 送（おく）ってください ➡️ | 送（おく）って。 | 傳送。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| ちょっと待（ま）って！ | うん、わかった。待（ま）つよ。 | 嗯，我知道了。我等喔。 |
| 心配（しんぱい）しないで。 | ありがとう、心配（しんぱい）しないよ。 | 謝謝，我不擔心喔。 |
| メッセージ送（おく）って。 | うん、後（あと）で送（おく）るね。 | 嗯，待會傳喔。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 待（ま）って。 | 等一下。 |
| +程度 | **ちょっと**待（ま）って。 | 等我一下。 |
| +時間 | **もう**ちょっと待（ま）って。 | 再等我一下。 |
| +主語 | **田中（たなか）さん**、もうちょっと待（ま）って。 | 田中先生，再等我一下。 |
