# 〜ないでください

## 規則與接續
- `動詞ない形 + でください`

## 用法說明
要求對方不要做某事（請不要...）。

## 實用例句
- 日文：<ruby>寫真<rt>しゃしん (shashin)</rt></ruby>を<ruby>撮<rt>と (to)</rt></ruby>らないでください。
- 中文：請不要拍照。
- 日文：<ruby>無理<rt>むり (muri)</rt></ruby>をしないでください。
- 中文：請不要勉強。

## 關係說明
- 表示請求「不要做某事」，與強求「請做某事」的 [[JP_Grammar_11_te_kudasai|〜てください @contradicts]] 為相反句型。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>写真<rt>しゃしん</rt></ruby>を<ruby>撮<rt>と</rt></ruby>らないでください。
> 請不要拍照。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| ここに入 (はい) ります | ここに**入らない**でください。 |
| 心配 (しんぱい) します | **心配しない**でください。 |
| タバコを吸 (す) います | タバコを**吸わない**でください。 |
| 忘 (わす) れます | **忘れない**でください。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 肯定請求 (〜てください) | 写真を**撮って**ください。 |
| → 義務 (〜なければなりません) | 写真を撮ら**なければなりません**。 |
| → 不需要 (〜なくてもいいです) | 写真を撮ら**なくてもいいです**。 |
| → 口語否定命令 (〜ないで) | 写真を**撮らないで**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 写真を撮ってもいいですか。 | いいえ、写真を撮らないでください。 |
| ここに入ってもいいですか。 | いいえ、ここに入らないでください。 |
| タバコを吸ってもいいですか。 | いいえ、タバコを吸わないでください。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | ください。 |
| +撮らないで | 撮らないでください。 |
| +写真を | 写真を撮らないでください。 |
| +ここでは | ここでは写真を撮らないでください。 |
