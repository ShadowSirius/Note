# 〜ないでください

## 規則與接續
- `動詞ない形 + でください`

## 用法說明
要求對方不要做某事（請不要...）。

## 實用例句
- 日文：<ruby>寫真<rt>しゃしん (shashin)</rt></ruby>を<ruby>撮<rt>と (to)</rt></ruby>らないでください。
- 中文：請不要拍照。
- 日文：<ruby>無理<rt>むり (muri)</rt></ruby>をしないでください。
- 中文：請不要勉強。

## 關係說明
- 表示請求「不要做某事」，與強求「請做某事」的 [[JP_Grammar_11_te_kudasai|〜てください @contradicts]] 為相反句型。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 写真（しゃしん）を撮（と）らないでください。
> （請不要拍照。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| タバコ | **タバコを吸（す）わないで**ください。 | 請不要吸菸。 |
| 入（はい）る | ここに**入（はい）らないで**ください。 | 請不要進去這裡。 |
| 無理（むり） | **無理（むり）をしないで**ください。 | 請不要勉強。 |
| 忘（わす）れる | **忘（わす）れないで**ください。 | 請不要忘記。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 肯定請求形 | 写真（しゃしん）を**撮（と）ってください**。 | 請拍照。 |
| → 義務必須形 | 写真（しゃしん）を**撮（と）らなければなりません**。 | 必須拍照。 |
| → 禁止形 | 写真（しゃしん）を**撮（と）ってはいけません**。 | 不准拍照。 |
| → 口語否定命令 | 写真（しゃしん）を**撮（と）らないで**。 | 不要拍照。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 写真（しゃしん）を撮（と）ってもいいですか。 | いいえ、写真（しゃしん）を撮（と）らないでください。 | 不，請不要拍照。 |
| ここに入（はい）ってもいいですか。 | いいえ、危（あぶ）ないですから、入（はい）らないでください。 | 不，因為危險，請不要進去。 |
| 明日（あした）の約束（やくそく）を忘（わす）れないでくださいね。 | はい、絶対に忘（わす）れません。 | 好的，絕對不會忘記。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | ください。 | 請。 |
| +動作否定 | **撮（と）らないで**ください。 | 請不要拍。 |
| +目的語 | **写真（しゃしん）を**撮（と）らないでください。 | 請不要拍照。 |
| +場所 | **ここで**写真（しゃしん）を撮（と）らないでください。 | 請不要在這裡拍照。 |
