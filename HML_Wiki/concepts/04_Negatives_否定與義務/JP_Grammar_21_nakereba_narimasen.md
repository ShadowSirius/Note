# 〜なければなりません

## 規則與接續
- `動詞ない形（去掉い）+ ければなりません`

## 用法說明
表示義務或必須（必須做某事 / 不得不...）。

## 實用例句
- 日文：<ruby>每日<rt>まいにち (mainichi)</rt></ruby><ruby>宿題<rt>しゅくだい (shukudai)</rt></ruby>をしなければなりません。
- 中文：每天都必須做功課。
- 日文：<ruby>藥<rt>くすり (kusuri)</rt></ruby>を<ruby>飲<rt>の (no)</rt></ruby>まなければなりません。
- 中文：必須吃藥。

## 關係說明
- 表示硬性義務，與非義務的 [[JP_Grammar_22_nakute_mo_ii_desu|〜なくてもいいです @contradicts]] 互相對應。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>薬<rt>くすり</rt></ruby>を<ruby>飲<rt>の</rt></ruby>まなければなりません。
> 必須吃藥。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 宿題をします | **宿題をしなければ**なりません。 |
| 学校に行きます | **学校に行かなければ**なりません。 |
| 早く帰ります | **早く帰らなければ**なりません。 |
| 日本語を勉強します | **日本語を勉強しなければ**なりません。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 不需要 (〜なくてもいいです) | 薬を飲ま**なくてもいいです**。 |
| → 口語義務 (〜なきゃいけない) | 薬を飲ま**なきゃいけない**。 |
| → 過去義務 (〜なりませんでした) | 薬を飲まなければ**なりませんでした**。 |
| → 疑問 (〜なりませんか) | 薬を飲まなければ**なりませんか**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 薬を飲まなければなりませんか。 | はい、薬を飲まなければなりません。 |
| 毎日宿題をしなければなりませんか。 | はい、毎日宿題をしなければなりません。 |
| 今日学校に行かなければなりませんか。 | いいえ、今日学校に行かなくてもいいです。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | なりません。 |
| +飲まなければ | 飲まなければなりません。 |
| +薬を | 薬を飲まなければなりません。 |
| +毎日 | 毎日薬を飲まなければなりません。 |
