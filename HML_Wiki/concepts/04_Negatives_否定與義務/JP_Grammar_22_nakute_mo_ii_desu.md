# 〜なくてもいいです

## 規則與接續
- `動詞ない形（去掉い）+ くてもいいです`

## 用法說明
表示不必要、沒必要（不...也可以）。

## 實用例句
- 日文：<ruby>明日<rt>あした (ashita)</rt></ruby>は<ruby>來<rt>こ (ko)</rt></ruby>なくてもいいです。
- 中文：明天不來也可以。
- 日文：<ruby>靴<rt>くつ (kutsu)</rt></ruby>を<ruby>脫<rt>ぬ (nu)</rt></ruby>がなくてもいいですよ。
- 中文：不用脫鞋子也沒關係喔。

## 關係說明
- 解除義務，與強烈要求義務的 [[JP_Grammar_21_nakereba_narimasen|〜なければなりません @contradicts]] 對立。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>靴<rt>くつ</rt></ruby>を<ruby>脱<rt>ぬ</rt></ruby>がなくてもいいです。
> 不用脫鞋子也沒關係。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 漢字を書きます | **漢字を書か**なくてもいいです。 |
| 急ぎます | **急が**なくてもいいです。 |
| 心配します | **心配し**なくてもいいです。 |
| 払います | **払わ**なくてもいいです。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 義務 (〜なければなりません) | 靴を脱が**なければなりません**。 |
| → 許可 (〜てもいいです) | 靴を脱いでも**いいです**。 |
| → 禁止 (〜てはいけません) | 靴を脱いでは**いけません**。 |
| → 普通形不需要 (〜なくてもいい) | 靴を脱がなくても**いいよ**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 靴を脱がなければなりませんか。 | いいえ、靴を脱がなくてもいいです。 |
| 明日は来なければなりませんか。 | いいえ、明日は来なくてもいいです。 |
| 漢字で書かなければなりませんか。 | いいえ、漢字で書かなくてもいいです。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | いいです。 |
