# 高頻詞彙 02：形容詞 50

> [!info] 兩類形容詞
> **い形容詞**：直接修飾名詞（高い山）；否定 → 〜くない。
> **な形容詞**：修飾名詞加な（静かな町）；否定 → 〜じゃない。

## 關係說明
- 「好き／嫌い／上手／下手／欲しい」皆用助詞 **が** 標記對象，見 [[JP_Grammar_09_ga|〜が @related_to]]、[[JP_Grammar_31_ga_hoshii_desu|〜が欲しいです @related_to]]。

## 🎯 練習建議
1. い形容詞背「反義詞成對」，一次記兩個。
2. 每天挑 3 個形容詞描述眼前事物：この部屋は明るいです。
3. 練習過去形：楽しい → 楽しかった；きれい → きれいでした。

> [!tip] Base Sentence
> この部屋（へや）は明（あか）るいです。
> （這個房間很明亮。）

### 2️⃣ Transformation Drill（轉換練習）— い形 vs な形對照

| Transformation | い形（明るい） | な形（静か） |
|---|---|---|
| → 否定形 | <span style="color:#3498DB">明るく</span>**<span style="color:#3498DB">ない</span>**<span style="color:#3498DB">です</span> | <span style="color:#3498DB">静か</span>**<span style="color:#3498DB">じゃない</span>**<span style="color:#3498DB">です</span> |
| → 過去形 | <span style="color:#3498DB">明る</span>**<span style="color:#3498DB">かった</span>**<span style="color:#3498DB">です</span> | <span style="color:#3498DB">静か</span>**<span style="color:#E74C3C">で</span><span style="color:#3498DB">した</span>** |
| → 過去否定 | <span style="color:#3498DB">明るく</span>**<span style="color:#3498DB">なかった</span>**<span style="color:#3498DB">です</span> | <span style="color:#3498DB">静か</span>**<span style="color:#3498DB">じゃなかった</span>**<span style="color:#3498DB">です</span> |
