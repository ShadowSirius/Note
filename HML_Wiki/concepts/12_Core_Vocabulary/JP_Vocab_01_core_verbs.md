## 🎯 練習建議
1. 每個動詞先確認「類別」，再口頭變化：ます形 → て形 → た形 → ない形。
2. 用「常用搭配」整句背，不要只背單字。
3. 搭配 [[JP_Grammar_12_te_imasu|〜ています]] 造句：今、何をしていますか。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 毎日（まいにち）ご飯（はん）を食（た）べます。
> （每天吃飯。）

### 1️⃣ Substitution Drill（替換練習）— 動詞＋受詞整組替換

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| お茶／飲む | <span style="color:#E67E22">毎日お茶（ちゃ）を</span>**<span style="color:#3498DB">飲（の）みます</span>**。 | 每天喝茶。 |
| 本／読む | <span style="color:#E67E22">毎日本（ほん）を</span>**<span style="color:#3498DB">読（よ）みます</span>**。 | 每天讀書。 |
| 音楽／聞く | <span style="color:#E67E22">毎日音楽（おんがく）を</span>**<span style="color:#3498DB">聞（き）きます</span>**。 | 每天聽音樂。 |
| 日本語／勉強する | <span style="color:#E67E22">毎日日本語（にほんご）を</span>**<span style="color:#3498DB">勉強（べんきょう）します</span>**。 | 每天學日文。 |
| 料理／作る | <span style="color:#E67E22">毎日料理（りょうり）を</span>**<span style="color:#3498DB">作（つく）ります</span>**。 | 每天做菜。 |

### 2️⃣ Transformation Drill（轉換練習）— 同一動詞跑遍高頻句型

以「食べます」為例，換成本副任何動詞重跑一輪：

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → て形請求 [[JP_Grammar_11_te_kudasai\ | <span style="color:#3498DB">〜てください]]</span> | ご飯を**食べてください**。 | 請吃飯。 |
| → 進行 [[JP_Grammar_12_te_imasu\ | <span style="color:#3498DB">〜ています]]</span> | 今、ご飯を**食べています**。 | 現在正在吃飯。 |
| → 願望 [[JP_Grammar_15_tai_desu\ | <span style="color:#3498DB">〜たいです]]</span> | ご飯を**食べたいです**。 | 想吃飯。 |
| → 經驗 [[JP_Grammar_18_ta_koto_ga_arimasu\ | <span style="color:#E67E22">〜たこと</span><span style="color:#2ECC71">が</span><span style="color:#3498DB">あります]]</span> | 寿司（すし）を**食べたことがあります**。 | 吃過壽司。 |
| → 禁止 [[JP_Grammar_14_te_wa_ikemasen\ | <span style="color:#2ECC71">〜ては</span><span style="color:#3498DB">いけません]]</span> | ここでご飯を**食べてはいけません**。 | 這裡不可以吃東西。 |
| → 邀約 [[JP_Grammar_34_masen_ka\ | <span style="color:#3498DB">〜ませんか]]</span> | 一緒（いっしょ）にご飯を**食べませんか**。 | 要不要一起吃飯？ |

| 今、何（なに）をしていますか。 | <span style="color:#E67E22">日本語を</span><span style="color:#3498DB">勉強しています。</span> | 正在學日文。 |
