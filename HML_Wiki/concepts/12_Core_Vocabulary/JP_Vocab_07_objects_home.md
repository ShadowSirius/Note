## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 机（つくえ）の上（うえ）に本（ほん）があります。
> （桌上有書。）

### 1️⃣ Substitution Drill（替換練習）— 物品＋位置替換

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| かばんの中／財布 | **<span style="color:#E67E22">かばんの</span><span style="color:#3498DB">中（なか）</span>**<span style="color:#E74C3C">に</span>**<span style="color:#3498DB">財布（さいふ）</span>**<span style="color:#2ECC71">が</span><span style="color:#3498DB">あります。</span> | 包包裡有錢包。 |
| 部屋／テレビ | **<span style="color:#3498DB">部屋（へや）</span>**<span style="color:#E74C3C">に</span>**<span style="color:#3498DB">テレビ</span>**<span style="color:#2ECC71">が</span><span style="color:#3498DB">あります。</span> | 房間裡有電視。 |
| 冷蔵庫の中／卵 | **<span style="color:#E67E22">冷蔵庫（れいぞうこ）の</span><span style="color:#3498DB">中</span>**<span style="color:#E74C3C">に</span>**<span style="color:#3498DB">卵（たまご）</span>**<span style="color:#2ECC71">が</span><span style="color:#3498DB">あります。</span> | 冰箱裡有蛋。 |
| ベッドの下／靴 | **<span style="color:#E67E22">ベッドの</span><span style="color:#3498DB">下（した）</span>**<span style="color:#E74C3C">に</span>**<span style="color:#3498DB">靴（くつ）</span>**<span style="color:#2ECC71">が</span><span style="color:#3498DB">あります。</span> | 床下有鞋子。 |

### 2️⃣ Transformation Drill（轉換練習）— 存在與狀態句型

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 疑問 | <span style="color:#E67E22">机の</span><span style="color:#E74C3C">上に</span><span style="color:#2ECC71">何（なに）が</span>**<span style="color:#3498DB">ありますか</span>**。 | 桌上有什麼？ |
| → 否定 | <span style="color:#E74C3C">部屋に</span><span style="color:#2ECC71">エアコンが</span>**<span style="color:#3498DB">ありません</span>**。 | 房間裡沒有冷氣。 |
| → 有生命（人／動物） | <span style="color:#E74C3C">部屋に</span><span style="color:#2ECC71">猫（ねこ）が</span>**<span style="color:#3498DB">います</span>**。 | 房間裡有貓。 |
| → 位置為主題 | <span style="color:#2ECC71">鍵（かぎ）は</span><span style="color:#3498DB">玄関（げんかん）</span>**<span style="color:#E74C3C">に</span><span style="color:#3498DB">あります</span>**。 | 鑰匙在玄關。 |
| → 狀態保留 [[JP_Grammar_36_te_arimasu\ | <span style="color:#3498DB">〜てあります]]</span> | 窓（まど）が**開（あ）けてあります**。 | 窗戶是開著的（有人開的）。 |
