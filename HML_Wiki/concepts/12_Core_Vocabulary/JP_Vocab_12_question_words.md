## 🎯 練習建議
1. 「疑問詞 + ですか」是旅遊生存核心，全部背到反射性脫口而出。
2. 自問自答練習：これは何ですか。— それは辞書です。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> これは何（なん）ですか。
> （這是什麼？）

### 1️⃣ Substitution Drill（替換練習）— 疑問詞替換

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 誰の傘 | <span style="color:#2ECC71">これは</span>**<span style="color:#E67E22">誰（だれ）の</span>**<span style="color:#3498DB">傘（かさ）ですか。</span> | 這是誰的傘？ |
| いくら | <span style="color:#2ECC71">これは</span>**<span style="color:#3498DB">いくら</span>**<span style="color:#3498DB">ですか。</span> | 這個多少錢？ |
| トイレ／どこ | <span style="color:#2ECC71">トイレは</span>**<span style="color:#3498DB">どこ</span>**<span style="color:#3498DB">ですか。</span> | 廁所在哪裡？ |
| 会議／いつ | <span style="color:#2ECC71">会議（かいぎ）は</span>**<span style="color:#3498DB">いつ</span>**<span style="color:#3498DB">ですか。</span> | 會議是什麼時候？ |
| 日本／どう | <span style="color:#2ECC71">日本（にほん）は</span>**<span style="color:#3498DB">どう</span>**<span style="color:#3498DB">ですか。</span> | 日本怎麼樣？ |

### 2️⃣ Transformation Drill（轉換練習）— 陳述句 → 疑問句

聽到答案，反推問題：

| 答案（Statement） | 你要問的問題 | Translation（中文翻譯） |
|---|---|---|
| 三時（さんじ）に始まります。 | **何時（なんじ）に**始（はじ）まりますか。 | 幾點開始？ |
| 駅（えき）の近（ちか）くにあります。 | **どこに**ありますか。 | 在哪裡？ |
| 電車（でんしゃ）で行きます。 | **何（なに）で**行きますか。 | 怎麼去？ |
| 五百円（ごひゃくえん）です。 | **いくら**ですか。 | 多少錢？ |
| 二時間（にじかん）ぐらいかかります。 | **どのくらい**かかりますか。 | 要花多久？ |

### 3️⃣ Response Drill（應答練習）— 生存場景快問快答

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| すみません、駅はどこですか。 | <span style="color:#E74C3C">まっすぐ行（い）って、左（ひだり）に</span><span style="color:#3498DB">あります。</span> | 直走，在左邊。 |
| お仕事（しごと）は何ですか。 | <span style="color:#3498DB">エンジニアです。</span> | 我是工程師。 |
| どうして日本語を勉強（べんきょう）していますか。 | <span style="color:#E67E22">日本の</span><span style="color:#2ECC71">文化（ぶんか）が</span><span style="color:#E74C3C">すから</span>。 | 因為喜歡日本文化。 |
| もう一度（いちど）お願（ねが）いします。 | <span style="color:#3498DB">い、ゆっくり話（はな）しますね。</span> | 好，我說慢一點喔。 |
