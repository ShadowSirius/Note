# 高頻詞彙 13：時間表現深化（〜に規則・順序詞・時間帶）

本卡深化 [[JP_Vocab_03_time_numbers|時間與數量 @extends]] 中已收的基礎時間詞（今日/昨日/明日/朝/昼/夜/午前/午後/今週/先週/来週等），**不重複建立基礎詞表**，聚焦三件事：〜に 該不該加的判斷規則、順序／連接類時間詞的細緻辨義、以及一日與距今遠近的時間帶系統。

## 🧠 記憶鉤
- **〜に 加不加，看「有沒有數字釘住一個點」**：三時・7月10日・2020年 都是可以用手指在日曆/時鐘上戳出來的「點」，點才需要 に 釘住；今日・明日・毎朝 這種相對詞本身已經指向明確的一天，不需要再釘。
- **この間・この頃・この前 三兄弟，用「時間箭頭方向」記**：この間＝箭頭指向過去某一點（前幾天發生的事，一次性）；この頃＝箭頭指向現在，是一段持續到現在的區間（最近這陣子）；この前＝箭頭指向過去某次事件（上一次，可重複發生的事件中的最近一次，如上次見面）。
- **さらいしゅう・さらいねん 的「さ」＝ 再**：再来週・再来年 是「來＋再來一次」＝未來 +2；一昨日・一昨年 是「昨＋再往前一次」＝過去 -2。「來」的加倍用「再」，「昨」的加倍用「一」，兩條路線用字完全不同，不要混著猜。
- **これから／それから／そして 用「時間軸位置」分**：これから＝從現在這一刻起（面向未來，常單獨開頭發語）；それから＝從剛講完的動作之後接下一個動作（強調先後順序，可替換成「その後」）；そして＝單純並列/添加下一件事（順序感較弱，也可表「而且」）。口訣：「これ」是現在,「それ」是剛剛講的那件事。

## ⚠️ 易混淆對比
| 易混組合 | 差異 |
|---|---|
| いつ vs なんじ | 兩者都可寫成「何時」，讀 いつ 時問的是「大範圍的時間」（何時行きますか＝什麼時候去，可能回答「來週」）；讀 なんじ 時問的是「精確幾點幾分」（何時ですか＝現在幾點）。**同一個漢字兩種讀音、兩種問法**，要靠上下文判斷。 |
| この間 vs 這段期間 | この間 **不是**「這段期間」，而是「前幾天／不久前的某一天」，是一個過去的時間點，不是持續中的區間。持續到現在的「這陣子」要用 この頃。 |
| 夕べ vs 夕方 | 夕べ（ゆうべ）＝昨晚（時間上＝昨夜）；夕方（ゆうがた）＝傍晚（一天中的時段，天天都有）。發音相近＋語意都跟「晚上」有關，極易混淆，記法：夕「べ」的べ跟「昨晚」一起背，夕「方」的方是「一日中的一個方位/時段」。 |
| あと vs ご（後） | 同一個漢字「後」有兩種讀法功能不同：あと／あとで 是訓讀，獨立副詞用（後で行きます＝待會去）；ご 是音讀，接在數量詞後面組成「N＋後」的固定結構（2日後＝兩天後、3年後＝三年後）。訓讀單獨站，音讀黏數字。 |
| 連休 vs 練習 | 連休（れんきゅう，連續假期）與 練習（れんしゅう，練習）發音只差一個母音（きゅう vs しゅう），聽力／口說時極易說錯，需刻意放慢分辨。 |

## 📐 〜に 加不加規則（決策表）
| 時間表現類型 | 加に？ | 例句 |
|---|---|---|
| 有具體數字的時間點（幾點／幾號／幾年／星期幾＋日期） | **必加** に | 三時に起きます。7月10日に行きます。2020年に生まれました。 |
| 無數字的相對時間詞（今日・明日・今週・毎朝・この間 等） | **不加** に | 今日行きます（× 今日に行きます）。毎朝走ります。 |
| 曜日（星期，單獨使用） | **可加可不加** | 月曜日（に）会議があります。 |
| 指定日期／劃定出一段明確時間（如：連休の間に、夏休み中に） | **必加** に | 夏休みの間に宿題をします。連休中に旅行します。 |

## 🕐 時間帶（一日時間軸 + 過去距離階梯）
一日時間軸（由早到晚）：

朝（あさ）→ 正午（しょうご）→ 昼（ひる）→ 夕方（ゆうがた）→ 薄暮（はくぼ）→ 晩／夜（ばん／よる）

|---|---|---|---|---|
| 朝 | あさ | asa | 早上 | 已收於 JP_Vocab_03 |
| 正午 | しょうご | shougo | 正午 | |
| 昼 | ひる | hiru | 中午、白天 | 已收於 JP_Vocab_03 |
| 夕方 | ゆうがた | yuugata | 傍晚 | 每天都有的時段，⚠️對比夕べ |
| 薄暮 | はくぼ | hakubo | 薄暮、黃昏 | 較書面，指日落後天未全黑 |
| 晩 | ばん | ban | 晚上 | 與夜近義，晩較口語日常（今晩） |
| 夜 | よる | yoru | 夜晚 | 已收於 JP_Vocab_03 |
| 今朝 | けさ | kesa | 今天早上 | ⚠️特殊讀音，非「いまあさ」 |
| 夕べ | ゆうべ | yuube | 昨晚 | ⚠️特殊讀音，非「今天傍晚」，見上方對比 |
| 今晩 | こんばん | konban | 今晚 | |
| 今夜 | こんや | kon'ya | 今夜 | 較書面／文學性 |
| 毎朝 | まいあさ | maiasa | 每天早上 | |
| 毎晩 | まいばん | maiban | 每天晚上 | |
| 朝昼晩御飯 | あさひるばんごはん | asa hiru ban gohan | 三餐（早中晚餐） | 記三餐固定用「晩」不用「夜」 |
| から | から | kara | 從（接持續動詞） | 與 [[JP_Grammar_08_kara_made]] 搭配 |
| まで | まで | made | 到 | |
| 近く | ちかく | chikaku | 附近、將近 | 時間語境：〜に近く＝快到〜的時候 |

**過去距離階梯**（由近到遠）：さっき（剛剛）→ この間（前幾天）→ 昔（むかし，很久以前）→ ずっと前（更早以前）→ 結構前（けっこうまえ，相當久以前，語感比昔更主觀近）→ 昔々（むかしむかし，故事開頭「很久很久以前」）

| 日文 | 假名 | 羅馬字 | 中文 |
|---|---|---|---|
| 昔 | むかし | mukashi | 以前、從前 |
| 昔々 | むかしむかし | mukashi mukashi | 很久很久以前（故事體） |
| ずっと前 | ずっとまえ | zutto mae | 很久以前 |
| 結構前 | けっこうまえ | kekkou mae | 相當久以前（口語） |

## 📅 相對日／週／年系統對照表
| 距今 | −2 | −1 | 0 | +1 | +2 |
|---|---|---|---|---|---|
| 日 | 一昨日（おととい） | 昨日（きのう） | 今日（きょう） | 明日（あした／あす） | 明後日（あさって） |
| （日，極端） | | | | | 明明後日（しあさって，+3） |
| 週 | ─ | 先週（せんしゅう） | 今週（こんしゅう） | 来週（らいしゅう） | 再来週（さらいしゅう） |
| 年 | 一昨年（おととし） | 去年（きょねん） | 今年（ことし） | 来年（らいねん） | 再来年（さらいねん） |

- 日/昨/明 已收於 [[JP_Vocab_03_time_numbers]]（今日・昨日・明日）；本表新增：一昨日、明後日、明明後日、再来週、一昨年、再来年。
- **「さ・らい規律」記憶鉤**：週與年系統呈規則對稱——未來 +1 用「来」，+2 在「来」前加「再」（さ）變成「再来」；過去 −1 用「先/去」，−2 用「一昨」。日系統則完全不遵循此規律，是獨立的一組固有詞（一昨日・明後日・明明後日），不能套用「再＋来日」去猜。
