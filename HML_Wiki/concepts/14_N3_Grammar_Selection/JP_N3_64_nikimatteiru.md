# 〜に決まっている

## 規則與接續
- `動詞（普通形） + に決まっている`
- `い形容詞 + に決まっている`
- `名詞 + に決まっている`
- 口語中常用來表示說話者根據情況做出的強烈斷定，有「肯定是、一定是」的語感。

## 用法說明
表示說話者根據眼前事實、常識或情況，帶著強烈把握地斷定「一定是……」。語氣比單純推測更強，常帶有主觀判斷、驚訝或不滿。

與 `〜はずだ` 相比，`〜に決まっている` 更直接、更斷定；與 `〜に違いない` 相比，`〜に決まっている` 更口語、語氣也更強。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼(かれ)は毎日(まいにち)遅(おく)くまで働(はたら)いているから、疲(つか)れているに決(き)まっている。
> （他每天工作到很晚，所以一定很累。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| おいしい | このケーキはおいしいに決まっている。 | 這個蛋糕一定很好吃。 |
| 忙しい | 彼は忙しいに決まっている。 | 他一定很忙。 |
| 有名だ | あの人は有名に決まっている。 | 那個人一定很有名。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 「彼は犯人だ。」を強く断定する | 彼は犯人に決まっている。 | 他一定是犯人。 |
| → 「このケーキはおいしい。」を強く断定する | このケーキはおいしいに決まっている。 | 這個蛋糕一定很好吃。 |
| → 「彼は忙しい。」を強く断定する | 彼は忙しいに決まっている。 | 他一定很忙。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
