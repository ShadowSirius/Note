# 〜でございます

## 規則與接續
- `名詞 + でございます`
- `な形容詞語幹 + でございます`
- `（商業／服務業場合）です 的鄭重語，常見於櫃檯、客服、店員介紹、電話應對`

## 用法說明
「〜でございます」是「〜です」的更鄭重版本，帶有明顯的商務、服務業語感，常用來向客人、上司或陌生人說明事物。與「ございます」相比，「ございます」多用在「〜がございます」表示「有／存在」；「〜でございます」則是判斷句的敬語形式，像是「〜です」的超正式說法，常與 [[JP_N3_52_osuru|お〜する @related_to]] 等謙讓表現搭配使用於服務業場合。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> こちらが新製品（しんせいひん）の案内（あんない）でございます。
> （這是新產品的介紹資料。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 予約表（よやくひょう） | こちらが予約表（よやくひょう）でございます。 | 這是預約表。 |
| 名刺（めいし） | こちらが名刺（めいし）でございます。 | 這是名片。 |
| 領収書（りょうしゅうしょ） | こちらが領収書（りょうしゅうしょ）でございます。 | 這是收據。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → です を でございます に変える | これは説明書（せつめいしょ）です。 → これは説明書（せつめいしょ）でございます。 | 這是說明書。 |
| → 敬語にする | ここは受付（うけつけ）です。 → ここは受付（うけつけ）でございます。 | 這裡是櫃檯。 |
| → 名詞述語 | 山田（やまだ）さんは担当（たんとう）です。 → 山田（やまだ）さんは担当（たんとう）でございます。 | 山田先生是負責人。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| これは何（なん）ですか。 | こちらはご案内（あんない）でございます。 | 這是說明資料。 |
| 受付（うけつけ）はどこですか。 | 受付（うけつけ）はあちらでございます。 | 櫃檯在那邊。 |
| お荷物（にもつ）はどうしますか。 | こちらでお預（あず）かりでございます。 | 行李由我們這邊保管。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | ご案内（あんない）でございます。 | 是說明資料。 |
| 展開1 | こちらはご案内（あんない）でございます。 | 這是說明資料。 |
| 展開2 | こちらが本日（ほんじつ）のご案内（あんない）でございます。 | 這是今天的說明資料。 |
| 展開3 | こちらが本日（ほんじつ）のご案内（あんない）資料（しりょう）でございます。 | 這是今天的說明資料。 |
