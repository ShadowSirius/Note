# 〜とみられる

## 規則與接續
- `動詞普通形 + とみられる`
- `い形容詞普通形 + とみられる`
- `な形容詞語幹 / 名詞 + とみられる`

## 用法說明
表示根據客觀事實、證據或情況判斷出來的結果，意思接近「被認為、被視為、推測為」。這是較正式、偏書面語的表達，常見於新聞、報告、分析文章中。

與 `〜と思われる` 相比，`〜とみられる` 更強調客觀報導與外部觀察，語氣較中立；`〜と思われる` 則可包含較廣的主觀判斷，使用範圍更一般。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> この事件(じけん)は計画的(けいかくてき)な犯行(はんこう)とみられる。
> （這起事件被認為是有計畫的犯行。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 事故の原因(げんいん) / 機械(きかい)の故障(こしょう) | 事故の原因(げんいん)は機械(きかい)の故障(こしょう)とみられる。 | 事故原因被認為是機械故障。 |
| 火災(かさい)の原因(げんいん) / 電気系統(でんきけいとう)の異常(いじょう) | 火災(かさい)の原因(げんいん)は電気系統(でんきけいとう)の異常(いじょう)とみられる。 | 火災原因被認為是電氣系統異常。 |
| 彼(かれ) / 事件(じけん)に関係(かんけい)している | 彼(かれ)は事件(じけん)に関係(かんけい)しているとみられる。 | 他被認為與事件有關。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 書面語・推測表達 | この建物(たてもの)は重要文化財(じゅうようぶんかざい)とみられる。 | 這棟建築被認為是重要文化財。 |
| → 口語「〜そうだ」的新聞式改寫 | 明日(あす)は雪(ゆき)が降(ふ)るとみられる。 | 預計明天會下雪。 |
| → 「〜だ」改成判斷表達 | その男(おとこ)は犯人(はんにん)とみられる。 | 那名男子被認為是犯人。 |
