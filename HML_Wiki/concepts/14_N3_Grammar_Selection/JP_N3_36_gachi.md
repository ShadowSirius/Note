# 〜がち

## 規則與接續
- `動詞ます形（去masu） + がち`（例：休む→休みがち）
- `名詞 + がち`（例：遠慮→遠慮がち）

## 用法說明
表示「容易...、動不動就...」，強調某種（多半是不好的）行為或狀態頻繁發生，語氣帶有負面評價，比 [[JP_N3_35_gimi|〜気味]] 更強調「次數多、反覆發生」而非單一時刻的狀態。

與 [[JP_N3_34_ppoi|〜っぽい]]、[[JP_N3_35_gimi|〜気味]] 的辨析：
- 「〜がち」：強調行為「反覆發生」的頻率傾向，幾乎都是負面事項（休みがち、忘れがち、遅刻がち）。
- 「〜気味」：描述「當下」身心狀態的輕微徵兆，語氣中性，不特別強調次數。
- 「〜っぽい」：描述整體性格或外觀特徵給人的印象，偏靜態、口語化。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）は最近（さいきん）、学校（がっこう）を休（やす）みがちだ。
> （他最近常常缺席學校。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 忘れる（わすれる） | 彼は約束を**忘れがち**だ。 | 他常常忘記約定。 |
| 遅刻する（ちこくする） | 彼は最近、**遅刻しがち**だ。 | 他最近常常遲到。 |
| 曇る（くもる） | この時期は空が**曇りがち**だ。 | 這個時期天空經常陰陰的。 |
| 病気（びょうき） | 子供のころは**病気がち**だった。 | 小時候常常生病。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 気味（單一狀態） | 彼は最近、疲れ**気味**だ。 | 他最近有點累的樣子。（改為描述當下狀態，非頻率） |
| → 丁寧形 | 彼は最近、学校を休みがち**です**。 | 他最近常常缺席學校。（禮貌形） |
| → 過去形 | 去年は学校を休みがち**だった**。 | 去年常常缺席學校。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 彼、最近学校に来ませんね。 | ええ、最近学校を休みがちなんです。 | 對啊，他最近常常缺席學校。 |
| 一人暮らしは大変ですか。 | ええ、食事が不規則になりがちですね。 | 是啊，飲食容易變得不規律呢。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 学校を休みがちだ。 | 常常缺席學校。 |
| +主語 | **彼は**学校を休みがちだ。 | 他常常缺席學校。 |
| +時間 | 彼は**最近、**学校を休みがちだ。 | 他最近常常缺席學校。 |
