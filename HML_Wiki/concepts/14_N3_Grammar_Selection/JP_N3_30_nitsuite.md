# 〜について

## 規則與接續
- `名詞 + について`

## 用法說明
表示思考、調查或陳述的話題對象，意為「關於…」，後方常接「話す、考える、調べる、書く」等動詞。不像 [[JP_N3_29_nitaishite|〜に対して @related_to]] 帶有對立或指向性，純粹用於提示討論主題。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E"><ruby>日本<rt>にほん</rt></ruby>の<ruby>社会<rt>しゃかい</rt></ruby><ruby>問題<rt>もんだい</rt></ruby>について、</span><span style="color:#3498DB">レポートを<ruby>書<rt>か</rt></ruby>かなければなりません。</span>
  中文：我必須寫一份關於日本社會問題的報告。
- 日文：<span style="color:#FF477E"><ruby>次<rt>つぎ</rt></ruby>の<ruby>会議<rt>かいぎ</rt></ruby>で、</span><span style="color:#FF477E"><ruby>新<rt>あたら</rt></ruby>しい<ruby>商品<rt>しょうひん</rt></ruby>の<ruby>企画<rt>きかく</rt></ruby>について</span><span style="color:#3498DB"><ruby>話<rt>はな</rt></ruby>し<ruby>合<rt>あ</rt></ruby>いましょう。</span>
  中文：在下次的會議上，一起討論關於新商品企劃的事吧。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 日本（にほん）の社会（しゃかい）問題（もんだい）について、レポートを書（か）かなければなりません。
> （我必須寫一份關於日本社會問題的報告。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 環境問題（かんきょうもんだい） | **環境問題について**調べています。 | 正在調查環境問題。 |
| 歴史（れきし） | この本は日本の**歴史について**書かれている。 | 這本書是關於日本歷史寫的。 |
| 将来の計画（しょうらいのけいかく） | **将来の計画について**話し合った。 | 討論了未來的計畫。 |
| 契約内容（けいやくないよう） | **契約内容について**確認してください。 | 請確認契約內容。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 名詞修飾用法 | それは日本**についての**本です。 | 那是一本關於日本的書。 |
| → 過去語境 | 社会問題について**レポートを書いた**。 | 已經寫了關於社會問題的報告。 |
| → 疑問形 | 何について**話しますか**。 | 要談關於什麼的事呢？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| レポートのテーマは何ですか。 | 日本の社会問題について書きます。 | 我要寫關於日本社會問題的內容。 |
| 次の会議では何を話しますか。 | 新しい商品の企画について話し合います。 | 討論關於新商品企劃的事。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | レポートを書きます。 | 要寫報告。 |
| +話題 | **日本の社会問題について、**レポートを書きます。 | 要寫關於日本社會問題的報告。 |
| +義務 | 日本の社会問題について、レポートを**書かなければなりません**。 | 必須寫一份關於日本社會問題的報告。 |
