# 〜わけだ

## 規則與接續
- `用言普通形 + わけだ`
- `な形容詞語幹 + な + わけだ`
- `名詞 + という/である + わけだ`

## 用法說明
表示根據已知事實推導出理所當然的結論，語感為「難怪、也就是說」。與偏向理性推測的 [[JP_Grammar_50_C45_hazu_desu_beki_desu|〜はずだ @related_to]] 不同，「〜わけだ」側重於得知原因後的恍然大悟與理解，而非單純的推理判斷。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71">彼は</span><span style="color:#FF477E"><ruby>日本<rt>にほん</rt></ruby>に10<ruby>年<rt>ねん</rt></ruby>も<ruby>住<rt>す</rt></ruby>んでいたから、</span><span style="color:#3498DB"><ruby>日本語<rt>にほんご</rt></ruby>がペラペラなわけだ。</span>
  中文：他在日本住了10年之久，難怪日文說得這麼流利。
- 日文：<span style="color:#FF477E"><ruby>消費税<rt>しょうひぜい</rt></ruby>が10%になったから、</span><span style="color:#3498DB"><ruby>買<rt>か</rt></ruby>い<ruby>物<rt>もの</rt></ruby>が<ruby>高<rt>たか</rt></ruby>く<ruby>感<rt>かん</rt></ruby>じるわけですね。</span>
  中文：因為消費稅變成10%了，也就是說買東西會覺得變貴了呢。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）は日本（にほん）に10年（ねん）も住（す）んでいたから、日本語（にほんご）がペラペラなわけだ。
> （他在日本住了10年之久，難怪日文說得這麼流利。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 料理が上手（じょうずだ） | 母が料理の先生だったから、**料理が上手なわけだ**。 | 難怪料理很拿手。 |
| 疲れている（つかれている） | 徹夜したから、**疲れているわけだ**。 | 難怪很累。 |
| 忙しい（いそがしい） | 仕事が増えたから、**忙しいわけだ**。 | 難怪很忙。 |
| 学生だ | まだ20歳（さい）だから、**学生というわけだ**。 | 也就是說還是學生。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 否定形（わけではない） | 誰でも日本語が話せる**わけではない**。 | 並非任何人都會說日文。 |
| → 丁寧形 | 日本語がペラペラな**わけです**。 | 難怪日文很流利。（禮貌形） |
| → 疑問形 | 日本語がペラペラな**わけですか**。 | 也就是說日文很流利嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 彼はどうして日本語が上手なんですか。 | 日本に10年も住んでいたから、上手なわけです。 | 因為在日本住了10年，難怪厲害。 |
| どうして最近買い物が高く感じるんですか。 | 消費税が10%になったから、高く感じるわけですね。 | 因為消費稅漲了，難怪覺得貴。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | ペラペラなわけだ。 | 難怪很流利。 |
| +主語 | **彼は**日本語がペラペラなわけだ。 | 難怪他日文很流利。 |
| +理由 | **日本に10年も住んでいたから、**彼は日本語がペラペラなわけだ。 | 因為在日本住了10年，難怪他日文流利。 |
