# 〜っぽい

## 規則與接續
- `動詞ます形（去masu） + っぽい`（例：忘れる→忘れっぽい）
- `名詞 + っぽい`（例：子供→子供っぽい）
- `い形容詞語幹（去い） + っぽい`（例：安い→安っぽい）

## 用法說明
表示「帶有...的傾向、看起來像...」，多用於口語，常帶輕微負面語感，用來描述人的性格、外觀或事物的性質接近某種特徵（但實際上不完全是）。

與 [[JP_N3_35_gimi|〜気味]]、[[JP_N3_36_gachi|〜がち]] 的辨析：
- 「〜っぽい」：描述「看起來像／帶有...性質」，偏向性格、外觀、材質等靜態特徵判斷，口語色彩濃，常帶輕微貶意（如子供っぽい＝幼稚）。
- 「〜気味」：描述身心狀態「有點...的徵兆」，語氣中性偏客觀，常用於健康、疲勞等暫時性狀態（如風邪気味）。
- 「〜がち」：描述「容易發生某種（多為負面）行為或狀態」的頻率傾向，強調發生次數偏多（如休みがち＝常常請假），比「っぽい」更聚焦於動作重複發生。

## 🎯 FSI Pattern Drill

> 彼（かれ）は忘（わす）れっぽい性格（せいかく）だ。
> （他個性健忘。）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 怒る（おこる） | 彼は**怒りっぽい**性格だ。 | 他個性容易生氣。 |
| 安い（やすい） | このかばんは**安っぽい**デザインだ。 | 這個包包設計看起來很廉價。 |
| 大人（おとな） | 彼女は**大人っぽい**服を着ている。 | 她穿著看起來很成熟的衣服。 |
| 油（あぶら） | この料理（りょうり）は**油っぽい**。 | 這道菜很油膩。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 気味（狀態較客觀） | 最近、忘れ**気味**だ。 | 最近有點健忘的徵兆。 |
| → 丁寧形 | 彼は忘れっぽい性格**です**。 | 他個性健忘。（禮貌形） |
| → 否定形 | 彼は忘れっぽく**ない**。 | 他不健忘。 |

| Prompt | Expected Response | Translation（中文翻譯） |
