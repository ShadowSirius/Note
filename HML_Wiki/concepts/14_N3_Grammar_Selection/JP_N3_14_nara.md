# 〜なら

## 規則與接續
- `普通形（な形容詞・名詞去だ） + なら`

## 用法說明
承接對方話題提出建議或意見，「如果是...的話」。與 [[JP_Grammar_50_C22_tara_conditional|〜たら @related_to]] 等一般條件句不同，「〜なら」強烈著重於「針對某個已知前提或對方剛提到的話題給予回饋」，前項不表示時間順序，可用於已發生或既定的事實。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#3498DB"><ruby>東京<rt>とうきょう</rt></ruby>へ<ruby>旅行<rt>りょこう</rt></ruby>に<ruby>行<rt>い</rt></ruby>くなら、</span><span style="color:#FF477E"><ruby>春<rt>はる</rt></ruby>の<ruby>桜<rt>さくら</rt></ruby>の<ruby>季節<rt>きせつ</rt></ruby>が</span><span style="color:#3498DB"><ruby>一番<rt>いちばん</rt></ruby>おすすめですよ。</span>
  中文：如果要去東京旅行的話，我最推薦春天櫻花的季節喔。
- 日文：<span style="color:#3498DB">そのパソコンを<ruby>捨<rt>す</rt></ruby>てるなら、</span><span style="color:#2ECC71">私に</span><span style="color:#3498DB"><ruby>譲<rt>ゆず</rt></ruby>ってくれませんか。</span>
  中文：如果你要丟掉那台電腦的話，可以讓給我嗎？

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 東京（とうきょう）へ旅行（りょこう）に行（い）くなら、春（はる）の桜（さくら）の季節（きせつ）が一番（いちばん）おすすめですよ。
> （如果要去東京旅行的話，我最推薦春天櫻花的季節喔。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 京都（きょうと） | **京都へ行くなら**、秋の紅葉（こうよう）がいいですよ。 | 如果要去京都，秋天的紅葉不錯喔。 |
| 車を買う（くるまをかう） | **車を買うなら**、この店がおすすめです。 | 如果要買車，我推薦這家店。 |
| 日本語を勉強する（べんきょうする） | **日本語を勉強するなら**、毎日聞くのがいいです。 | 如果要學日文，每天聽最好。 |
| 引っ越す（ひっこす） | **引っ越すなら**、駅の近くがいいですよ。 | 如果要搬家，車站附近比較好喔。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 換用たら（前提性較弱） | 東京へ行っ**たら**、桜を見てください。 | 如果去了東京，請看看櫻花。 |
| → 否定形 | 東京へ**行かないなら**、この本はいりません。 | 如果不去東京的話，就不需要這本書了。 |
| → 疑問形 | 東京へ行く**なら、いつがいいですか**。 | 如果要去東京的話，什麼時候好呢？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 東京へ旅行に行きたいんですが、いつがいいですか。 | 行くなら、春の桜の季節が一番おすすめですよ。 | 如果要去的話，我最推薦春天櫻花季喔。 |
| このパソコン、もう使わないんです。 | 捨てるなら、私に譲ってくれませんか。 | 如果要丟掉的話，可以讓給我嗎？ |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 行くなら、おすすめですよ。 | 如果要去的話，我推薦。 |
| +對象 | 東京へ旅行に行くなら、**春の桜の季節が**おすすめですよ。 | 如果要去東京旅行，我推薦春天櫻花季。 |
| +程度 | 東京へ旅行に行くなら、春の桜の季節が**一番**おすすめですよ。 | 如果要去東京旅行，我最推薦春天櫻花季。 |
