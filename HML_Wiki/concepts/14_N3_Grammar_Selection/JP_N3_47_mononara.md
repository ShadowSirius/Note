# 〜ものなら

## 規則與接續
- `動詞可能形 + ものなら`
- `できるものなら + 〜`
- 常用於表達「如果真的做得到就試試看」的挑釁、反諷或警告語氣
- 前句多半是假設「幾乎不可能實現」的情況，後句常接命令、意志、感嘆、警告等表達

## 用法說明
「〜ものなら」表示說話者假設某件事**如果辦得到**，就想去做，或反過來用來對對方發出挑釁與警告。語感比單純的 `〜たら` 更強，帶有「有本事就試試看」的態度。  
相近的 `〜ようものなら` 也常表警告，但 `〜ものなら` 更常見於「如果能…的話，我也想…／你要是能…就來啊」這種帶情緒的說法。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 行けるものなら行ってみたい。
> （如果能去的話，我真想去看看。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 空を飛べる | 空を飛べるものなら、飛んでみたい。 | 如果能飛上天空的話，我真想飛看看。 |
| 一人で直せる | 一人で直せるものなら、直してみなさい。 | 如果你真能一個人修好，就修修看吧。 |
| 本当に会える | 本当に会えるものなら、会ってみたい。 | 如果真的能見面的話，我想見一面。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 挑釁語氣 | その問題が解けるものなら、解いてみろ。 | 那道題如果你能解，就解來看看啊。 |
| → 願望表達 | 海に住めるものなら、住んでみたい。 | 如果能住在海裡的話，我真想住看看。 |
| → 警告語氣 | その薬を勝手に飲めるものなら、飲んでみなさい。 | 那個藥如果你敢亂吃，就吃看看吧。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| その荷物、持てますか。 | 持てるものなら、持ってみてください。 | 那個行李，你拿得動嗎？如果拿得動就試試看吧。 |
| この難問、解けますか。 | 解けるものなら、解いてみたいです。 | 這道難題，你解得了嗎？如果能解，我想試試。 |
| その扉、開けられますか。 | 開けられるものなら、開けてみろ。 | 那扇門你開得了嗎？有本事就打開啊。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 行けるものなら。 | 如果能去的話。 |
| 擴展 1 | 行けるものなら、行ってみたい。 | 如果能去的話，我想去看看。 |
| 擴展 2 | 今すぐ行けるものなら、行ってみたい。 | 如果現在立刻能去的話，我很想去看看。 |
