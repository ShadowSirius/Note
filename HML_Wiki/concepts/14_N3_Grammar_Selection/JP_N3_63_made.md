# 〜まで

## 規則與接續
- `名詞 + まで`
- `時間名詞 + まで`
- `場所名詞 + まで`
- `A から B まで`（表示範圍「從 A 到 B」）
- `名詞 + まで` 也可表示「甚至連……都」，帶有出乎意料、加強語氣的感覺。

## 用法說明
`〜まで` 的核心意思是「到……為止／一直到……」，可用來表示時間、場所、數量或範圍的終點。  
和 `〜までに` 相比，`〜まで` 只表示「持續到某個點」，`〜までに` 則強調「在某個期限之前完成」；另外，`〜まで` 也常用來表示「甚至連……都」，語氣比 `〜さえ`、`〜でも` 更偏向範圍延伸與意外感。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 電車(でんしゃ)は深夜(しんや)まで走(はし)っています。
> （這班電車一直運行到深夜。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 午後11時(ごごじゅういちじ) | 電車(でんしゃ)は午後11時(ごごじゅういちじ)まで走(はし)っています。 | 電車一直開到晚上11點。 |
| 駅前(えきまえ) | バスは駅前(えきまえ)まで行(い)きます。 | 公車會到車站前。 |
| 子ども | この映画(えいが)は子どもから大人(おとな)まで楽し(たの)しめます。 | 這部電影從小孩到大人都能享受。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 持續到 8 點 | 8時(はちじ)まで続(つづ)きます。 | 持續到8點。 |
| → 從小孩到大人都能進入 | 子どもから大人(おとな)まで入(はい)れます。 | 從小孩到大人都能進入。 |
| → 直到車站前 | 駅前(えきまえ)まで行(い)きます。 | 直到車站前。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| この店は何時(なんじ)まで開(あ)いていますか。 | 午後10時(ごごじゅうじ)まで開(あ)いています。 | 開到晚上10點。 |
| 誰(だれ)が参加(さんか)できますか。 | 子どもから大人(おとな)まで参加(さんか)できます。 | 從小孩到大人都能參加。 |
| 何(なに)をくれたんですか。 | 花(はな)だけでなく、ケーキまでくれました。 | 不只送花，甚至連蛋糕都送了。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 開(あ)いています。 | 開著。 |
| 擴展 1 | 午後10時(ごごじゅうじ)まで開(あ)いています。 | 開到晚上10點。 |
| 擴展 2 | この店(みせ)は午後10時(ごごじゅうじ)まで開(あ)いています。 | 這間店開到晚上10點。 |
| 擴展 3 | この店(みせ)は週末(しゅうまつ)まで午後10時(ごごじゅうじ)まで開(あ)いています。 | 這間店連週末都開到晚上10點。 |
