# 〜末に

## 規則與接續
- `動詞た形 + 末に`
- `名詞 + の末に`
- 常用於表示「經過長時間思考、討論、努力之後，最後做出某個決定或得到某個結果」。

## 用法說明
表示「經過...之後，最終...」，重點在於前面有一段反覆思考、猶豫、協商或努力的過程，之後才出現最後的決定或結果。與 `〜結果` 相比，`〜末に` 更強調前面的過程；與 `〜あげく` 相比，`〜末に` 較中性，`〜あげく` 多半帶有徒勞或不理想的結果。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼女（かのじょ）は何度（なんど）も迷（まよ）った末（すえ）に、留学（りゅうがく）することにした。
> （她再三猶豫後，最終決定去留學。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 転職（てんしょく）することにした | 彼女は何度も迷った末に、転職することにした。 | 她再三猶豫後，最終決定轉職。 |
| 日本（にほん）へ帰（かえ）ることにした | 彼女は何度も迷った末に、日本へ帰ることにした。 | 她再三猶豫後，最終決定回日本。 |
| 研究（けんきゅう）を続（つづ）けることにした | 彼女は何度も迷った末に、研究を続けることにした。 | 她再三猶豫後，最終決定繼續研究。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 使「〜末に」表達 | 何度も考えて、彼女は留学することにした。 → 彼女は何度も考えた末に、留学することにした。 | 經過反覆思考後，她最終決定去留學。 |
| → 使「〜末に」表達 | みんなで相談して、私たちは計画を変更した。 → 私たちはみんなで相談した末に、計画を変更した。 | 大家討論之後，我們最終修改了計畫。 |
| → 使「〜末に」表達 | 長く話し合って、彼らは結論を出した。 → 彼らは長く話し合った末に、結論を出した。 | 他們長時間討論後，最終得出了結論。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうして留学することにしたんですか。 | 何度も迷った末に、留学することにしたんです。 | 為什麼決定去留學？——因為再三猶豫後才決定的。 |
| その会議（かいぎ）の結果（けっか）は？ | 長い会議の末に、新しい方針が決まりました。 | 那個會議的結果呢？——經過長會議後，決定了新方針。 |
| その会社（かいしゃ）を辞（や）めた理由（りゆう）は？ | いろいろ悩んだ末に、辞めたんです。 | 你辭掉那間公司的理由是？——是苦惱之後才辭掉的。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 留学（りゅうがく）することにした。 | 決定去留學。 |
| 擴展 1 | 何度（なんど）も迷（まよ）った末（すえ）に、留学（りゅうがく）することにした。 | 再三猶豫後，最終決定去留學。 |
| 擴展 2 | 彼女（かのじょ）は何度（なんど）も迷（まよ）った末（すえ）に、日本（にほん）へ留学（りゅうがく）することにした。 | 她再三猶豫後，最終決定去日本留學。 |
