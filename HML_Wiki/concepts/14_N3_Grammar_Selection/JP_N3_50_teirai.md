# 〜て以来

## 規則與接續
- `動詞て形 + 以来（いらい）`
- 也可見於「名詞 + 以来（いらい）」：如 `卒業以来`、`先月以来`
- 表示「自從……之後，一直到現在持續著某種狀態／變化」

## 用法說明
「〜て以来」用來表示某個動作、事件發生之後，後面的狀態一直延續到現在，語感偏向「自從……以來就一直……」。常和「ずっと」「〜ていない」「変わった」這類持續性表達一起使用。

與「〜てから」相比，「〜てから」只是單純表示先後順序，不一定有「持續到現在」的意味；「〜て以来」則更強調從那個時點開始的持續狀態。若起點本身是名詞事件，也可以直接用「N以来」，如「卒業以来」。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 私（わたし）は日本（にほん）に来（き）て以来（いらい）、毎日（まいにち）日本語（にほんご）を勉強（べんきょう）しています。
> （自從來日本以來，我每天都在學日文。）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 東京（とうきょう）に引（ひ）っ越（こ）す | 東京（とうきょう）に引（ひ）っ越（こ）して以来（いらい）、毎日（まいにち）電車（でんしゃ）で通勤（つうきん）しています。 | 自從搬到東京以來，我每天都搭電車通勤。 |
| 会社（かいしゃ）を変（か）える | 会社（かいしゃ）を変（か）えて以来（いらい）、仕事（しごと）が忙（いそが）しくなりました。 | 自從換公司以來，工作變忙了。 |
| 子（こ）どもが生（う）まれる | 子（こ）どもが生（う）まれて以来（いらい）、早（はや）く帰（かえ）るようになりました。 | 自從孩子出生以來，我變得會早點回家。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → `〜てから` 改成 `〜て以来` | 日本（にほん）に来（き）て以来（いらい）、ずっと日本語（にほんご）を勉強（べんきょう）しています。 | 自從來日本以來，我一直在學日文。 |
| → `〜たあと` 改成 `〜て以来` | 卒業（そつぎょう）して以来（いらい）、先生（せんせい）に会（あ）っていません。 | 自從畢業以來，就沒有見過老師。 |
| → `〜たあと` 改成 `〜て以来` | 結婚（けっこん）して以来（いらい）、生活（せいかつ）が変（か）わりました。 | 自從結婚以來，生活改變了。 |
