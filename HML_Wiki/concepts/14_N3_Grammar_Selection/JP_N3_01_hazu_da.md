# 〜はずだ

## 規則與接續
- `動詞/い形容詞 普通形 + はずだ`
- `な形容詞語幹 + な + はずだ`
- `名詞 + の + はずだ`

## 用法說明
表示基於客觀事實或邏輯推導出的強烈確信「理應、應該會」。與偏主觀猜測的「〜だろう」不同，「〜はずだ」帶有充分的推理根據；也和單純表推測的 [[JP_Grammar_50_C33_conjecture_deshou_kamoshirenai|でしょう/かもしれない @related_to]] 語氣強度不同，はずだ更接近「照道理應該如此」。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71">田中さんは</span><span style="color:#FF477E">昨日からハワイに旅行に行っているので、</span><span style="color:#FF477E">今日は</span><span style="color:#3498DB">会社にいないはずだ。</span>
  中文：田中先生從昨天就去<ruby>夏威夷<rt>ハワイ</rt></ruby>旅行了，今天應該不在公司。
- 日文：<span style="color:#2ECC71">このレストランは</span><span style="color:#FF477E">いつも人が<ruby>並<rt>なら</rt></ruby>んでいるから、</span><span style="color:#3498DB"><ruby>絶対<rt>ぜったい</rt></ruby>においしいはずだ。</span>
  中文：這家餐廳總是大排長龍，絕對應該很好吃。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 田中（たなか）さんは今日（きょう）は会社（かいしゃ）にいないはずだ。
> （田中先生今天應該不在公司。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| もう帰った（かえった） | 田中さんはもう**帰ったはず**だ。 | 田中先生應該已經回去了。 |
| 知っている（しっている） | 田中さんは**知っているはず**だ。 | 田中先生應該知道。 |
| 元気だ（げんきだ） | 田中さんは**元気なはず**だ。 | 田中先生應該很好。 |
| 学生だ（がくせいだ） | 田中さんは**学生のはず**だ。 | 田中先生應該是學生。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 田中さんは会社にいない**はずだった**。 | 田中先生當時應該不在公司。 |
| → 否定形（はずがない） | 田中さんが休むはずが**ない**。 | 田中先生不可能請假。 |
| → 疑問形 | 田中さんは会社にいない**はずですか**。 | 田中先生應該不在公司嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 田中さんは会社にいますか。 | いいえ、旅行中なのでいないはずです。 | 不，因為在旅行，應該不在。 |
| このレストランはおいしいですか。 | 人がいつも並んでいるから、おいしいはずです。 | 因為總是大排長龍，應該很好吃。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | いないはずだ。 | 應該不在。 |
| +主語 | 田中さんは**いないはずだ**。 | 田中先生應該不在。 |
| +理由 | 旅行に行っているので、**田中さんはいないはずだ**。 | 因為在旅行，田中先生應該不在。 |
| +時間 | **今日は**旅行に行っているので、田中さんはいないはずだ。 | 因為今天在旅行，田中先生應該不在。 |
