# 〜ことなく

## 規則與接續
- `動詞原形（辞書形） + ことなく`
- 常見於書面語，表示「不...地、沒有...就...」
- 也常見固定搭配：`一度も〜ことなく`、`休むことなく`、`会うことなく`

## 用法說明
「〜ことなく」表示在沒有做前項動作的情況下，就進行後項動作，語氣較正式、帶有書面感。  
和「〜ずに／〜ないで」意思相近，但「〜ことなく」更常出現在文章、演說、敘述文；日常會話通常較少用。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）は休（やす）むことなく働（はたら）いた。
> （他不休息地工作了。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 寝る（ねる） | 彼は寝ることなく働いた。 | 他沒有睡覺就工作了。 |
| 休む（やすむ） | 彼は休むことなく働いた。 | 他不休息地工作了。 |
| 食事をする（しょくじをする） | 彼は食事をすることなく働いた。 | 他沒吃飯就工作了。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 〜ずに | 彼は眠らずに働いた。 → 彼は眠ることなく働いた。 | 他沒睡就工作了。 |
| → 〜ないで | 先生に会わないで卒業した。 → 先生に会うことなく卒業した。 | 沒有見到老師就畢業了。 |
| → 〜ずに | 一度も泣かずに話した。 → 一度も泣くことなく話した。 | 一次都沒哭就說了話。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうして彼はそんなに疲（つか）れているんですか。 | 毎日休むことなく働いているからです。 | 因為他每天都不停歇地工作。 |
| その二人（ふたり）は会（あ）えましたか。 | いいえ、一度も会うことなく別れました。 | 沒有，兩人一次都沒見到就分開了。 |
| 彼女は泣（な）かなかったんですか。 | はい、泣くことなく事実（じじつ）を話しました。 | 是的，她沒哭就說出了事實。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 休むことなく働いた。 | 不休息地工作了。 |
| 擴展1 | 彼は休むことなく働いた。 | 他不休息地工作了。 |
| 擴展2 | 彼は毎日休むことなく働いた。 | 他每天都不停歇地工作。 |
| 擴展3 | 彼は毎日朝から夜まで休むことなく働いた。 | 他每天從早到晚都不停歇地工作。 |
