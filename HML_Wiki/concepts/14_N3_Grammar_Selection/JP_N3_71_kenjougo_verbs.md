# 敬語動詞（謙讓語）参る／伺う／申す／〜ておる

## 規則與接續
- `行く／来る → 参る・参ります`
- `訪問する／聞く／尋ねる → 伺う・伺います`
- `言う／名乗る → 申す・申します`
- `〜ている → 〜ておる・〜ております`（現代口語與商務場合多用 `〜ております`）
- 以上都屬於「謙讓語」：把**自己**或**自己一方**的動作說得更低，以抬高對方。

## 用法說明
這一組敬語動詞用來降低說話者的身分，常見於商務、服務業、自我介紹、向上對象說話時；其中 `参る` 是 `行く／来る` 的謙讓語，`伺う` 可表示「拜訪、請教、詢問」，`申す` 常用於「自我介紹、說、稱作」。和 [[JP_N3_70_sonkeigo_verbs|尊敬語 @related_to]] 不同，謙讓語不是把對方抬高，而是把**自己的動作**放低；另外，`〜ておる` 是 `〜ている` 的謙讓／鄭重表達，實際會話中通常更常見 `〜ております`。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 私（わたし）は明日（あした）、先生（せんせい）のお宅（たく）へ伺（うかが）います。
> （我明天會去老師家拜訪。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 明日（あした）、社長（しゃちょう）のところへ | 明日（あした）、社長（しゃちょう）のところへ伺（うかが）います。 | 我明天會去拜訪社長。 |
| 午後三時（ごごさんじ）に、資料（しりょう）について | 午後三時（ごごさんじ）に、資料（しりょう）について伺（うかが）います。 | 我會在下午三點請教關於資料的事。 |
| 来週（らいしゅう）、大阪（おおさか）へ | 来週（らいしゅう）、大阪（おおさか）へ伺（うかが）います。 | 我下週會去大阪。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 行（い）きます → | 参（まい）ります。 | 我去／前往。 |
| 言（い）います → | 申（もう）します。 | 我說／我叫作。 |
| しています → | しております。 | 我正在做／我正持續……。 |
| 聞（き）きます → | 伺（うかが）います。 | 我請教／詢問。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 自分（じぶん）の名前（なまえ）を丁寧（ていねい）に言（い）うときは？ | 山田（やまだ）と申（もう）します。 | 我叫山田。 |
| 相手（あいて）に訪問（ほうもん）を伝（つた）えるときは？ | 明日（あした）伺（うかが）います。 | 我明天會去拜訪。 |
| 今（いま）していることを丁寧（ていねい）に言（い）うと？ | ただいま準備（じゅんび）しております。 | 我現在正在準備。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 伺（うかが）います。 | 我會去拜訪／請教。 |
| 擴張1 | 明日（あした）伺（うかが）います。 | 我明天去拜訪。 |
| 擴張2 | 明日（あした）、先生（せんせい）のお宅（たく）へ伺（うかが）います。 | 我明天去老師家拜訪。 |
