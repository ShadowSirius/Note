# 〜にしては

## 規則與接續
- `名詞 + にしては`
- `動詞・い形容詞普通形 + にしては`
- 常接在表示「身分、年齡、程度、條件」等名詞或普通形後面，表示「以…來說卻…、按…標準看有點不合預期」。

## 用法說明
「〜にしては」用來表示：以某個標準、身份、年齡、條件來看，後面的結果和一般預期不太一樣，帶有「沒想到、算是意外」的語感。  
和 [[JP_N3_33_wari_ni|〜わりに（は）@related_to]] 很接近，但「〜にしては」更常強調「以這個條件來說卻出乎意料」；「〜わりに(は)」則較偏向單純比較程度。

  中文：他以外表來說算是年輕。

- 日文：<span style="color:#FF477E">この<ruby>値段<rt>ねだん</rt></ruby>にしては、</span><span style="color:#3498DB"><ruby>味<rt>あじ</rt></ruby>がいいです。</span>
  中文：以這個價格來說，味道算是不錯。

- 日文：<span style="color:#2ECC71"><ruby>子供<rt>こども</rt></ruby>にしては</span><span style="color:#FF477E"><ruby>字<rt>じ</rt></ruby>が</span><span style="color:#E67E22">とても</span><span style="color:#3498DB">きれいです。</span>
  中文：以小孩子來說，字寫得非常漂亮。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）は見（み）た目（め）にしては若（わか）いです。
> （他以外表來說算是年輕。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| この店（みせ）は | この店（みせ）は値段（ねだん）にしては、おいしいです。 | 這家店以價格來說算是好吃。 |
| 子供（こども）は | 子供（こども）にしては、字（じ）がきれいです。 | 以小孩子來說，字寫得很漂亮。 |
| 日本人（にほんじん）は | 日本人（にほんじん）にしては、英語（えいご）が上手（じょうず）です。 | 以日本人來說，英文很不錯。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 使用「〜にしては」 | 子供（こども）ですが、字（じ）が上手（じょうず）です。 → 子供（こども）にしては、字（じ）が上手（じょうず）です。 | 以小孩子來說，字很拿手。 |
| → 使用「〜にしては」 | 新人（しんじん）ですが、仕事（しごと）が速（はや）いです。 → 新人（しんじん）にしては、仕事（しごと）が速（はや）いです。 | 以新人來說，工作很快。 |
| → 使用「〜にしては」 | このケーキは高（たか）いですが、小（ちい）さいです。 → このケーキは高（たか）いにしては、小（ちい）さいです。 | 這個蛋糕以價格來說算小。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| このレストラン、どうでしたか。 | 値段（ねだん）にしては、おいしかったです。 | 這家餐廳以價格來說算好吃。 |
| 彼（かれ）は若（わか）く見（み）えますね。 | 見（み）た目（め）にしては、若（わか）いです。 | 以外表來說，他算年輕。 |
| あの子（こ）は字（じ）が上手（じょうず）ですね。 | 子供（こども）にしては、字（じ）が上手（じょうず）ですね。 | 以那孩子來說，字很拿手。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | おいしいです。 | 很好吃。 |
| 條件 | この値段（ねだん）にしては | 以這個價格來說 |
| 完成 | この値段（ねだん）にしては、おいしいです。 | 以這個價格來說很好吃。 |
| 追加 | この値段（ねだん）にしては、おいしくて、量（りょう）も多（おお）いです。 | 以這個價格來說，不但好吃，份量也很多。 |
