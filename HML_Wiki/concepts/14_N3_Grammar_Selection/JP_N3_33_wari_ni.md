# 〜わりに（は）

## 規則與接續
- `動詞普通形 + わりに（は）`
- `い形容詞普通形 + わりに（は）`
- `な形容詞語幹 + な + わりに（は）`
- `名詞 + の + わりに（は）`

## 用法說明
表示「以...而言」，將前項的一般基準（年齡、價格、努力程度等）與後項的實際結果相比，指出兩者不成比例，結果比預期的好或壞都可以使用，語氣中立、偏客觀對比，不像 [[JP_N3_32_kuseni|〜くせに]] 那樣帶輕蔑責備情緒。

與 [[JP_N3_32_kuseni|〜くせに]]、[[JP_N3_39_noni|〜のに]] 的辨析：
- 「〜わりに」：中立的「比較不成比例」語感，可用於正面驚喜（例如便宜卻好吃）或負面落差，不帶道德評價。
- 「〜くせに」：帶強烈輕蔑、責備語氣，幾乎只用於負面評價他人。
- 「〜のに」：帶主觀不滿或驚訝情緒，但不特別強調「基準與結果的比例落差」，單純陳述與預期相反的結果。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> このレストランは安（やす）いわりに、料理（りょうり）がおいしい。
> （這家餐廳雖然便宜，菜卻很好吃，超乎預期。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 練習していない（れんしゅうしていない） | 練習していない**わりに**、上手だ。 | 明明沒怎麼練習，卻很厲害。 |
| 簡単だ（かんたんだ） | 簡単な**わりに**、効果がある。 | 雖然簡單，卻很有效果。 |
| 新人だ | 新人の**わりに**、仕事が早い。 | 雖然是新人，工作卻很快。 |
| 狭い（せまい） | 狭い**わりに**、収納が多い。 | 雖然空間小，收納卻很多。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 〜わりには（強調對比） | このレストランは安い**わりには**、料理がおいしい。 | 這家餐廳以便宜的程度來說，菜算是很好吃。 |
| → くせに（轉為負面輕蔑；くせに 多用於人，故換例） | 彼（かれ）は知（し）っている**くせに**、教（おし）えてくれない。 | 他明明知道，卻不告訴我。（帶責備語氣） |
| → 丁寧形 | このレストランは安いわりに、料理がおいし**いです**。 | 這家餐廳雖然便宜，菜卻很好吃。（禮貌形） |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| このレストラン、安いのにおいしいですね。 | ええ、安いわりに料理がおいしいですよね。 | 是啊，雖然便宜但菜很好吃呢。 |
| 彼、若く見えますね。 | ええ、年齢のわりに若く見えますね。 | 是啊，就年齡來說算年輕的。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 料理がおいしい。 | 菜很好吃。 |
| +主題 | このレストランは料理がおいしい。 | 這家餐廳菜很好吃。 |
| +對比基準 | このレストランは**安いわりに**、料理がおいしい。 | 這家餐廳雖然便宜，菜卻很好吃。 |
