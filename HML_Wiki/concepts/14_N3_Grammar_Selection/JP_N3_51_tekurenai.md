# 〜てくれない

## 規則與接續
- `動詞て形 + くれない？`
- `動詞て形 + くれないかな`
- `動詞て形 + くれない？` は口語的な請求表現で，「可以幫我……嗎」「能不能……」的語感
- 比 `〜てください` 更隨口、親近；比直接命令更委婉
- `〜てもらえない？` 也可表達請求，但 `〜てくれない？` 更常用於熟人之間，口氣更直接自然

## 用法說明
`〜てくれない？` 是把「〜てくれる」改成否定疑問的口語請求句，表面上是「不幫我嗎？」實際上多半是委婉地拜託對方做某事。常用於朋友、家人、同事等熟悉關係中；若要更正式，則可用 `〜てくれませんか` 或 `〜てもらえませんか`。

與 `〜てください` 相比，`〜てくれない？` 更口語、親近；與 [[JP_N3_22_temorau|〜てもらう @related_to]] 系統的 `〜てもらえない？` 相比，前者更直接，後者則稍微保留距離、也更禮貌。也可視為 [[JP_N3_23_tekureru|〜てくれる @related_to]] 的口語疑問形。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> この荷物（にもつ）を持（も）ってくれない？
> （可以幫我拿這個行李嗎？）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| この箱（はこ） | この箱（はこ）を持（も）ってくれない？ | 可以幫我拿這個箱子嗎？ |
| ドア（どあ） | ドア（どあ）を開（あ）けてくれない？ | 可以幫我開門嗎？ |
| 先生（せんせい）に連絡（れんらく）する | 先生（せんせい）に連絡（れんらく）してくれない？ | 可以幫我聯絡老師嗎？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 更口語的請求 | この荷物（にもつ）を持（も）ってくれない？ | 可以幫我拿這個行李嗎？ |
| → 改成更正式的說法 | この荷物（にもつ）を持（も）ってくれませんか。 | 可以幫我拿這個行李嗎？ |
| → 改成更委婉的說法 | この荷物（にもつ）を持（も）ってもらえない？ | 可以請你幫我拿這個行李嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| すみません、これを見（み）てくれない？ | いいよ。どこを見（み）ればいい？ | 可以啊。你要我看哪裡？ |
| この資料（しりょう）をコピーしてくれない？ | わかった。今（いま）すぐやるよ。 | 好的，我現在就做。 |
| ちょっと写真（しゃしん）を撮（と）ってくれない？ | うん、いいよ。 | 嗯，可以。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 持（も）ってくれない？ | 可以幫我拿嗎？ |
| 擴展 1 | この荷物（にもつ）を持（も）ってくれない？ | 可以幫我拿這個行李嗎？ |
| 擴展 2 | ちょっと、この荷物（にもつ）を持（も）ってくれない？ | 可以幫我拿一下這個行李嗎？ |
| 擴展 3 | ちょっと、駅（えき）までこの荷物（にもつ）を持（も）ってくれない？ | 可以幫我把這個行李拿到車站嗎？ |
