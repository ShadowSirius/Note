# 〜のに（逆接）

## 規則與接續
- `用言普通形 + のに`
- `な形容詞語幹 + な + のに`
- `名詞 + な + のに`

## 用法說明
表示與預期相反的結果，帶有不滿、遺憾或驚訝等強烈主觀情緒，是本組逆接句型中口語使用頻率最高、情緒最直接的一種。

與 [[JP_N3_31_mono_no|〜ものの]]、[[JP_N3_38_nagara_mo|〜ながらも]]、[[JP_N3_32_kuseni|〜くせに]] 的辨析：
- 「〜のに」：主觀情緒最直接，可用於任何人稱，語氣可從單純遺憾到強烈不滿都適用，口語常用。
- 「〜ものの」：陳述前後矛盾的事實，語氣客觀、書面，情緒色彩淡。
- 「〜ながらも」：強調兩件事「同時並存」的矛盾，書面語感，情緒比「のに」溫和。
- 「〜くせに」：比「のに」情緒更強烈、更帶輕蔑責備，且幾乎只用於評價他人（第二、三人稱）。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 一生懸命（いっしょうけんめい）勉強（べんきょう）したのに、試験（しけん）に落（お）ちた。
> （明明拚命讀書了，卻沒考過。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 頑張った（がんばった） | 頑張った**のに**、結果が出なかった。 | 明明努力了，卻沒有結果。 |
| 約束した（やくそくした） | 約束した**のに**、彼は来なかった。 | 明明約好了，他卻沒來。 |
| 暇だ（ひまだ） | 今日は暇な**のに**、誰も誘ってくれない。 | 今天明明有空，卻沒人約我。 |
| 休日だ | 今日は休日な**のに**、仕事をしている。 | 今天明明是假日，卻在工作。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → ものの（語氣趨於客觀） | 一生懸命勉強した**ものの**、試験に落ちた。 | 雖然拚命讀書了，但沒考過。（較客觀） |
| → くせに（用於責備他人時） | 勉強していない**くせに**、いい点を取った。 | 明明沒讀書，卻拿了好分數。（帶輕微酸諷語氣） |
| → 丁寧形 | 一生懸命勉強したのに、試験に落ち**ました**。 | 明明拚命讀書了，卻沒考過。（禮貌形） |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 試験、どうでしたか。 | 一生懸命勉強したのに、落ちてしまいました。 | 明明拚命讀書了，卻還是沒考過。 |
| 今日、寒いですね。 | ええ、もう春なのに、まだ寒いですね。 | 是啊，明明已經春天了，卻還是很冷。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 試験に落ちた。 | 沒考過。 |
| +逆接理由 | **一生懸命勉強したのに、**試験に落ちた。 | 明明拚命讀書了，卻沒考過。 |
| +情緒補述 | 一生懸命勉強したのに、試験に落ちて**悔しい**。 | 明明拚命讀書了，卻沒考過，真不甘心。 |
