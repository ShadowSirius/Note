# 〜たばかりだ

## 規則與接續
- `動詞た形 + ばかりだ`

## 用法說明
表示說話者「心理上」覺得動作才剛發生不久，即使客觀時間已經過了數個月甚至更久，只要主觀感覺短暫就能使用。與強調客觀絕對短暫的 [[JP_N3_18_ta_tokoro_da|〜たところだ @related_to]] 不同，它不限於動作結束的當下瞬間。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71">このパソコンは</span><span style="color:#FF477E"><ruby>先月<rt>せんげつ</rt></ruby></span><span style="color:#3498DB"><ruby>買<rt>か</rt></ruby>ったばかりなのに、</span><span style="color:#3498DB">もう<ruby>壊<rt>こわ</rt></ruby>れました。</span>
  中文：這台筆電上個月才剛買，卻已經壞了。
- 日文：<span style="color:#FF477E"><ruby>日本<rt>にほん</rt></ruby>に<ruby>来<rt>き</rt></ruby>たばかりなので、</span><span style="color:#3498DB">まだ<ruby>生活<rt>せいかつ</rt></ruby>に<ruby>慣<rt>な</rt></ruby>れていません。</span>
  中文：因為剛來日本不久，所以還不習慣這裡的生活。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> このパソコンは先月（せんげつ）買（か）ったばかりなのに、もう壊（こわ）れました。
> （這台筆電上個月才剛買，卻已經壞了。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 結婚する（けっこんする） | 二人は**結婚したばかり**です。 | 兩人才剛結婚。 |
| 入社する（にゅうしゃする） | 彼は**入社したばかり**なので、まだ慣れていない。 | 他才剛進公司，還不習慣。 |
| 引っ越す（ひっこす） | **引っ越したばかり**で、荷物がまだ片付いていない。 | 才剛搬家，行李還沒整理好。 |
| 生まれる（うまれる） | この犬（いぬ）は**生まれたばかり**です。 | 這隻狗才剛出生。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 名詞修飾 | 買った**ばかりの**パソコンが壊れた。 | 剛買的筆電壞了。 |
| → 過去語境 | あの頃（ころ）、日本に来た**ばかりだった**。 | 那時候剛來日本不久。 |
| → 疑問形 | もう壊れた**んですか**。買ったばかりなのに。 | 已經壞了嗎？明明才剛買。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| このパソコン、古いんですか。 | いいえ、先月買ったばかりなのに、もう壊れました。 | 不，上個月才剛買，卻已經壞了。 |
| 日本の生活には慣れましたか。 | まだです。日本に来たばかりなので。 | 還沒，因為剛來日本不久。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 買ったばかりなのに壊れました。 | 才剛買就壞了。 |
| +主語 | **このパソコンは**先月買ったばかりなのに、壊れました。 | 這台筆電上個月才剛買，卻壞了。 |
| +時間 | このパソコンは**先月**買ったばかりなのに、もう壊れました。 | 這台筆電上個月才剛買，現在就壞了。 |
