# 〜ことにする

## 規則與接續
- `動詞辞書形/ない形 + ことにする`

## 用法說明
表示說話者經過思考後，靠自己主觀意志所下達的決定或決心。與 [[JP_N3_03_koto_ni_naru|客觀決定的〜ことになる @related_to]] 相對；若表示已成習慣的持續決定，常用「〜ことにしている」。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E"><ruby>夜<rt>よる</rt></ruby>8<ruby>時<rt>じ</rt></ruby><ruby>以降<rt>いこう</rt></ruby>は</span><span style="color:#3498DB"><ruby>何<rt>なに</rt></ruby>も<ruby>食<rt>た</rt></ruby>べないことにしました。</span>
  中文：因為正在減肥，我決定晚上8點以後什麼都不吃。
- 日文：<span style="color:#FF477E"><ruby>今年<rt>ことし</rt></ruby>の<ruby>夏休<rt>なつやす</rt></ruby>みは、</span><span style="color:#2ECC71"><ruby>家族<rt>かぞく</rt></ruby>と<ruby>一緒<rt>いっしょ</rt></ruby>に</span><span style="color:#3498DB"><ruby>北海道<rt>ほっかいどう</rt></ruby>へ<ruby>行<rt>い</rt></ruby>くことにした。</span>
  中文：今年的暑假，我決定要和家人一起去北海道。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> ダイエット中（ちゅう）なので、夜（よる）8時（じ）以降（いこう）は何（なに）も食（た）べないことにしました。
> （因為正在減肥，我決定晚上8點以後什麼都不吃。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 毎朝走る（まいあさはしる） | **毎朝走ることに**しました。 | 決定每天早上跑步。 |
| タバコをやめる | **タバコをやめることに**しました。 | 決定戒菸了。 |
| お酒を飲まない（のまない） | **お酒を飲まないことに**しました。 | 決定不喝酒了。 |
| 早く寝る（はやくねる） | **早く寝ることに**しました。 | 決定早點睡覺。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 習慣表現（ています） | 夜は何も食べない**ことにしています**。 | 一直以來都習慣晚上不吃東西。 |
| → 否定形 | 食べない**ことにはしません**でした。 | 結果沒有決定不吃。 |
| → 疑問形 | 何も食べない**ことにしました**か。 | 決定什麼都不吃了嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうして夜食べないんですか。 | ダイエット中なので、食べないことにしました。 | 因為在減肥，決定不吃了。 |
| 夏休みはどこへ行きますか。 | 家族と北海道へ行くことにしました。 | 決定和家人去北海道。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 食べないことにしました。 | 決定不吃了。 |
| +時間 | **夜8時以降は**食べないことにしました。 | 決定晚上8點以後不吃了。 |
| +理由 | ダイエット中なので、**夜8時以降は何も食べないことにしました**。 | 因為在減肥，決定晚上8點後什麼都不吃。 |
