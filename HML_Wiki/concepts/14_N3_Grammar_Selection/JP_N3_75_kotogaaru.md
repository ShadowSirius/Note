# 〜ことがある（経験）

## 規則與接續
- `動詞た形 + ことがある`
- `動詞た形 + ことがない`（否定：從來沒有...過）

## 用法說明
表示「曾經做過某事／有過某經驗」，常和「一度、以前、子どものころに」等時間詞一起出現。  
要注意不要和「動詞辞書形 + ことがある」混淆；後者是「有時候、偶爾會...」，而這裡一定是「た形」，重點在經驗。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71">私は</span><span style="color:#E67E22">一度</span><span style="color:#3498DB"><ruby>北海道<rt>ほっかいどう</rt></ruby>へ行ったことがあります。</span>
  中文：我曾經去過北海道一次。

- 日文：<span style="color:#E67E22"><ruby>子供<rt>こども</rt></ruby>のころに</span><span style="color:#2ECC71">私は</span><span style="color:#3498DB"><ruby>海<rt>うみ</rt></ruby>で<ruby>泳<rt>およ</rt></ruby>いだことがあります。</span>
  中文：我小時候曾經在海裡游泳過。

- 日文：<span style="color:#2ECC71">私は</span><span style="color:#E67E22"><ruby>以前<rt>いぜん</rt></ruby></span><span style="color:#3498DB">この<ruby>店<rt>みせ</rt></ruby>のラーメンを<ruby>食<rt>た</rt></ruby>べたことがあります。</span>
  中文：我以前曾經吃過這家店的拉麵。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 私（わたし）は北海道（ほっかいどう）へ行（い）ったことがあります。
> （我曾經去過北海道。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 韓国（かんこく）へ行（い）った | 私は韓国へ行ったことがあります。 | 我曾經去過韓國。 |
| 富士山（ふじさん）に登（のぼ）った | 私は富士山に登ったことがあります。 | 我曾經登過富士山。 |
| この映画（えいが）を見（み）た | 私はこの映画を見たことがあります。 | 我曾經看過這部電影。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 經驗表現 | 彼（かれ）は<ruby>寿司<rt>すし</rt></ruby>を食（た）べました。 → 彼は寿司を食べたことがあります。 | 他曾經吃過壽司。 |
| → 經驗表現 | 彼女（かのじょ）は富士山（ふじさん）に登（のぼ）りました。 → 彼女は富士山に登ったことがあります。 | 她曾經登過富士山。 |
| → 否定形 | 私たちは京都（きょうと）へ行（い）きました。 → 私たちは京都へ行ったことがありません。 | 我們沒有去過京都。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 北海道（ほっかいどう）へ行ったことがありますか。 | はい、一度行ったことがあります。 | 你曾經去過北海道嗎？／是的，去過一次。 |
| <ruby>寿司<rt>すし</rt></ruby>を食べたことがありますか。 | はい、食べたことがあります。 | 你曾經吃過壽司嗎？／有，吃過。 |
| 富士山（ふじさん）に登ったことがありますか。 | いいえ、登ったことがありません。 | 你曾經登過富士山嗎？／不，沒有。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 行（い）ったことがあります。 | 曾經去過。 |
| 擴展 1 | 北海道（ほっかいどう）へ行ったことがあります。 | 曾經去過北海道。 |
| 擴展 2 | 私（わたし）は一度北海道へ行ったことがあります。 | 我曾經去過北海道一次。 |
