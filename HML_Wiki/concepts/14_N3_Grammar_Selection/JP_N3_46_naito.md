# 〜ないと（いけない）

## 規則與接續
- `動詞ない形 + と（いけない）`
- `動詞ない形 + と`

## 用法說明
表示「不……不行、必須……」的口語義務表現，前面接動詞ない形。完整說法是 `〜ないといけない`，日常會話中常把後半省略成 `〜ないと`，語氣比 `〜なければならない` 更口語、更自然。  
另外，`〜なくちゃ`／`〜なきゃ` 是更隨便的縮略說法，常見於非常口語的對話。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 明日（あした）までに宿題（しゅくだい）を終（お）わらないといけない。
> （我必須在明天之前把作業做完。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| レポートを出す | 今日中（きょうじゅう）にレポートを出（だ）さないといけない。 | 必須在今天之內交報告。 |
| 電話をする | もう遅（おそ）いから、電話（でんわ）をしないといけない。 | 因為已經晚了，必須打電話。 |
| 予約を取る | 来週（らいしゅう）までに予約（よやく）を取（と）らないといけない。 | 必須在下週之前訂位。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 「〜なければならない」に言い換える | 明日（あした）までに宿題（しゅくだい）を終（お）わらなければならない。 | 必須在明天之前把作業做完。 |
| → もっとくだけた形にする | 書類（しょるい）を出（だ）さなきゃ。 | 得交文件。 |
| → 「しないと」を使う | 電話（でんわ）をしないといけない。 | 必須打電話。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| もう帰（かえ）ってもいいですか。 | まだ会議（かいぎ）があるので、まだ帰（かえ）らないといけません。 | 還不行，因為還有會議，所以必須回去。 |
| この書類（しょるい）、今日中（きょうじゅう）ですか。 | はい、今日中（きょうじゅう）に出（だ）さないといけません。 | 是的，必須今天之內交。 |
| 今日は遊（あそ）べますか。 | すみません、宿題（しゅくだい）をしないといけないんです。 | 抱歉，我得做作業。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 行（い）かないといけない。 | 必須去。 |
| 場所を追加 | 会社（かいしゃ）に行（い）かないといけない。 | 必須去公司。 |
| 時間を追加 | 朝（あさ）8時（じ）までに会社（かいしゃ）に行（い）かないといけない。 | 必須在早上8點前去公司。 |
