# 〜てからでないと

## 規則與接續
- `動詞 て形 + からでないと`
- 常用於後句的否定表達：`〜ない / 〜できない / 〜れない`
- `〜てからでなければ`：意思相近，語氣較書面、正式

## 用法說明
表示「必須先完成前一個動作，後面的事情才可能成立」，語感就是「不先做…就不能…」。這個句型常用來表達條件、規則或限制，後面多接 `できない`、`入れない`、`使えない` 之類的否定句。

和 `〜ないと` 相比，`〜てからでないと` 更強調「先後順序」與「前提條件」；和 `〜てからでなければ` 意思相近，但 `〜てからでないと` 在口語與書面中都很常見。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 会議室（かいぎしつ）は予約（よやく）してからでないと、入（はい）れません。
> （會議室不先預約就不能進入。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| パスワードを変更（へんこう）する | パスワードを変更してからでないと、ログインできません。 | 不先更改密碼就不能登入。 |
| 係員（かかりいん）に確認（かくにん）する | 係員に確認してからでないと、会場（かいじょう）に入（はい）れません。 | 不先向工作人員確認就不能進會場。 |
| 説明書（せつめいしょ）を読む | 説明書を読んでからでないと、使えません。 | 不先讀說明書就不能使用。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| パスワードを変更（へんこう）する / ログインできる | パスワードを変更してからでないと、ログインできません。 | 不先更改密碼就不能登入。 |
| 予約（よやく）する / 会議室（かいぎしつ）を使（つか）う | 予約してからでないと、会議室を使えません。 | 不先預約就不能使用會議室。 |
| 先生（せんせい）に聞（き）く / 提出（ていしゅつ）する | 先生に聞いてからでないと、提出できません。 | 不先問老師就不能提交。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| この会議室（かいぎしつ）は、いつ使（つか）えますか。 | 予約してからでないと、使えません。 | 要先預約才能使用。 |
| このアプリはすぐ使えますか。 | 登録（とうろく）してからでないと、使えません。 | 不先註冊就不能使用。 |
| どうしたら会場（かいじょう）に入（はい）れますか。 | 受付（うけつけ）で名前（なまえ）を書（か）いてからでないと、入れません。 | 不先在櫃檯寫名字就不能進去。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 予約（よやく）してからでないと、入（はい）れません。 | 不先預約就不能進入。 |
| 擴展 1 | この会議室（かいぎしつ）は、予約してからでないと、入れません。 | 這個會議室不先預約就不能進入。 |
| 擴展 2 | この会議室は、前日（ぜんじつ）までに予約してからでないと、入れません。 | 這個會議室不先在前一天以前預約就不能進入。 |
