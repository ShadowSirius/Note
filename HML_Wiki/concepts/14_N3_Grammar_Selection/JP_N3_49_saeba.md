# 〜さえ〜ば

## 規則與接續
- `名詞 + さえ + あれば／いれば`
- `動詞て形 + さえ + いれば`
- `動詞ます形 + さえ + すれば`
- `い形容詞く + さえ + あれば`

## 用法說明
表示把條件壓縮到「最低限度」：只要具備那一點，後面的結果就能成立，常帶有「其他條件都不重要」的語感。  
和「〜だけで」相比，「〜さえ〜ば」更強調「這一點就足夠了」；和「〜たら」相比，重點不是時間上的「如果」，而是必要條件。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 時間(じかん)さえあれば、旅行(りょこう)に行(い)ける。
> （只要有時間，就能去旅行。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| お金(かね)がある | お金(かね)さえあれば、海外旅行(かいがいりょこう)に行(い)ける。 | 只要有錢，就能去海外旅行。 |
| 日本語(にほんご)ができる | 日本語(にほんご)さえできれば、この仕事(しごと)はできる。 | 只要會日文，就能做這份工作。 |
| 先生(せんせい)がいる | 先生(せんせい)さえいれば、質問(しつもん)できます。 | 只要有老師，就可以提問。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
