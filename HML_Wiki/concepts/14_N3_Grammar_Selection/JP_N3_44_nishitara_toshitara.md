# 〜にしたら／〜としたら

## 規則與接續
- `名詞 + にしたら`
- `動詞普通形 + としたら`
- `い形容詞普通形 + としたら`
- `な形容詞 + だとしたら／だったとしたら`
- `名詞 + だとしたら／だったとしたら`

## 用法說明
表示「如果站在某人的立場來看」或「假設某種情況成立的話」。  
- 「〜にしたら」多用來從某人的立場、感受、想法來說明，帶有主觀視角。  
- 「〜としたら」則是先假設某事為真，再推測結果或判斷。  
和「〜にとって」相比，「〜にしたら」更口語，也更容易帶出當事人的情緒；和「〜なら」相比，「〜としたら」更強調「假設真的如此」。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 親（おや）にしたら、子（こ）どもが元気（げんき）でいるのがいちばんうれしい。  
> （站在父母的立場來說，孩子平安健康最讓人開心。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 先生（せんせい） | 先生（せんせい）にしたら、学生（がくせい）がよく勉強（べんきょう）するのがいちばんうれしい。 | 站在老師的立場來說，學生好好念書最讓人開心。 |
| 店員（てんいん） | 店員（てんいん）にしたら、あいさつしてくれるお客（きゃく）さんはありがたい。 | 站在店員的立場來說，會打招呼的客人很令人感激。 |
| 親友（しんゆう） | 親友（しんゆう）にしたら、急（きゅう）に連絡（れんらく）が取（と）れなくなるのは心配（しんぱい）だ。 | 站在摯友的立場來說，突然聯絡不上會很擔心。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 立場を表す言い方 | 子どもは長（なが）い宿題（しゅくだい）がいやです。 → 子どもにしたら、長（なが）い宿題（しゅくだい）はいやです。 | 孩子會覺得，長長的作業很討厭。 |
| → 仮定表現 | 雨（あめ）が降（ふ）るなら、試合（しあい）は中止（ちゅうし）です。 → 雨（あめ）が降（ふ）るとしたら、試合（しあい）は中止（ちゅうし）です。 | 如果假設會下雨，比賽就中止。 |
| → 立場の表現に変える | 彼（かれ）の立場（たちば）では、断（ことわ）るのはむずかしいです。 → 彼（かれ）にしたら、断（ことわ）るのはむずかしいです。 | 站在他的立場來說，拒絕很難。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 子（こ）どもが夜（よる）おそく帰（かえ）りたがっています。 | 親（おや）にしたら、心配（しんぱい）です。 | 站在父母立場來說，會很擔心。 |
| 旅行（りょこう）の予定（よてい）が変（か）わるかもしれません。 | 変（か）わるとしたら、早（はや）めに連絡（れんらく）します。 | 如果行程要變，就會提早聯絡。 |
| 新人（しんじん）がこの仕事（しごと）をします。 | 新人（しんじん）にしたら、この仕事（しごと）はむずかしいです。 | 站在新人的立場來說，這份工作很難。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | もし行（い）けるとしたら、行（い）きます。 | 如果能去的話，我就去。 |
| 加上時間 | もし来週（らいしゅう）行（い）けるとしたら、行（い）きます。 | 如果下週能去的話，我就去。 |
| 加上具體行動 | もし来週（らいしゅう）の土曜日（どようび）に行（い）けるとしたら、朝（あさ）早（はや）く出発（しゅっぱつ）します。 | 如果下週六能去的話，我會一早出發。 |
