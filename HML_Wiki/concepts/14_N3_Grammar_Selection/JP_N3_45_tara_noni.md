# 〜たら〜のに

## 規則與接續
- `動詞た形 + ら + のに`
- `い形容詞：〜かったら + のに`
- `な形容詞・名詞：〜だったら + のに`

## 用法說明
表示「如果當時那樣做／那樣的情況成立就好了」，帶有與事實相反的遺憾、惋惜或後悔，常譯為「要是...就好了」。  
與「〜ばよかった」相比，兩者都能表達後悔；「〜たら〜のに」更強調對「如果前件成立，結果就會改變」的假設遺憾，「〜ばよかった」則更直接表示「當初應該做...」的自責。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 私(わたし)がもっと早(はや)く出発(しゅっぱつ)したら、会議(かいぎ)に間(ま)に合(あ)ったのに。
> （我要是更早出發，就能趕上會議了。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 傘(かさ)を持(も)つ | 傘(かさ)を持(も)っていたら、濡(ぬ)れなかったのに。 | 要是有帶傘，就不會淋濕了。 |
| もっと早(はや)く着(つ)く | もっと早(はや)く着(つ)いていたら、席(せき)が取(と)れたのに。 | 要是早點到，就能先佔到位子了。 |
| 連絡(れんらく)する | 先(さき)に連絡(れんらく)していたら、待(ま)たせずにすんだのに。 | 要是先聯絡，就不用讓對方等了。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 後悔(こうかい)を表(あらわ)す | もっと早(はや)く起(お)きていたら、会議(かいぎ)に間(ま)に合(あ)ったのに。 | 要是更早起床，就能趕上會議了。 |
| → 遺憾(いかん)を表(あらわ)す | 事前(じぜん)に知(し)っていたら、準備(じゅんび)できたのに。 | 要是事先知道，就能準備了。 |
| → もっと勉強(べんきょう)した場合 | もっと勉強(べんきょう)していたら、合格(ごうかく)できたのに。 | 要是更認真讀書，就能合格了。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうして来(こ)なかったの？ | もっと早(はや)く知(し)っていたら、行(い)けたのに。 | 要是早點知道，我就能去了。 |
| どうして傘(かさ)を持(も)ってこなかったの？ | 雨(あめ)が降(ふ)ると分(わ)かっていたら、持(も)ってきたのに。 | 要是知道會下雨，我就帶來了。 |
| どうしてその場(ば)で言(い)わなかったの？ | そのとき言(い)えたら、誤解(ごかい)されなかったのに。 | 要是當場能說，就不會被誤會了。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 間(ま)に合(あ)ったのに。 | 就能趕上了。 |
| 擴展 1 | もっと早(はや)く出発(しゅっぱつ)したら、間(ま)に合(あ)ったのに。 | 要是更早出發，就能趕上了。 |
| 擴展 2 | 私(わたし)がもっと早(はや)く出発(しゅっぱつ)したら、会議(かいぎ)に間(ま)に合(あ)ったのに。 | 我要是更早出發，就能趕上會議了。 |
