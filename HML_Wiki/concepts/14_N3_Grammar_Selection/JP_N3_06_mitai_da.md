# 〜みたいだ

## 規則與接續
- `名詞/動詞普通形/い形容詞/な形容詞語幹 + みたいだ`

## 用法說明
表主觀推測或比喻，語氣比 [[JP_N3_08_you_da|〜ようだ @related_to]] 更口語、隨意，常用於日常對話。與根據傳聞的 [[JP_N3_07_rashii|〜らしい @related_to]] 不同，みたいだ較偏個人直覺感受。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71">彼は</span><span style="color:#3498DB"><ruby>疲<rt>つか</rt></ruby>れているみたいだから、</span><span style="color:#FF477E"><ruby>少<rt>すこ</rt></ruby>し</span><span style="color:#3498DB"><ruby>休<rt>やす</rt></ruby>ませてあげよう。</span>
  中文：他看起來好像很累，讓他稍微休息一下吧。
- 日文：<span style="color:#FF477E"><ruby>今日<rt>きょう</rt></ruby>の</span><span style="color:#2ECC71"><ruby>空<rt>そら</rt></ruby>は</span><span style="color:#3498DB">まるで<ruby>絵<rt>え</rt></ruby>の<ruby>具<rt>ぐ</rt></ruby>で<ruby>描<rt>えが</rt></ruby>いたみたいだ。</span>
  中文：今天的天空簡直就像是用顏料畫出來的一樣。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）は疲（つか）れているみたいだから、少（すこ）し休（やす）ませてあげよう。
> （他看起來好像很累，讓他稍微休息一下吧。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 熱がある（ねつがある） | 彼は**熱があるみたいだ**。 | 他好像發燒了。 |
| 元気がない（げんきがない） | 彼は**元気がないみたいだ**。 | 他好像沒精神。 |
| 学生（がくせい） | あの人は**学生みたいだ**。 | 那個人好像是學生。 |
| 忙しい（いそがしい） | 彼女は**忙しいみたいだ**。 | 她好像很忙。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 彼は疲れている**みたいだった**。 | 他當時好像很累。 |
| → 疑問形 | 彼は疲れている**みたいですか**。 | 他好像很累嗎？ |
| → 名詞修飾 | 疲れている**みたいな**顔（かお）をしている。 | 露出一副好像很累的表情。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 彼はどうしたんですか。 | 疲れているみたいだから、休ませてあげよう。 | 他好像很累，讓他休息吧。 |
| 今日の空はきれいですか。 | はい、絵の具で描いたみたいですよ。 | 是的，簡直像畫出來的一樣喔。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 疲れているみたいだ。 | 好像很累。 |
| +主語 | **彼は**疲れているみたいだ。 | 他好像很累。 |
| +行動 | 彼は疲れているみたいだから、**少し休ませてあげよう**。 | 他好像很累，讓他稍微休息一下吧。 |
