# 〜あげく（に）

## 規則與接續
- `動詞た形 + あげく（に）`
- `名詞 + の + あげく（に）`

## 用法說明
表示經過一番波折、長時間的努力或糾結之後，最終得出的結果（多半是負面、不理想的），強調過程的曲折與結果的落差，語氣偏書面，帶有無奈或後悔的意味。

與「〜末（すえ）に」的辨析：
- 「〜あげく（に）」：強調經過一番波折後產生的結果，**結果多為負面、不理想**，帶無奈或後悔語感。
- 「〜末に」：同樣表示「經過一段過程後的最終結果」，但**語感中性**，結果可好可壞（例：話し合いの末に、合意に達した），更常見於書面與正式場合。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> さんざん悩（なや）んだあげく、彼（かれ）は仕事（しごと）を辞（や）めることにした。
> （苦惱了很久之後，他最終決定辭職。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 迷う（まよう） | さんざん**迷ったあげく**、結局買わなかった。 | 猶豫了很久之後，最後還是沒買。 |
| 喧嘩する（けんかする） | ひどい**喧嘩をしたあげく**、二人は別れた。 | 大吵一架之後，兩人分手了。 |
| 長い話し合い（ながいはなしあい） | **長い話し合いのあげく**、計画は中止になった。 | 經過長時間的商談，計畫最終取消了。 |
| 検討する（けんとうする） | 何度も**検討したあげく**、案を却下した。 | 反覆研究之後，最終否決了方案。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 末に（較中性書面） | さんざん悩んだ**末に**、彼は仕事を辞めることにした。 | 經過一番苦惱之後，他決定辭職。（語感較「あげく」中性，不一定隱含負面） |
| → 丁寧形 | さんざん悩んだあげく、仕事を辞めることにし**ました**。 | 苦惱了很久之後，決定辭職了。（禮貌形） |
| → 過去形強調結果 | さんざん悩んだあげく、彼は仕事を**辞めてしまった**。 | 苦惱了很久之後，他終究還是辭職了。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 彼、会社辞めたんですか。 | ええ、さんざん悩んだあげく、辞めることにしたそうです。 | 是的，聽說他苦惱很久之後決定辭職了。 |
| 会議の結論は出ましたか。 | いいえ、長い議論のあげく、結論は出ませんでした。 | 沒有，經過長時間討論，最終還是沒有結論。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 仕事を辞めることにした。 | 決定辭職了。 |
| +主語 | **彼は**仕事を辞めることにした。 | 他決定辭職了。 |
| +過程 | **さんざん悩んだあげく、**彼は仕事を辞めることにした。 | 苦惱了很久之後，他決定辭職了。 |
