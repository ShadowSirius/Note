# 〜ていただけませんか

## 規則與接續
- `動詞て形 + いただけませんか`

## 用法說明
用於禮貌地請求他人做某事。比「〜てください」委婉，與同課的 [[JP_Grammar_50_C35_te_moraemasen_ka_request|〜てもらえませんか @related_to]] 相比敬意更高，常用於對上級、客戶或不熟識的人。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#3498DB"><ruby>申<rt>もう</rt></ruby>し<ruby>訳<rt>わけ</rt></ruby>ありませんが、</span><span style="color:#FF477E">この<ruby>書類<rt>しょるい</rt></ruby>の<ruby>書<rt>か</rt></ruby>き<ruby>方<rt>かた</rt></ruby>を</span><span style="color:#3498DB"><ruby>教<rt>おし</rt></ruby>えていただけませんか。</span>
  中文：真的很抱歉，能不能請您教我這份文件的寫法呢？
- 日文：<span style="color:#FF477E"><ruby>会議<rt>かいぎ</rt></ruby>の<ruby>資料<rt>しりょう</rt></ruby>を</span><span style="color:#3498DB">メールで<ruby>送<rt>おく</rt></ruby>っていただけませんか。</span>
  中文：能不能請您把會議資料用電子郵件寄給我呢？

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 申（もう）し訳（わけ）ありませんが、この書類（しょるい）の書（か）き方（かた）を教（おし）えていただけませんか。
> （真的很抱歉，能不能請您教我這份文件的寫法呢？）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 少し待つ（すこしまつ） | **少し待っていただけませんか**。 | 能請您稍等一下嗎？ |
| もう一度説明する（もういちどせつめいする） | **もう一度説明していただけませんか**。 | 能請您再說明一次嗎？ |
| 書類を確認する（しょるいをかくにんする） | **書類を確認していただけませんか**。 | 能請您確認一下文件嗎？ |
| サインする | ここに**サインしていただけませんか**。 | 能請您在這裡簽名嗎？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 較不禮貌形 | 書き方を教えて**もらえませんか**。 | 能不能教我寫法呢（較不禮貌）。 |
| → 最高禮貌形 | 書き方を教えて**いただけますでしょうか**。 | 是否能請您教我寫法呢（最禮貌）。 |
| → 肯定回答 | はい、**喜んで教えさせていただきます**。 | 好的，我很樂意為您說明。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| すみません、この書類の書き方が分かりません。 | 教えていただけませんか。 | 能請您教我嗎？ |
| 会議の資料はどこですか。 | メールで送っていただけませんか。 | 能請您用電子郵件寄給我嗎？ |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 教えていただけませんか。 | 能請您教我嗎？ |
| +對象 | この**書類の書き方を**教えていただけませんか。 | 能請您教我這份文件的寫法嗎？ |
| +前言 | **申し訳ありませんが、**この書類の書き方を教えていただけませんか。 | 真的很抱歉，能請您教我這份文件的寫法嗎？ |
