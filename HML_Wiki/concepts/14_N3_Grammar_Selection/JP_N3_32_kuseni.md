# 〜くせに

## 規則與接續
- `動詞普通形 + くせに`
- `い形容詞普通形 + くせに`
- `な形容詞語幹 + な + くせに`
- `名詞 + の + くせに`

## 用法說明
表示「明明...卻...」，帶有輕蔑、責備或諷刺語氣的逆接，多用於對第二、三人稱的負面評價，說話者主觀不滿情緒強烈。一般不用於稱讚場合，也很少用在自己身上（除非是自嘲）。

與 [[JP_N3_39_noni|〜のに]]、[[JP_N3_31_mono_no|〜ものの]] 的辨析：
- 「〜くせに」：語氣最強烈、最口語化，帶明顯輕蔑／責備感情，主語常是對方或第三者。
- 「〜のに」：同樣帶主觀情緒（不滿、驚訝、遺憾），但語氣比「くせに」溫和，可用於任何人稱，包括自己。
- 「〜ものの」：語氣最客觀、書面，幾乎不帶責備情緒，純粹陳述前後矛盾的事實。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）は自分（じぶん）でやらないくせに、文句（もんく）ばかり言（い）う。
> （他明明自己不做，卻老是抱怨。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 下手だ（へただ） | 下手な**くせに**、いつも自信満々だ。 | 明明很不擅長，卻總是很有自信。 |
| お金がない | お金がない**くせに**、高い物ばかり買う。 | 明明沒有錢，卻老是買貴的東西。 |
| 知らない（しらない） | 何も知らない**くせに**、口を出すな。 | 明明什麼都不懂，別多嘴。 |
| 学生だ | 学生の**くせに**、毎晩遊び歩いている。 | 明明是學生，卻每晚到處玩。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → のに（語氣較溫和） | 彼は自分でやらない**のに**、文句ばかり言う。 | 他明明自己不做，卻老是抱怨。（語氣較緩和） |
| → 過去形 | 彼は自分でやらなかった**くせに**、文句ばかり言った。 | 他明明當時沒做，卻一直抱怨。 |
| → 疑問形（責備語氣） | 自分でやらない**くせに**、なんで文句を言うの？ | 明明自己不做，為什麼要抱怨？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 彼はどうしていつも文句を言うんですか。 | 自分でやらないくせに、文句ばかり言うんですよ。 | 因為他明明自己不做，卻老是抱怨。 |
| あの子、生意気ですね。 | ええ、子供のくせに生意気なことばかり言いますね。 | 是啊，明明是小孩卻老說些狂妄的話。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 文句ばかり言う。 | 老是在抱怨。 |
| +主語 | **彼は**文句ばかり言う。 | 他老是在抱怨。 |
| +輕蔑理由 | **彼は自分でやらないくせに、**文句ばかり言う。 | 他明明自己不做，卻老是抱怨。 |
