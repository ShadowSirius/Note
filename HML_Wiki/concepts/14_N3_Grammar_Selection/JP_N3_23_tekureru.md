# 〜てくれる

## 規則與接續
- `動詞て形 + くれる`

## 用法說明
表示他人主動為說話者（或說話者一方）做某事，主詞為「施恩者」。與 [[JP_N3_22_temorau|〜てもらう @related_to]] 相比，「〜てくれる」更強調對方是主動幫忙，帶有強烈的感謝之意，語法結構為「（對方）が（我）に〜てくれる」。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E"><ruby>荷物<rt>にもつ</rt></ruby>が<ruby>重<rt>おも</rt></ruby>かったので、</span><span style="color:#2ECC71"><ruby>親切<rt>しんせつ</rt></ruby>な<ruby>人<rt>ひと</rt></ruby>が</span><span style="color:#3498DB"><ruby>運<rt>はこ</rt></ruby>んでくれた。</span>
  中文：因為行李很重，有位好心人幫我搬了。
- 日文：<span style="color:#FF477E"><ruby>誕生日<rt>たんじょうび</rt></ruby>の<ruby>日<rt>ひ</rt></ruby>に、</span><span style="color:#2ECC71"><ruby>彼氏<rt>かれし</rt></ruby>が</span><span style="color:#3498DB"><ruby>綺麗<rt>きれい</rt></ruby>なネックレスを<ruby>買<rt>か</rt></ruby>ってくれた。</span>
  中文：生日那天，男朋友買了一條漂亮的項鍊給我。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 荷物（にもつ）が重（おも）かったので、親切（しんせつ）な人（ひと）が運（はこ）んでくれた。
> （因為行李很重，有位好心人幫我搬了。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 説明する（せつめいする） | 先生が**説明してくれた**。 | 老師幫我說明了。 |
| 迎えに来る（むかえにくる） | 母が駅まで**迎えに来てくれた**。 | 媽媽來車站接我了。 |
| 案内する（あんないする） | 現地の人が**案内してくれた**。 | 當地人幫我帶路了。 |
| 手伝う（てつだう） | 同僚が**手伝ってくれた**。 | 同事幫我忙了。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 敬語形（くださる） | 先生が説明して**くださった**。 | 老師為我說明了（敬語）。 |
| → 否定形 | 誰も手伝って**くれなかった**。 | 沒有人幫忙。 |
| → 疑問形 | 誰か運んで**くれましたか**。 | 有人幫忙搬了嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 荷物、重かったでしょう。 | はい、でも親切な人が運んでくれました。 | 是的，但有位好心人幫我搬了。 |
| 誕生日プレゼントは何でしたか。 | 彼氏が綺麗なネックレスを買ってくれました。 | 男朋友買了漂亮的項鍊給我。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 運んでくれた。 | 幫我搬了。 |
| +主語 | **親切な人が**運んでくれた。 | 有位好心人幫我搬了。 |
| +理由 | **荷物が重かったので、**親切な人が運んでくれた。 | 因為行李很重，有位好心人幫我搬了。 |
