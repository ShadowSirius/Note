# 〜てあげる

## 規則與接續
- `動詞て形 + あげる`

## 用法說明
表示說話者為他人做某事。與 [[JP_N3_23_tekureru|〜てくれる @related_to]] 方向相反：てあげる是「我為他人」，てくれる是「他人為我」。對長輩或上司使用「〜てあげる」容易顯得高高在上、多管閒事，因此應避免對上位者使用，通常僅用於平輩、晚輩或親密關係之間。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 妹（いもうと）が泣（な）いていたので、お菓子（かし）を買（か）ってあげた。
> （妹妹剛在哭，所以我買了零食給她。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 宿題を教える（しゅくだいをおしえる） | 弟に**宿題を教えてあげた**。 | 教弟弟寫功課了。 |
| 本を貸す（ほんをかす） | 友達に**本を貸してあげた**。 | 借書給朋友了。 |
| 荷物を持つ（にもつをもつ） | 後輩の**荷物を持ってあげた**。 | 幫學弟妹拿行李了。 |
| 写真を撮る（しゃしんをとる） | 観光客に**写真を撮ってあげた**。 | 幫觀光客拍照了。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 對上位者改用差し上げる | 先生に資料を送って**差し上げた**。 | 為老師送上了資料（謙讓）。 |
| → 否定形 | 妹に何も買って**あげなかった**。 | 沒有買任何東西給妹妹。 |
| → 疑問形 | お菓子を買って**あげましたか**。 | 有買零食給她嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 妹はなぜ喜んでいるんですか。 | お菓子を買ってあげたからです。 | 因為我買了零食給她。 |
| 友達の引っ越しはどうでしたか。 | 少し手伝ってあげました。 | 我稍微幫了忙。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 買ってあげた。 | 買給她了。 |
| +對象 | **妹に**お菓子を買ってあげた。 | 買零食給妹妹了。 |
| +理由 | **妹が泣いていたので、**お菓子を買ってあげた。 | 因為妹妹在哭，買零食給她了。 |
