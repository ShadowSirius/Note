# 〜させてもらう

## 規則與接續
- `動詞使役て形（五段：〜せて／一段・する・来る：〜させて） + もらう`

## 用法說明
謙讓表達，表示請求或獲得對方許可做某事，並對此感到感激。奠基於 [[JP_Grammar_50_C47_causative_voice|使役形 @related_to]] 和 [[JP_N3_22_temorau|〜てもらう @related_to]]，比單純的「〜する」更禮貌，常代替「〜します」用於商務或鄭重場合表達自己要做的事。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E"><ruby>体調<rt>たいちょう</rt></ruby>が<ruby>悪<rt>わる</rt></ruby>いので、</span><span style="color:#FF477E"><ruby>今日<rt>きょう</rt></ruby>は</span><span style="color:#3498DB"><ruby>早<rt>はや</rt></ruby>く<ruby>帰<rt>かえ</rt></ruby>らせてもらえませんか。</span>
  中文：因為身體不舒服，今天能讓我早點回家嗎？
- 日文：<span style="color:#FF477E">ここに</span><span style="color:#3498DB"><ruby>自転車<rt>じてんしゃ</rt></ruby>を<ruby>停<rt>と</rt></ruby>めさせてもらってもいいですか。</span>
  中文：請問我可以把腳踏車停在這裡嗎？

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 体調（たいちょう）が悪（わる）いので、今日（きょう）は早（はや）く帰（かえ）らせてもらえませんか。
> （因為身體不舒服，今天能讓我早點回家嗎？）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 発言する（はつげんする） | 少し**発言させてもらえませんか**。 | 可以讓我發個言嗎？ |
| 休む（やすむ） | 明日**休ませてもらえませんか**。 | 明天可以讓我請假嗎？ |
| 考える（かんがえる） | 少し**考えさせてもらえませんか**。 | 可以讓我考慮一下嗎？ |
| 説明する（せつめいする） | 私に**説明させてもらえませんか**。 | 可以讓我來說明嗎？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 直述形（較不禮貌） | 早く**帰らせてもらう**。 | 讓我早點回去（陳述）。 |
| → 過去形（感謝） | 早く帰らせて**もらいました**。ありがとうございました。 | 讓我早退了，謝謝您。 |
| → 更謙讓形 | 発言させて**いただけませんか**。 | 能否讓我發言呢（更禮貌）。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうしたんですか、顔色が悪いですよ。 | 体調が悪いので、早く帰らせてもらえませんか。 | 因為身體不舒服，能讓我早點回家嗎？ |
| ここに自転車を停めてもいいですか。 | はい、停めさせてもらってもいいですよ。 | 是的，可以停在這裡。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 帰らせてもらえませんか。 | 能讓我回去嗎？ |
| +理由 | **体調が悪いので、**帰らせてもらえませんか。 | 因為身體不舒服，能讓我回去嗎？ |
| +時間 | 体調が悪いので、**今日は早く**帰らせてもらえませんか。 | 因為身體不舒服，今天能讓我早點回去嗎？ |
