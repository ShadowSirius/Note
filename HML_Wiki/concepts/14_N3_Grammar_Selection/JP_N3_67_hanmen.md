# 〜反面

## 規則與接續
- `動詞普通形 + 反面`
- `い形容詞普通形 + 反面`
- `な形容詞 + な反面`
- `名詞 + である反面`（口語中也常見 `名詞 + の反面` 的說法，但正式教材以 `である反面` 較能清楚表現）

## 用法說明
表示同一事物有「相反的一面」，前半句先提出一種特徵，後半句再補出與之對照的另一面，語感常接近「雖然……，但另一方面……」。  
和 `〜一方（で）` 相近，但 `〜反面` 更常用來強調同一主體的正反對比，帶有「優點／缺點、好處／麻煩」這類明顯反差的語感。

- 日文：<span style="color:#2ECC71"><ruby>彼<rt>かれ</rt></ruby>は</span><span style="color:#FF477E"><ruby>優<rt>やさ</rt></ruby>しい<ruby>反面<rt>はんめん</rt></ruby></span><span style="color:#3498DB">、<ruby>決<rt>き</rt></ruby>めたことはなかなか<ruby>変<rt>か</rt></ruby>えない。</span>

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）は優（やさ）しい反面（はんめん）、決（き）めたことはなかなか変（か）えない。
> （他雖然很溫柔，但一旦決定了就不太會改變。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| おとなしい | 彼はおとなしい反面、決めたことはなかなか変えない。 | 他雖然很文靜，但一旦決定了就不太會改變。 |
| 厳しい（きびしい） | 彼は厳しい反面、面倒見がいい。 | 他雖然嚴格，但很會照顧人。 |
| 忙しい（いそがしい） | 彼は忙しい反面、休みが少ない。 | 他雖然很忙，但休息時間很少。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 合併兩句 | この薬は眠くなる。 この薬は効き目が強い。 → この薬は眠くなる反面、効き目が強い。 | 這款藥會讓人想睡，但另一方面藥效很強。 |
| → 合併兩句 | 彼はやさしい。 決めたことはなかなか変えない。 → 彼はやさしい反面、決めたことはなかなか変えない。 | 他雖然溫柔，但一旦決定了就不太會改變。 |
| → 合併兩句 | 勉強は楽しい。 時間が足りなくなることもある。 → 勉強は楽しい反面、時間が足りなくなることもある。 | 學習雖然有趣，但另一方面也可能會時間不夠。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 彼はどんな人ですか。 | 優しい反面、決めたことはなかなか変えない人です。 | 他是什麼樣的人？——他雖然溫柔，但一旦決定就不太改變的人。 |
| この仕事の特徴は何ですか。 | 面白い反面、忙しい仕事です。 | 這份工作的特點是什麼？——雖然有趣，但也很忙。 |
| このアプリはどうですか。 | 便利な反面、使いすぎると時間を取られます。 | 這個 App 怎麼樣？——雖然方便，但用太多會佔用時間。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 優しい反面、 | 雖然溫柔，但另一方面…… |
| 擴展 1 | 彼は優しい反面、 | 他雖然溫柔，但另一方面…… |
| 擴展 2 | 彼は優しい反面、決めたことはなかなか変えない。 | 他雖然溫柔，但一旦決定了就不太會改變。 |
| 擴展 3 | 彼は優しい反面、決めたことはなかなか変えないし、少し頑固だ。 | 他雖然溫柔，但一旦決定就不太會改變，而且有點固執。 |
