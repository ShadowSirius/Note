# 〜ないでもらいたい

## 規則與接續
- `動詞ない形 + でもらいたい`
- `（人）に + 動詞ない形 + でもらいたい`
- 常用於表達說話者「希望對方不要做某事」的委婉要求。

## 用法說明
「〜ないでもらいたい」表示說話者出於自己的立場，對對方提出希望或請求，語氣比直接命令更婉轉，帶有「希望你不要……」的意味。  
和「〜ないでほしい」很接近，但「〜ないでもらいたい」更像是把重點放在「請你幫我不要做這件事」的請求感，語法上源自 [[JP_N3_22_temorau|〜てもらう @related_to]] 的否定願望形；「〜ないでください」則更直接、正式，常用於提醒或禁止。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 会議中（かいぎちゅう）に電話（でんわ）を使（つか）わないでもらいたい。
> （希望你在會議中不要使用電話。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 会議中（かいぎちゅう）に電話（でんわ）を使（つか）う | 会議中に電話を使わないでもらいたい。 | 希望你在會議中不要使用電話。 |
| 電車（でんしゃ）の中（なか）で話（はな）す | 電車の中で話さないでもらいたい。 | 希望你不要在電車裡說話。 |
| ここで写真（しゃしん）を撮（と）る | ここで写真を撮らないでもらいたい。 | 希望你不要在這裡拍照。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 希望式表達 | もう少し静かにしてもらいたい。 | 希望你能再安靜一點。 |
| → 否定請求 | ここでタバコを吸わないでもらいたい。 | 希望你不要在這裡抽菸。 |
| → 過去式 | その件は、二度と話さないでもらいたかった。 | 那件事，我希望你再也不要提起。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうして電話を使わないでほしいのですか。 | 会議中なので、電話を使わないでもらいたいです。 | 為什麼希望不要用電話？因為在開會。 |
| ここで写真を撮ってもいいですか。 | すみません、ここでは写真を撮らないでもらいたいです。 | 可以在這裡拍照嗎？不好意思，希望不要拍。 |
| 少し待ってもらえますか。 | はい、ただし大声で話さないでもらいたいです。 | 可以稍等嗎？可以，但希望不要大聲說話。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 話さないでもらいたい。 | 希望你不要說話。 |
| 擴展 1 | ここで話さないでもらいたい。 | 希望你不要在這裡說話。 |
| 擴展 2 | 会議中にここで話さないでもらいたい。 | 希望你在會議中不要在這裡說話。 |
| 擴展 3 | 申し訳ありませんが、会議中にここで話さないでもらいたいです。 | 不好意思，希望你在會議中不要在這裡說話。 |
