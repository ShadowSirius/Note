# 〜わけではない

## 規則與接續
- `普通形（な形容詞+な/である・名詞+の/な/である） + わけではない`

## 用法說明
表示「並非完全如此」，常與「いつも、全部、必ず」等詞連用作部分否定，是在承認某事實前提下的澄清說明。與全面否定的 [[JP_N3_10_wake_ga_nai|〜わけがない @related_to]] 意義完全不同：わけがない是「不可能」，わけではない是「並非全部/並不是」，兩者常被學習者混淆，需特別留意。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#3498DB"><ruby>肉<rt>にく</rt></ruby>が<ruby>嫌<rt>きら</rt></ruby>いなわけではないが、</span><span style="color:#FF477E"><ruby>最近<rt>さいきん</rt></ruby>は</span><span style="color:#3498DB"><ruby>野菜<rt>やさい</rt></ruby>を<ruby>多<rt>おお</rt></ruby>く<ruby>食<rt>た</rt></ruby>べるようにしている。</span>
  中文：我並不是討厭吃肉，只是最近盡量多吃蔬菜。
- 日文：<span style="color:#FF477E">お<ruby>金<rt>かね</rt></ruby>があれば</span><span style="color:#3498DB"><ruby>必<rt>かなら</rt></ruby>ず<ruby>幸<rt>しあわ</rt></ruby>せになれるわけではない。</span>
  中文：並不是有錢就一定能得到幸福。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 肉（にく）が嫌（きら）いなわけではないが、最近（さいきん）は野菜（やさい）を多（おお）く食（た）べるようにしている。
> （我並不是討厭吃肉，只是最近盡量多吃蔬菜。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| いつも忙しい（いそがしい） | **いつも忙しいわけではない**。 | 並不是一直都很忙。 |
| お金が全部悪い（かねがぜんぶわるい） | お金が**全部悪いわけではない**。 | 錢並不是全都是壞的。 |
| 嫌いだ（きらいだ） | 別に**嫌いなわけではない**。 | 並不是特別討厭。 |
| 反対だ（はんたいだ） | 私は**反対なわけではない**。 | 我並不是反對。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 昔から嫌いだった**わけではない**。 | 並不是從以前就討厭。 |
| → 疑問形 | 肉が嫌いな**わけではないんですか**。 | 並不是討厭吃肉嗎？ |
| → 強調（決して〜わけではない） | 決して肉が嫌いな**わけではない**。 | 絕對不是討厭吃肉。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 肉が嫌いなんですか。 | いいえ、嫌いなわけではないですが、野菜を多く食べています。 | 不，並不是討厭，只是多吃蔬菜而已。 |
| お金があれば幸せになれますか。 | いいえ、必ず幸せになれるわけではありません。 | 不，並不是一定能得到幸福。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 嫌いなわけではない。 | 並不是討厭。 |
| +對象 | **肉が**嫌いなわけではない。 | 並不是討厭吃肉。 |
| +補充 | 肉が嫌いなわけではないが、**最近は野菜を多く食べるようにしている**。 | 並不是討厭吃肉，只是最近盡量多吃蔬菜。 |
