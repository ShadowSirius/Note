# 〜うちに

## 規則與接續
- `動詞辞書形/ている形/ない形 + うちに`
- `い形容詞 + うちに`／`な形容詞 + な + うちに`／`名詞 + の + うちに`

## 用法說明
表示「趁著某狀態尚未改變前做某事」，若錯過這段時間該行為將變得困難，常帶有說話者的主觀急迫感。與 [[JP_N3_17_aida_ni|〜間に @related_to]] 相比，「〜うちに」更強調時機的急迫性與可能錯失的遺憾，「〜間に」則單純描述期間內發生的動作。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71">スープが</span><span style="color:#3498DB"><ruby>冷<rt>さ</rt></ruby>めないうちに、</span><span style="color:#FF477E"><ruby>早<rt>はや</rt></ruby>く</span><span style="color:#3498DB"><ruby>飲<rt>の</rt></ruby>んでください。</span>
  中文：趁著湯還沒冷，請快點喝吧。
- 日文：<span style="color:#FF477E"><ruby>日本<rt>にほん</rt></ruby>に<ruby>留学<rt>りゅうがく</rt></ruby>しているうちに、</span><span style="color:#3498DB"><ruby>色々<rt>いろいろ</rt></ruby>な<ruby>所<rt>ところ</rt></ruby>へ<ruby>旅行<rt>りょこう</rt></ruby>したいです。</span>
  中文：趁著在日本留學的時候，想去很多地方旅行。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> スープが冷（さ）めないうちに、早（はや）く飲（の）んでください。
> （趁著湯還沒冷，請快點喝吧。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 若い（わかい） | **若いうちに**、色々な経験をしたほうがいい。 | 趁年輕的時候，多累積各種經驗比較好。 |
| 元気だ（げんきだ） | **元気なうちに**、旅行に行きたい。 | 趁還健康的時候，想去旅行。 |
| 忘れない（わすれない） | **忘れないうちに**、メモしておこう。 | 趁還沒忘記，先記下來吧。 |
| 明るい（あかるい） | **明るいうちに**、家に帰りましょう。 | 趁天還亮的時候回家吧。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 否定接續 | 冷め**ないうちに**、早く飲んでください。 | 趁還沒冷掉，請快喝。 |
| → 肯定接續（対比） | 冷め**てから**飲むと、おいしくない。 | 冷掉之後才喝就不好喝了。 |
| → 疑問形 | スープが冷めないうちに**飲んだほうがいいですか**。 | 趁湯還沒冷喝比較好嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| このスープ、いつ飲めばいいですか。 | 冷めないうちに、早く飲んでください。 | 趁還沒冷，請快點喝。 |
| 留学中に何をしたいですか。 | 日本に留学しているうちに、色々な所へ旅行したいです。 | 趁留學期間，想去很多地方旅行。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 冷めないうちに飲んでください。 | 趁還沒冷請喝。 |
| +主語 | **スープが**冷めないうちに、飲んでください。 | 趁湯還沒冷，請喝。 |
| +程度 | スープが冷めないうちに、**早く**飲んでください。 | 趁湯還沒冷，請快點喝。 |
