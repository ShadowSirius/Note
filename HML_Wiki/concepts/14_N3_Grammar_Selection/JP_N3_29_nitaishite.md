# 〜に対して

## 規則與接續
- `名詞 + に対して`
- `普通形（名詞・な形容詞 + の） + に対して`

## 用法說明
表示動作施加的情感或行為對象（對於…），或前後兩者的對比（與…相反）。與單純提示話題的 [[JP_N3_30_nitsuite|〜について @related_to]] 不同，「〜に対して」帶有明確的對立或指向性；也不同於評價性的 [[JP_N3_28_nitotte|〜にとって @related_to]]。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71">彼は</span><span style="color:#FF477E"><ruby>誰<rt>だれ</rt></ruby>に<ruby>対<rt>たい</rt></ruby>しても</span><span style="color:#3498DB"><ruby>親切<rt>しんせつ</rt></ruby>で、みんなから<ruby>好<rt>す</rt></ruby>かれています。</span>
  中文：他對任何人都很親切，非常受大家歡迎。
- 日文：<span style="color:#2ECC71">兄が</span><span style="color:#FF477E"><ruby>活発<rt>かっぱつ</rt></ruby>なのに<ruby>対<rt>たい</rt></ruby>して、</span><span style="color:#2ECC71">弟は</span><span style="color:#3498DB">とてもおとなしい<ruby>性格<rt>せいかく</rt></ruby>です。</span>
  中文：哥哥很活潑，相對地，弟弟的個性則非常文靜。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）は誰（だれ）に対（たい）しても親切（しんせつ）で、みんなから好（す）かれています。
> （他對任何人都很親切，非常受大家歡迎。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 部下（ぶか） | 彼は**部下に対して**厳しい。 | 他對下屬很嚴格。 |
| 質問（しつもん） | 先生は**質問に対して**丁寧に答える。 | 老師對問題會仔細回答。 |
| 顧客（こきゃく） | このサービスは**顧客に対して**手厚い。 | 這項服務對顧客很周到。 |
| 意見（いけん） | 私の**意見に対して**反対する人もいる。 | 也有人反對我的意見。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 名詞修飾用法 | これは私**に対する**評価です。 | 這是對我的評價。 |
| → 對比用法（のに対して） | 兄が活発な**のに対して**、弟は静かだ。 | 相對於哥哥活潑，弟弟很安靜。 |
| → 疑問形 | 彼は誰**に対しても親切ですか**。 | 他對任何人都很親切嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 彼はどんな性格ですか。 | 誰に対しても親切で、みんなから好かれています。 | 他對任何人都很親切，很受歡迎。 |
| 兄弟の性格は似ていますか。 | いいえ、兄が活発なのに対して、弟はおとなしいです。 | 不，哥哥很活潑，弟弟卻很文靜。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 親切です。 | 很親切。 |
| +對象 | **誰に対しても**親切です。 | 對任何人都很親切。 |
| +結果 | 彼は誰に対しても親切で、**みんなから好かれています**。 | 他對任何人都很親切，很受大家歡迎。 |
