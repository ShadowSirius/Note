# 授受敬語〜ていただく／〜させていただく／〜くださる／〜てくださいませんか

## 規則與接續
- `動詞て形 + いただく`  
  用於表達「請對方幫我做……／承蒙對方幫忙……」，是自謙、感謝的敬語。
- `動詞使役形 + いただく`  
  用於表達「請允許我……／讓我……」，帶有「在對方允許下進行」的謙讓語感。
- `動詞て形 + くださる`  
  用於表達「對方為我做……」，是對對方行為的尊敬說法。
- `動詞て形 + くださいませんか`  
  用於表達「可以請您……嗎？」比 `〜てください` 更委婉、客氣。

## 用法說明
這組敬語屬於「授受表現」的禮貌說法，常用在請求、感謝、正式場合自我表述。`〜ていただく／〜てくださる` 分別比 `〜てもらう／〜てくれる` 更正式；`〜てくださいませんか` 則比 `〜てください` 更委婉。  
另外，`〜させていただく` 常用來表示「在對方允許下做某事」，適合正式發言，但若過度使用會顯得過於客套。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> この資料（しりょう）を確認（かくにん）してくださいませんか。
> （可以請您確認這份資料嗎？）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| こちらの書類（しょるい）を読（よ）む | こちらの書類を読んでくださいませんか。 | 可以請您讀一下這份文件嗎？ |
| もう一度（いちど）説明（せつめい）する | もう一度説明してくださいませんか。 | 可以請您再說明一次嗎？ |
| この席（せき）を少（すこ）し空（あ）ける | この席を少し空けてくださいませんか。 | 可以請您稍微空出這個座位嗎？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → ていただく | 山田（やまだ）さんに資料（しりょう）を見（み）てもらう。→ 山田さんに資料を見ていただく。 | 承蒙山田先生幫我看資料。 |
| → させていただく | 私（わたし）が説明（せつめい）する。→ 私が説明させていただきます。 | 由我來說明。 |
| → てくださいませんか | ここに名前（なまえ）を書く。→ ここに名前を書いてくださいませんか。 | 可以請您在這裡寫上名字嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 資料（しりょう）を丁寧（ていねい）に確認（かくにん）してほしいとき | この資料を確認してくださいませんか。 | 想請您仔細確認這份資料。 |
| 会議（かいぎ）で自分（じぶん）が発表（はっぴょう）すると言（い）いたいとき | 会議では、私が発表させていただきます。 | 在會議上，我來發表。 |
| 上司（じょうし）が手伝（てつだ）ってくれたことに感謝（かんしゃ）したいとき | 上司が手伝ってくださいました。 | 上司很親切地幫了我。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 確認（かくにん）してくださいませんか。 | 可以請您確認嗎？ |
| 前置 | 恐（おそ）れ入（い）りますが、確認してくださいませんか。 | 不好意思，可以請您確認嗎？ |
| 完整 | 恐れ入りますが、この資料（しりょう）を確認してくださいませんか。 | 不好意思，可以請您確認這份資料嗎？ |
