# 〜てもらう

## 規則與接續
- `動詞て形 + もらう`

## 用法說明
表示說話者拜託他人做某事並從中受益，主詞為「接受恩惠者（通常是我）」。與 [[JP_N3_23_tekureru|〜てくれる @related_to]] 不同，「〜てもらう」的視角是「我從對方那裡得到幫忙」，語法上常見「（我）は（對方）に〜てもらう」的結構；禮貌形請求見 [[JP_Grammar_50_C35_te_moraemasen_ka_request|〜てもらえませんか @related_to]]。

  中文：如果有不懂的地方，我們去請老師教吧。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> パソコンの調子（ちょうし）が悪（わる）いので、友達（ともだち）に直（なお）してもらった。
> （因為電腦有點怪怪的，所以我請朋友幫我修好了。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 荷物を運ぶ（にもつをはこぶ） | 友達に**荷物を運んでもらった**。 | 請朋友幫我搬行李了。 |
| 道を教える（みちをおしえる） | 通行人（つうこうにん）に**道を教えてもらった**。 | 請路人幫我指路了。 |
| 写真を撮る（しゃしんをとる） | 店員に**写真を撮ってもらった**。 | 請店員幫我拍照了。 |
| 手伝う（てつだう） | 母に**手伝ってもらった**。 | 請媽媽幫忙了。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 敬語形（いただく） | 先生に教えて**いただいた**。 | 承蒙老師指導。 |
| → 否定形 | 誰にも直して**もらえなかった**。 | 沒有任何人願意幫我修。 |
| → 疑問形 | 友達に直して**もらいましたか**。 | 有請朋友幫忙修好嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| パソコンはどうやって直したんですか。 | 友達に直してもらいました。 | 請朋友幫我修好的。 |
| 分からないところはどうしますか。 | 先生に教えてもらいましょう。 | 去請老師教我們吧。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 直してもらった。 | 請人幫我修好了。 |
| +對象 | **友達に**直してもらった。 | 請朋友幫我修好了。 |
| +理由 | **パソコンの調子が悪いので、**友達に直してもらった。 | 因為電腦不太對勁，請朋友幫我修好了。 |
