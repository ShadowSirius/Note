# 〜わけがない

## 規則與接續
- `動詞/い形容詞 普通形 + わけがない`
- `な形容詞 + な / 名詞 + の + わけがない`

## 用法說明
根據常理或強烈的主觀判斷，全面否定某事的可能性，意為「不可能、絕對不會」，語氣十分堅決。與部分否定的 [[JP_N3_11_wake_dewa_nai|〜わけではない @related_to]] 意思完全不同，切勿混淆。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> ピーマンが嫌（きら）いな彼（かれ）が、そんな野菜炒（やさいいた）めを食（た）べるわけがない。
> （討厭青椒的他，不可能會吃那種炒青菜的。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| そんな高い物（たかいもの）を買う（かう） | 彼が**そんな高い物を買うわけがない**。 | 他不可能會買那麼貴的東西。 |
| 一人で行く（ひとりでいく） | 彼女が**一人で行くわけがない**。 | 她不可能一個人去。 |
| 嘘をつく（うそをつく） | あの人が**嘘をつくわけがない**。 | 那個人不可能會說謊。 |
| 簡単だ（かんたんだ） | この試験（しけん）が**簡単なわけがない**。 | 這場考試不可能簡單。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 較口語形（わけない） | そんなの食べる**わけない**。 | 不可能吃那種東西啦。 |
| → 過去形 | あの日、彼が行った**わけがない**。 | 那天他不可能有去。 |
| → 疑問形（反問語氣） | そんなことがある**わけがないでしょう**。 | 怎麼可能會有那種事嘛。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 彼はピーマンを食べますか。 | いいえ、嫌いだから食べるわけがない。 | 不，他討厭青椒不可能會吃。 |
| 練習していないのに勝てますか。 | 全然練習していないのに、勝てるわけがないでしょう。 | 完全沒練習，怎麼可能會贏嘛。 |

| Step | Sentence | Translation（中文翻譯） |

| 核心 | 食べるわけがない。 | 不可能會吃。 |

| +對象 | ピーマンが嫌いな彼が、**そんな野菜炒めを**食べるわけがない。 | 討厭青椒的他不可能會吃那種炒青菜。 |
