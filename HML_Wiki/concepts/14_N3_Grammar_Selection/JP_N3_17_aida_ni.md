# 〜間に

## 規則與接續
- `動詞辞書形/ている形 + 間に`
- `い形容詞 + 間に`／`な形容詞 + な + 間に`／`名詞 + の + 間に`

## 用法說明
表示在某個持續狀態的期間內，發生了另一件短暫、瞬間的動作。與不加「に」的「〜間」不同：「〜間」指整段期間動作皆持續發生，「〜間に」則是期間內某個時間點才發生。與 [[JP_N3_16_uchi_ni|〜うちに @related_to]] 相比，「〜間に」較客觀、不強調急迫感。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E"><ruby>子供<rt>こども</rt></ruby>が<ruby>寝<rt>ね</rt></ruby>ている<ruby>間<rt>あいだ</rt></ruby>に、</span><span style="color:#3498DB"><ruby>家事<rt>かじ</rt></ruby>を<ruby>終<rt>お</rt></ruby>わらせました。</span>
  中文：趁著小孩睡著的期間，我把家事做完了。
- 日文：<span style="color:#FF477E"><ruby>留守<rt>るす</rt></ruby>の<ruby>間<rt>あいだ</rt></ruby>に、</span><span style="color:#2ECC71"><ruby>荷物<rt>にもつ</rt></ruby>が</span><span style="color:#3498DB"><ruby>届<rt>とど</rt></ruby>いたようです。</span>
  中文：不在家的期間，好像有包裹送來了。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 子供（こども）が寝（ね）ている間（あいだ）に、家事（かじ）を終（お）わらせました。
> （趁著小孩睡著的期間，我把家事做完了。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 授業中（じゅぎょうちゅう） | **授業の間に**、居眠り（いねむり）をしてしまった。 | 上課期間不小心打瞌睡了。 |
| 会議（かいぎ） | **会議の間に**、電話が鳴った（なった）。 | 開會期間電話響了。 |
| 出かけている（でかけている） | **出かけている間に**、雨が降った。 | 外出期間下了雨。 |
| 休み（やすみ） | **休みの間に**、部屋を片付けた（かたづけた）。 | 放假期間整理了房間。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 去に（強調整段期間） | 子供が寝ている**間**、ずっと本を読んでいた。 | 小孩睡覺期間一直在看書。 |
| → 過去形 | 留守の間に、荷物が**届いていた**。 | 不在家期間，包裹已經送到了。 |
| → 疑問形 | 子供が寝ている間に**何をしましたか**。 | 小孩睡覺的時候你做了什麼？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| いつ家事をしたんですか。 | 子供が寝ている間に終わらせました。 | 趁小孩睡著的期間做完了。 |
| 荷物はいつ届いたんですか。 | 留守の間に届いたようです。 | 好像是不在家的時候送到的。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
