# 〜たびに

## 規則與接續
- `動詞辞書形 + たびに`
- `名詞 + の + たびに`

## 用法說明
表示「每當...就一定會...」，強調某動作或事件發生時必然伴隨另一現象，語感較 [[JP_Grammar_50_C18_toki|〜とき @related_to]] 更強調規律性重複。不適合用於如「每天起床、每天吃飯」等單純的日常規律行為。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E">この<ruby>曲<rt>きょく</rt></ruby>を<ruby>聴<rt>き</rt></ruby>くたびに、</span><span style="color:#2ECC71"><ruby>学生時代<rt>がくせいじだい</rt></ruby>の<ruby>楽<rt>たの</rt></ruby>しかった<ruby>思<rt>おも</rt></ruby>い<ruby>出<rt>で</rt></ruby>が</span><span style="color:#3498DB"><ruby>蘇<rt>よみがえ</rt></ruby>る。</span>
  中文：每當聽這首歌，就會喚起學生時代快樂的回憶。
- 日文：<span style="color:#2ECC71">彼は</span><span style="color:#FF477E"><ruby>出張<rt>しゅっちょう</rt></ruby>のたびに、</span><span style="color:#3498DB"><ruby>珍<rt>めずら</rt></ruby>しいお<ruby>土産<rt>みやげ</rt></ruby>を<ruby>買<rt>か</rt></ruby>ってきてくれる。</span>
  中文：他每次出差，都會買稀奇的名產回來給我們。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> この曲（きょく）を聴（き）くたびに、学生時代（がくせいじだい）の楽（たの）しかった思（おも）い出（で）が蘇（よみがえ）る。
> （每當聽這首歌，就會喚起學生時代快樂的回憶。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| この写真（しゃしん）を見る（みる） | **この写真を見るたびに**、懐かしい気持ちになる。 | 每次看這張照片，就會覺得懷念。 |
| 台湾へ帰る（たいわんへかえる） | **台湾へ帰るたびに**、太った気がする。 | 每次回台灣，都覺得胖了。 |
| 彼女に会う（かのじょにあう） | **彼女に会うたびに**、元気をもらう。 | 每次見到她，就會獲得元氣。 |
| 試験（しけん） | **試験のたびに**、緊張してしまう。 | 每次考試都會緊張。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 名詞形式 | **出張のたびに**、お土産を買ってくる。 | 每次出差都會買名產回來。 |
| → 過去語境 | この曲を聴くたびに、**思い出が蘇った**。 | 每次聽這首歌，就會喚起回憶。 |
| → 疑問形 | この曲を聴く**たびに、悲しくなりますか**。 | 每次聽這首歌都會難過嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうしてこの曲が好きなんですか。 | 聴くたびに、学生時代の楽しい思い出が蘇るからです。 | 因為每次聽都會喚起學生時代的快樂回憶。 |
| 彼はいつもお土産をくれますか。 | はい、出張のたびに買ってきてくれます。 | 是的，每次出差都會買回來給我們。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 聴くたびに蘇る。 | 每次聽就會喚起。 |
| +主語 | この曲を聴くたびに、**学生時代の思い出が**蘇る。 | 每次聽這首歌，學生時代的回憶就會浮現。 |
| +修飾 | この曲を聴くたびに、学生時代の**楽しかった**思い出が蘇る。 | 每次聽這首歌，學生時代快樂的回憶就會浮現。 |
