# 〜たとたんに

## 規則與接續
- `動詞た形 + とたんに`

## 用法說明
表示前項動作剛結束的瞬間，立刻引發後項預期外的突發事件。與「〜か〜ないかのうちに」相比，此句型更強調後項動作的意外性與驚訝感，且後項不能接意志句、命令句或請求句。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#3498DB"><ruby>家<rt>いえ</rt></ruby>を<ruby>出<rt>で</rt></ruby>たとたんに、</span><span style="color:#3498DB"><ruby>大雨<rt>おおあめ</rt></ruby>が<ruby>降<rt>ふ</rt></ruby>ってきた。</span>
  中文：剛一出門，就突然下起大雨了。
- 日文：<span style="color:#FF477E"><ruby>疲<rt>つか</rt></ruby>れていたのか、</span><span style="color:#3498DB">ベッドに<ruby>入<rt>はい</rt></ruby>ったとたんに<ruby>眠<rt>ねむ</rt></ruby>ってしまった。</span>
  中文：可能是太累了，剛鑽進被窩就馬上睡著了。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 家（いえ）を出（で）たとたんに、大雨（おおあめ）が降（ふ）ってきた。
> （剛一出門，就突然下起大雨了。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| ドアを開ける（あける） | **ドアを開けたとたんに**、犬が飛び出してきた。 | 剛一開門，狗就衝出來了。 |
| 座る（すわる） | **座ったとたんに**、電話が鳴った。 | 剛一坐下，電話就響了。 |
| 見る（みる） | 彼女を**見たとたんに**、恋に落ちた。 | 一看到她就墜入愛河了。 |
| 発表する（はっぴょうする） | 結果を**発表したとたんに**、会場がざわめいた。 | 一宣布結果，會場就騷動起來。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 換用瞬間句型 | 家を出るか出ない**かのうちに**、雨が降ってきた。 | 幾乎在出門的同時就下起雨了。 |
| → 過去語境延伸 | 家を出たとたんに、大雨が**降ってきてしまった**。 | 一出門，就突然下起大雨了（無奈語氣）。 |
| → 疑問形 | 家を出た**とたんに、雨が降ってきたんですか**。 | 是剛出門就下起雨了嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうして濡れているんですか。 | 家を出たとたんに、大雨が降ってきたんです。 | 因為剛出門就突然下起大雨了。 |
| なぜすぐ寝てしまったんですか。 | 疲れていたのか、ベッドに入ったとたんに眠ってしまいました。 | 可能太累了，一鑽進被窩就睡著了。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 出たとたんに降ってきた。 | 剛出門就下起來了。 |
| +主語 | 家を出たとたんに、**大雨が**降ってきた。 | 剛出門，就下起大雨了。 |
| +地點 | **家を**出たとたんに、大雨が降ってきた。 | 剛從家裡出門，就下起大雨了。 |
