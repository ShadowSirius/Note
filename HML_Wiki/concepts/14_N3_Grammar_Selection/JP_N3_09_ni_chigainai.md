# 〜に違いない

## 規則與接續
- `普通形（名詞・な形容詞去だ） + に違いない`

## 用法說明
表說話者基於客觀根據，帶有強烈確信的推測，意為「一定是、絕對是」。語氣比 [[JP_Grammar_50_C33_conjecture_deshou_kamoshirenai|〜でしょう @related_to]] 更肯定，接近書面／正式表達，口語中常換用「絶対に〜と思う」。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E"><ruby>毎日<rt>まいにち</rt></ruby><ruby>夜遅<rt>よるおそ</rt></ruby>くまで<ruby>勉強<rt>べんきょう</rt></ruby>していたから、</span><span style="color:#2ECC71">彼は</span><span style="color:#3498DB"><ruby>合格<rt>ごうかく</rt></ruby>するに<ruby>違<rt>ちが</rt></ruby>いない。</span>
  中文：他每天都讀書讀到深夜，一定能考上的。
- 日文：<span style="color:#3498DB"><ruby>鍵<rt>かぎ</rt></ruby>がない！</span><span style="color:#FF477E">さっきの<ruby>店<rt>みせ</rt></ruby>に</span><span style="color:#3498DB"><ruby>落<rt>お</rt></ruby>としたに<ruby>違<rt>ちが</rt></ruby>いない。</span>
  中文：鑰匙不見了！一定是在剛剛那家店弄丟的。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 毎日（まいにち）夜遅（よるおそ）くまで勉強（べんきょう）していたから、彼（かれ）は合格（ごうかく）するに違（ちが）いない。
> （他每天都讀書讀到深夜，一定能考上的。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 成功する（せいこうする） | 彼は**成功するに違いない**。 | 他一定會成功。 |
| 疲れている | 彼は**疲れているに違いない**。 | 他一定很累。 |
| 犯人だ（はんにんだ） | あの人が**犯人に違いない**。 | 那個人一定是犯人。 |
| 嘘だ（うそだ） | それは**嘘に違いない**。 | 那一定是謊話。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 彼は合格した**に違いない**。 | 他一定是考上了。 |
| → 語氣弱化（かもしれない） | 彼は合格する**かもしれない**。 | 他也許能考上。 |
| → 疑問形（少見，多用きっと） | きっと合格するでしょう**ね**。 | 一定會考上吧。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 彼は試験に受かると思いますか。 | 毎日勉強していたから、合格するに違いない。 | 他每天都讀書，一定能考上。 |
| 鍵はどこで落としたんですか。 | さっきの店に落としたに違いない。 | 一定是在剛剛那家店弄丟的。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 合格するに違いない。 | 一定能考上。 |
| +主語 | **彼は**合格するに違いない。 | 他一定能考上。 |
| +理由 | **毎日夜遅くまで勉強していたから、**彼は合格するに違いない。 | 因為每天讀到深夜，他一定能考上。 |
