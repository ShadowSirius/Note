# 〜における

## 規則與接續
- `名詞 + における + 名詞`
- 表示「在……方面／在……情境下／於……之中的」，多用於書面語、正式表達
- 可視為 `において` 的連體形，用來修飾後面的名詞

## 用法說明
「〜における」用來表示某個範圍、領域、場面或情境中的事物，語氣正式、偏書面，常見於論文、報告、新聞或公告。  
和「〜において」相比，「〜において」是句中使用的副詞形式；「〜における」則是放在名詞前面修飾名詞。  
另外，和「〜に関する」相比，「〜における」更強調「在某一領域／場面之中」，而「〜に関する」則是單純表示「關於」。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 現代社会（げんだいしゃかい）における子育（こそだ）ての課題（かだい）を説明（せつめい）します。
> （說明現代社會中的育兒課題。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 会社における情報管理 | 会社（かいしゃ）における情報（じょうほう）管理（かんり）を説明（せつめい）します。 | 說明公司中的資訊管理。 |
| 教育現場における問題 | 教育（きょういく）現場（げんば）における問題（もんだい）を説明（せつめい）します。 | 說明教育現場中的問題。 |
| 国際会議における英語使用 | 国際（こくさい）会議（かいぎ）における英語（えいご）使用（しよう）を説明（せつめい）します。 | 說明國際會議上的英語使用。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 「〜において」的句型 | 現代社会（げんだいしゃかい）において、子育（こそだ）ての課題（かだい）を説明（せつめい）します。 | 在現代社會中，說明育兒課題。 |
| → 改成更正式的名詞修飾 | 会社（かいしゃ）における情報（じょうほう）管理（かんり）は重要（じゅうよう）です。 | 公司中的資訊管理很重要。 |
| → 否定表現 | ここにおける規則（きそく）だけでは十分（じゅうぶん）ではありません。 | 只有這裡的規則還不夠。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| その研究は何についてですか。 | 現代社会（げんだいしゃかい）における子育（こそだ）ての課題（かだい）についてです。 | 是關於現代社會中的育兒課題。 |
| 会社では何が大切ですか。 | 会社（かいしゃ）における情報（じょうほう）管理（かんり）が大切（たいせつ）です。 | 公司中的資訊管理很重要。 |
| 国際会議で何が必要ですか。 | 国際（こくさい）会議（かいぎ）における英語（えいご）の使用（しよう）が必要（ひつよう）です。 | 國際會議上需要使用英語。 |

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 子育ての課題（こそだてのかだい） | 育兒課題 |
| 擴展 1 | 現代社会（げんだいしゃかい）における子育ての課題（こそだてのかだい） | 現代社會中的育兒課題 |
| 擴展 2 | 現代社会（げんだいしゃかい）における子育ての課題（こそだてのかだい）を説明（せつめい）します。 | 說明現代社會中的育兒課題 |
