# 〜ということだ

## 規則與接續
- `普通形 + ということだ`
- `名詞 + だ + ということだ`
- `な形容詞 + だ + ということだ`
- 可用於表示「聽說…／據說…」或「也就是說、換句話說…」

## 用法說明
`〜ということだ` 主要有兩種用法：一是根據來源轉述資訊，表示「聽說／據說」；二是對前文做整理歸納，表示「也就是說、換句話說」。和 `〜そうだ` 相比，`〜ということだ` 較正式，常見於新聞、公告、說明或報告語氣。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 先生（せんせい）によると、明日（あした）は休講（きゅうこう）だということだ。  
> （據老師說，明天停課。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 来月（らいげつ）から値段（ねだん）が上（あ）がる | 新聞（しんぶん）によると、来月（らいげつ）から値段（ねだん）が上（あ）がるということだ。 | 據報紙說，下個月價格會上漲。 |
| この薬（くすり）は食後（しょくご）に飲（の）む | 医者（いしゃ）によると、この薬（くすり）は食後（しょくご）に飲（の）むということだ。 | 據醫生說，這藥要飯後吃。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 聽說體 | 「明日（あした）は休講（きゅうこう）です」→ 先生（せんせい）によると、明日（あした）は休講（きゅうこう）だということだ。 | 「明天停課」→ 據老師說，明天停課。 |
| → 聽說體 | 「来月（らいげつ）から値上（ねあ）げします」→ 新聞（しんぶん）によると、来月（らいげつ）から値上（ねあ）げするということだ。 | 「下個月要漲價」→ 據報紙說，下個月要漲價。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 明日（あした）は授業（じゅぎょう）がありますか。 | 先生（せんせい）によると、明日（あした）は休講（きゅうこう）だということです。 | 明天有課嗎？據老師說，明天停課。 |
| この薬（くすり）はいつ飲（の）みますか。 | 医者（いしゃ）によると、食後（しょくご）に飲（の）むということです。 | 這藥什麼時候吃？據醫生說，飯後吃。 |
