# 〜つつある

## 規則與接續
- `動詞ます形（去ます） + つつある`

## 用法說明
表示某種變化正在進行中、尚未完成，屬於較生硬正式的書面語，常見於新聞報導或評論文章。與口語常見的「〜ている」相比，更強調「逐漸變化」的動態過程，且常接續如「増える、減る、進む」等表示變化趨勢的動詞，不用於單純動作的持續（如「食べている」不可換成「食べつつある」）。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

  中文：隨著科技進步，傳統文化正逐漸消失。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 台湾（たいわん）の少子化（しょうしか）問題（もんだい）は、深刻（しんこく）になりつつあります。
> （台灣的少子化問題正逐漸變得嚴重。）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 増える（ふえる） | 人口（じんこう）が**増えつつあります**。 | 人口正逐漸增加。 |
| 回復する（かいふくする） | 景気（けいき）が**回復しつつあります**。 | 景氣正逐漸恢復。 |
| 忘れられる（わすれられる） | 昔の習慣（しゅうかん）は**忘れられつつある**。 | 過去的習慣正逐漸被遺忘。 |
| 進む（すすむ） | 高齢化（こうれいか）が**進みつつある**。 | 高齡化正逐漸加劇。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去語境 | 問題は深刻になり**つつあった**。 | 問題當時正逐漸變得嚴重。 |

| → 疑問形 | 問題は深刻になり**つつありますか**。 | 問題正逐漸變得嚴重嗎？ |
