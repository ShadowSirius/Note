# 〜として

## 規則與接續
- `名詞 + として`

## 用法說明
表示「以...身份、立場或資格」，強調主體本身具備的客觀身分或名義。與表示評價基準的 [[JP_N3_28_nitotte|〜にとって @related_to]] 不同：として關注「是什麼身份」，にとって關注「對誰而言如何評價」。

- 日文：<span style="color:#2ECC71">彼は</span><span style="color:#FF477E"><ruby>留学生<rt>りゅうがくせい</rt></ruby>として</span><span style="color:#FF477E"><ruby>台湾<rt>たいわん</rt></ruby>から</span><span style="color:#3498DB"><ruby>日本<rt>にほん</rt></ruby>へ<ruby>来<rt>き</rt></ruby>ました。</span>
  中文：他以留學生的身分從台灣來到了日本。
- 日文：<span style="color:#2ECC71">この<ruby>建物<rt>たてもの</rt></ruby>は</span><span style="color:#FF477E"><ruby>現在<rt>げんざい</rt></ruby>、</span><span style="color:#3498DB"><ruby>美術館<rt>びじゅつかん</rt></ruby>として<ruby>使<rt>つか</rt></ruby>われています。</span>
  中文：這棟建築物現在被當作美術館使用。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）は留学生（りゅうがくせい）として台湾（たいわん）から日本（にほん）へ来（き）ました。
> （他以留學生的身分從台灣來到了日本。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 代表（だいひょう） | 彼は会社の**代表として**会議に出た。 | 他以公司代表的身分出席會議了。 |
| 通訳（つうやく） | 私は**通訳として**働いています。 | 我以口譯員的身分工作。 |
| 記念（きねん） | この写真は**記念として**大切にしている。 | 我把這張照片當作紀念珍藏著。 |
| 一員（いちいん） | チームの**一員として**頑張ります。 | 我會以團隊一員的身分努力。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 名詞修飾用法（Nとしての） | これは**社会人としての**責任です。 | 這是身為社會人士的責任。 |
| → 過去語境 | 彼は留学生として**日本へ来た**。 | 他過去以留學生身分來了日本。 |
| → 疑問形 | 彼は留学生**として来たんですか**。 | 他是以留學生的身分來的嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 彼はなぜ日本にいるんですか。 | 留学生として台湾から来ました。 | 他是以留學生的身分從台灣來的。 |
| この建物は今、何に使われていますか。 | 美術館として使われています。 | 現在被當作美術館使用。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 留学生として来ました。 | 以留學生身分來的。 |
| +主語 | **彼は**留学生として来ました。 | 他以留學生身分來的。 |
| +起點 | 彼は留学生として**台湾から**日本へ来ました。 | 他以留學生身分從台灣來到日本。 |
