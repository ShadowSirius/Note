# 〜やすい／〜にくい

## 規則與接續
- `動詞ます形（去masu） + やすい`（容易...做）
- `動詞ます形（去masu） + にくい`（難以...做）

## 用法說明
表示動作進行的難易度：「〜やすい」表示做該動作很容易、順手，也可延伸表示「容易發生某種傾向」（例：壊れやすい＝容易壞）；「〜にくい」表示做該動作有困難、不順手。兩者活用皆比照い形容詞。

與 [[JP_N3_36_gachi|〜がち]] 的辨析（延伸傾向用法時）：
- 「〜やすい」延伸表傾向時（如壊れやすい、間違えやすい），指「該事物本身的性質容易導致某結果」，是對物品/事物特性的客觀描述。
- 「〜がち」指「某主體的行為或狀態反覆發生」（如休みがち、遅れがち），聚焦於發生頻率而非事物本身特性，且多帶負面評價。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> このペンは軽（かる）くて使（つか）いやすい。
> （這支筆很輕，很好用。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 分かる（わかる） | この説明は**分かりやすい**。 | 這個說明很好懂。 |
| 割れる（われる） | このコップは**割れやすい**。 | 這個杯子容易破。 |
| 読む（よむ） | 彼の字は**読みにくい**。 | 他的字很難讀。 |
| 発音する（はつおんする） | この単語は**発音しにくい**。 | 這個單字很難發音。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 否定形 | このペンは使い**やすくない**。 | 這支筆不好用。 |
| → 過去形 | このペンは使い**やすかった**。 | 這支筆之前很好用。 |
| → やすい⇄にくい 對調 | このペンは重（おも）くて使い**にくい**。 | 這支筆很重，不好用。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| このペン、どうですか。 | 軽くて使いやすいですよ。 | 很輕、很好用喔。 |
| この漢字、読めますか。 | いいえ、複雑で覚えにくいです。 | 不會，太複雜了很難記。 |
