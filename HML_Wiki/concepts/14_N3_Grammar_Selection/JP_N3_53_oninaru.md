# お〜になる

## 規則與接續
- `お + 動詞ます形去掉「ます」 + になる`
- 常見敬語也可用 `ご + サ變動詞語幹 + になる`，如：`ご利用になる`、`ご覧になる`

## 用法說明
表示對對方或第三者的動作抬高，屬於尊敬語，意思是「您做……／他做……」。  
與 `〜られる` 相比，`お〜になる` 也能表示尊敬，但語感通常較明確、較正式；而 [[JP_N3_70_sonkeigo_verbs|おっしゃる・いらっしゃる・召し上がる @related_to]] 等則是不規則的尊敬表現，需另外記憶。與降低自身動作的謙讓語 [[JP_N3_52_osuru|お〜する @related_to]] 方向相反。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 社長（しゃちょう）は今（いま）お帰（かえ）りになります。  
> （社長現在要回去了。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 部長（ぶちょう） | 部長（ぶちょう）は今（いま）お帰（かえ）りになります。 | 部長現在要回去了。 |
| お客様（きゃくさま） | お客様（きゃくさま）は今（いま）お帰（かえ）りになります。 | 客人現在要回去了。 |
| 先生（せんせい） | 先生（せんせい）は今（いま）お帰（かえ）りになります。 | 老師現在要回去了。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 尊敬語 | 社長は会議に出ます。 → 社長は会議にお出（で）になります。 | 社長參加會議。→ 社長會出席會議。 |
| → 尊敬語 | 先生は本を読みます。 → 先生は本をお読（よ）みになります。 | 老師看書。→ 老師閱讀書籍。 |
| → 尊敬語 | 部長は資料を使います。 → 部長は資料をお使（つか）いになります。 | 部長使用資料。→ 部長會使用資料。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 社長はもう帰りましたか。 | はい、もうお帰（かえ）りになりました。 | 是的，已經回去了。 |
| 先生は何をしていますか。 | 先生は資料をご覧（らん）になっています。 | 老師正在看資料。 |
| 部長はいつ来ますか。 | 部長は10時にお見（み）えになります。 | 部長十點會來。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | お読みになります。 | 您閱讀。 |
| 擴充 1 | 先生はお読みになります。 | 老師閱讀。 |
| 擴充 2 | 先生はこの資料をお読みになります。 | 老師閱讀這份資料。 |
| 擴充 3 | 先生は今（いま）この資料をお読みになります。 | 老師現在正在閱讀這份資料。 |
