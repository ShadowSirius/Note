# 〜んじゃないか

## 規則與接續
- `動詞・い形容詞普通形 + んじゃないか`
- `な形容詞語幹 + なんじゃないか`
- `名詞 + なんじゃないか`
- `ない形 + んじゃないか`（例：行かないんじゃないか）

## 用法說明
表示根據眼前情況、線索或常識做出的**口語推測**，語感接近「不是嗎、我覺得是……吧」。也常用在想確認自己判斷是否正確時，帶有「大概就是這樣吧？」的輕微反問感。

與 `〜のではないか` 相比，`〜んじゃないか` 更口語、更常用於對話；`〜のではないか` 則較正式、書面。與單純的 `〜だろう` 相比，`〜んじゃないか` 更像是「根據理由做出的判斷」，語氣更貼近說話者的直覺。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> この値段（ねだん）は、少（すこ）し高（たか）すぎるんじゃないか。
> （這個價格，是不是有點太貴了？）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 安（やす）すぎる | この値段（ねだん）は、少（すこ）し安（やす）すぎるんじゃないか。 | 這個價格，是不是有點太便宜了？ |
| 速（はや）すぎる | この作業（さぎょう）は、少（すこ）し速（はや）すぎるんじゃないか。 | 這個工作，是不是有點太快了？ |
| 難（むずか）しすぎる | この問題（もんだい）は、少（すこ）し難（むずか）しすぎるんじゃないか。 | 這個問題，是不是有點太難了？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 推測語氣 | この店（みせ）は閉（しま）まっているんじゃないか。 | 這間店，是不是已經關了？ |
| → 名詞句 | 彼（かれ）の荷物（にもつ）なんじゃないか。 | 不是他的行李嗎？ |
| → な形容詞句 | この部屋（へや）は静（しず）かなんじゃないか。 | 這個房間，是不是很安靜？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうして電気（でんき）が消（き）えているんですか。 | もう誰（だれ）もいないんじゃないか。 | 為什麼燈是關著的？ 可能已經沒人了吧。 |
| この音（おと）、何（なん）ですか。 | 上（うえ）の階（かい）で何（なに）か落（お）ちたんじゃないか。 | 這個聲音是什麼？ 可能是樓上有東西掉了。 |
| あの人（ひと）は来（こ）ないのかな。 | もう帰（かえ）ったんじゃないか。 | 那個人會不會不來了？ 可能已經回去了吧。 |
