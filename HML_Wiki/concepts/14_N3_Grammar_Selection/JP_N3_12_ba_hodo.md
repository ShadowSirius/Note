# 〜ば〜ほど

## 規則與接續
- `動詞ば形 + 同一動詞辞書形 + ほど`
- `い形容詞（去い）ければ + い形容詞 + ほど`
- `な形容詞 + なら + な形容詞 + な + ほど`

## 用法說明
表示「越...越...」，前項程度改變後項也隨之增減。建立在 [[JP_Grammar_50_C36_ba_conditional|ば條件形 @related_to]] 之上，前後常重複同一詞彙。與單純表示比例變化的「〜につれて」不同，此句型更強調條件與結果的正比關係，帶有較強的規律性語感。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71"><ruby>外国語<rt>がいこくご</rt></ruby>は、</span><span style="color:#3498DB"><ruby>話<rt>はな</rt></ruby>せば<ruby>話<rt>はな</rt></ruby>すほど<ruby>上手<rt>じょうず</rt></ruby>になりますよ。</span>
  中文：外語是越說會越進步的喔。
- 日文：<span style="color:#2ECC71">パソコンの<ruby>操作<rt>そうさ</rt></ruby>は</span><span style="color:#3498DB"><ruby>簡単<rt>かんたん</rt></ruby>なら<ruby>簡単<rt>かんたん</rt></ruby>なほどいいです。</span>
  中文：電腦的操作越簡單越好。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 外国語（がいこくご）は、話（はな）せば話（はな）すほど上手（じょうず）になりますよ。
> （外語是越說會越進步的喔。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 考える（かんがえる） | **考えれば考えるほど**分からなくなる。 | 越想越搞不懂。 |
| 高い（たかい） | 値段（ねだん）は**高ければ高いほど**品質（ひんしつ）がいいとは限らない。 | 價格不一定越高品質越好。 |
| 練習する（れんしゅうする） | **練習すればするほど**上手になる。 | 越練習就越熟練。 |
| 早い（はやい） | **早ければ早いほど**いいです。 | 越早越好。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去語境 | 話せば話すほど、**上手になった**。 | 越說就變得越熟練了。 |
| → 否定語境 | あまり話さなければ、**上手になりません**。 | 如果不太說，就不會變熟練。 |
| → 疑問形 | 話せば話すほど**上手になりますか**。 | 越說會越進步嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 外国語はどうすれば上手になりますか。 | 話せば話すほど上手になりますよ。 | 越說會越進步喔。 |
| パソコンの操作はどんなのがいいですか。 | 簡単なら簡単なほどいいです。 | 越簡單越好。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 話すほど上手になる。 | 越說越熟練。 |
| +條件 | **話せば話すほど**上手になる。 | 越說越是熟練。 |
| +主語 | **外国語は**、話せば話すほど上手になりますよ。 | 外語是越說會越進步的喔。 |
