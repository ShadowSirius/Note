# 〜としても

## 規則與接續
- `普通形 + としても`

## 用法說明
表示假設性的讓步，「就算...也...」。與基本讓步句型 [[JP_Grammar_50_C23_temo_conditional|〜ても @related_to]] 相似，但「〜としても」更強調該假設本身成立可能性極低，帶有「退一步就算真是這樣」的語氣，常用於反駁或加強論點。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E">たとえ</span><span style="color:#3498DB"><ruby>失敗<rt>しっぱい</rt></ruby>したとしても、</span><span style="color:#2ECC71">この<ruby>経験<rt>けいけん</rt></ruby>は</span><span style="color:#3498DB"><ruby>絶対<rt>ぜったい</rt></ruby>に<ruby>無駄<rt>むだ</rt></ruby>にはならない。</span>
  中文：就算失敗了，這個經驗也絕對不會白費。
- 日文：<span style="color:#FF477E"><ruby>明日<rt>あした</rt></ruby><ruby>雨<rt>あめ</rt></ruby>が<ruby>降<rt>ふ</rt></ruby>るとしても、</span><span style="color:#3498DB"><ruby>予定通<rt>よていどお</rt></ruby>りに<ruby>出発<rt>しゅっぱつ</rt></ruby>します。</span>
  中文：即使明天下雨，我們也會照原定計畫出發。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> たとえ失敗（しっぱい）したとしても、この経験（けいけん）は絶対（ぜったい）に無駄（むだ）にはならない。
> （就算失敗了，這個經驗也絕對不會白費。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| お金がある（かねがある） | たとえ**お金があったとしても**、幸せとは限らない。 | 就算有錢，也不一定幸福。 |
| 反対される（はんたいされる） | たとえ**反対されたとしても**、私はやります。 | 就算被反對，我還是要做。 |
| 忙しい（いそがしい） | たとえ**忙しいとしても**、連絡してください。 | 就算很忙，也請聯絡我。 |
| 有名だ（ゆうめいだ） | たとえ**有名だとしても**、努力は必要だ。 | 就算有名，努力還是必要的。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 失敗した**としても**、後悔（こうかい）はない。 | 就算是失敗了，也不後悔。 |
| → 換用ても | 失敗し**ても**、経験は無駄にならない。 | 就算失敗，經驗也不會白費。 |
| → 疑問形 | 失敗した**としても**大丈夫ですか。 | 就算失敗了也沒關係嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 失敗が怖くないんですか。 | たとえ失敗したとしても、経験は無駄にならないと思います。 | 就算失敗了，我覺得經驗也不會白費。 |
| 明日は行きますか。雨が降りそうですが。 | 雨が降るとしても、予定通りに出発します。 | 即使下雨，也會照原定計畫出發。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 失敗したとしても。 | 就算失敗了。 |
| +結果 | 失敗したとしても、**無駄にはならない**。 | 就算失敗了，也不會白費。 |
| +強調 | **たとえ**失敗したとしても、この経験は**絶対に**無駄にはならない。 | 就算失敗了，這個經驗也絕對不會白費。 |
