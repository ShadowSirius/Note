# 〜気味

## 規則與接續
- `動詞ます形（去masu） + 気味`（例：疲れる→疲れ気味）
- `名詞 + 気味`（例：風邪→風邪気味）

## 用法說明
表示「有點...的徵兆／傾向」，用來客觀描述身心或事物目前呈現的輕微狀態變化，語氣中性，較少帶主觀評價，常見於健康、疲勞、緊張等暫時性狀態的描述（如風邪気味＝有點感冒的徵兆）。

與 [[JP_N3_34_ppoi|〜っぽい]]、[[JP_N3_36_gachi|〜がち]] 的辨析：
- 「〜気味」：客觀描述「當下狀態」的輕微傾向，語氣中性，多用於健康、心理狀態（疲れ気味、緊張気味）。
- 「〜っぽい」：描述「性格或外觀」給人的整體印象，偏口語且常帶輕微貶意。
- 「〜がち」：描述某行為「頻繁發生」的傾向，強調次數多，且多為負面（如遅刻がち）。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 最近（さいきん）、風邪気味（かぜぎみ）で体（からだ）がだるい。
> （最近有點感冒的徵兆，身體發沉。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 太る（ふとる） | 最近、**太り気味**だ。 | 最近有點發胖的傾向。 |
| 緊張する（きんちょうする） | 面接の前は**緊張気味**になる。 | 面試前會有點緊張。 |
| 寝不足（ねぶそく） | 今週はずっと**寝不足気味**だ。 | 這週一直有點睡眠不足。 |
| 遅れる（おくれる） | 電車が**遅れ気味**だ。 | 電車有點誤點的徵兆。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → がち（頻率化） | 最近、体調を崩し**がち**だ。 | 最近身體狀況常常變差。（改為強調頻繁發生） |
| → 丁寧形 | 最近、風邪気味で体がだるい**です**。 | 最近有點感冒的徵兆，身體發沉。（禮貌形） |
| → 過去形 | 先週は風邪気味**だった**。 | 上週有點感冒的徵兆。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 顔色が悪いですね、大丈夫ですか。 | ちょっと風邪気味なんです。 | 有點感冒的徵兆。 |
| 彼、最近疲れているみたいですね。 | ええ、疲れ気味だから休ませたほうがいいですね。 | 對啊，他有點累，讓他休息比較好。 |
