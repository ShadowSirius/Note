# 〜たまま

## 規則與接續
- `動詞た形 + まま`
- `名詞 + の + まま`
- 表示「做完 A 之後，A 所形成的狀態保持不變，接著進行 B」；也可表示某種狀態一直維持到說話時。

## 用法說明
「〜たまま」用來表達動作或變化發生後，原本的狀態沒有改變，例如「開著窗」、「穿著鞋」、「開著電視」等。  
和「〜ながら」不同，「〜ながら」是兩個動作同時進行；「〜たまま」則重點在「狀態持續不變」。另外，和「〜っぱなし」相比，「〜っぱなし」常帶有「放著不管、忘了處理」的負面語感，「〜たまま」則較中性。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 窓(まど)を開(あ)けたまま寝(ね)てしまった。  
> （我開著窗就睡著了。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 電気(でんき)をつける | 電気(でんき)をつけたまま出(で)かけた。 | 開著燈就出門了。 |
| 靴(くつ)を履(は)く | 靴(くつ)を履(は)いたまま家(いえ)に入(はい)った。 | 穿著鞋就進了家門。 |
| テレビをつける | テレビをつけたまま寝(ね)てしまった。 | 開著電視就睡著了。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 〜たままにする | ドア(どあ)を開(あ)けたままにしておいてください。 | 請先把門開著。 |
| → AてB → AたままB | 彼(かれ)はドア(どあ)を開(あ)けて部屋(へや)を出(で)た。 → 彼(かれ)はドア(どあ)を開(あ)けたまま部屋(へや)を出(で)た。 | 他開著門就離開了房間。 |
| → AてB → AたままB | 子(こ)どもは靴(くつ)をはいて寝(ね)た。 → 子(こ)どもは靴(くつ)を履(は)いたまま寝(ね)た。 | 孩子穿著鞋睡了。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうして部屋(へや)が寒(さむ)かったんですか。 | 窓(まど)を開(あ)けたまま寝(ね)てしまったからです。 | 因為我開著窗就睡著了。 |
| どうして電気(でんき)がついたままなんですか。 | つけたまま出(で)かけてしまったんです。 | 因為我出門時忘了關。 |
| どうして靴(くつ)のまま入(はい)ったんですか。 | 急(いそ)いでいたので、靴(くつ)を履(は)いたまま入(はい)ってしまいました。 | 因為太趕了，所以就穿著鞋進去了。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 窓(まど)を開(あ)けたまま。 | 開著窗。 |
| 擴充 1 | 窓(まど)を開(あ)けたまま寝(ね)てしまった。 | 開著窗就睡著了。 |
| 擴充 2 | 窓(まど)を開(あ)けたまま寝(ね)てしまったので、朝(あさ)は寒(さむ)かった。 | 因為開著窗就睡著了，所以早上很冷。 |
