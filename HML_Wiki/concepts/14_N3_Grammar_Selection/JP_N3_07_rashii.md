# 〜らしい

## 規則與接續
- `名詞/普通形（名詞・な形容詞去だ） + らしい`

## 用法說明
基於客觀事實或傳聞得出的推測，帶有「聽說、似乎」的語感，與傳聞句型 [[JP_Grammar_50_C46_hearsay_sou_desu|〜そうだ（傳聞）@related_to]] 相近但更含蓄。與主觀比喻的 [[JP_N3_08_you_da|〜ようだ @related_to]] 不同，らしい更側重客觀依據而非個人直覺。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E"><ruby>天気予報<rt>てんきよほう</rt></ruby>によると、</span><span style="color:#FF477E"><ruby>明日<rt>あした</rt></ruby>は</span><span style="color:#3498DB"><ruby>大雨<rt>おおあめ</rt></ruby>になるらしい。</span>
  中文：根據天氣預報，明天似乎會下大雨。
- 日文：<span style="color:#FF477E"><ruby>噂<rt>うわさ</rt></ruby>によると、</span><span style="color:#2ECC71">あの<ruby>二人<rt>ふたり</rt></ruby>は</span><span style="color:#FF477E"><ruby>来月<rt>らいげつ</rt></ruby></span><span style="color:#3498DB"><ruby>結婚<rt>けっこん</rt></ruby>するらしいよ。</span>
  中文：聽傳聞說，那兩個人下個月好像要結婚喔。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 天気予報（てんきよほう）によると、明日（あした）は大雨（おおあめ）になるらしい。
> （根據天氣預報，明天似乎會下大雨。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 台風が来る（たいふうがくる） | 明日は**台風が来るらしい**。 | 明天似乎有颱風要來。 |
| 晴れる（はれる） | 明日は**晴れるらしい**。 | 明天似乎會放晴。 |
| 寒い（さむい） | 明日は**寒いらしい**。 | 明天似乎很冷。 |
| 休みだ | 明日は会社が**休みらしい**。 | 公司明天似乎放假。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 昨日は大雨だった**らしい**。 | 昨天似乎下了大雨。 |
| → 否定形 | 明日は雨にならない**らしい**。 | 明天似乎不會下雨。 |
| → 疑問形 | 明日は大雨になる**らしいですか**。 | 明天似乎會下大雨嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 明日の天気はどうですか。 | 天気予報によると、大雨になるらしいです。 | 根據天氣預報，似乎會下大雨。 |
| あの二人はどうなるんですか。 | 噂によると、来月結婚するらしいですよ。 | 聽說下個月好像要結婚喔。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 大雨になるらしい。 | 似乎會下大雨。 |
| +時間 | **明日は**大雨になるらしい。 | 明天似乎會下大雨。 |
| +根據 | **天気予報によると、**明日は大雨になるらしい。 | 根據天氣預報，明天似乎會下大雨。 |
