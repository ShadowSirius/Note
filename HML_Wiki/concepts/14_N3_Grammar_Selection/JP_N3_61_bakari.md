# 〜ばかり

## 規則與接續
- 名詞 + ばかり
- 動詞て形 + ばかり（してばかりいる）
- 〜ばかりで / 〜ばかりだ

## 用法說明
表示某種人事物或行為「占得太多、太偏」，帶有抱怨、嫌多、失衡的語氣，可譯為「淨是、光是...」。

和「だけ」相比，「ばかり」更強調比例偏重與不滿；和「しか〜ない」相比，「ばかり」不一定要接否定，但常有「怎麼又是這個」的感覺。另外，「動詞た形 + ばかり」也可表示「剛...」，要和這裡的「偏重」用法分開。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> この部屋(へや)は本(ほん)ばかりで、玩具(がんぐ)が全然(ぜんぜん)ない。
> （這個房間淨是書，完全沒有玩具。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 野菜(やさい) | 冷蔵庫(れいぞうこ)は野菜(やさい)ばかりで、肉(にく)がない。 | 冰箱裡淨是蔬菜，沒有肉。 |
| 仕事(しごと) | 彼(かれ)は仕事(しごと)ばかりで、休(やす)めない。 | 他淨是工作，沒辦法休息。 |
| テレビ | 子どもはテレビばかり見(み)ている。 | 小孩淨是在看電視。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| お菓子(かし)だけ食(た)べる。 → | お菓子(かし)ばかり食(た)べる。 | 淨是吃零食。 |
| テレビを見る。 → | テレビばかり見(み)ている。 | 淨是看電視。 |
| 仕事(しごと)をする。 → | 仕事(しごと)ばかりしている。 | 淨是在工作。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 最近(さいきん)、何(なに)をしていますか。 | 仕事(しごと)ばかりしています。 | 最近在做什麼？——淨是在工作。 |
| 何(なに)を食(た)べていますか。 | 野菜(やさい)ばかり食(た)べています。 | 你在吃什麼？——淨是吃蔬菜。 |
| どうして疲(つか)れているんですか。 | 仕事(しごと)ばかりで、休(やす)めないからです。 | 為什麼這麼累？——因為淨是工作，沒法休息。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 仕事(しごと)ばかりだ。 | 淨是工作。 |
| 擴展 1 | 仕事(しごと)ばかりで、休(やす)めない。 | 淨是工作，沒辦法休息。 |
| 擴展 2 | 仕事(しごと)ばかりで、休(やす)めないうえに、友達(ともだち)に会(あ)う時間(じかん)もない。 | 淨是工作，不但沒辦法休息，連見朋友的時間也沒有。 |
