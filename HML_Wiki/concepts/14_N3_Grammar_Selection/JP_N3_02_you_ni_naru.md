# 〜ようになる

## 規則與接續
- `動詞辞書形/可能形 + ようになる`
- `動詞ない形 + なくなる`

## 用法說明
表示能力、習慣或狀態隨時間逐漸產生的轉變，如「不會」變成「會」、「沒做」變成「有做」。與客觀決定產生結果的「〜ことになる」不同，「〜ようになる」強調變化本身的漸進過程，而非決定的結果。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E"><ruby>毎日<rt>まいにち</rt></ruby><ruby>練習<rt>れんしゅう</rt></ruby>したら、</span><span style="color:#FF477E">少しずつ</span><span style="color:#3498DB"><ruby>日本語<rt>にほんご</rt></ruby>が<ruby>話<rt>はな</rt></ruby>せるようになりました。</span>
  中文：每天練習之後，漸漸變得會說日文了。
- 日文：<span style="color:#FF477E"><ruby>最近<rt>さいきん</rt></ruby>は</span><span style="color:#FF477E"><ruby>健康<rt>けんこう</rt></ruby>のために、</span><span style="color:#2ECC71"><ruby>毎朝<rt>まいあさ</rt></ruby></span><span style="color:#3498DB"><ruby>野菜<rt>やさい</rt></ruby>を<ruby>食<rt>た</rt></ruby>べるようになりました。</span>
  中文：最近為了健康，變得每天早上都會吃蔬菜了。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 毎日（まいにち）練習（れんしゅう）したら、日本語（にほんご）が話（はな）せるようになりました。
> （每天練習之後，漸漸變得會說日文了。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 漢字（かんじ）が読める（よめる） | 練習したら、**漢字が読めるように**なりました。 | 練習之後變得會讀漢字了。 |
| 一人で料理（りょうり）ができる | 練習したら、**一人で料理ができるように**なりました。 | 練習之後變得會自己做菜了。 |
| 早く起きる（おきる） | 練習したら、**早く起きるように**なりました。 | 練習之後變得會早起了。 |
| 甘い物（あまいもの）を食べない | 練習したら、**甘い物を食べなく**なりました。 | 練習之後變得不吃甜食了。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 否定形 | まだ日本語が話せる**ようになりません**。 | 還沒有變得會說日文。 |
| → 疑問形 | 日本語が話せる**ようになりました**か。 | 變得會說日文了嗎？ |
| → 過去否定 | 去年（きょねん）は話せる**ようになっていませんでした**。 | 去年還沒有變得會說。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 日本語が話せるようになりましたか。 | はい、練習したら話せるようになりました。 | 是的，練習之後就會說了。 |
| どうして野菜を食べるようになったんですか。 | 健康のために食べるようになりました。 | 為了健康才變得會吃了。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 話せるようになりました。 | 變得會說了。 |
| +主語 | 日本語が**話せるようになりました**。 | 日文變得會說了。 |
| +條件 | 毎日練習したら、**日本語が話せるようになりました**。 | 每天練習後，日文變得會說了。 |
| +程度 | 毎日練習したら、**少しずつ**日本語が話せるようになりました。 | 每天練習後，漸漸地日文變得會說了。 |
