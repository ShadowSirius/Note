# 〜ようだ

## 規則與接續
- `動詞/い形容詞 普通形 + ようだ`
- `な形容詞 + な / 名詞 + の + ようだ`

## 用法說明
表主觀推測或比喻，口語與書面語皆可用。比 [[JP_N3_07_rashii|〜らしい @related_to]] 更憑直覺與主觀感受，且較口語的 [[JP_N3_06_mitai_da|〜みたいだ @related_to]] 更正式、書面。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#3498DB">誰もいないね。どうやら<ruby>道<rt>みち</rt></ruby>を<ruby>間違<rt>まちが</rt></ruby>えたようだ。</span>
  中文：都沒有人在呢。我們好像走錯路了。
- 日文：<span style="color:#2ECC71">彼女の<ruby>笑顔<rt>えがお</rt></ruby>は、</span><span style="color:#3498DB">まるで<ruby>太陽<rt>たいよう</rt></ruby>のように<ruby>明<rt>あか</rt></ruby>るい。</span>
  中文：她的笑容宛如太陽般明朗。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> どうやら道（みち）を間違（まちが）えたようだ。
> （我們好像走錯路了。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 風邪をひいた（かぜをひいた） | どうやら**風邪をひいたようだ**。 | 好像感冒了。 |
| 誰かいる（だれかいる） | どうやら**誰かいるようだ**。 | 好像有人在。 |
| 元気だ（げんきだ） | どうやら**元気なようだ**。 | 好像很有精神。 |
| 雨だ | どうやら**雨のようだ**。 | 好像在下雨。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 道を間違えた**ようだった**。 | 當時好像走錯路了。 |
| → 比喻用法（まるで〜ように） | まるで夢（ゆめ）のように美しい。 | 宛如夢境般美麗。 |
| → 名詞修飾 | 間違えた**ような**気（き）がする。 | 有種好像走錯的感覺。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 誰もいませんね、どうしたんですか。 | どうやら道を間違えたようです。 | 我們好像走錯路了。 |
| 彼女の笑顔はどうですか。 | まるで太陽のように明るいですよ。 | 宛如太陽般明朗喔。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 間違えたようだ。 | 好像弄錯了。 |
| +對象 | **道を**間違えたようだ。 | 好像走錯路了。 |
| +語氣 | **どうやら**道を間違えたようだ。 | 看來好像是走錯路了。 |
