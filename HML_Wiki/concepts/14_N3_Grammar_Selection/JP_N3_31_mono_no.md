# 〜ものの

## 規則與接續
- `動詞普通形 + ものの`
- `い形容詞普通形 + ものの`
- `な形容詞語幹 + な + ものの`
- `名詞 + である + ものの`

## 用法說明
表示前項事實已經成立，但後項出現與預期不符的結果，屬於逆接用法，語感偏書面、客觀，較少帶強烈情緒。

與 [[JP_N3_39_noni|〜のに]]、[[JP_N3_38_nagara_mo|〜ながらも]] 的辨析：
- 「〜ものの」：客觀陳述前後矛盾的事實，語氣較平淡、書面，常見於新聞、評論等正式場合。
- 「〜のに」：帶有強烈的不滿、驚訝或遺憾等主觀情緒，口語中使用頻繁。
- 「〜ながらも」：強調兩件事「同時並存」卻互相矛盾，語感介於「ものの」與「のに」之間，也偏書面，常見「知りながらも」等固定搭配。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 給料（きゅうりょう）は上（あ）がったものの、生活（せいかつ）は楽（らく）にならない。
> （雖然薪水漲了，但生活並沒有變輕鬆。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 頭では分かっている（あたまではわかっている） | **頭では分かっているものの**、実行できない。 | 雖然道理都懂，但做不到。 |
| 忙しい（いそがしい） | **忙しいものの**、毎日充実している。 | 雖然很忙，但每天都很充實。 |
| まだ若い（わかい） | **まだ若いものの**、経験は豊富だ。 | 雖然還年輕，但經驗豐富。 |
| 会社員だ | **会社員であるものの**、副業もしている。 | 雖然是上班族，但也有在做副業。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → のに（更強烈的不滿） | 給料は上がった**のに**、生活は楽にならない。 | 明明薪水漲了，生活卻沒有變輕鬆。（帶不滿語氣） |
| → ながらも | 給料は上がり**ながらも**、生活は楽にならない。 | 薪水雖然漲了，生活卻依然沒有變輕鬆。 |
| → 丁寧形 | 給料は上がったものの、生活は楽になり**ません**。 | 雖然薪水漲了，但生活並沒有變輕鬆。（禮貌形） |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 給料は上がったんですか。 | ええ、上がったものの、生活は楽になりませんね。 | 是啊，雖然漲了，但生活沒有變輕鬆呢。 |
| 彼は謝ったんですか。 | はい、謝ったものの、まだ怒っている人もいますよ。 | 是的，雖然道歉了，但還是有人在生氣。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 楽にならない。 | 不會變輕鬆。 |
| +主題 | **生活は**楽にならない。 | 生活不會變輕鬆。 |
| +逆接條件 | **給料は上がったものの、**生活は楽にならない。 | 雖然薪水漲了，但生活並沒有變輕鬆。 |
