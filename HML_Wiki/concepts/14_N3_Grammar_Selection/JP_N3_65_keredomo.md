# 〜けれど（も）

## 規則與接續
- `動詞普通形 + けれど（も）`
- `い形容詞普通形 + けれど（も）`
- `な形容詞語幹 + だ + けれど（も）`
- `名詞 + だ + けれど（も）`

## 用法說明
表示前後兩句內容相反或有對照，中文可譯為「雖然……但是……」「……不過……」。`けれど（も）` 比 `しかし` 更口語，也比帶有明顯不滿、遺憾語氣的 `〜のに` 來得溫和。和 `〜が` 類似，但 `けれど（も）` 在日常會話中更常見，句尾單獨使用時也可用來婉轉收尾、留有餘地。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> この靴(くつ)は高(たか)いけれど、長(なが)く使(つか)えます。
> （這雙鞋雖然貴，但是很耐用。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| この映画(えいが)は長(なが)い | この映画(えいが)は長(なが)いけれど、最後(さいご)まで面白(おもしろ)いです。 | 這部電影雖然很長，但一直都很有趣。 |
| この部屋(へや)は狭(せま)い | この部屋(へや)は狭(せま)いけれど、静(しず)かです。 | 這個房間雖然小，但很安靜。 |
| この本(ほん)は難(むずか)しい | この本(ほん)は難(むずか)しいけれど、勉強(べんきょう)になります。 | 這本書雖然難，但很有學習價值。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 句尾加上「けれど」 | 今日は寒(さむ)いです。→ 今日は寒(さむ)いけれど、出(で)かけます。 | 今天雖然冷，但還是要出門。 |
| → 轉成逆接表達 | 彼(かれ)は親切(しんせつ)です。→ 彼(かれ)は親切(しんせつ)だけれど、少(すこ)し無口(むくち)です。 | 他雖然親切，但有點沉默寡言。 |
| → 連接兩句 | 値段(ねだん)は高(たか)いです。品質(ひんしつ)はいいです。→ 値段(ねだん)は高(たか)いけれど、品質(ひんしつ)はいいです。 | 價格雖高，但品質很好。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうしてそのカバンを買(か)ったんですか。 | <span>安(やす)くはないけれど、</span>毎日(まいにち)使(つか)いやすいからです。 | 雖然不便宜，但因為每天都很好用。 |
| 日本語(にほんご)の勉強(べんきょう)は楽(たの)しいですか。 | <span>難(むずか)しいけれど、</span>楽(たの)しいです。 | 雖然難，但很有趣。 |
| 週末(しゅうまつ)は休(やす)めますか。 | <span>少(すこ)し忙(いそが)しいけれど、</span>休(やす)めます。 | 雖然有點忙，但可以休息。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 行(い)きます。 | 要去。 |
| 擴展 1 | 行(い)きたいけれど、 | 雖然想去，但是…… |
| 擴展 2 | 行(い)きたいけれど、用事(ようじ)があります。 | 雖然想去，但是有事。 |
| 擴展 3 | 友達(ともだち)と<ruby>会<rt>あ</rt></ruby>いたいけれど、用事(ようじ)があります。 | 雖然想和朋友見面，但是有事。 |
