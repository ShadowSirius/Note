# 〜ゆえの

## 規則與接續
- `名詞 + ゆえの + 名詞`（連體修飾用法）

## 用法說明
是 [[JP_N1_54_gayueni|〜がゆえに @related_to]] 的連體修飾變化形，意為「因為...而產生的...」，直接置於名詞前修飾後面的名詞，表示該名詞所指的狀態或事物是前面原因所導致的結果，書面語感強，常見於文學性敘述與評論文章。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 | 🔵核心/謂語 | 🟠修飾語 | 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E">それは</span><span style="color:#3498DB"><ruby>若<rt>わか</rt></ruby>さゆえの<ruby>過<rt>あやま</rt></ruby>ちだった。</span>
  中文：那是年輕才會犯的過錯。
- 日文：<span style="color:#2ECC71">これは</span><span style="color:#3498DB"><ruby>無知<rt>むち</rt></ruby>ゆえの<ruby>発言<rt>はつげん</rt></ruby>だ。</span>
  中文：這是因為無知才說出的話。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> それは若（わか）さゆえの過（あやま）ちだった。
> （那是年輕才會犯的過錯。）

### 1️⃣ Substitution Drill（替換練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 愛情（あいじょう） | それは**愛情ゆえの**厳しさだった。 | 那是因為愛才有的嚴格。 |
| 無知（むち） | **無知ゆえの**恐怖もある。 | 也有因無知而產生的恐懼。 |

### 2️⃣ Transformation Drill（轉換練習）
| Transformation | Expected Response | Translation |
|---|---|---|
| → 換用がゆえに | 若い**がゆえに**、過ちを犯した。 | 因為年輕，犯了過錯。（述語版） |

### 3️⃣ Response Drill（應答練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| なぜあんな発言をしたのですか。 | 無知ゆえの発言でした。 | 那是因為無知才說出的話。 |

### 4️⃣ Expansion Drill（擴展練習）
| Step | Sentence | Translation |
|---|---|---|
| 核心 | 過ちだった。 | 是個過錯。 |
| +原因修飾 | **若さゆえの**過ちだった。 | 是年輕才會犯的過錯。 |
