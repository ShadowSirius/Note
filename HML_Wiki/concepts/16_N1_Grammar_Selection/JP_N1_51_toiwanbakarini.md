# 〜と言わんばかりに

## 規則與接續
- `普通形・引用句 + と言わんばかりに`

## 用法說明
意為「簡直像是要說...似地」，是 [[JP_N1_35_tobakarini|〜とばかりに @related_to]] 的完整形（明示「言う」動詞＋んばかりに），語意完全相同，僅語感上更清楚點出「彷彿要說出口」的動作主體，兩者可視為同一文法點的完整形／省略形關係，實務中「とばかりに」使用頻率略高。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼女（かのじょ）は「これで終（お）わり」と言（い）わんばかりにドアを閉（し）めた。
> （她簡直像是要說「就此結束」似地關上了門。）

### 1️⃣ Substitution Drill（替換練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 「関係ない」 | 「**関係ない**」と言わんばかりに、無視した。 | 一副「不相干」似地無視了。 |
| 「勝った」 | 「**勝った**」と言わんばかりに、笑った。 | 一副「贏了」似地笑了。 |

### 2️⃣ Transformation Drill（轉換練習）
| Transformation | Expected Response | Translation |
|---|---|---|
| → 換用とばかりに | 「終わり」**とばかりに**ドアを閉めた。 | 簡化版（同義互換）。 |

### 3️⃣ Response Drill（應答練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 彼女はどんな様子でドアを閉めたのですか。 | 「これで終わり」と言わんばかりに閉めました。 | 一副「就此結束」似地關上的。 |

### 4️⃣ Expansion Drill（擴展練習）
| Step | Sentence | Translation |
|---|---|---|
| 核心 | ドアを閉めた。 | 關上了門。 |
| +暗示語氣 | **「これで終わり」と言わんばかりに**ドアを閉めた。 | 一副「就此結束」似地關上了門。 |
