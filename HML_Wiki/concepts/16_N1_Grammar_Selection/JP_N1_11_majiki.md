# 〜まじき

## 規則與接續
- `動詞辭書形 + まじき + 名詞`（する→するまじき為現代主流；すまじき為文語古形，仍見於固定書面表現，兩形並存）

## 用法說明
連體修飾用法，意為「不該有的...」，用於強烈譴責某行為與其身分、立場完全不相稱，常見於「〜にあるまじき行為」（身為...不該有的行為）的固定搭配。與 [[JP_N1_01_bekarazu|〜べからず @related_to]] 同屬文語否定家族，但「まじき」限定連體修飾名詞、語氣更偏向道德譴責而非規則禁止。（見上「規則與接續」的說明）

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 教師（きょうし）にあるまじき行為（こうい）だ。
> （這是身為教師不該有的行為。）

### 1️⃣ Substitution Drill（替換練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 医者（いしゃ） | **医者にあるまじき**言動だ。 | 這是身為醫生不該有的言行。 |
| 公務員（こうむいん） | **公務員にあるまじき**態度だ。 | 這是身為公務員不該有的態度。 |

### 2️⃣ Transformation Drill（轉換練習）
| Transformation | Expected Response | Translation |
|---|---|---|
| → 感嘆句 | なんと**あるまじき**行為だろう。 | 這是多麼不該有的行為啊。 |

### 3️⃣ Response Drill（應答練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| その行為をどう思いますか。 | 教師にあるまじき行為だと思います。 | 我認為那是身為教師不該有的行為。 |

### 4️⃣ Expansion Drill（擴展練習）
| Step | Sentence | Translation |
|---|---|---|
| 核心 | あるまじき行為だ。 | 是不該有的行為。 |
| +身分 | **教師に**あるまじき行為だ。 | 是身為教師不該有的行為。 |
