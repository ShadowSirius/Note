# 〜んばかりに

## 規則與接續
- `動詞ない形（去ない） + んばかりに`（する→せんばかりに）
- 亦可作文末用法「〜んばかりだ」

## 用法說明
表示「幾乎要...的樣子」，用於誇張比喻某動作或狀態幾乎（但實際未）發生，帶有生動的臨場感，常見於描寫表情、動作的文學性敘述。與 [[JP_N1_04_ngatame|〜んがため @related_to]] 同樣源自古語未然形接續，但語意方向完全不同：一個表目的、一個表狀態逼真程度。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）は今（いま）にも泣（な）き出（だ）さんばかりに顔（かお）を歪（ゆが）めた。
> （他一副幾乎要哭出來的樣子扭曲著臉。）

### 1️⃣ Substitution Drill（替換練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 倒（たお）れる | **倒れんばかりに**疲れていた。 | 累得幾乎要倒下。 |
| 飛（と）び上（あ）がる | **飛び上がらんばかりに**喜んだ。 | 高興得幾乎要跳起來。 |

### 2️⃣ Transformation Drill（轉換練習）
| Transformation | Expected Response | Translation |
|---|---|---|
| → 文末用法 | 彼の喜びようは**飛び上がらんばかりだった**。 | 他高興得幾乎要跳起來。 |

### 3️⃣ Response Drill（應答練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 彼はどんな顔をしていましたか。 | 今にも泣き出さんばかりの顔でした。 | 一副幾乎要哭出來的臉。 |

### 4️⃣ Expansion Drill（擴展練習）
