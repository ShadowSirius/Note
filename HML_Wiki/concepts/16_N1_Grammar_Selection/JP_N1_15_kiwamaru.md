# 〜極まる

## 規則與接續
- `な形容詞語幹 + 極まる`

## 用法說明
與 [[JP_N1_14_kiwamarinai|〜極まりない @related_to]] 同義，皆表「...到極點」，差別在「極まる」是動詞用法（原意「達到極限」），可直接修飾名詞（例：極まる態度），書面強調語氣略高於「極まりない」，兩者實務上可互換使用。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）の行動（こうどう）は無責任（むせきにん）極（きわ）まるものだった。
> （他的行動是極其不負責任的。）

### 1️⃣ Substitution Drill（替換練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 危険（きけん） | それは**危険極まる**行為だ。 | 那是極其危險的行為。 |
| 傲慢（ごうまん） | 彼の言葉は**傲慢極まる**ものだった。 | 他的話極其傲慢。 |

### 2️⃣ Transformation Drill（轉換練習）
| Transformation | Expected Response | Translation |
|---|---|---|
| → 換用極まりない | 無責任**極まりない**行動だった。 | 是極其不負責任的行動。 |

### 3️⃣ Response Drill（應答練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 彼の行動をどう評価しますか。 | 無責任極まるものだと思います。 | 我認為是極其不負責任的。 |
