# 〜にとどまらず

## 規則與接續
- `名詞・動詞辭書形 + にとどまらず`

## 用法說明
意為「不僅限於...」，用於指出某影響、範圍超越了原先較小的界線，擴及更廣的層面，書面評論語氣，常見於論述影響力擴大的文脈（例：問題は国内にとどまらず、世界へと広がった）。與 [[JP_N1_36_nikagittakotodewanai|〜に限ったことではない @related_to]] 意義相近，但「にとどまらず」更強調範圍的擴張延伸，語氣更正式。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> その影響（えいきょう）は国内（こくない）にとどまらず、世界（せかい）へと広（ひろ）がった。
> （那個影響不僅限於國內，還擴及到了全世界。）

### 1️⃣ Substitution Drill（替換練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 経済（けいざい） | 被害は**経済にとどまらず**、社会全体に及んだ。 | 損害不僅限於經濟，還波及整個社會。 |
| 若者（わかもの） | この問題は**若者にとどまらず**、高齢者にも見られる。 | 這個問題不僅限於年輕人，高齡者也能看到。 |

### 2️⃣ Transformation Drill（轉換練習）
| Transformation | Expected Response | Translation |
|---|---|---|
| → 換用に限らず | 国内**に限らず**、世界へ広がった。 | 不僅限於國內，擴及全世界。（近義） |
