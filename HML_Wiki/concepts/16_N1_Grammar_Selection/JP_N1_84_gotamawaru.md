# ご〜賜る

## 規則與接續
- `ご + 動作性名詞 + を + 賜る`（たまわる）

## 用法說明
最高敬語表現，意為「承蒙賜予...」，用於下對上、極度鄭重地表達「承蒙對方賜予（恩惠、指教、光臨等）」，比一般謙讓語「いただく」更古典、更莊重，常見於致辭、感謝狀、公文書信的開頭或結尾。與 [[JP_N1_86_omoushiageru|お〜申し上げる @related_to]] 同屬最高敬意表現，但「賜る」聚焦於「承受對方給予的恩惠」，「お〜申し上げる」聚焦於「自己動作對上位者的謙遜表達」。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 本日（ほんじつ）はご出席（しゅっせき）を賜（たまわ）り、誠（まこと）にありがとうございます。
> （今日承蒙各位蒞臨出席，由衷感謝。）

### 1️⃣ Substitution Drill（替換練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| ご指導（しどう） | 日頃より**ご指導を賜り**、感謝申し上げます。 | 平日承蒙指導，致上感謝。 |
| ご厚情（こうじょう） | 皆様の**ご厚情を賜り**、心より御礼申し上げます。 | 承蒙各位的厚愛，由衷致謝。 |

| Transformation | Expected Response | Translation |
|---|---|---|
| → 換用いただく | ご出席**をいただき**、ありがとうございます。 | 承蒙出席，謝謝。（一般謙讓語版） |
