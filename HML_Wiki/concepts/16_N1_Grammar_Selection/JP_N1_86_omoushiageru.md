# お〜申し上げる

## 規則與接續
- `お + 動詞ます形 + 申し上げる`

## 用法說明
謙讓語最高敬意表現，用於對上位者表達自己的動作，是「お〜する」的更高敬意版本，常見於商務書信、正式致辭等最鄭重的場合，如「お願い申し上げます・お礼申し上げます・お祈り申し上げます」。與 [[JP_N1_84_gotamawaru|ご〜賜る @related_to]] 同屬最高敬意表現，但方向相反：「お〜申し上げる」是自己動作的謙遜表達，「ご〜賜る」是承受對方給予的恩惠。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 皆様（みなさま）の健康（けんこう）を心（こころ）よりお祈（いの）り申（もう）し上（あ）げます。
> （由衷祈願各位身體健康。）

### 1️⃣ Substitution Drill（替換練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 感謝（かんしゃ）する | 皆様のご協力に**感謝申し上げます**。 | 對各位的協助致上感謝。 |
| お詫（わ）びする | この度のご迷惑を**お詫び申し上げます**。 | 對此次造成的困擾致上歉意。 |

### 2️⃣ Transformation Drill（轉換練習）
| Transformation | Expected Response | Translation |
|---|---|---|
| → 換用お〜する | 皆様のご健康を**お祈りします**。 | 祈願各位身體健康。（一般敬語版） |
