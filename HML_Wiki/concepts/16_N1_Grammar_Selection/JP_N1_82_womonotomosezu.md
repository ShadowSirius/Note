# 〜をものともせず

## 規則與接續
- `名詞 + をものともせず（に）`（多接「困難・逆境・批判・悪天候」等障礙、逆境類名詞）

## 用法說明
意為「不顧、不畏懼...」，表示不畏懼困難、逆境、障礙等，堅強地克服並前進，帶有積極、勇敢、正面評價的語感。與 [[JP_N1_34_woyosoni|〜をよそに @related_to]] 形成語感對比：「をものともせず」是正面稱讚的堅強不屈（克服困難），「をよそに」則多帶批判色彩（漠視他人的擔憂或反對）。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）は周囲（しゅうい）の批判（ひはん）をものともせず、自分（じぶん）の道（みち）を貫（つらぬ）いた。
> （他不畏周遭的批評，堅持走自己的路。）

### 1️⃣ Substitution Drill（替換練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 障害（しょうがい） | 彼女は**障害をものともせず**、夢を叶えた。 | 她不畏障礙，實現了夢想。 |
| 痛（いた）み | 選手は**痛みをものともせず**、走り続けた。 | 選手不畏疼痛，持續奔跑。 |

### 2️⃣ Transformation Drill（轉換練習）
| Transformation | Expected Response | Translation |
|---|---|---|
| → 換用にもかかわらず | 批判がある**にもかかわらず**、道を貫いた。 | 儘管有批評，仍堅持走自己的路。（近義表現） |
