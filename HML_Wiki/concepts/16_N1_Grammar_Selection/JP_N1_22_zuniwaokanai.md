# 〜ずにはおかない

## 規則與接續
- `動詞ない形（去ない） + ずにはおかない`（する→せずにはおかない）

## 用法說明
意為「一定會...、必然導致...」，表示某種外部力量或自然發展必然引發某結果，帶有「不由自主、無法阻止」的語感，可用於自然情感（感動させずにはおかない）或客觀必然結果，書面語氣強烈。與 [[JP_N1_23_naidewaokanai|〜ないではおかない @related_to]] 意義完全相同，為口語書寫變體，二者可互換。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> この映画（えいが）は観（み）る者（もの）を感動（かんどう）させずにはおかない。
> （這部電影必然會令觀眾感動。）

### 1️⃣ Substitution Drill（替換練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 考（かんが）えさせる | この本は読者を**考えさせずにはおかない**。 | 這本書必然會令讀者思考。 |
| 驚（おどろ）かせる | その発表は人々を**驚かせずにはおかない**。 | 那項發表必然會令人們驚訝。 |

### 2️⃣ Transformation Drill（轉換練習）
| Transformation | Expected Response | Translation |
|---|---|---|
| → 換用ないではおかない | 感動**させないではおかない**。 | 必然會令人感動。（同義互換） |

### 3️⃣ Response Drill（應答練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| この映画はどんな作品ですか。 | 観る者を感動させずにはおかない作品です。 | 是必然會令觀眾感動的作品。 |

### 4️⃣ Expansion Drill（擴展練習）
| Step | Sentence | Translation |
|---|---|---|
