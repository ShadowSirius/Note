# 〜に限ったことではない

## 規則與接續
- `名詞 + に限ったことではない`

## 用法說明
意為「不僅限於...」，用於指出某問題或現象並非只發生於特定對象／時間，而是普遍存在的情況，語氣帶有補充說明、糾正過於狹隘看法的意味，常見於評論、討論文脈。與 [[JP_N1_44_nitodomarazu|〜にとどまらず @related_to]] 意義相近，皆表「不僅限於」，但「に限ったことではない」更口語化、常用於反駁「這只是個案」的說法。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> この問題（もんだい）は日本（にほん）に限（かぎ）ったことではない。
> （這個問題不僅限於日本。）

### 1️⃣ Substitution Drill（替換練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 今回（こんかい） | このミスは**今回に限ったことではない**。 | 這個失誤不僅限於這一次。 |
| 彼（かれ） | 遅刻は**彼に限ったことではない**。 | 遲到不僅限於他一人。 |

### 2️⃣ Transformation Drill（轉換練習）
| Transformation | Expected Response | Translation |
|---|---|---|
| → 否定強調 | これは決して**日本に限ったことではない**。 | 這絕不僅限於日本。 |

### 3️⃣ Response Drill（應答練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| これは日本だけの問題ですか。 | いいえ、日本に限ったことではありません。 | 不，不僅限於日本。 |

### 4️⃣ Expansion Drill（擴展練習）
| Step | Sentence | Translation |
|---|---|---|
| 核心 | 限ったことではない。 | 不僅限於此。 |
| +主題 | **この問題は日本に**限ったことではない。 | 這個問題不僅限於日本。 |
