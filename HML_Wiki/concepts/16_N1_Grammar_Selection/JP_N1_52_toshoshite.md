# 〜と称して

## 規則與接續
- `名詞・動詞辭書形 + と称して`

## 用法說明
意為「號稱、以...名義」，用於指出某人以某種名目或藉口行事，但暗示該名目可能與實際行為不符，帶有懷疑、揭發的批判語氣，常見於報導詐騙、藉口行為的新聞文脈。

  中文：他號稱是投資顧問，騙取了高齡者的金錢。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）は投資顧問（とうしこもん）と称（しょう）して、高齢者（こうれいしゃ）からお金（かね）をだまし取（と）った。
> （他號稱是投資顧問，騙取了高齡者的金錢。）

### 1️⃣ Substitution Drill（替換練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 弁護士（べんごし） | **弁護士と称して**、報酬をだまし取った。 | 號稱是律師，騙取了報酬。 |
| 研修（けんしゅう） | **研修と称して**、実際は遊んでいた。 | 以研修為名，實際上在玩樂。 |

| Transformation | Expected Response | Translation |
|---|---|---|
| → 過去繼續形 | ずっと投資顧問**と称してきた**。 | 一直號稱是投資顧問。 |
