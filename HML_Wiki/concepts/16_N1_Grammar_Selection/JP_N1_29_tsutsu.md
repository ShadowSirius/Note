# 〜つ〜つ

## 規則與接續
- `動詞ます形 + つ + 動詞ます形（多為對照動詞） + つ`（固定成對，如「行きつ戻りつ」「持ちつ持たれつ」）

## 用法說明
意為「一邊...一邊...」，用於描述兩個相對或交替的動作反覆發生，屬高度固定化的四字慣用格式，現代日語僅存少數幾個固定搭配（行きつ戻りつ・持ちつ持たれつ・抜きつ抜かれつ），不可自由造句，文學、敘事語感濃厚。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）は部屋（へや）の中（なか）を行（い）きつ戻（もど）りつしていた。
> （他在房間裡走來走去、來回踱步。）

### 1️⃣ Substitution Drill（替換練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 抜（ぬ）く／抜（ぬ）かれる | 二台の車は**抜きつ抜かれつ**していた。 | 兩輛車互相超車、被超車。 |
| 持（も）つ／持（も）たれる | 会社同士は**持ちつ持たれつ**の関係にある。 | 公司之間是互相扶持的關係。 |

### 2️⃣ Transformation Drill（轉換練習）
| Transformation | Expected Response | Translation |
|---|---|---|
| → 名詞修飾 | **行きつ戻りつする**姿が印象的だった。 | 來回踱步的樣子令人印象深刻。 |

|---|---|---|
