# 〜には当たらない

## 規則與接續
- `動詞辭書形／する動詞語幹 + には当たらない`

## 用法說明
意為「不至於、不值得...」，表示某事的程度不到需要做出後項反應（驚訝、稱讚、責備等）的地步，帶有「不必反應過度」的冷靜判斷語氣，常搭配「驚く・褒める・非難する」等動詞。與 [[JP_N1_70_mademonai|〜までもない @related_to]] 對照：「には当たらない」聚焦「反應程度不需要那麼強烈」，「までもない」聚焦「動作本身沒必要做」。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> この程度（ていど）のミスなら、非難（ひなん）するには当（あ）たらない。
> （這種程度的失誤，不至於要責備。）

### 1️⃣ Substitution Drill（替換練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 褒（ほ）める | この結果は特に**褒めるには当たらない**。 | 這個結果不值得特別稱讚。 |
| 感心（かんしん）する | その程度なら**感心するには当たらない**。 | 那種程度不值得佩服。 |

### 2️⃣ Transformation Drill（轉換練習）
| Transformation | Expected Response | Translation |
|---|---|---|
| → 換用ほどのことではない | 驚く**ほどのことではない**。 | 不是值得驚訝的事。（近義表現） |

### 3️⃣ Response Drill（應答練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 彼を非難すべきでしょうか。 | いいえ、この程度なら非難するには当たりません。 | 不，這種程度不至於要責備。 |

### 4️⃣ Expansion Drill（擴展練習）
| Step | Sentence | Translation |
|---|---|---|
| 核心 | 驚くには当たらない。 | 不值得驚訝。 |
| +主題 | **彼の成功は**、驚くには当たらない。 | 他的成功，不值得驚訝。 |
