# 〜の余り／〜あまり

## 規則與接續
- `名詞 + の余り`（書面語感較強）
- `動詞辭書形／い形容詞／な形容詞（な） + あまり（に）`（適用範圍較廣，接名詞亦可：`名詞の + あまり`）

## 用法說明
意為「過於...以致於...」，表示因情感或狀態過度而導致某種（多為異常、意外或負面的）結果。「の余り」較文語、偏書面，僅接名詞；「あまり」除接名詞外還可接動詞、形容詞，日常書面皆可用，適用範圍更廣、更為常見。兩者語意相通，可視為同一文法的書面／通用變體。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 嬉（うれ）しさのあまり、涙（なみだ）が出（で）た。
> （太過高興，流下了眼淚。）

### 1️⃣ Substitution Drill（替換練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 悲（かな）しさ | **悲しさのあまり**、言葉を失った。 | 太過悲傷，說不出話來。 |
| 緊張（きんちょう）する | **緊張するあまり**、声が震えた。 | 太過緊張，聲音發抖。 |

### 2️⃣ Transformation Drill（轉換練習）
| Transformation | Expected Response | Translation |
|---|---|---|
| → 換用の余り | 驚きの**余り**、声も出なかった。 | 太過驚訝，連聲音都發不出來。（書面版） |

### 3️⃣ Response Drill（應答練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| どうして涙が出たのですか。 | 嬉しさのあまり、涙が出たのです。 | 因為太過高興，才流下眼淚的。 |

### 4️⃣ Expansion Drill（擴展練習）
| Step | Sentence | Translation |
|---|---|---|
| 核心 | 涙が出た。 | 流下了眼淚。 |
| +原因 | **嬉しさのあまり**、涙が出た。 | 太過高興，流下了眼淚。 |
