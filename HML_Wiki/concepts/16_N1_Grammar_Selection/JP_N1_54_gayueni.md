# 〜がゆえに

## 規則與接續
- `動詞・い形容詞普通形 + がゆえに`
- `名詞・な形容詞（である）+ がゆえに`

## 用法說明
文語書面因果表現，意為「正因為...、由於...」，用於客觀陳述前項是後項的原因，語氣比「〜から」「〜ので」更硬、更具論說文／判決書色彩。與 [[JP_N1_30_bakoso|〜ばこそ @related_to]] 同屬N1書面因果表現，但語感不同：「がゆえに」偏向客觀陳述因果關係，「ばこそ」則強調「正因為...才...」的主觀決定性理由。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 経験（けいけん）が浅（あさ）いがゆえに、失敗（しっぱい）を繰（く）り返（かえ）した。
> （因為經驗尚淺，所以屢屢失敗。）

### 1️⃣ Substitution Drill（替換練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 未熟（みじゅく）だ | **未熟であるがゆえに**、失敗を繰り返した。 | 因為尚未成熟，所以屢屢失敗。 |
| 若（わか）い | **若いがゆえに**、無茶をしてしまった。 | 因為年輕，做了魯莽的事。 |

### 2️⃣ Transformation Drill（轉換練習）
| Transformation | Expected Response | Translation |
|---|---|---|
| → 換用ので | 経験が浅い**ので**、失敗を繰り返した。 | 因為經驗尚淺，屢屢失敗。（口語化） |
