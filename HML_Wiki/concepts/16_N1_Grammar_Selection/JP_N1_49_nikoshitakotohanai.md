# 〜に越したことはない

## 規則與接續
- `動詞辭書形・ない形・な形容詞・形容詞普通形 + に越したことはない`

## 用法說明
意為「最好是...、...是最理想的」，用於表達雖非必要、但若能達成該條件會是最好的選擇，語氣溫和、帶常識性建議色彩，常見於給予建議或表達普遍認可的道理（例：用心に越したことはない＝小心點總是好的）。與 [[JP_N1_48_niwaoyobanai|〜には及ばない @related_to]] 都涉及程度比較，但「に越したことはない」是正面推薦最佳選項，而非否定必要性。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 健康（けんこう）のためには早寝早起（はやねはやお）きに越（こ）したことはない。
> （為了健康，早睡早起是最理想的。）

### 1️⃣ Substitution Drill（替換練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 用心（ようじん）する | **用心するに越したことはない**。 | 小心點總是最好的。 |
| 安全（あんぜん） | **安全に越したことはない**。 | 安全是最重要的。 |

| Transformation | Expected Response | Translation |
|---|---|---|
| → 比較級形式 | 早ければ**早いに越したことはない**。 | 越早越好。 |

|---|---|---|
