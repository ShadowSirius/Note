# 〜に足る

## 規則與接續
- `動詞辭書形・名詞 + に足る + 名詞`
- 句尾用法：`〜に足る` / `〜に足りない`（否定）

## 用法說明
意為「足以...、值得...」，用於評價某人事物具備充分的資格或價值，可承受某種評價或信任，語氣正面、書面色彩濃厚，常見於「信頼するに足る」「尊敬するに足る」等固定搭配。與 [[JP_N1_47_nitaeru|〜に堪える @related_to]] 意義相近，皆表「值得、經得起」，但「に足る」偏重「資格充分」，「に堪える」偏重「承受得住考驗」。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）は信頼（しんらい）するに足（た）る人物（じんぶつ）だ。
> （他是值得信賴的人物。）

### 1️⃣ Substitution Drill（替換練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 尊敬（そんけい）する | 彼は**尊敬するに足る**先生だ。 | 他是值得尊敬的老師。 |
| 満足（まんぞく）する | 結果は**満足するに足る**ものだった。 | 結果是值得滿意的。 |

### 2️⃣ Transformation Drill（轉換練習）
| Transformation | Expected Response | Translation |
|---|---|---|
| → 否定形 | 信頼する**に足りない**人物だ。 | 是不值得信賴的人物。 |

### 3️⃣ Response Drill（應答練習）
| Prompt | Expected Response | Translation |
|---|---|---|
| 彼は信頼できる人ですか。 | はい、信頼するに足る人物です。 | 是的，他是值得信賴的人物。 |

### 4️⃣ Expansion Drill（擴展練習）
| Step | Sentence | Translation |
|---|---|---|
| 核心 | 信頼するに足る。 | 值得信賴。 |
| +主語 | **彼は**信頼するに足る人物だ。 | 他是值得信賴的人物。 |
