# 〜てみます

## 規則與接續
- `動詞て形 + みます`

## 用法說明
表示嘗試性地做某個動作，藉此看看結果如何，或確認情況（試著做某事）。
- 常見變形 `〜てみてもいいですか` 結合了許可句型，用於非常客氣地詢問「我可以試試看…嗎？」（如試穿、試吃等）。

## 實用例句
- 日文：この<ruby>服<rt>ふく (fuku)</rt></ruby>を<ruby>着<rt>き (ki)</rt></ruby>てみてもいいですか。
- 中文：我可以試穿這件衣服嗎？
- 日文：もう<ruby>一度<rt>いちど (ichido)</rt></ruby>やってみます。
- 中文：我再試一次。
- 日文：この<ruby>料理<rt>りょうり (ryouri)</rt></ruby>を<ruby>作<rt>つく (tsuku)</rt></ruby>ってみました。
- 中文：我試著做了這道料理。
- 日文：そのアプリを<ruby>使<rt>つか (tsuka)</rt></ruby>ってみてください。
- 中文：請試用那個應用程式看看。

## 關係說明
- 常與表達許可的 [[JP_Grammar_13_te_mo_ii_desu|〜てもいいです @related_to]] 結合為 `〜てみてもいいですか`，廣泛應用於購物、餐飲等日常對話場景。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> もう一度（いちど）やってみます。
> （我再試一次。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 書（か）く | もう一度（いちど）**書（か）いて**みます。 | 我再寫一次看看。 |
| 話（はな）す | もう一度（いちど）**話（はな）して**みます。 | 我再說一次看看。 |
| 聞（き）く | もう一度（いちど）**聞（き）いて**みます。 | 我再聽/問一次看看。 |
| 考（かんが）える | もう一度（いちど）**考（かんが）えて**みます。 | 我再思考一次看看。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 過去形（かこけい） | もう一度（いちど）やって**みました**。 | 我再試了一次。 |
| 否定形（ひていけい） | もう一度（いちど）やって**みません**。 | 我不再試了。 |
| 許可（きょか） | もう一度（いちど）やって**みてもいいですか**。 | 我可以再試一次看看嗎？ |
| 請求（せいきゅう） | もう一度（いちど）やって**みてください**。 | 請再試一次看看。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| この料理（りょうり）を食べて（たべて）みてもいいですか。 | はい、食べて（たべて）みてください。 | 我可以試吃這道料理嗎？ / 是的，請試吃看看。 |
| そのアプリを使（つか）ってみましたか。 | はい、使（つか）ってみました。 | 你試用那個應用程式了嗎？ / 是的，我試用過了。 |
| この服（ふく）を着（き）てみますか。 | はい、着（き）てみます。 | 你要試穿這件衣服嗎？ / 是的，我試穿看看。 |
| 日本語（にほんご）で話（はな）してみますか。 | はい、日本語（にほんご）で話（はな）してみます。 | 你要試著用日語交談嗎？ / 是的，我試著用日語交談看看。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心（かくしん） | やってみます。 | 試試看。 |
| +頻度（ひんど） | もう一度（いちど）やってみます。 | 再試一次看看。 |
| +主体（しゅたい） | 自分で（じぶんで）もう一度（いちど）やってみます。 | 自己再試一次看看。 |
| +時間（じかん） | 明日（あした）自分で（じぶんで）もう一度（いちど）やってみます。 | 明天自己再試一次看看。 |
