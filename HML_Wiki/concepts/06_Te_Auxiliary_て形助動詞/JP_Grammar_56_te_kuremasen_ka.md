# 〜てくれませんか

## 規則與接續
- `動詞て形 + くれませんか`

## 用法說明
表示請求他人為自己做某事（要不要幫我做某事？）。
- **口語日常**：這是一種溫和、有禮的日常口語請求方式。相較於單向指令的 `てください`，它採用否定疑問，給予了對方考慮與拒絕的空間，聽起來較客氣。
- **適用對象**：常用於家人、朋友、平輩或熟人之間。

### 📊 請求句型禮貌程度比較 (從低到高)
1. `〜て`（口語，最隨意）
2. `〜てください`（直接請求，帶有指示色彩）
3. **`〜てくれませんか`**（溫和 of 日常口語請求 🌟）
4. `〜てもらえませんか` / `〜ていただけますか`（客氣的委婉請求）
5. `〜ていただけないでしょうか`（極其客氣的商務敬語請求）

## 關係說明
- 作為日常生活中必備的請求句型，是直接請求 [[JP_Grammar_11_te_kudasai|〜てください @refines]] 的委婉口語細化。
- 與更為莊重客氣的 [[JP_Grammar_33_te_moraemasen_ka|〜てもらえませんか @related_to]] 具有緊密的語意對應與替換關係。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 窓（まど）を開（あ）けてくれませんか。
> （可以幫我把窗戶打開嗎？）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 窓（まど）を閉（し）めて | 窓（まど）を**閉（し）めてくれませんか**。 | 可以幫我把窗戶關上嗎？ |
| ドアを開（あ）けて | ドアを**開（あ）けてくれませんか**。 | 可以幫我把門打開嗎？ |
| 電気（でんき）をつけて | 電気（でんき）を**つけてくれませんか**。 | 可以幫我把燈打開嗎？ |
| エアコンを消（け）して | エアコンを**消（け）してくれませんか**。 | 可以幫我把冷氣關掉嗎？ |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 丁寧指示（ていねいしじ） | 窓（まど）を開（あ）けて**ください**。 | 請打開窗戶。 |
| 口語形（こうごけい） | 窓（まど）を開（あ）けて**くれない**？ | 能幫忙開個窗戶嗎？（口語） |
| 謙遜請求（けんそんせいきゅう） | 窓（まど）を開（あ）けて**もらえませんか**。 | 能請您幫我開窗戶嗎？ |
| 最上敬語請求（さいじょうけいごせいきゅう） | 窓（まど）を開（あ）けて**いただけないでしょうか**。 | 是否能麻煩您幫忙開窗戶呢？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 窓（まど）を開（あ）けてくれませんか。 | はい、開（あ）けますよ。 | 可以幫我把窗戶打開嗎？ / 好的，我來開。 |
| この文章（ぶんしょう）を読んで（よんで）くれませんか。 | はい、読み（よみ）ますよ。 | 能幫我看一下這段文章嗎？ / 好的，我看一下。 |
| 写真（しゃしん）を送（おく）ってくれませんか。 | はい、送（おく）りますよ。 | 能把照片發給我嗎？ / 好的，我發給你。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心（かくしん） | くれませんか。 | 能幫我做嗎？ |
| +動作（どうさ） | 開（あ）けてくれませんか。 | 能幫我打開嗎？ |
| +目的語（もくてきご） | 窓（まど）を開（あ）けてくれませんか。 | 能幫我把窗戶打開嗎？ |
| +丁寧（ていねい） | すみませんが、窓（まど）を開（あ）けてくれませんか。 | 不好意思，能幫我把窗戶打開嗎？ |
