# 〜てもらいます

## 規則與接續
- `（受惠者/我）が + （施惠者/他人）に + 動詞て形 + もらいます`

## 用法說明
表示「我（或我方人員）承蒙他人做某事，並從中得到了幫助 or 好處」。
- **主詞**：是受惠的「我」或我方人員（通常省略 `私に`）。
- **動作主體**：提供好意的「他人」必須用助詞 `に` 標記（表示動作的來源）。
- 語氣比主動發发生的 `〜てくれます` 更強調「我主動請求或安排了這件事，因而得到對方的幫忙」。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#3498DB"><ruby>先生<rt>せんせい (sensei)</rt></ruby>にレポートを<ruby>直<rt>nao (nao)</rt></ruby>してもらいました。</span>
- 中文：我請老師幫我修改了報告。（承蒙老師的好意）
- 日文：<span style="color:#3498DB"><ruby>友達<rt>ともだち (tomodachi)</rt></ruby>に<ruby>引越し<rt>ひっこし (hikkoshi)</rt></ruby>を<ruby>手伝<rt>てつだ (tetsuda)</rt></ruby>ってもらいます。</span>
- 中文：我請朋友幫忙我搬家。
- 日文：<span style="color:#3498DB"><ruby>工事部<rt>こうじぶ (koujibu)</rt></ruby>に<ruby>見積もり<rt>みつもり (mitsumori)</rt></ruby>を<ruby>出<rt>だ (da)</rt></ruby>してもらえますか。</span>
- 中文：能請工務部開個估價單給我嗎？（委婉請求變形）
- 日文：<span style="color:#3498DB"><ruby>客先<rt>きゃくさき (kyakusaki)</rt></ruby>に</span><span style="color:#F39C12"><ruby>交渉<rt>こうしょう (koushou)</rt></ruby>の</span><span style="color:#3498DB"><ruby>機会<rt>きかい (kikai)</rt></ruby>を<ruby>作<rt>つく (tsuku)</rt></ruby>ってもらいました。</span>
- 中文：客戶（幫忙）為我安排了談判的機會。

## 關係說明
- 與同樣表達得到幫助的 [[JP_Grammar_42_te_kuremasu|〜てくれます @replaces]] 互為近義替換，但**主詞視角不同**（`てもらいます` 主詞為受惠者，用 `に` 標記對方；`てくれます` 主詞為施惠者，用 `が` 標記）。
- 該句型改為疑問形即為日常極其高頻的委婉請求句型 [[JP_Grammar_33_te_moraemasen_ka|〜てもらえませんか @parent_of]]。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 先生（せんせい）にレポートを直（なお）してもらいました。
> （我請老師幫我修改了報告。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 先輩（せんぱい） | **先輩（せんぱい）**にレポートを直（なお）してもらいました。 | 我請前輩幫我修改了報告。 |
| 友達（ともだち） | **友達（ともだち）**にレポートを直（なお）してもらいました。 | 我請朋友幫我修改了報告。 |
| 彼女（かのじょ） | **彼女（かのじょ）**にレポートを直（なお）してもらいました. | 我請女朋友幫我修改了報告。 |
| 部長（ぶちょう） | **部長（ぶちょう）**にレポートを直（なお）してもらいました。 | 我請經理幫我修改了報告。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 非過去形（ひかこけい） | 先生（せんせい）にレポートを直（なお）して**もらいます**。 | 我請老師幫我修改報告。 |
| 丁寧可能質問（ていねいかのうしつもん） | 先生（せんせい）にレポートを直（なお）して**もらえますか**。 | 能請老師幫我修改報告嗎？ |
| 普通形過去（ふつうけいかこ） | 先生（せんせい）にレポートを直（なお）して**もらった**。 | 請老師幫我修改了報告。（常體） |
| 否定形過去（ひていけいかこ） | 先生（せんせい）にレポートを直（なお）して**もらえませんでした**。 | 沒有請老師幫我修改報告。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 誰（だれ）にレポートを直（なお）してもらいましたか。 | 先生（せんせい）に直（なお）してもらいました。 | 你請誰幫你修改報告了？ / 請老師幫我修改了。 |
| 友達（ともだち）に引越し（ひっこし）を手伝（てつだ）ってもらいますか。 | はい、手伝（てつだ）ってもらいます。 | 你會請朋友幫忙搬家嗎？ / 是的，我會請他幫忙。 |
| 彼（かれ）に教えて（おしえて）もらいましたか。 | はい、教えて（おしえて）もらいました。 | 你請他教你了嗎？ / 是的，我請他教我了。 |
