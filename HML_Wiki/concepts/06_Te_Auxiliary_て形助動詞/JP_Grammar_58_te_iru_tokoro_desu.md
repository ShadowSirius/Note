# 〜ているところです

## 規則與接續
- `動詞て形 + いるところです`

## 用法說明
表示某個動作「正處於進行中的那一個精確瞬間/當下時間點」（正在做某事的最忙碌時刻）。
- **強調此刻瞬間**：比起一般敘述進行狀態的 `〜ています`，`〜ているところです` 更加強調「此時此刻，該動作正在發生的那一秒/那一刻」，常用於回答他人「你現在在幹嘛？」或交代工作進度的緊急感。

## 實用例句
- 日文：<ruby>今<rt>いま (ima)</rt></ruby>メールを<ruby>書<rt>か (ka)</rt></ruby>いているところです。
- 中文：我現在正在寫電子郵件。（正在寫的那一刻）
- 日文：ただ<ruby>今<rt>いま (ima)</rt></ruby>、<ruby>資料<rt>しりょう (shiryou)</rt></ruby>を<ruby>準備<rt>じゅんび (junbi)</rt></ruby>しているところです。
- 中文：我目前正忙著準備資料。
- 日文：<ruby>今<rt>いま (ima)</rt></ruby>は<ruby>掃除<rt>そうじ (souji)</rt></ruby>をしているところです。
- 中文：我現在正在打掃呢。
- 日文：すみません。<ruby>出<rt>de (de)</rt></ruby>かける<ruby>準備<rt>じゅんび (junbi)</rt></ruby>をしているところです。
- 中文：不好意思，我正準備要出門。（正處於出門準備的當下）

## 關係說明
- 是進行狀態 [[JP_Grammar_12_te_imasu|〜ています @refines]] 的精確細化，特別強調「當下那個特定的時刻點」。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 今（いま）メールを書（か）いているところです。
> （我現在正在寫電子郵件。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 電話（でんわ）をかけて | 今（いま）**電話（でんわ）をかけているところです**。 | 我現在正在打電話。 |
| 会議資料（かいぎしりょう）をコピー（こぴー）して | 今（いま）**会議資料（かいぎしりょう）をコピー（こぴー）しているところです**。 | 我現在正在影印會議資料。 |
| 報告書（ほうこくしょ）を直（なお）して | 今（いま）**報告書（ほうこくしょ）を直（なお）しているところです**。 | 我現在正在修改報告書。 |
| 予定表（よていひょう）を確認（かくにん）して | 今（いま）**予定表（よていひょう）を確認（かくにん）しているところです**。 | 我現在正在確認行程表。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 普通形（ふつうけい） | 今（いま）メールを書（か）いている**ところだ**。 | 我現在正在寫電子郵件。（常體） |
| 過去形（かこけい） | 今（いま）メールを書（か）いている**ところでした**。 | 我剛才正在寫電子郵件。 |
| 否定形（ひていけい） | 今（いま）メールを書（か）いている**ところではありません**。 | 我現在不是正在寫電子郵件。 |
| 一般進行形（いっぱんしんこうけい） | 今（いま）メールを**書（か）いています**。 | 我現在正在寫電子郵件。（一般進行） |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 今（いま）メールを書（か）しているところですか。 | はい、書（か）いているところです。 | 我現在正在寫電子郵件嗎？ / 是的，我正在寫。 |
| ただ今（いま）資料（しりょう）を準備（じゅんび）しているところですか。 | はい、準備（じゅんび）しているところです。 | 您目前正忙著準備資料嗎？ / 是的，我正在準備。 |
| 出（で）かける準備（じゅんび）をしているところですか。 | はい、準備（じゅんび）しているところです。 | 您正處於出門準備的當下嗎？ / 是的，我正準備要出門。 |
