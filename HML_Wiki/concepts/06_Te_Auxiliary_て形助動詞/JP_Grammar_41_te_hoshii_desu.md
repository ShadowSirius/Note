# 〜てほしいです

## 規則與接續
- `（他人）に + 動詞て形 + 欲しいです`
- 否定形式：`〜ないでほしいです`（希望他人不要做某事）

## 用法說明
表示說話者「希望或要求他人做某個動作」（對他人行為的期盼）。
- 說話者是希望的主體，而實際執行動作的他人通常用對象助詞 `に` 標記。
- 由於此說法較為直接，主要用於對晚輩、平輩或熟人。對長輩或商務場合，應使用更禮貌的表達。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E"><ruby>明日<rt>あした (ashita)</rt></ruby></span><span style="color:#FF477E">早く</span><span style="color:#3498DB"><ruby>連絡<rt>れんらく (renraku)</rt></ruby>して<ruby>欲<rt>ほ (ho)</rt></ruby>しいです。</span>
- 中文：希望你明天早點聯絡我。
- 日文：<span style="color:#F39C12">この</span><span style="color:#3498DB"><ruby>問題<rt>もんだい (mondai)</rt></ruby>を</span><span style="color:#FF477E"><ruby>一緒に<rt>いっしょに (isshoni)</rt></ruby></span><span style="color:#3498DB"><ruby>解決<rt>かいけつ (kaiketsu)</rt></ruby>してほしい。</span>
- 中文：希望我們能一起解決這個問題。
- 日文：<span style="color:#FF477E"><ruby>時間<rt>じかん (jikan)</rt></ruby>があれば</span><span style="color:#3498DB">アドバイスして<ruby>欲<rt>ほ (ho)</rt></ruby>しいです。</span>
- 中文：如果有時間的話，希望你能給我一些建議。
- 日文：<span style="color:#3498DB"><ruby>資料<rt>しりょう (shiryou)</rt></ruby>を</span><span style="color:#FF477E"><ruby>今夜<rt>こんや (konya)</rt></ruby>までに</span><span style="color:#3498DB">まとめて<ruby>欲<rt>ほ (ho)</rt></ruby>しいです。</span>
- 中文：希望你今晚之前把資料整理好。

## 關係說明
- 與表達自己想做某事的 [[JP_Grammar_15_tai_desu|〜たいです @related_to]] 互為補充（前者希望他人做，後者自己想做）。
- 與希望得到某物品的 [[JP_Grammar_31_ga_hoshii_desu|〜が欲しいです @related_to]] 同源，均為願望的延伸。
- 商務及對上關係中，可精確深化為極具敬意的 [[JP_Grammar_55_te_itadakitai|〜ていただきたい @parent_of]]。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 明日（あした）早く（はやく）連絡（れんらく）して欲（ほ）しいです。
> （希望你明天早點聯絡我。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 来（く）る | 明日（あした）早く（はやく）**来（き）て**欲（ほ）しいです。 | 希望你明天早點來。 |
| 言（い）う | 明日（あした）早く（はやく）**言（い）って**欲（ほ）しいです。 | 希望你明天早點說。 |
| 始（はじ）める | 明日（あした）早く（はやく）**始（はじ）めて**欲（ほ）しいです。 | 希望你明天早點開始。 |
| メールする | 明日（あした）早く（はやく）**メールして**欲（ほ）しいです。 | 希望你明天早點發郵件。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 普通形（ふつうけい） | 明日（あした）早く（はやく）連絡（れんらく）して**欲（ほ）しい**。 | 希望你明天早點聯絡我。（常體） |
| 否定形（ひていけい） | 明日（あした）早く（はやく）連絡（れんらく）**しないで欲（ほ）しいです**。 | 希望你明天不要早點聯絡我。 |
| 過去形（かこけい） | 明日（あした）早く（はやく）連絡（れんらく）して**欲（ほ）しかったです**。 | 以前希望你明天早點聯絡我。 |
| 疑問形（ぎもんけい） | 明日（あした）早く（はやく）連絡（れんらく）して欲（ほ）しいです**か**。 | 你希望明天早點聯絡嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| あなたは誰（だれ）に手伝（てつだ）ってほしいですか。 | 友達（ともだち）に手伝（てつだ）ってほしいです。 | 你希望誰來幫忙？ / 希望朋友來幫忙。 |
| 資料（しりょう）を今夜（こんや）までにまとめてほしいですか。 | はい、今夜（こんや）までにまとめてほしいです。 | 你希望今晚之前把資料整理好嗎？ / 是的，希望今晚之前整理好。 |
| 早く（はやく）来て（きて）ほしいですか。 | はい、早く（はやく）来て（きて）ほしいです。 | 你希望早點來嗎？ / 是的，希望早點來。 |

| +副詞（ふくし） | 早く（はやく）連絡（れんらく）して欲（ほ）しいです。 | 希望早點聯絡我。 |
