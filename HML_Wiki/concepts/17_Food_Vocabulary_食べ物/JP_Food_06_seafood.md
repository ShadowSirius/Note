# 食材詞彙 06：海鮮

## 🧠 記憶鉤
- **魚類漢字幾乎全是「魚部訓読み」單字**：鮪(まぐろ)、鮭(さけ)、鰹(かつお)、鯖(さば)、鰻(うなぎ)、鮑(あわび)、鰡(ぼら) 都是「魚＋聲符」的形聲字，但**讀音是純日式訓読み，不能用音読規則猜**，必須逐字硬背。訣竅：先記部首「魚」代表水產類，再記右邊聲符對應的固有詞。
- **秋刀魚(さんま)是少數「魚字部＋非單字漢字組合」的例外**：由「秋」＋「刀」＋「魚」三字會意組成，因形似刀、盛產於秋而得名，屬於**訓読み+意象合成詞**，好記口訣「秋天的刀形魚」。
- **貝類、蝦蟹多為和語固有詞**：蛤(はまぐり)、牡蠣(かき)、帆立(ほたて)、海老(えび)，帆立取名自其貝殼像帆船。
- **魚卵類詞彙常為外來語或轉用詞**：とびこ(飛魚卵，字面「跳び子」轉寫)、いくら(鮭魚卵，源自俄語ikra！)——是少數海鮮詞來自俄語的特例，因北海道漁業與俄國交流而傳入。
- **生食文化詞彙集中在「刺身」系統**：刺身(さしみ)＝生魚片，盛り合わせ(もりあわせ)＝拼盤，理解這組詞可以推知大部分生魚料理菜單用語。
- **出世魚(しゅっせうお)概念**：同一種魚隨成長改名，如鰤魚 ワカシ→イナダ→ワラサ→**ブリ**(關東)。壽司店的 **ハマチ** 多指養殖鰤魚或關西系稱呼——記住「ハマチ和ブリ是同一種魚的不同人生階段」。
- **鮪魚部位金字塔**：赤身(あかみ,瘦肉)→中トロ(中腹)→大トロ(おおとろ,大腹),脂肪越多越貴。トロ源自「とろける」(入口即化)。
- **壽司店三件套**:シャリ(醋飯)+ネタ(料)+ガリ(薑片)——全是壽司職人行話,菜單看不到但點餐聽得到。

## ⚠️ 易混淆對比
| 易混組合 | 差異 |
|---|---|
| 魚(さかな) vs 肴(さかな) | 同音同源！「肴」原指配酒的下酒菜，因古時常以魚類下酒，「さかな」逐漸轉指「魚」本身，兩字現今並存，語境判斷：喝酒配菜語境用肴，一般食材語境用魚 |
| 鮭(さけ) vs 酒(さけ) | 純粹同音異字，「鮭魚」與「酒」發音完全相同，只能靠上下文或漢字辨義，是初學者常見的聽力陷阱 |
| 鮭 vs サーモン | 鮭(さけ)是日式傳統稱呼（多指鹹鮭魚、烤鮭魚等日本料理脈絡）；サーモン(salmon)是西式外來語，常用於生食（如サーモン刺身），兩者所指魚種相同但使用情境有別 |
| とびこ vs いくら | 都是魚卵，但とびこ＝飛魚卵（顆粒小、橙紅色、口感脆），いくら＝鮭魚卵（顆粒大、鮮紅色、口感彈潤），壽司店常見誤認 |
| 鰹(かつお) vs 鮭(さけ) | 漢字形近（右邊部件不同：堅 vs 圭），讀音也完全不同（かつお vs さけ），書寫時容易寫錯偏旁 |
| 牡蠣(かき) vs 柿(かき) | 同音異字！「牡蠣」是生蠔，「柿」是柿子，兩者讀音完全相同，只能靠漢字或語境判斷 |
| ハマチ vs ブリ | 同一種魚！ハマチ=幼魚/養殖/關西稱呼,ブリ=成魚(出世魚改名)。壽司店兩者都常見 |
| ヒラメ vs カレイ | 都是比目魚。口訣「左ヒラメに右カレイ」:眼睛在左=ヒラメ(高級壽司料),在右=カレイ(多用煮付) |
| 穴子(あなご) vs 鰻(うなぎ) | 穴子=海鰻,清淡便宜,壽司常客;鰻=淡水鰻,油脂重,多做蒲焼丼飯 |
