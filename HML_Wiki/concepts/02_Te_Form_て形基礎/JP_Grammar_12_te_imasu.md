# 〜ています

## 規則與接續
- `動詞て形 + います`

## 用法說明
1. 表示正在進行的動作（正在...）；2. 表示動作後的狀態殘留。

## 實用例句
- 日文：<ruby>今<rt>いま (ima)</rt></ruby>、<ruby>日本語<rt>にほんご (nihongo)</rt></ruby>を<ruby>勉強<rt>べんきょう (benkyou)</rt></ruby>しています。
- 中文：現在正在學習日語。
- 日文：<ruby>田中<rt>たなか (tanaka)</rt></ruby>さんは<ruby>結婚<rt>けっこん (kekkon)</rt></ruby>しています。
- 中文：田中先生已經結婚了（狀態）。
- 日文：<ruby>今晩<rt>こんばん (konban)</rt></ruby>ご<ruby>飯<rt>はん (han)</rt></ruby>を<ruby>作<rt>つく (tsuku)</rt></ruby>っています。
- 中文：我今晚正在做飯。
- 日文：<ruby>彼<rt>かれ (kare)</rt></ruby>は<ruby>日本語<rt>にほんご (nihongo)</rt></ruby>を<ruby>勉強<rt>べんきょう (benkyou)</rt></ruby>しています。
- 中文：他正在學習日語。
- 日文：<ruby>友達<rt>ともだち (tomodachi)</rt></ruby>は<ruby>今<rt>いま (ima)</rt></ruby><ruby>映画<rt>えいが (eiga)</rt></ruby>を<ruby>見<rt>み (mi)</rt></ruby>ています。
- 中文：朋友現在正在看電影。
- 日文：<ruby>犬<rt>いぬ (inu)</rt></ruby>が<ruby>外<rt>そと (soto)</rt></ruby>で<ruby>遊<rt>あそ (aso)</rt></ruby>んでいます。
- 中文：狗正在外面玩耍。

## 關係說明
- 皆屬於動詞て形延伸句型，與 [[JP_Grammar_11_te_kudasai|〜てください @sibling_of]] 共用基礎變形。
- 與人為意圖狀態結果的 [[JP_Grammar_36_te_arimasu|〜てあります @contradicts]] 形成對比（自動詞狀態 vs. 他動詞人為狀態）。
- 可被進一步精確細化為當下瞬間進行的 [[JP_Grammar_58_te_iru_tokoro_desu|〜ているところです @refines]]，或帶有負面責備情緒的 [[JP_Grammar_49_te_bakari_imasu|〜てばかりいます @refines]]。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>今<rt>いま</rt></ruby>、<ruby>日本語<rt>にほんご</rt></ruby>を<ruby>勉強<rt>べんきょう</rt></ruby>しています。
> 現在正在學習日語。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 英語 (えいご) | 今、**英語**を勉強しています。 |
| 漢字 (かんじ) | 今、**漢字**を勉強しています。 |
| 数学 (すうがく) | 今、**数学**を勉強しています。 |
| 歴史 (れきし) | 今、**歴史**を勉強しています。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 過去進行 (〜ていました) | 今、日本語を勉強して**いました**。 |
| → 否定 (〜ていません) | 今、日本語を勉強して**いません**。 |
| → 普通形 (〜ている) | 今、日本語を勉強して**いる**。 |
| → 普通形過去 (〜ていた) | 今、日本語を勉強して**いた**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 今、何をしていますか。 | 日本語を勉強しています。 |
| 日本語を勉強していますか。 | はい、日本語を勉強しています。 |
| 毎日日本語を勉強していますか。 | はい、毎日日本語を勉強しています。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | 勉強しています。 |
| +日本語を | 日本語を勉強しています。 |
| +今 | 今、日本語を勉強しています。 |
| +図書館で | 図書館で今、日本語を勉強しています。 |
