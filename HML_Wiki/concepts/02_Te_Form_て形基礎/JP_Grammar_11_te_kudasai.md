# 〜てください

## 規則與接續
- `動詞て形 + ください`

## 用法說明
表示禮貌的請求或指示（請做某事）。

## 實用例句
- 日文：ここに<ruby>名前<rt>なまえ (namae)</rt></ruby>を<ruby>書<rt>か (ka)</rt></ruby>いてください。
- 中文：請在這裡寫上名字。
- 日文：ちょっと<ruby>待<rt>ま (ma)</rt></ruby>ってください。
- 中文：請稍等一下。
- 日文：この<ruby>書類<rt>しょるい (shorui)</rt></ruby>を<ruby>印刷<rt>いんさつ (insatsu)</rt></ruby>してください。
- 中文：請列印這份文件。

- 日文：できたらメールで<ruby>送<rt>おく (oku)</rt></ruby>ってください。
- 中文：完成後請以電子郵件寄出。
- 日文：<ruby>問題<rt>もんだい (mondai)</rt></ruby>があれば<ruby>連絡<rt>れんらく (renraku)</rt></ruby>してください。
- 中文：如果有問題請聯絡我。
- 日文：<ruby>明日<rt>あした (ashita)</rt></ruby>までに<ruby>決定<rt>けってい (kettei)</rt></ruby>を<ruby>教<rt>おしえ (oshie)</rt></ruby>てください。
- 中文：請在明天之前告訴我決定。

## 關係說明
- 表示請求做某事，其否定形式為 [[JP_Grammar_20_nai_de_kudasai|〜ないでください @contradicts]]。
- 可被進一步精確細化為更委婉有禮的 [[JP_Grammar_33_te_moraemasen_ka|〜てもらえませんか @parent_of]] 或口語平輩間的 [[JP_Grammar_56_te_kuremasen_ka|〜てくれませんか @parent_of]]。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> ここに<ruby>名前<rt>なまえ</rt></ruby>を<ruby>書<rt>か</rt></ruby>いてください。
> 請在這裡寫上名字。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 住所 (じゅうしょ) | ここに**住所**を書いてください。 |
| 連絡先 (れんらくさき) | ここに**連絡先**を書いてください。 |
| 電話番号 (でんわばんごう) | ここに**電話番号**を書いてください。 |
| サイン | ここに**サイン**を書いてください。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定 (〜ないでください) | ここに名前を**書かないで**ください。 |
| → 委婉 (〜てもらえませんか) | ここに名前を書いて**もらえませんか**。 |
| → 口語 (命令/請求) | ここに名前を**書いて**。 |
| → 禁止 (〜てはいけません) | ここに名前を書いては**いけません**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| ここに名前を書きますか。 | はい、ここに名前を書いてください。 |
| どこに住所を書けばいいですか。 | ここに住所を書いてください。 |
| サインも必要ですか。 | はい、ここにサインを書いてください。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | 書いてください。 |
| +名前を | 名前を書いてください。 |
| +ここに | ここに名前を書いてください。 |
| +すみませんが | すみませんが、ここに名前を書いてください。 |
