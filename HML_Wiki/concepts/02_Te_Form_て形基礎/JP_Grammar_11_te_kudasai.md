# 〜てください

## 規則與接續
- `動詞て形 + ください`

## 用法說明
表示禮貌的請求或指示（請做某事）。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E">ここに</span><span style="color:#3498DB"><ruby>名前<rt>なまえ (namae)</rt></ruby>を<ruby>書<rt>か (ka)</rt></ruby>いてください。</span>
- 中文：請在這裡寫上名字。
- 日文：<span style="color:#FF477E">ちょっと</span><span style="color:#3498DB"><ruby>待<rt>ま (ma)</rt></ruby>ってください。</span>
- 中文：請稍等一下。
- 日文：<span style="color:#F39C12">この</span><span style="color:#3498DB"><ruby>書類<rt>しょるい (shorui)</rt></ruby>を<ruby>印刷<rt>いんさつ (insatsu)</rt></ruby>してください。</span>
- 中文：請列印這份文件。

- 日文：<span style="color:#FF477E">できたらメールで</span><span style="color:#3498DB"><ruby>送<rt>おく (oku)</rt></ruby>ってください。</span>
- 中文：完成後請以電子郵件寄出。
- 日文：<span style="color:#FF477E"><ruby>問題<rt>もんだい (mondai)</rt></ruby>があれば</span><span style="color:#3498DB"><ruby>連絡<rt>れんらく (renraku)</rt></ruby>してください。</span>
- 中文：如果有問題請聯絡我。
- 日文：<span style="color:#FF477E"><ruby>明日<rt>あした (ashita)</rt></ruby>までに</span><span style="color:#3498DB"><ruby>決定<rt>けってい (kettei)</rt></ruby>を<ruby>教<rt>おしえ (oshie)</rt></ruby>てください。</span>
- 中文：請在明天之前告訴我決定。

## 關係說明
- 表示請求做某事，其否定形式為 [[JP_Grammar_20_nai_de_kudasai|〜ないでください @contradicts]]。
- 可被進一步精確細化為更委婉有禮的 [[JP_Grammar_33_te_moraemasen_ka|〜てもらえませんか @parent_of]] 或口語平輩間的 [[JP_Grammar_56_te_kuremasen_ka|〜てくれませんか @parent_of]]。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 名前（なまえ）を書（か）いてください。
> （請寫名字。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 住所（じゅうしょ） | **住所（じゅうしょ）**を書（か）いてください。 | 請寫住址。 |
| 電話番号（でんわばんごう） | **電話番号（でんわばんごう）**を書（か）いてください。 | 請寫電話號碼。 |
| 生年月日（せいねんがっぴ） | **生年月日（せいねんがっぴ）**を書（か）いてください。 | 請寫出生年月日。 |
| 国籍（こくせき） | **国籍（こくせき）**を書（か）いてください。 | 請寫國籍。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 否定請求形 | 名前（なまえ）を**書（か）かないでください**。 | 請不要寫名字。 |
| → 肯定命令形 | 名前（なまえ）を**書（か）いて**。 | 寫名字。 |
| → 丁寧表現形 | 名前（なまえ）を**書（か）いていただけませんか**。 | 能不能請您寫下名字呢？ |
| → 動詞進行形 | 名前（なまえ）を**書（か）いています**。 | 正在寫名字。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| ここに名前（なまえ）を書（か）いてください。 | はい、わかりました。ここに書（か）きます。 | 好的，我知道了。寫在這裡。 |
| 住所（じゅうしょ）も書（か）かなければなりませんか。 | はい、住所（じゅうしょ）も書（か）いてください。 | 是的，也請寫上住址。 |
| ちょっと待（ま）ってください。 | はい、ここで待（ま）ちます。 | 好的，我在這裡等。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 書（か）いてください。 | 請寫。 |
| +目的語 | **名前（なまえ）を**書（か）いてください。 | 請寫名字。 |
| +場所 | **ここに**名前（なまえ）を書（か）いてください。 | 請在這裡寫上名字。 |
| +詳細修飾 | ここに**お名前（なまえ）を黒（くろ）いペンで**書（か）いてください。 | 請在這裡用黑色鋼筆寫上您的名字。 |
