# 〜てもいいです

## 規則與接續
- `動詞て形 + もいいです`

## 用法說明
表示許可（可以做某事）。

## 實用例句
- 日文：<ruby>写真<rt>しゃしん (shashin)</rt></ruby>を<ruby>撮<rt>と (to)</rt></ruby>ってもいいですか。
- 中文：可以拍照嗎？
- 日文：ここで<ruby>休<rt>やす (yasu)</rt></ruby>んでもいいです。
- 中文：可以在這裡休息。
- 日文：エアコンをつけてもいいですよ。
- 中文：可以開冷氣喔。（給予對方許可）
- 日文：<ruby>質問<rt>しつもん (shitsumon)</rt></ruby>してもいいですか。
- 中文：我可以提問嗎？
- 日文：<ruby>今日<rt>きょう (kyou)</rt></ruby>は<ruby>早く<rt>はや (haya)</rt></ruby><ruby>帰<rt>かえ (kae)</rt></ruby>ってもいいです。
- 中文：今天可以早點回家。

## 關係說明
- 表示許可，其相反的禁止形式為 [[JP_Grammar_14_te_wa_ikemasen|〜てはいけません @contradicts]]。
- 其疑問形式 `〜てもいいですか` 用於詢問許可，相關的肯定與否定答覆參見 [[JP_Dialogue_Responses|情境對話應答 @related_to]]。
- 面對緊急事態時，其從容狀態會被 [[JP_Grammar_50_te_wa_irarenai|〜てはいられない @prevents]] 所排除（不能再做某事，無法容許放鬆）。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>写真<rt>しゃしん</rt></ruby>を<ruby>撮<rt>と</rt></ruby>ってもいいですか。
> 可以拍照嗎？

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 質問 (しつもん) をします | **質問して**もいいですか。 |
| ここに入 (はい) ります | ここに**入って**もいいですか。 |
| テレビを見 (み) ます | テレビを**見て**もいいですか。 |
| 帰 (かえ) ります | **帰って**もいいですか。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 丁寧肯定 (〜てもいいです) | 写真を撮っても**いいです**。 |
| → 禁止 (〜てはいけません) | 写真を撮っては**いけません**。 |
| → 普通形疑問 (〜てもいい？) | 写真を撮っても**いい**？ |
| → 義務 (〜なければなりません) | 写真を撮ら**なければなりません**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 写真を撮ってもいいですか。 | はい、撮ってもいいですよ。 |
| ここで休んでもいいですか。 | すみません、ここでは休んではいけません。 |
| 帰ってもいいですか。 | はい、早く帰ってもいいです。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | いいですか。 |
| +撮っても | 撮ってもいいですか。 |
| +写真を | 写真を撮ってもいいですか。 |
| +ここで | ここで写真を撮ってもいいですか。 |
