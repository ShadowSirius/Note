# 〜ながら

## 規則與接續
- `動詞ます形（去掉ます）+ nagara + 主要動詞`

## 用法說明
表示同時進行兩個動作（一邊...一邊...），後半段為主要動作。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E"><ruby>音樂<rt>おんがく (ongaku)</rt></ruby>を<ruby>聽<rt>き (ki)</rt></ruby>きながら</span><span style="color:#3498DB"><ruby>勉強<rt>べんきょう (benkyou)</rt></ruby>します。</span>
- 中文：一邊聽音樂一邊學習。
- 日文：<span style="color:#FF477E"><ruby>步<rt>ある (aru)</rt></ruby>きながら</span><span style="color:#3498DB"><ruby>電話<rt>でんわ (denwa)</rt></ruby>をかけます。</span>
- 中文：一邊走路一邊打電話。

## 關係說明
- 表示伴隨動作。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 音楽（おんがく）を聴（き）きながら勉強（べんきょう）します。
> （一邊聽音樂一邊學習。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| テレビ | **テレビを見（み）ながら**勉強（べんきょう）します。 | 一邊看電視一邊學習。 |
| お茶（ちゃ） | **お茶（ちゃ）を飲（の）みながら**勉強（べんきょう）します。 | 一邊喝茶一邊學習。 |
| 散歩（さんぽ） | **音楽（おんがく）を聴（き）きながら散歩（さんぽ）します**。 | 一邊聽音樂一邊散步。 |
| 電話（でんわ） | **歩（ある）きながら電話（でんわ）をかけます**。 | 一邊走路一邊打電話。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 音楽（おんがく）を聴（き）きながら勉強（べんきょう）**しました**。 | 一邊聽音樂一邊學習了。 |
| → 否定形 | 音楽（おんがく）を聴（き）きながら勉強（べんきょう）**しません**。 | 不一邊聽音樂一邊學習。 |
| → 進行形 | 音楽（おんがく）を聴（き）きながら勉強（べんきょう）**しています**。 | 正一邊聽音樂一邊學習。 |
| → 疑問形 | あなたはいつも音楽（おんがく）を聴（き）きながら勉強（べんきょう）します**か**。 | 你總是一邊聽音樂一邊學習嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|

### 4️⃣ Expansion Drill（擴展練習）

| 核心 | 勉強（べんきょう）します。 | 學習。 |
