# 〜ながら

## 規則與接續
- `動詞ます形（去掉ます）+ nagara + 主要動詞`

## 用法說明
表示同時進行兩個動作（一邊...一邊...），後半段為主要動作。

## 實用例句
- 日文：<ruby>音樂<rt>おんがく (ongaku)</rt></ruby>を<ruby>聽<rt>き (ki)</rt></ruby>きながら<ruby>勉強<rt>べんきょう (benkyou)</rt></ruby>します。
- 中文：一邊聽音樂一邊學習。
- 日文：<ruby>步<rt>ある (aru)</rt></ruby>きながら<ruby>電話<rt>でんわ (denwa)</rt></ruby>をかけます。
- 中文：一邊走路一邊打電話。

## 關係說明
- 表示伴隨動作。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>音楽<rt>おんがく</rt></ruby>を<ruby>聞<rt>き</rt></ruby>きながら<ruby>勉強<rt>べんきょう</rt></ruby>します。
> 一邊聽音樂一邊學習。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| コーヒーを飲みます | **コーヒーを飲みながら**勉強します。 |
| テレビを見ます | **テレビを見ながら**勉強します。 |
| ガムを噛みます | **ガムを噛みながら**勉強します。 |
| お菓子を食べます | **お菓子を食べながら**勉強します。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 過去形 (〜ました) | 音楽を聞きながら**勉強しました**。 |
| → 否定 (〜しません) | 音楽を聞きながら**勉強しません**。 |
| → 普通形 (〜する) | 音楽を聞きながら**勉強する**。 |
| → 進行形 (〜ています) | 音楽を聞きながら**勉強しています**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 音楽を聞きながら勉強しますか。 | はい、音楽を聞きながら勉強します。 |
| 何をしながら勉強しますか。 | 音楽を聞きながら勉強します。 |
| 歩きながら電話をかけますか。 | いいえ、歩きながら電話をかけません。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | 勉強します。 |
| +聞きながら | 聞きながら勉強します。 |
| +音楽を | 音楽を聞きながら勉強します。 |
| +毎日 | 毎日、音楽を聞きながら勉強します。 |
