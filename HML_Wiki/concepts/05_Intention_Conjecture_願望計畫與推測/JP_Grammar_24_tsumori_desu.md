# 〜つもりです

## 規則與接續
- `動詞原形/ない形 + つもりです`

## 用法說明
表示說話者個人的計畫、意圖或打算（打算...）。

## 實用例句
- 日文：<ruby>來年<rt>らいねん (rainen)</rt></ruby>、<ruby>日本<rt>にほん (nihon)</rt></ruby>へ<ruby>留學<rt>りゅうがく (ryuugaku)</rt></ruby>するつもりです。
- 中文：我打算明年去日本留學。
- 日文：タバコはもう<ruby>吸<rt>す (su)</rt></ruby>わないつもりです。
- 中文：我打算不再吸菸了。

## 關係說明
- 表示個人打算，與較強烈的主觀願望 [[JP_Grammar_15_tai_desu|〜たいです @related_to]] 概念相關。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>来年<rt>らいねん</rt></ruby>、<ruby>日本<rt>にほん</rt></ruby>へ<ruby>留学<rt>りゅうがく</rt></ruby>するつもりです。
> 我打算明年去日本留學。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 旅行します (りょこうします) | 来年、日本へ**旅行する**つもりです。 |
| 行きます (いきます) | 来年、日本へ**行く**つもりです。 |
| アメリカ | 来年、**アメリカ**へ留学するつもりです。 |
| イギリス | 来年、**イギリス**へ留学するつもりです。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定打算 (〜しないつもりです) | 来年、日本へ留学**しないつもりです**。 |
| → 過去打算 (〜つもりでした) | 来年、日本へ留学する**つもりでした**。 |
| → 過去否定打算 (〜しないつもりでした) | 来年、日本へ留学しない**つもりでした**。 |
| → 願望 (〜したいです) | 来年、日本へ留学**したいです**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 来年、日本へ留学するつもりですか。 | はい、日本へ留学するつもりです。 |
| いつ日本へ留学するつもりですか。 | 来年、日本へ留学するつもりです。 |
| タバコはもう吸わないつもりですか。 | はい、もう吸わないつもりです。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | つもりです。 |
| +留学するつもり | 留学するつもりです。 |
| +日本へ | 日本へ留学するつもりです。 |
| +来年 | 来年、日本へ留学するつもりです。 |
