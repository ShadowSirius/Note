# 〜ことができます

## 規則與接續
- `動詞原形 + ことができます`

## 用法說明
表示能力或客觀環境的允許（能夠做某事 / 會...）。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71"><ruby>私<rt>わたし (watashi)</rt></ruby>は</span><span style="color:#F39C12"><ruby>車<rt>くるま (kuruma)</rt></ruby>の</span><span style="color:#3498DB"><ruby>運轉<rt>うんてん (unten)</rt></ruby>ができます。</span>
- 中文：我會開車。
- 日文：<span style="color:#FF477E">ここで</span><span style="color:#3498DB">チケットを<ruby>買<rt>か (ka)</rt></ruby>うことができます。</span>
- 中文：可以在這裡買票。

## 關係說明
- 表示客觀能力或可能，相較於 [[JP_Grammar_13_te_mo_ii_desu|〜てもいいです @refines]]，它著重於「能力」而非主觀「許可」。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> ここでチケットを買（か）うことができます。
> （可以在這裡買票。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 本（ほん） | ここで**本（ほん）を借（か）りる**ことができます。 | 可以在這裡借書。 |
| 写真（しゃしん） | ここで**写真（しゃしん）を撮（と）る**ことができます。 | 可以在這裡拍照。 |
| 日本語（にほんご） | ここで**日本語（にほんご）を勉強（べんきょう）する**ことができます。 | 可以在這裡學習日語。 |
| お金（かね） | ここで**お金（かね）を換（か）える**ことができます。 | 可以在這裡換錢。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 否定形 | ここでチケットを買（か）うことが**できません**。 | 在這裡不能買票。 |
| → 過去肯定 | ここでチケットを買（か）うことが**できました**。 | 以前可以在這裡買票。 |
| → 名詞直接能力形 | 私（わたし）は車（くるま）の**運転（うんてん）ができます**。 | 我會開車。 |
| → 疑問形式 | ここでチケットを買（か）うことができます**か**。 | 可以在這裡買票嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| ここでチケットを買（か）うことができますか。 | はい、ここでチケットを買（か）うことができます。 | 是的，可以在這裡買票。 |
| あなたは車（くるま）の運転（うんてん）ができますか。 | いいえ、車（くるま）の運転（うんてん）はできません。 | 不，我不會開車。 |
| カードを使（つか）うことができますか。 | はい、カードを使（つか）うことができます。 | 是的，可以使用信用卡。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | できます。 | 可以。 |
| +句型 | **ことができます**。 | 能夠做...。 |
| +動作 | **買（か）う**ことができます。 | 能夠買。 |
| +地點目的 | **ここでチケットを**買（か）うことができます。 | 可以在這裡買票。 |
