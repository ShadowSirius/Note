# 〜ことができます

## 規則與接續
- `動詞原形 + ことができます`

## 用法說明
表示能力或客觀環境的允許（能夠做某事 / 會...）。

## 實用例句
- 日文：<ruby>私<rt>わたし (watashi)</rt></ruby>は<ruby>車<rt>くるま (kuruma)</rt></ruby>の<ruby>運轉<rt>うんてん (unten)</rt></ruby>ができます。
- 中文：我會開車。
- 日文：ここでチケットを<ruby>買<rt>か (ka)</rt></ruby>うことができます。
- 中文：可以在這裡買票。

## 關係說明
- 表示客觀能力或可能，相較於 [[JP_Grammar_13_te_mo_ii_desu|〜てもいいです @refines]]，它著重於「能力」而非主觀「許可」。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> ここでチケットを<ruby>買<rt>か</rt></ruby>うことができます。
> 可以在這裡買票。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 写真を撮ります | ここで**写真を撮る**ことができます。 |
| お金を払います | ここで**お金を払う**ことができます。 |
| 日本語を勉強します | ここで**日本語を勉強する**ことができます。 |
| 車を止めます | ここで**車を止める**ことができます。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定 (〜できません) | ここでチケットを買うことが**できません**。 |
| → 過去 (〜できました) | ここでチケットを買うことが**できました**。 |
| → 普通形 (〜できる) | ここでチケットを買うことが**できる**。 |
| → 普通形否定 (〜できない) | ここでチケットを買うことが**できない**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| ここでチケットを買うことができますか。 | はい、買うことができます。 |
| 車の運転ができますか。 | はい、運転ができます。 |
| どこでチケットを買うことができますか。 | ここでチケットを買うことができます。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | できます。 |
| +ことができます | ことができます。 |
| +買うことが | 買うことができます。 |
| +チケットを | チケットを買うことができます。 |
| +ここで | ここでチケットを買うことができます。 |
