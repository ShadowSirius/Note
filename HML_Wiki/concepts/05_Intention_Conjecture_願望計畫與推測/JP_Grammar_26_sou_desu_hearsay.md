# 〜そうです（傳聞）

## 規則與接續
- `動詞/形容詞普通形 + そうです`

## 用法說明
表示轉述聽來的消息（聽說... / 據說...）。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71"><ruby>明日<rt>あした (ashita)</rt></ruby>は</span><span style="color:#2ECC71"><ruby>雨<rt>あめ (ame)</rt></ruby>が</span><span style="color:#3498DB"><ruby>降<rt>ふ (fu)</rt></ruby>るそうです。</span>
- 中文：聽說明天下雨。
- 日文：<span style="color:#2ECC71"><ruby>田中<rt>たなか (tanaka)</rt></ruby>さんは</span><span style="color:#3498DB"><ruby>元氣<rt>げんき (genki)</rt></ruby>だそうです。</span>
- 中文：聽說田中先生精神很好。

## 關係說明
- 表示傳聞，與基於視覺推測的 [[JP_Grammar_25_sou_desu_conjecture|〜そうです（樣態） @sibling_of]] 有細微接續差異。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 明日（あした）は雨（あめ）が降（ふ）るそうです。
> （聽說明天下雨。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 雪（ゆき） | 明日（あした）は**雪（ゆき）が降（ふ）る**そうです。 | 聽說明天會下雪。 |
| 台風（たいふう） | 明日（あした）は**台風（たいふう）が来（く）る**そうです。 | 聽說明天颱風會來。 |
| 天気（てんき） | 明日（あした）は**天気（てんき）がいい**そうです。 | 聽說明天天氣很好。 |
| 風（かぜ） | 明日（あした）は**風（かぜ）が強（つよ）い**そうです。 | 聽說明天風很大。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去傳聞 | 昨日（きのう）は雨（あめ）が**降（ふ）った**そうです。 | 聽說昨天下了雨。 |
| → 否定傳聞 | 明日（あした）は雨（あめ）が**降（ふ）らない**そうです。 | 聽說明天不下雨。 |
| → 形容動詞傳聞 | 田中（たなか）さんは**元気（げんき）だ**そうです。 | 聽說田中先生精神很好。 |
| → 傳聞來源顯示 | **天気予報（てんきよほう）によると**、明日（あした）は雨（あめ）が降（ふ）るそうです。 | 根據天氣預報，聽說明天下雨。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 明日（あした）は雨（あめ）が降（ふ）るそうです。 | え、本当ですか。じゃ、傘（かさ）を持（も）っていきましょう。 | 誒，真的嗎？那帶傘去吧。 |
| 田中（たなか）さんはお元気（げんき）ですか。 | はい、とてもお元気（げんき）だそうです。 | 是的，聽說他精神非常好。 |
| 昨日（きのう）のテストは難（むずか）しかったですか。 | いいえ、あまり難（むずか）しくなかったそうです。 | 不，聽說沒有很難。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | そうです。 | 聽說。 |
| +動作 | **降（ふ）る**そうです。 | 聽說下。 |
| +主格 | **雨（あめ）が**降（ふ）るそうです。 | 聽說下雨。 |
| +時間 | **明日（あした）は**雨（あめ）が降（ふ）るそうです。 | 聽說明天下雨。 |
