# 〜そうです（傳聞）

## 規則與接續
- `動詞/形容詞普通形 + そうです`

## 用法說明
表示轉述聽來的消息（聽說... / 據說...）。

## 實用例句
- 日文：<ruby>明日<rt>あした (ashita)</rt></ruby>は<ruby>雨<rt>あめ (ame)</rt></ruby>が<ruby>降<rt>ふ (fu)</rt></ruby>るそうです。
- 中文：聽說明天下雨。
- 日文：<ruby>田中<rt>たなか (tanaka)</rt></ruby>さんは<ruby>元氣<rt>げんき (genki)</rt></ruby>だそうです。
- 中文：聽說田中先生精神很好。

## 關係說明
- 表示傳聞，與基於視覺推測的 [[JP_Grammar_25_sou_desu_conjecture|〜そうです（樣態） @sibling_of]] 有細微接續差異。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>明日<rt>あした</rt></ruby>は<ruby>雨<rt>あめ</rt></ruby>が<ruby>降<rt>ふ</rt></ruby>るそうです。
> 聽說明天下雨。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 雪 (ゆき) が降ります | 明日は**雪が降る**そうです。 |
| 晴れます (はれます) | 明日は**晴れる**そうです。 |
| 寒い (さむい) です | 明日は**寒い**そうです。 |
| 暑い (あつい) です | 明日は**暑い**そうです。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定傳聞 (〜ないそうです) | 明日は雨が**降らないそうです**。 |
| → 過去傳聞 (〜たそうです) | 昨日は雨が**降ったそうです**。 |
| → 樣態 (〜そうです) | 今にも雨が**降りそうです**。 |
| → 普通形傳聞 (〜そうだ) | 明日は雨が降る**そうだ**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 明日の天気はどうですか。 | 明日は雨が降るそうです。 |
| 田中さんは元気ですか。 | はい、田中さんは元気だそうです。 |
| 明日は寒いですか。 | はい、明日は寒いそうです。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | そうです。 |
| +降るそうです | 降るそうです。 |
| +雨が | 雨が降るそうです。 |
| +明日は | 明日は雨が降るそうです。 |
