# 〜たいです

## 規則與接續
- `動詞ます形（去掉ます）+ たいです`

## 用法說明
表示第一人稱或詢問第二人稱的願望（想做某事）。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E"><ruby>日本<rt>にほん (nihon)</rt></ruby>へ</span><span style="color:#FF477E"><ruby>旅行<rt>りょこう (ryokou)</rt></ruby>に</span><span style="color:#3498DB"><ruby>行<rt>い (i)</rt></ruby>きたいです。</span>
- 中文：我想去日本旅行。
- 日文：<span style="color:#3498DB">お<ruby>茶<rt>ちゃ (cha)</rt></ruby>が<ruby>飲<rt>の (no)</rt></ruby>みたいです。</span>
- 中文：我想喝茶。

## 關係說明
- 表示動詞願望，與表示具體行動計畫的 [[JP_Grammar_24_tsumori_desu|〜つもりです @related_to]] 互為補充。
- 與表示名詞願望（想要某物）的 [[JP_Grammar_31_ga_hoshii_desu|〜が欲しいです @parent_of]] 組成完整的日語願望表達系統。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 日本（にほん）へ旅行（りょこう）に行（い）きたいです。
> （我想去日本旅行。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 京都（きょうと） | **京都（きょうと）**へ旅行（りょこう）に行（い）きたいです。 | 我想去京都旅行。 |
| 北海道（ほっかいどう） | **北海道（ほっかいどう）**へ旅行（りょこう）に行（い）きたいです。 | 我想去北海道旅行。 |
| 沖縄（おきなわ） | **沖縄（おきなわ）**へ旅行（りょこう）に行（い）きたいです。 | 我想去沖繩旅行。 |
| 東京（とうきょう） | **東京（とうきょう）**へ旅行（りょこう）に行（い）きたいです。 | 我想去東京旅行。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 否定形 | 日本（にほん）へ旅行（りょこう）に行（い）**たくないです**。 | 我不想去日本旅行。 |
| → 過去形 | 日本（にほん）へ旅行（りょこう）に行（い）**たかったです**。 | 我以前想去日本旅行。 |
| → 過去否定形 | 日本（にほん）へ旅行（りょこう）に行（い）**たくなかったです**。 | 我以前不想去日本旅行。 |
| → 疑問形 | あなたも日本（にほん）へ旅行（りょこう）に行（い）きたいです**か**。 | 你也想去日本旅行嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 日本（にほん）へ旅行（りょこう）に行（い）きたいですか。 | はい、日本（にほん）へ旅行（りょこう）に行（い）きたいです。 | 是的，我想去日本旅行。 |
| お茶（ちゃ）が飲（の）みたいですか。 | いいえ、お茶（ちゃ）は飲（の）みたくないです。コーヒーが飲（の）みたいです。 | 不，我不想喝茶。我想喝咖啡。 |
| 何（なに）を食べ（たべ）たいですか。 | お寿司（すし）を食べ（たべ）たいです。 | 我想吃壽司。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 行（い）きたいです。 | 想去。 |
| +目的 | **旅行（りょこう）に**行（い）きたいです。 | 想去旅行。 |
| +目的地 | **日本（にほん）へ**旅行（りょこう）に行（い）きたいです。 | 想去日本旅行。 |
| +同伴 | **家族（かぞく）と**日本（にほん）へ旅行（りょこう）に行（い）きたいです。 | 我想和家人去日本旅行。 |
