# 〜たいです

## 規則與接續
- `動詞ます形（去掉ます）+ たいです`

## 用法說明
表示第一人稱或詢問第二人稱的願望（想做某事）。

## 實用例句
- 日文：<ruby>日本<rt>にほん (nihon)</rt></ruby>へ<ruby>旅行<rt>りょこう (ryokou)</rt></ruby>に<ruby>行<rt>い (i)</rt></ruby>きたいです。
- 中文：我想去日本旅行。
- 日文：お<ruby>茶<rt>ちゃ (cha)</rt></ruby>が<ruby>飲<rt>の (no)</rt></ruby>みたいです。
- 中文：我想喝茶。

## 關係說明
- 表示動詞願望，與表示具體行動計畫的 [[JP_Grammar_24_tsumori_desu|〜つもりです @related_to]] 互為補充。
- 與表示名詞願望（想要某物）的 [[JP_Grammar_31_ga_hoshii_desu|〜が欲しいです @parent_of]] 組成完整的日語願望表達系統。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>日本<rt>にほん</rt></ruby>へ<ruby>旅行<rt>りょこう</rt></ruby>に<ruby>行<rt>い</rt></ruby>きたいです。
> 我想去日本旅行。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 北海道 (ほっかいどう) | **北海道**へ旅行に行きたいです。 |
| 京都 (きょうと) | **京都**へ旅行に行きたいです。 |
| 沖縄 (おきなわ) | **沖縄**へ旅行に行きたいです。 |
| 買い物 (かいもの) | 日本へ**買い物**に行きたいです。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定 (〜たくないです) | 日本へ旅行に**行きたくないです**。 |
| → 過去 (〜たかったです) | 日本へ旅行に**行きたかったです**。 |
| → 過去否定 (〜たくなかったです) | 日本へ旅行に**行きたくなかったです**。 |
| → 普通形 (〜たい) | 日本へ旅行に**行きたい**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 日本へ旅行に行きたいですか。 | はい、日本へ旅行に行きたいです。 |
| どこへ旅行に行きたいですか。 | 日本へ旅行に行きたいです。 |
| お茶が飲みたいですか。 | はい、お茶が飲みたいです。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | 行きたいです。 |
| +旅行に | 旅行に行きたいです。 |
| +日本へ | 日本へ旅行に行きたいです。 |
| +来年 | 来年、日本へ旅行に行きたいです。 |
