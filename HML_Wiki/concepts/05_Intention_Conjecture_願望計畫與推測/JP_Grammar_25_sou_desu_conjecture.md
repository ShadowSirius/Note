# 〜そうです（樣態）

## 規則與接續
- `動詞ます形（去ます）/ 形容詞（去い/よ→よさ）+ そうです`

## 用法說明
表示根據眼前情況所做的直覺判斷（看起來好像... / 快要...）。

## 實用例句
- 日文：<ruby>今<rt>いま (ima)</rt></ruby>にも<ruby>雨<rt>あめ (ame)</rt></ruby>が<ruby>降<rt>ふ (fu)</rt></ruby>りそうです。
- 中文：看起來隨時會下雨的樣子。
- 日文：このケーキは<ruby>美味<rt>おい (oi)</rt></ruby>しそうです。
- 中文：這個蛋糕看起來很好吃。

## 關係說明
- 表示目視樣態，與表示傳聞 hearsay 的 [[JP_Grammar_26_sou_desu_hearsay|〜そうです（傳聞） @sibling_of]] 形式與意義皆不同。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> このケーキは<ruby>美味<rt>おい</rt></ruby>しそうです。
> 這個蛋糕看起來很好吃。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 料理 (りょうり) | この**料理**は美味しそうです。 |
| リンゴ | この**リンゴ**は美味しそうです。 |
| 甘い (あまい) | このケーキは**甘**そうです。 |
| 高い (たかい) | このケーキは**高**そうです。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定 (〜そうではありません) | このケーキは**美味しそうではありません**。 |
| → 過去樣態 (〜そうでした) | このケーキは**美味しそうでした**。 |
| → 普通形 (〜そうだ) | このケーキは**美味しそうだ**。 |
| → 傳聞 (〜いそうです) | このケーキは**美味しいそうです**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| このケーキは美味しそうですか。 | はい、とても美味しそうです。 |
| 今にも雨が降りそうですか。 | はい、今にも雨が降りそうです。 |
| あの料理は辛 (から) そうですか。 | はい、とても辛そうです。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | そうです。 |
| +美味しそう | 美味しそうです。 |
| +ケーキは | ケーキは美味しそうです。 |
| +このケーキは | このケーキは美味しそうです。 |
