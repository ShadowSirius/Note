# 〜そうです（樣態）

## 規則與接續
- `動詞ます形（去ます）/ 形容詞（去い/よ→よさ）+ そうです`

## 用法說明
表示根據眼前情況所做的直覺判斷（看起來好像... / 快要...）。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E"><ruby>今<rt>いま (ima)</rt></ruby>にも</span><span style="color:#2ECC71"><ruby>雨<rt>あめ (ame)</rt></ruby>が</span><span style="color:#3498DB"><ruby>降<rt>ふ (fu)</rt></ruby>りそうです。</span>
- 中文：看起來隨時會下雨的樣子。
- 日文：<span style="color:#F39C12">この</span><span style="color:#2ECC71">ケーキは</span><span style="color:#3498DB"><ruby>美味<rt>おい (oi)</rt></ruby>しそうです。</span>
- 中文：這個蛋糕看起來很好吃。

## 關係說明
- 表示目視樣態，與表示傳聞 hearsay 的 [[JP_Grammar_26_sou_desu_hearsay|〜そうです（傳聞） @sibling_of]] 形式與意義皆不同。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> このケーキは美味し（おいし）そうです。
> （這個蛋糕看起來很好吃。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 辛い（から） | このラーメンは**辛（から）そうです**。 | 這個拉麵看起來很辣。 |
| よさそう | このお弁当（べんとう）は**よさそうです**。 | 這個便當看起來很棒。 |
| 甘い（あま） | この果物（くだもの）は**甘（あま）そうです**。 | 這個水果看起來很甜。 |
| 美味しい（おいし） | この料理（りょうり）は**美味し（おいし）そうです**。 | 這道料理看起來很好吃。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 否定形 | このケーキは美味し（おいし）**そうではありません**。 | 這個蛋糕看起來不好吃。 |
| → 過去形 | このケーキは美味し（おいし）**そうでした**。 | 這個蛋糕那時看起來很好吃。 |
| → 動作推測 | 今（いま）にも雨（あめ）が**降（ふ）りそうです**。 | 看起來隨時要下雨的樣子。 |
| → 疑問形 | そのケーキは美味し（おいし）そうです**か**。 | 那個蛋糕看起來很好吃嗎？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| このケーキは美味し（おいし）そうです。 | はい、本当に美味し（おいし）そうですね。食べ（たべ）ましょう。 | 是的，真的看起來很好吃。吃吧。 |
| 雨（あめ）が降（ふ）りそうです。 | はい、傘（かさ）を持って行（い）ったほうがいいですね。 | 是的，最好帶傘去呢。 |
| このカメラは高（たか）そうです。 | いいえ、これはあまり高（たか）そうではありません。 | 不，這台看起來沒有很貴。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | そうです。 | 看起來。 |
| +狀態描述 | **美味し（おいし）**そうです。 | 看起來好吃。 |
| +主題 | **ケーキは**美味し（おいし）そうです。 | 蛋糕看起來好吃。 |
| +指示修飾 | **この**ケーキは美味し（おいし）そうです。 | 這個蛋糕看起來好吃。 |
