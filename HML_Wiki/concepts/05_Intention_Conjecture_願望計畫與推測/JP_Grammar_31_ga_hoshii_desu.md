# 〜が欲しいです

## 規則與接續
- `名詞 + が + 欲しいです`

## 用法說明
表示第一人稱想要某個「物品」（名詞），或用於詢問第二人稱。
- 否定形式為：`〜は欲しくないです` 或 `〜は入りません`（不需要/不用）。

## 實用例句
- 日文：お<ruby>酒<rt>さけ (sake)</rt></ruby>が<ruby>欲<rt>ほ (ho)</rt></ruby>しいです。
- 中文：我想要酒。
- 日文：お<ruby>酒<rt>さけ (sake)</rt></ruby>は<ruby>欲<rt>ほ (ho)</rt></ruby>しくないです。
- 中文：我不要酒。
- 日文：お<ruby>酒<rt>さけ (sake)</rt></ruby>は<ruby>入<rt>い (i)</rt></ruby>りません。
- 中文：我不需要酒。
- 日文：このパンが<ruby>欲<rt>ほ (ho)</rt></ruby>しいです。
- 中文：我想要這個麵包。
- 日文：あんな<ruby>服<rt>ふく (fuku)</rt></ruby>が<ruby>欲<rt>ほ (ho)</rt></ruby>しいです。
- 中文：我想要那樣的衣服。

## 關係說明
- 與同樣表達願望的 [[JP_Grammar_15_tai_desu|〜たいです @child_of]] 互為補充（`〜たいです` 用於動詞，`〜が欲しいです` 用於名詞）。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> お<ruby>酒<rt>さけ</rt></ruby>が<ruby>欲<rt>ほ</rt></ruby>しいです。
> 我想要酒。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 新しい車 (あたらしいくるま) | **新しい車**が欲しいです。 |
| パソコン | **パソコン**が欲しいです。 |
| 時間 (じかん) | **時間**が欲しいです。 |
| 休み (やすみ) | **休み**が欲しいです。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定 (〜は欲しくないです) | お酒は**欲しくないです**。 |
| → 過去 (〜が欲しかったです) | お酒が**欲しかったです**。 |
| → 過去否定 (〜は欲しくなかったです) | お酒は**欲しくなかったです**。 |
| → 不需要 (〜は入りません) | お酒は**入りません**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| お酒が欲しいですか。 | はい、お酒が欲しいです。 |
| 新しい車が欲しいですか。 | いいえ、新しい車は欲しくないです。 |
| 何が欲しいですか。 | パソコンが欲しいです。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | 欲しいです。 |
| +が欲しいです | が欲しいです。 |
| +お酒が | お酒が欲しいです。 |
| +今 | 今、お酒が欲しいです。 |
