# 〜ので

## 規則與接續
- `名詞+な/形容詞/動詞普通形 + ので`

## 用法說明
表示原因或理由（因為...）。語氣比「から」更柔和客觀，常用於委婉託辭或商務場合。

## 實用例句
- 日文：<ruby>雨<rt>あめ (ame)</rt></ruby>が<ruby>降<rt>ふ (fu)</rt></ruby>るので、<ruby>遲<rt>おく (oku)</rt></ruby>れました。
- 中文：因為下雨所以遲到了。
- 日文：<ruby>頭<rt>あたま (atama)</rt></ruby>が<ruby>痛<rt>いた (ita)</rt></ruby>いので、<ruby>先<rt>さき (saki)</rt></ruby>に<ruby>帰<rt>かえ (kae)</rt></ruby>ります。
- 中文：因為頭痛，先回去了。

## 關係說明
- 表示客觀原因，常用於商務客氣口氣以替代主觀的 [[JP_Grammar_10_kara_reason|〜から @replaces]]。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>頭<rt>あたま</rt></ruby>が<ruby>痛<rt>いた</rt></ruby>いので、<ruby>先<rt>さき</rt></ruby>に<ruby>帰<rt>かえ</rt></ruby>ります。
> 因為頭痛，先回去了。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| お腹 (おなか) が痛いです | **お腹が痛い**ので、先に帰ります。 |
| 風邪をひきました | **風邪をひいた**ので、先に帰ります。 |
| 用事 (ようじ) があります | **用事がある**ので、先に帰ります。 |
| 気分が悪いです | **気分が悪い**ので、先に帰ります。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 主觀原因 (〜から) | 頭が痛い**から**、先に帰ります。 |
| → 過去結果 (〜帰りました) | 頭が痛いので、先に**帰りました**。 |
| → 詢問許可 (〜帰ってもいいですか) | 頭が痛いので、先に**帰ってもいいですか**。 |
| → 普通形 (〜帰る) | 頭が痛いので、先に**帰る**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| どうして先に帰りますか。 | 頭が痛いので、先に帰ります。 |
| 頭が痛いですが、どうしますか。 | 頭が痛いので、先に帰ります。 |
| 雨が降るので、遅れましたか。 | はい、雨が降るので、遅れました。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | 帰ります。 |
| +先に | 先に帰ります。 |
| +痛いので | 痛いので、先に帰ります。 |
| +頭が | 頭が痛いので、先に帰ります。 |
