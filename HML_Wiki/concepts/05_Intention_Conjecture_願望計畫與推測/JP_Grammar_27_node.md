# 〜ので

## 規則與接續
- `名詞+な/形容詞/動詞普通形 + ので`

## 用法說明
表示原因或理由（因為...）。語氣比「から」更柔和客觀，常用於委婉託辭或商務場合。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71"><ruby>雨<rt>あめ (ame)</rt></ruby>が</span><span style="color:#FF477E"><ruby>降<rt>ふ (fu)</rt></ruby>るので、</span><span style="color:#3498DB"><ruby>遲<rt>おく (oku)</rt></ruby>れました。</span>
- 中文：因為下雨所以遲到了。
- 日文：<span style="color:#2ECC71"><ruby>頭<rt>あたま (atama)</rt></ruby>が</span><span style="color:#FF477E"><ruby>痛<rt>いた (ita)</rt></ruby>いので、</span><span style="color:#FF477E"><ruby>先<rt>さき (saki)</rt></ruby>に</span><span style="color:#3498DB"><ruby>帰<rt>かえ (kae)</rt></ruby>ります。</span>
- 中文：因為頭痛，先回去了。

## 關係說明
- 表示客觀原因，常用於商務客氣口氣以替代主觀的 [[JP_Grammar_10_kara_reason|〜から @replaces]]。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 雨（あめ）が降（ふ）るので、遅（おく）れました。
> （因為下雨所以遲到了。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 風邪（かぜ） | **風邪（かぜ）をひいた**ので、遅（おく）れました。 | 因為感冒所以遲到了。 |
| 事故（じこ） | **事故（じこ）があった**ので、遅（おく）れました。 | 因為發生事故所以遲到了。 |
| バス | **バスが来（こ）なかった**ので、遅（おく）れました。 | 因為公車沒來所以遲到了。 |
| 道（みち） | **道（みち）が混（こ）んでいた**ので、遅（おく）れました。 | 因為塞車所以遲到了。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 丁寧形結尾前置 | 雨（あめ）が**降（ふ）りますので**、遅（おく）れました。 | 因為下雨，所以遲到了。 |
| → 過去結果否定 | 雨（あめ）が降（ふ）るので、遅（おく）**れませんでした**。 | 雖然下雨，但我沒有遲到。 |
| → 形容詞原因句型 | **頭（あたま）が痛（いた）いので、先（さき）に帰（かえ）ります**。 | 因為頭痛，我先回去了。 |
| → 普通形疑問原因 | どうして遅（おく）れましたか。雨（あめ）が降（ふ）った**からです**。 | 為什麼遲到了？因為下雨了。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうして遅（おく）れましたか。 | 雨（あめ）が降（ふ）ったので、遅（おく）れました。 | 因為下雨所以遲到了。 |
| 頭（あたま）が痛（いた）いのですか。 | はい、頭（あたま）が痛（いた）いので、先（さき）に帰（かえ）ります。 | 是的，因為頭痛，我先回去了。 |
| どうしてエアコンをつけましたか。 | 暑（あつ）いので、エアコンをつけました。 | 因為熱，所以開了冷氣。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 遅（おく）れました。 | 遲到了。 |
| +因果 | **ので**、遅（おく）れました。 | 因為...所以遲到了。 |
| +動作 | **降（ふ）るので**、遅（おく）れました。 | 因為下...所以遲到了。 |
| +主格原因 | **雨（あめ）が**降（ふ）るので、遅（おく）れました。 | 因為下雨所以遲到了。 |
