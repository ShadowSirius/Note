# 〜を

## 規則與接續
- `名詞 + を + 他動詞`

## 用法說明
標記動作的直接對象（賓語）。

## 實用例句
- 日文：ご<ruby>飯<rt>はん (han)</rt></ruby>を<ruby>食<rt>た (ta)</rt></ruby>べます。
- 中文：吃飯。
- 日文：<ruby>水<rt>みず (mizu)</rt></ruby>を<ruby>飲<rt>の (no)</rt></ruby>みます。
- 中文：喝水。

## 關係說明
- 搭配他動詞使用，標記動作承受者。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> ご<ruby>飯<rt>はん</rt></ruby>を<ruby>食<rt>た</rt></ruby>べます。
> 吃飯。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| パン | **パン**を食べます。 |
| <ruby>寿司<rt>すし</rt></ruby> | **<ruby>寿司<rt>すし</rt></ruby>**を食べます。 |
| <ruby>水<rt>みず</rt></ruby>を<ruby>飲<rt>の</rt></ruby>みます | **<ruby>水<rt>みず</rt></ruby>を<ruby>飲<rt>の</rt></ruby>みます**。 |
| <ruby>本<rt>ほん</rt></ruby>を<ruby>読<rt>よ</rt></ruby>みます | **<ruby>本<rt>ほん</rt></ruby>を<ruby>読<rt>よ</rt></ruby>みます**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定形 | ご飯を**食べません**。 |
| → 過去形 | ご飯を**食べました**。 |
| → 過去否定形 | ご飯を**食べませんでした**。 |
| → 疑問形 | ご飯を食べます**か**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 何を食べますか。 | ご飯を食べます。 |
| 朝ご飯を食べましたか。 | はい、朝ご飯を食べました。 |
| 水を飲みますか。 | いいえ、水を飲みません。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>食<rt>た</rt></ruby>べます。 |
| +目的語 | **ご飯を**<ruby>食<rt>た</rt></ruby>べます。 |
| +時間 | **朝**ご飯を食べます。 |
| +場所 | **食堂で**朝ご飯を食べます。 |
