# 〜を

## 規則與接續
- `名詞 + を + 他動詞`

## 用法說明
標記動作的直接對象（賓語）。

## 實用例句
- 日文：ご<ruby>飯<rt>はん (han)</rt></ruby>を<ruby>食<rt>た (ta)</rt></ruby>べます。
- 中文：吃飯。
- 日文：<ruby>水<rt>みず (mizu)</rt></ruby>を<ruby>飲<rt>の (no)</rt></ruby>みます。
- 中文：喝水。

## 關係說明
- 搭配他動詞使用，標記動作承受者。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 水（みず）を飲（の）みます。
> （喝水。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| お茶（ちゃ） | お**茶（ちゃ）**を飲（の）みます。 | 喝茶。 |
| 牛乳（ぎゅうにゅう） | **牛乳（ぎゅうにゅう）**を飲（の）みます。 | 喝牛奶。 |
| コーヒー | **コーヒー**を飲（の）みます。 | 喝咖啡。 |
| お酒（さけ） | お**酒（さけ）**を飲（の）みます。 | 喝酒。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 水（みず）を**飲（の）みました**。 | 喝了水。 |
| → 否定形 | 水（みず）を**飲（の）みません**。 | 不喝水。 |
| → 勸誘形 | 水（みず）を**飲（の）みましょう**。 | 一起喝水吧。 |
| → 願望形 | 水（みず）が**飲（の）みたいです**。 | 我想喝水。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 何（なに）を飲（の）みますか。 | 水（みず）を飲（の）みます。 | 喝水。 |
| 朝ごはんを食べましたか。 | はい、ご飯（はん）を食べました。 | 是的，吃了飯。 |
| コーヒーを飲みませんか。 | はい、飲みましょう。 | 好的，一起喝吧。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 飲（の）みます。 | 喝。 |
| +目的語 | **水（みず）を**飲（の）みます。 | 喝水。 |
| +副詞 | 水（みず）を**たくさん**飲（の）みます。 | 喝很多水。 |
| +時間 | **毎日（まいにち）**水（みず）をたくさん飲（の）みます。 | 每天喝很多水。 |
