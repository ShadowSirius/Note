# 〜に / 〜へ

## 規則與接續
- `場所名詞 + に / へ + 動詞（行く、来るなど）`

## 用法說明
表示移動的目的地（に）或方向（へ）。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E"><ruby>日本<rt>にほん (nihon)</rt></ruby>に</span><span style="color:#3498DB"><ruby>行<rt>い (i)</rt></ruby>きます。</span>
- 中文：去日本。
- 日文：<span style="color:#FF477E"><ruby>學校<rt>がっこう (gakkou)</rt></ruby>へ</span><span style="color:#3498DB"><ruby>來<rt>き (ki)</rt></ruby>ました。</span>
- 中文：來到學校。

## 關係說明
- 表示移動目的地，須注意與表示動作發生場所的 [[JP_Grammar_05_de|〜で @contradicts]] 區分。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 日本（にほん）に行（い）きます。
> （去日本。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 学校（がっこう） | **学校（がっこう）**に行（い）きます。 | 去學校。 |
| 駅（えき） | **駅（えき）**に行（い）きます。 | 去車站。 |
| 病院（びょういん） | **病院（びょういん）**に行（い）きます。 | 去醫院。 |
| 空港（くうこう） | **空港（くうこう）**に行（い）きます。 | 去機場。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | 日本（にほん）に**行（い）きました**。 | 去了日本。 |
| → 否定形 | 日本（にほん）に**行（い）きません**。 | 不去日本。 |
| → 過去否定形 | 日本（にほん）に**行（い）きませんでした**。 | 以前沒去日本。 |
| → 方向助詞替換（へ） | 日本（にほん）**へ**行（い）きます。 | 往日本去。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| いつ日本（にほん）に行（い）きますか。 | 来月（らいげつ）日本（にほん）に行（い）きます。 | 下個月去日本。 |
| 誰（だれ）と日本（にほん）に行（い）きますか。 | 友達（ともだち）と日本（にほん）に行（い）きます。 | 和朋友去日本。 |
| 学校（がっこう）へ来（き）ましたか。 | はい、もう学校（がっこう）へ来（き）ました。 | 是的，已經來到學校了。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 行（い）きます。 | 去。 |
| +地點 | **日本（にほん）に**行（い）きます。 | 去日本。 |
| +同伴 | **友達（ともだち）と**日本（にほん）に行（い）きます。 | 和朋友去日本。 |
| +時間 | **来年（らいねん）**友達（ともだち）と日本（にほん）に行（い）きます。 | 明年和朋友去日本。 |
