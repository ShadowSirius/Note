# 〜に / 〜へ

## 規則與接續
- `場所名詞 + に / へ + 動詞（行く、来るなど）`

## 用法說明
表示移動的目的地（に）或方向（へ）。

## 實用例句
- 日文：<ruby>日本<rt>にほん (nihon)</rt></ruby>に<ruby>行<rt>い (i)</rt></ruby>きます。
- 中文：去日本。
- 日文：<ruby>學校<rt>がっこう (gakkou)</rt></ruby>へ<ruby>來<rt>き (ki)</rt></ruby>ました。
- 中文：來到學校。

## 關係說明
- 表示移動目的地，須注意與表示動作發生場所的 [[JP_Grammar_05_de|〜で @contradicts]] 區分。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>日本<rt>にほん</rt></ruby>に<ruby>行<rt>い</rt></ruby>きます。
> 去日本。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| <ruby>学校<rt>がっこう</rt></ruby> | **<ruby>学校<rt>がっこう</rt></ruby>**に行きます。 |
| <ruby>会社<rt>かいしゃ</rt></ruby> | **<ruby>会社<rt>かいしゃ</rt></ruby>**に行きます。 |
| <ruby>帰<rt>かえ</rt></ruby>ります | 日本に**<ruby>帰<rt>かえ</rt></ruby>ります**。 |
| <ruby>来<rt>き</rt></ruby>ます | 日本に**<ruby>来<rt>き</rt></ruby>ます**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定形 | 日本に**行きません**。 |
| → 過去形 | 日本に**行きました**。 |
| → 過去否定形 | 日本に**行きませんでした**。 |
| → 方向助詞「へ」 | 日本**へ**行きます。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| どこに行きますか。 | 日本に行きます。 |
| いつ日本に行きますか。 | 来週、日本に行きます。 |
| 誰と日本に行きますか。 | 友達と日本に行きます。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>行<rt>い</rt></ruby>きます。 |
| +目的地 | **日本に**<ruby>行<rt>い</rt></ruby>きます。 |
| +時間 | **来週**日本に行きます。 |
| +共同主體 | 来週**友達と**日本に行きます。 |
