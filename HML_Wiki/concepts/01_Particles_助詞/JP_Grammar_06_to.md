# 〜と

## 規則與接續
- `名詞 + と + 名詞 / 名詞 + と + 動詞`

## 用法說明
1. 表示並列（和）；2. 表示動作的共同參與者（和...一起）。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#3498DB">りんごとみかんを<ruby>買<rt>か (ka)</rt></ruby>いました。</span>
- 中文：買了蘋果和橘子。
- 日文：<span style="color:#FF477E"><ruby>友達<rt>ともだち (tomodachi)</rt></ruby>と</span><span style="color:#3498DB"><ruby>映畫<rt>えいが (eiga)</rt></ruby>を<ruby>見<rt>み (mi)</rt></ruby>ました。</span>
- 中文：和朋友一起看了電影。

## 關係說明
- 表示共同參與者。

### 補充：や〜など（舉例並列，非完全列舉）
- `名詞 A + や + 名詞 B + （…）+ など`：從眾多同類事物中**舉出幾個例子**，暗示「還有其他沒說的」。
- **與 と 的差異**：と 是**完全列舉**（列出的就是全部）；や〜など 是**部分舉例**（只是舉例，實際上可能更多）。
  - 日文：<span style="color:#3498DB">りんごと</span><span style="color:#3498DB">みかんを買いました。</span> 中文：買了蘋果和橘子。（**只買了這兩樣，完全列舉**）
  - 日文：<span style="color:#3498DB"><ruby>机<rt>つくえ (tsukue)</rt></ruby>の<ruby>上<rt>うえ (ue)</rt></ruby>に</span><span style="color:#E67E22">本や<ruby>ノート<rt>のーと</rt></ruby>など</span><span style="color:#3498DB">があります。</span> 中文：桌上有書、筆記本等等（東西）。（**只是舉例，桌上可能還有其他物品**）
- 補充：口語／書面亦常見「〜等々（とうとう）」表示「…等等」，語感較正式；日常口語則更常直接讀作「などなど」（雖非標準讀音，但廣泛使用）。與 など 意思相通，皆表示部分舉例。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 友達（ともだち）と映画（えいが）を見（み）ました。
> （和朋友一起看了電影。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 家族（かぞく） | **家族（かぞく）**と映画（えいが）を見（み）ました。 | 和家人一起看了電影。 |
| 先生（せんせい） | **先生（せんせい）**と映画（えいが）を見（み）ました。 | 和老師一起看了電影。 |
| 彼女（かのじょ） | **彼女（かのじょ）**と映画（えいが）を見（み）ました。 | 和女朋友一起看了電影。 |
| 同僚（どうりょう） | **同僚（どうりょう）**と映画（えいが）を見（み）ました。 | 和同事一起看了電影。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 現在形 | 友達（ともだち）と映画（えいが）を**見（み）ます**。 | 和朋友一起看電影。 |
| → 否定形 | 友達（ともだち）と映画（えいが）を**見（み）ません**。 | 不和朋友一起看電影。 |
| → 勸誘形 | 友達（ともだち）と映画（えいが）を**見（み）ましょう**。 | 和朋友一起看電影吧。 |
| → 並列助詞替換 | りんごとみかんを**買（か）いました**。 | 買了蘋果和橘子。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 誰（だれ）と映画（えいが）を見（み）ましたか。 | 友達（ともだち）と映画（えいが）を見（み）ました。 | 和朋友一起看了電影。 |
| 友達（ともだち）と何（なに）を見（み）ましたか。 | 映画（えいが）を見（み）ました。 | 看了電影。 |
| りんごとみかんを買（か）いましたか。 | はい、りんごとみかんを買（か）いました。 | 是的，買了蘋果和橘子。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 見（み）ました。 | 看了。 |
| +目的語 | **映画（えいが）を**見（み）ました。 | 看了電影。 |
| +同伴 | **友達（ともだち）と**映画（えいが）を見（み）ました。 | 和朋友一起看了電影。 |
| +時間 | **昨日（きのう）**友達（ともだち）と映画（えいが）を見（み）ました。 | 昨天和朋友一起看了電影。 |
