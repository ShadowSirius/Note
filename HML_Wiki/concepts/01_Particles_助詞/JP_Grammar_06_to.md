# 〜と

## 規則與接續
- `名詞 + と + 名詞 / 名詞 + と + 動詞`

## 用法說明
1. 表示並列（和）；2. 表示動作的共同參與者（和...一起）。

## 實用例句
- 日文：りんごとみかんを<ruby>買<rt>か (ka)</rt></ruby>いました。
- 中文：買了蘋果和橘子。
- 日文：<ruby>友達<rt>ともだち (tomodachi)</rt></ruby>と<ruby>映畫<rt>えいが (eiga)</rt></ruby>を<ruby>見<rt>み (mi)</rt></ruby>ました。
- 中文：和朋友一起看了電影。

## 關係說明
- 表示共同參與者。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>友達<rt>ともだち</rt></ruby>と<ruby>映畫<rt>えいが</rt></ruby>を<ruby>見<rt>み</rt></ruby>ました。
> 和朋友一起看了電影。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| <ruby>家族<rt>かぞく</rt></ruby> | **<ruby>家族<rt>かぞく</rt></ruby>**と映画を見ました。 |
| <ruby>先生<rt>せんせい</rt></ruby> | **<ruby>先生<rt>せんせい</rt></ruby>**と映画を見ました。 |
| <ruby>ご飯<rt>はん</rt></ruby>を<ruby>食<rt>た</rt></ruby>べました | 友達と**ご<ruby>飯<rt>はん</rt></ruby>を<ruby>食<rt>た</rt></ruby>べました**。 |
| <ruby>日本<rt>にほん</rt></ruby>に<ruby>行<rt>い</rt></ruby>きました | 友達と**日本に<ruby>行<rt>い</rt></ruby>きました**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 現在形 | 友達と映画を**見ます**。 |
| → 否定形 | 友達と映画を**見ません**。 |
| → 過去否定形 | 友達と映画を**見ませんでした**。 |
| → 並列の「と」 | りんご**と**みかんを買いました。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 誰と映画を見ましたか。 | 友達と映画を見ました。 |
| 友達と何をしましたか。 | 友達と映画を見ました。 |
| 一人で映画を見ましたか。 | いいえ、友達と見ました。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>見<rt>み</rt></ruby>ました。 |
| +目的語 | **映画を**<ruby>見<rt>み</rt></ruby>ました。 |
| +共同主體 | **友達と**映画を見ました。 |
| +時間 | **昨日**友達と映画を見ました。 |
