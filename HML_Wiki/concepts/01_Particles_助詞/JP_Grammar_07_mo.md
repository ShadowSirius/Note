# 〜も

## 規則與接續
- `名詞 + も`

## 用法說明
表示「也」。會取代或疊加於「は」和「が」之上。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#2ECC71"><ruby>彼<rt>かれ (kare)</rt></ruby>も</span><span style="color:#3498DB"><ruby>學生<rt>がくせい (gakusei)</rt></ruby>です。</span>
- 中文：他也是學生。
- 日文：<span style="color:#3498DB"><ruby>日本語<rt>にほんご (nihongo)</rt></ruby>も<ruby>英語<rt>えいご (eigo)</rt></ruby>も<ruby>話<rt>はな (hana)</rt></ruby>せます。</span>
- 中文：日文和英文都會說。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 彼（かれ）も学生（がくせい）です。
> （他也是學生。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 私（わたし） | **私（わたし）**も学生（がくせい）です。 | 我也是學生。 |
| 彼女（かのじょ） | **彼女（かのじょ）**も学生（がくせい）です。 | 她也是學生。 |
| 李（り）さん | **李（り）さん**も学生（がくせい）です。 | 李先生也是學生。 |
| 留学生（りゅうがくせい） | **留学生（りゅうがくせい）**も学生（がくせい）です。 | 留學生也是學生。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 否定形 | 彼（かれ）も学生（がくせい）**ではありません**。 | 他也不是學生。 |
| → 過去形 | 彼（かれ）も学生（がくせい）**でした**。 | 他以前也是學生。 |
| → 複數主語否定形式 | 彼（かれ）も私（わたし）も学生（がくせい）**ではありません**。 | 和我都不是學生。 |
| → 雙重能力形式 | 日本語（にほんご）も英語（えいご）も**話（はな）せます**。 | 日文和英文都會說。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| あなたは学生（がくせい）ですか。 | はい、私は学生（がくせい）です。彼（かれ）も学生（がくせい）です。 | 是的，我是學生。他也是學生。 |
| 李（り）さんも韓国人（かんこくじん）ですか。 | いいえ、李（り）さんは韓国人（かんこくじん）ではありません。 | 不，李先生不是韓國人。 |
| 日本語（にほんご）が話（はな）せますか。 | はい、日本語（にほんご）も英語（えいご）も話（はな）せます。 | 是的，日語和英語都會說。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 学生（がくせい）です。 | 是學生。 |
| +主題 | 彼（かれ）は学生（がくせい）です。 | 他是學生。 |
| +追加主語 | 彼（かれ）は学生（がくせい）です。**私（わたし）も**学生（がくせい）です。 | 他是學生。我也是學生。 |
| +否定追加 | 彼（かれ）は学生（がくせい）ではありません。**私（わたし）も**学生（がくせい）ではありません。 | 他不是學生。我也不是學生。 |
