# 〜も

## 規則與接續
- `名詞 + も`

## 用法說明
表示「也」。會取代或疊加於「は」和「が」之上。

## 實用例句
- 日文：<ruby>彼<rt>かれ (kare)</rt></ruby>も<ruby>學生<rt>がくせい (gakusei)</rt></ruby>です。
- 中文：他也是學生。
- 日文：<ruby>日本語<rt>にほんご (nihongo)</rt></ruby>も<ruby>英語<rt>えいご (eigo)</rt></ruby>も<ruby>話<rt>はな (hana)</rt></ruby>せます。
- 中文：日文和英文都會說。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>彼<rt>かれ</rt></ruby>も<ruby>學生<rt>がくせい</rt></ruby>です。
> 他也是學生。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| <ruby>私<rt>わたし</rt></ruby> | **<ruby>私<rt>わたし</rt></ruby>**も學生です。 |
| <ruby>李さん<rt>りさん</rt></ruby> | **<ruby>李さん<rt>りさん</rt></ruby>**も學生です。 |
| <ruby>先生<rt>せんせい</rt></ruby> | 彼も**<ruby>先生<rt>せんせい</rt></ruby>**です。 |
| <ruby>日本人<rt>にほんじん</rt></ruby> | 彼も**<ruby>日本人<rt>にほんじん</rt></ruby>**です。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定形 | 彼も學生**ではありません**。 |
| → 過去形 | 彼も學生**でした**。 |
| → AもBも | 日本語**も**英語**も**話せます。 |
| → 疑問形 | 彼も學生です**か**。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| あなたは学生ですか。 | はい、私は学生です。彼も学生です。 |
| 彼女も日本人ですか。 | いいえ、彼女は日本人ではありません。 |
| 日本語が話せますか。英語はどうですか。 | 日本語も英語も話せます。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>學生<rt>がくせい</rt></ruby>です。 |
| +主語 | **彼も**<ruby>學生<rt>がくせい</rt></ruby>です。 |
| +修飾 | 彼も**この大学の**<ruby>學生<rt>がくせい</rt></ruby>です。 |
| +副詞 | 彼も**すでに**この大学の<ruby>學生<rt>がくせい</rt></ruby>です。 |
