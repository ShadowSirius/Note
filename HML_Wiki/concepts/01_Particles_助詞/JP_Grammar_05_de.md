# 〜で

## 規則與接續
- `名詞 + で`

## 用法說明
1. 表示動作進行的場所；2. 表示手段、方法、工具、材料。

## 實用例句
- 日文：<ruby>圖書館<rt>としょかん (toshokan)</rt></ruby>で<ruby>勉強<rt>べんきょう (benkyou)</rt></ruby>します。
- 中文：在圖書館學習。
- 日文：ペンで<ruby>書<rt>か (ka)</rt></ruby>きます。
- 中文：用鋼筆寫。

## 關係說明
- 表示動作發生的場所，與表示存在位置或移動終點的 [[JP_Grammar_03_ni_he|〜に @contradicts]] 不同。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>圖書館<rt>としょかん</rt></ruby>で<ruby>勉強<rt>べんきょう</rt></ruby>します。
> 在圖書館學習。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| <ruby>学校<rt>がっこう</rt></ruby> | **<ruby>学校<rt>がっこう</rt></ruby>**で勉強します。 |
| <ruby>部屋<rt>へや</rt></ruby> | **<ruby>部屋<rt>へや</rt></ruby>**で勉強します。 |
| <ruby>本<rt>ほん</rt></ruby>を<ruby>読<rt>よ</rt></ruby>みます | 圖書館で**<ruby>本<rt>ほん</rt></ruby>を<ruby>読<rt>よ</rt></ruby>みます**。 |
| <ruby>友達<rt>ともだち</rt></ruby>と<ruby>話<rt>はな</rt></ruby>します | 圖書館で**<ruby>友達<rt>ともだち</rt></ruby>と<ruby>話<rt>はな</rt></ruby>します**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定形 | 圖書館で勉強**しません**。 |
| → 過去形 | 圖書館で勉強**しました**。 |
| → 過去否定形 | 圖書館で勉強**しませんでした**。 |
| → 工具/手段 | ペン**で**書きます。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| どこで勉強しますか。 | 圖書館で勉強します。 |
| 圖書館で何をしますか。 | 圖書館で勉強します。 |
| 家で勉強しますか。 | いいえ、圖書館で勉強します。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>勉強<rt>べんきょう</rt></ruby>します。 |
| +場所 | **圖書館で**<ruby>勉強<rt>べんきょう</rt></ruby>します。 |
| +時間 | **午後**圖書館で勉強します。 |
| +共同主體 | 午後**友達と**圖書館で勉強します。 |
