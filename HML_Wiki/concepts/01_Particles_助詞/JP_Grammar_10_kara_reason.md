# 〜から（原因）

## 規則與接續
- `小句（普通形/敬體形）+ から`

## 用法說明
表示主觀的原因或理由（因為...）。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E"><ruby>危<rt>あぶ (abu)</rt></ruby>ないから、</span><span style="color:#3498DB"><ruby>入<rt>はい (hai)</rt></ruby>らないでください。</span>
- 中文：因為危險，請不要進去。
- 日文：<span style="color:#FF477E"><ruby>忙<rt>いそが (isoga)</rt></ruby>しいですから、</span><span style="color:#3498DB"><ruby>行<rt>い (i)</rt></ruby>けません。</span>
- 中文：因為很忙，所以去不了。

## 關係說明
- 表示原因，在禮貌口吻中常被更客觀的 [[JP_Grammar_27_node|〜ので @replaces]] 代替。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> 忙（いそが）しいですから、行（い）けません。
> （因為很忙，所以去不了。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 危（あぶ）ない | **危（あぶ）ない**ですから、行（い）けません。 | 因為危險，所以去不了。 |
| 遠（とお）い | **遠（とお）い**ですから、行（い）けません。 | 因為很遠，所以去不了。 |
| 遅（おそ）い | **遅（おそ）い**ですから、行（い）けません。 | 因為太晚了，所以去不了。 |
| 寒（さむ）い | **寒（さむ）い**ですから、行（い）けません。 | 因為很冷，所以去不了。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 普通形結尾（から） | 忙（いそが）しい**から**、行（い）けません。 | 因為很忙，所以去不了。 |
| → 過去原因 | 忙（いそが）し**かったですから**、行（い）けませんでした。 | 因為以前很忙，所以以前去不了。 |
| → 祈使句型 | 危（あぶ）ないから、**入（はい）らないでください**。 | 因為危險，請不要進去。 |
| → 原因結果倒裝 | どうして行（い）けませんか。忙（いそが）しい**からです**。 | 為什麼去不了？因為很忙。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| どうして今日（きょう）来（き）ませんか。 | 忙（いそが）しいですから、行（い）けません。 | 因為很忙，所以去不了。 |
| どうしてビールを飲（の）みませんか。 | 車（くるま）を運転（うんてん）しますから、飲（の）みません。 | 因為要開車，所以不喝。 |
| ここに入（はい）ってもいいですか。 | いいえ、危（あぶ）ないですから、入（はい）らないでください。 | 不，因為危險，請不要進去。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 行（い）けません。 | 去不了。 |
| +原因 | 忙（いそが）しいですから、**行（い）けません**。 | 因為很忙，所以去不了。 |
| +同伴 | 忙（いそが）しいですから、**友達（ともだち）と**行（い）けません。 | 因為很忙，所以不能和朋友一起去。 |
| +地點 | 忙（いそが）しいですから、友達（ともだち）と**日本（にほん）へ**行（い）けません。 | 因為很忙，所以不能和朋友一起去日本。 |
