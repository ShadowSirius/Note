# 〜から（原因）

## 規則與接續
- `小句（普通形/敬體形）+ から`

## 用法說明
表示主觀的原因或理由（因為...）。

## 實用例句
- 日文：<ruby>危<rt>あぶ (abu)</rt></ruby>ないから、<ruby>入<rt>はい (hai)</rt></ruby>らないでください。
- 中文：因為危險，請不要進去。
- 日文：<ruby>忙<rt>いそが (isoga)</rt></ruby>しいですから、<ruby>行<rt>い (i)</rt></ruby>けません。
- 中文：因為很忙，所以去不了。

## 關係說明
- 表示原因，在禮貌口吻中常被更客觀的 [[JP_Grammar_27_node|〜ので @replaces]] 代替。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>忙<rt>いそが</rt></ruby>しいですから、<ruby>行<rt>い</rt></ruby>けません。
> 因為很忙，所以去不了。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| <ruby>時間<rt>じかん</rt></ruby>がありません | **<ruby>時間<rt>じかん</rt></ruby>がありません**から、行けません。 |
| <ruby>用事<rt>ようじ</rt></ruby>があります | **<ruby>用事<rt>ようじ</rt></ruby>があります**から、行けません。 |
| <ruby>買<rt>か</rt></ruby>えません | 忙しいですから、**<ruby>買<rt>か</rt></ruby>えません**。 |
| <ruby>勉強<rt>べんきょう</rt></ruby>できません | 忙しいですから、**<ruby>勉強<rt>べんきょう</rt></ruby>できません**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 普通形接續 | 忙しい**から**、行けません。 |
| → 過去形 | 忙し**かったです**から、行けませんでした。 |
| → 疑問形 | どうして行けませんか。 |
| → 倒裝句 | 行けません、忙しいですから。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| どうして行けませんか。 | 忙しいですから、行けません。 |
| 一緒に映画を見に行きませんか。 | すみません、用事がありますから、行けません。 |
| どうして食べませんか。 | お腹がいっぱいですから、食べません。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>行<rt>い</rt></ruby>けません。 |
| +結果理由連結 | **忙しいですから**、<ruby>行<rt>い</rt></ruby>けません。 |
| +詳細理由 | **今日**忙しいですから、行けません。 |
| +道歉語 | **すみません**、今日忙しいですから、行けません。 |
