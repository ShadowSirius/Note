# 〜から 〜まで

## 規則與接續
- `名詞/時間 + から + 名詞/時間 + まで`

## 用法說明
表示時間或空間的起點（から）與終點（まで）。

## 實用例句
> [!tip] 句子結構色彩：🟢主語/主題 \| 🔵核心/謂語 \| 🟠修飾語 \| 🔴時間/場所/副詞

- 日文：<span style="color:#FF477E">９<ruby>時<rt>じ (ji)</rt></ruby>から５<ruby>時<rt>じ (ji)</rt></ruby>まで</span><span style="color:#3498DB"><ruby>働<rt>はたら (hatara)</rt></ruby>きます。</span>
- 中文：從9點工作到5點。
- 日文：<span style="color:#FF477E"><ruby>東京<rt>とうきょう (toukyou)</rt></ruby>から<ruby>大阪<rt>おおさか (oosaka)</rt></ruby>まで</span><span style="color:#3498DB"><ruby>行<rt>い (i)</rt></ruby>きます。</span>
- 中文：從東京去到大阪。

## 關係說明
- 用於標示起點與終點。

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> ９時（くじ）から５時（ごじ）まで働（はたら）きます。
> （從9點工作到5點。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| ８時 / ４時 | **８時（はちじ）**から**４時（よじ）**まで働（はたら）きます。 | 從8點工作到4點。 |
| 10時 / ６時 | **10時（じゅうじ）**から**６時（ろくじ）**まで働（はたら）きます。 | 從10點工作到6點。 |
| 月曜日 / 金曜日 | **月曜日（げつようび）**から**金曜日（きんようび）**まで働（はたら）きます。 | 從星期一工作到星期五。 |
| 朝 / 夜 | **朝（あさ）**から**夜（よる）**まで働（はたら）きます。 | 從早上工作到晚上。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 過去形 | ９時（くじ）から５時（ごじ）まで**働（はたら）きました**。 | 從9點工作到了5點。 |
| → 否定形 | ９時（くじ）から５時（ごじ）まで**働（はたら）きません**。 | 不從9點工作到5點。 |
| → 疑問形 | 何時（なんじ）から何時（なんじ）まで働（はたら）きますか。 | 從幾點工作到幾點？ |
| → 地點助詞句型 | 東京（とうきょう）から大阪（おおさか）まで**行（い）きます**。 | 從東京去到大阪。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| 何時（なんじ）から何時（なんじ）まで働（はたら）きますか。 | ９時（くじ）から５時（ごじ）まで働（はたら）きます。 | 從9點工作到5點。 |
| 月曜日（げつようび）から金曜日（きんようび）まで勉強（べんきょう）しますか。 | はい、月曜日（げつようび）から金曜日（きんようび）まで勉強（べんきょう）します。 | 是的，從星期一學習到星期五。 |
| 東京（とうきょう）から大阪（おおさか）までどうやって行（い）きますか。 | 新幹線（しんかんせん）で行（い）きます。 | 搭新幹線去。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | 働（はたら）きます。 | 工作。 |
| +起點 | **９時（くじ）から**働（はたら）きます。 | 從9點開始工作。 |
| +終點 | ９時（くじ）から**５時（ごじ）まで**働（はたら）きます。 | 從9點工作到5點。 |
| +主語 | **私（わたし）は**９時（くじ）から５時（ごじ）まで働（はたら）きます。 | 我從9點工作到5點。 |
