# 〜から 〜まで

## 規則與接續
- `名詞/時間 + から + 名詞/時間 + まで`

## 用法說明
表示時間或空間的起點（から）與終點（まで）。

## 實用例句
- 日文：９<ruby>時<rt>じ (ji)</rt></ruby>から５<ruby>時<rt>じ (ji)</rt></ruby>まで<ruby>働<rt>はたら (hatara)</rt></ruby>きます。
- 中文：從9點工作到5點。
- 日文：<ruby>東京<rt>とうきょう (toukyou)</rt></ruby>から<ruby>大阪<rt>おおさか (oosaka)</rt></ruby>まで<ruby>行<rt>い (i)</rt></ruby>きます。
- 中文：從東京去到大阪。

## 關係說明
- 用於標示起點與終點。

---

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> ９<ruby>時<rt>じ</rt></ruby>から５<ruby>時<rt>じ</rt></ruby>まで<ruby>働<rt>はたら</rt></ruby>きます。
> 從9點工作到5點。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| 10時 / 6時 | **10時**から**6時**まで働きます。 |
| 8時 / 5時半 | **8時**から**5時半**まで働きます。 |
| 月曜日 / 金曜日 | **月曜日**から**金曜日**まで働きます。 |
| 東京 / 大阪 + 行きます | **東京**から**大阪**まで**行きます**。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 過去形 | ９時から５時まで**働きました**。 |
| → 否定形 | ９時から５時まで**働きません**。 |
| → 疑問形 | 何時から何時まで働きます**か**。 |
| → 起點單獨使用 | ９時**から**働きます。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| 何時から何時まで働きますか。 | ９時から５時まで働きます。 |
| 土曜日も働きますか。 | いいえ、月曜日から金曜日まで働きます。 |
| 東京から大阪まで何で行きますか。 | 新幹線で行きます。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>働<rt>はたら</rt></ruby>きます。 |
| +終點 | **５時まで**<ruby>働<rt>はたら</rt></ruby>きます。 |
| +起點 | **９時から**５時まで働きます。 |
| +副詞 | **毎日**９時から５時まで働きます。 |
