# 〜の

## 規則與接續
- `名詞 + の + 名詞`

## 用法說明
表示所有格、屬性、關係或修飾（的）。

## 實用例句
- 日文：<ruby>私<rt>わたし (watashi)</rt></ruby>の<ruby>本<rt>ほん (hon)</rt></ruby>です。
- 中文：是我的書。
- 日文：<ruby>日本語<rt>にほんご (nihongo)</rt></ruby>の<ruby>先生<rt>せんせい (sensei)</rt></ruby>です。
- 中文：是日語老師。

## 關係說明
- 用於名詞修飾，為 [[JP_Grammar_01_wa_desu|〜は〜です @related_to]] 主題句中不可或缺的基礎助詞。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> <ruby>私<rt>わたし</rt></ruby>の<ruby>本<rt>ほん</rt></ruby>です。
> 是我的書。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| <ruby>車<rt>くるま</rt></ruby> | 私の**<ruby>車<rt>くるま</rt></ruby>**です。 |
| <ruby>鍵<rt>かぎ</rt></ruby> | 私の**<ruby>鍵<rt>かぎ</rt></ruby>**です。 |
| <ruby>携帯<rt>けいたい</rt></ruby> | 私の**<ruby>携帯<rt>けいたい</rt></ruby>**です。 |
| <ruby>先生<rt>せんせい</rt></ruby> | **<ruby>先生<rt>せんせい</rt></ruby>**の本です。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定形 | 私の本**ではありません**。 |
| → 過去形 | 私の本**でした**。 |
| → 疑問形 | あなたの本です**か**。 |
| → 所有代名詞化 | これは私の**の**です。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| これは誰の本ですか。 | それは私の本です。 |
| それは日本語の先生の本ですか。 | いいえ、これは英語の先生の本です。 |
| この携帯はあなたのですか。 | はい、それは私のです。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>本<rt>ほん</rt></ruby>です。 |
| +所有 | **私の**<ruby>本<rt>ほん</rt></ruby>です。 |
| +屬性 | 私の**日本語の**<ruby>本<rt>ほん</rt></ruby>です。 |
| +指示代名詞 | **これは**私の日本語の本です。 |
