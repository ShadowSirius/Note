# 〜が

## 規則與接續
- `名詞 + が + 動詞/形容詞`

## 用法說明
標記主語，常用於新資訊的導入，或與形容詞/能力/願望搭配。

## 實用例句
- 日文：<ruby>雨<rt>あめ (ame)</rt></ruby>が<ruby>降<rt>ふ (fu)</rt></ruby>っています。
- 中文：正在下雨。
- 日文：<ruby>私<rt>わたし (watashi)</rt></ruby>は<ruby>犬<rt>いぬ (inu)</rt></ruby>が<ruby>好<rt>す (su)</rt></ruby>きです。
- 中文：我喜歡狗。

- 作為主格助詞，經常與表示大主題的 [[JP_Grammar_01_wa_desu|〜は @contradicts]] 進行語義對比。

## 🎯 FSI Pattern Drill

> [!tip] 基礎句
> 私は<ruby>犬<rt>いぬ</rt></ruby>が<ruby>好<rt>す</rt></ruby>きです。
> 我喜歡狗。

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response |
|---|---|
| <ruby>猫<rt>ねこ</rt></ruby> | 私は**<ruby>猫<rt>ねこ</rt></ruby>**が好きです。 |
| <ruby>寿司<rt>すし</rt></ruby> | 私は**<ruby>寿司<rt>すし</rt></ruby>**が好きです。 |
| <ruby>日本語<rt>にほんご</rt></ruby> | 私は**<ruby>日本語<rt>にほんご</rt></ruby>**が好きです。 |
| <ruby>音楽<rt>おんがく</rt></ruby> | 私は**<ruby>音楽<rt>おんがく</rt></ruby>**が好きです。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response |
|---|---|
| → 否定形 | 私は犬が好き**ではありません**。 |
| → 過去形 | 私は犬が好き**でした**。 |
| → 疑問形 | あなたは犬が好きです**か**。 |
| → 主格の「が」 | 雨**が**降っています。 |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response |
|---|---|
| どんな動物が好きですか。 | 私は犬が好きです。 |
| 猫が好きですか。 | いいえ、猫は好きではありません。犬が好きです。 |
| 日本料理が好きですか。 | はい、私は日本料理が好きです。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence |
|---|---|
| 核心 | <ruby>好<rt>す</rt></ruby>きです。 |
| +對象 | **犬が**<ruby>好<rt>す</rt></ruby>きです。 |
| +主語 | **私は**犬が<ruby>好<rt>す</rt></ruby>きです。 |
| +程度 | 私は犬が**とても**好きです。 |
