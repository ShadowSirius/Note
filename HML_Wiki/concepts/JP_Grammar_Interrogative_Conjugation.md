# 🇯🇵 日語疑問詞與詞性對應指南 (Interrogatives & Word Classes)

> [!important] 句子結構色彩標記：
> 🟢 **主語/主題** (`#2ECC71`) \| 🔵 **核心/謂語** (`#3498DB`) \| 🟠 **修飾語/賓語** (`#E67E22`) \| 🔴 **時間/場所/副詞** (`#E74C3C`)

日語的疑問詞（何、誰、どこ 等）在句子中扮演特定的語法角色。相較於英文疑問詞（Who, What, Where 等）有時詞性多變，日語疑問詞的**品詞（詞性）對應非常精確**。理解這些對應關係有助於正確接續助詞與組織句型。

---

## 📂 第一部分：日、英、中疑問詞與詞性對照表

| 英文疑問詞 | 日語對應單字 | 日語品詞 (詞性) | 語法特性與接續規則 | 日語結構例句 |
| :--- | :--- | :--- | :--- | :--- |
| **Who** | **誰（だれ）** | **名詞 / 代名詞** | 可以像普通名詞一樣後接各式助詞（が、を、に、と）。 | <span style="color:#2ECC71">誰が</span><span style="color:#3498DB">来ますか。</span> <br>（誰要來？） |
| **What** | **何（なに / なん）** | **名詞 / 代名詞** | 接 t、d、n 行音（如 だ、の）時讀 `なん`，其餘讀 `なに`。 | <span style="color:#E67E22">何を</span><span style="color:#3498DB">食べますか。</span> <br>（你要吃什麼？） |
| **Where** | **どこ** | **場所代名詞** | 指代空間場所。後接場所助詞（に、で、へ）。 | <span style="color:#E74C3C">どこに</span><span style="color:#3498DB">住んでいますか。</span> <br>（你住在哪裡？） |
| **When** | **いつ** | **時間名詞 / 副詞** | 當時間名詞用時，**後方不加助詞「に」**。 | <span style="color:#E74C3C">いつ</span><span style="color:#3498DB">日本へ行きますか。</span> <br>（你什麼時候去日本？） |
| **Why** | **なぜ / どうして** | **副詞** | 放在句首或動詞前方，用來修飾後方的謂語（動詞/形容詞）。 | **どうして**<span style="color:#3498DB">遅刻しましたか。</span> <br>（為什麼遲到了？） |
| **How (修飾動詞)** <br>(如何做某事) | **どう / いかが** | **副詞** | 修飾動詞動作的進行方式。`いかが` 為 `どう` 的敬語。 | <span style="color:#E67E22">これを</span><span style="color:#E74C3C">どう</span><span style="color:#3498DB">使いますか。</span> <br>（這個要怎麼用？） |
| **How (修飾名詞)** <br>(怎樣的名詞) | **どんな + 名詞** | **連體詞** | **不可單獨使用**，後面必須緊接名詞。類似 English Determiner。 | <span style="color:#E67E22">どんな音楽を</span><span style="color:#3498DB">聞きますか。</span> <br>（你聽怎樣的音樂？） |
| **Which (多選一)** <br>(指代事物) | **どれ** | **代名詞** | 在三個以上事物中進行挑選。獨立作主語/賓語。 | <span style="color:#2ECC71">あなたの傘は</span><span style="color:#3498DB">どれですか。</span> <br>（你的傘是哪一把？） |
| **Which (多選一)** <br>(修飾名詞) | **どの + 名詞** | **連體詞** | 後面必須緊接名詞。 | <span style="color:#E67E22">どの本を</span><span style="color:#3498DB">買いましたか。</span> <br>（你買了哪本書？） |
| **How much/long** <br>(數量/長度/價格) | **どのくらい** <br>**いくら (價格)** | **數量副詞 / 名詞** <br>**名詞** | 修飾數量或程度。`いくら` 專指價格。 | <span style="color:#E74C3C">どのくらい</span><span style="color:#3498DB">かかりますか。</span> <br>（需要花費大約多久/多少？） |

## 📂 第二部分：疑問詞的助詞接續與「主語」鐵律

### 1. 疑問詞作「主語」時，助詞必須用 が
當疑問詞（如 誰、何、どこ）在句子中擔任主語時，**絕對不能使用「は」**，必須使用 **「が」**。同時，回答該問題時，答案的主語也必須用 **「が」**。
*   **正確**：<span style="color:#2ECC71">誰が</span><span style="color:#3498DB">来ましたか。</span> (誰來了？)
*   **錯誤**：~~誰は来ましたか。~~
*   **回答**：<span style="color:#2ECC71">山田さんが</span><span style="color:#3498DB">来ました。</span> (山田先生來了。)

### 2. 疑問詞 + か（不確定代詞）
當疑問詞後方加上助詞 **か** 時，會轉化為不確定的代名詞（相當於英文的 something, someone, somewhere）：
*   **何か（なに collision か / something）**：某些事物。
    *   例句：<span style="color:#E67E22">何か</span><span style="color:#3498DB">食べたいです。</span> (我想吃點什麼。)
*   **誰か（だれか / someone）**：某個人。
    *   例句：教室に<span style="color:#2ECC71">誰か</span><span style="color:#3498DB">いますか。</span> (教室裡有人嗎？)
*   **どこか（somewhere）**：某個地方。
    *   例句：<span style="color:#E74C3C">どこかへ</span><span style="color:#3498DB">行きたいです。</span> (我想去某個地方。)

### 3. 疑問詞 + も + 否定（全面否定）
當疑問詞後方加上 **も**，且句尾使用**否定形**時，代表全面否定（相當於英文的 nothing, no one, nowhere）：
*   **何も + 否定（nothing）**：什麼都不...。
    *   例句：<span style="color:#E67E22">何も</span><span style="color:#3498DB">食べません。</span> (我什麼都不吃。)
*   **誰聯も + 否定（no one）**：誰都不...。
    *   例句：教室に<span style="color:#2ECC71">誰も</span><span style="color:#3498DB">いません。</span> (教室裡誰都不在。)
*   **どこも + 否定（nowhere）**：哪裡都不...。
    *   例句：<span style="color:#E74C3C">どこへも</span><span style="color:#3498DB">行きませんでした。</span> (我哪裡都沒去。)
