# 🇯🇵 指示詞こそあど系統：記憶與應用指南

> [!important] 句子結構色彩標記：
> 🟢 **主語/主題** (`#2ECC71`) \| 🔵 **核心/謂語** (`#3498DB`) \| 🟠 **修飾語/賓語** (`#E67E22`) \| 🔴 **時間/場所/副詞** (`#E74C3C`)

---

## 🧠 記憶鉤（一律套用到所有系列）

> **こ = 近己（離我近）／そ = 近聽者（離對方近）／あ = 雙方皆遠（雙方都看得到但都離得遠，或彼此共同記憶中的事物）／ど = 疑問（不知道是哪一個）**

這一組鉤子同時套用在：これ系（物）、ここ系（地點）、こちら系（方向・人・禮貌）、この系（連體，修飾名詞）、こんな系（性狀，補充）。只要記住「己→近／聽→中／雙遠→遠／疑問→ど」，五套系列的字尾規律（これ・ここ・こちら・この・こんな）就能直接套用字首推出。

---

## 📂 系統總表（5 類別 × こ・そ・あ・ど）

| 類別 | こ系（近己） | そ系（近聽者） | あ系（雙方皆遠） | ど系（疑問） |
| :--- | :--- | :--- | :--- | :--- |
| **① 事物代名詞**<br>（獨立作主詞/受詞） | これ (kore／這個) | それ (sore／那個) | あれ (are／那個・遠處) | どれ (dore／哪個，三選一) |
| **② 場所代名詞** | ここ (koko／這裡) | そこ (soko／那裡) | あそこ (asoko／那裡・遠處) | どこ (doko／哪裡) |
| **③ 方向／人稱**<br>（禮貌指代） | こちら／こっち (kochira／kocchi／這邊・我方) | そちら／そっち (sochira／socchi／那邊・你方) | あちら／あっち (achira／acchi／那邊・遠方) | どちら／どっち (dochira／docchi／哪邊，二選一) |
| **④ 連體詞**<br>（不可獨立，須接名詞） | この (kono／這…) | その (sono／那…) | あの (ano／那…・遠處) | どの (dono／哪個…＋名詞) |
| **⑤ 性狀（補充）**<br>（＋な＋名詞，狀態如何） | こんな (konna／這樣的) | そんな (sonna／那樣的) | あんな (anna／那樣的・遠) | どんな (donna／怎樣的，見 [[JP_Grammar_Interrogative_Conjugation\|疑問詞卡]]) |

### 例句（來自使用者手抄筆記，逐句核對無誤）
- 日文：<span style="color:#E74C3C">ここに</span><span style="color:#3498DB">座ってください。</span> 中文：請坐這裡。
- 日文：<span style="color:#E74C3C">そこに</span><span style="color:#3498DB">本があります。</span> 中文：那裡有書。
- 日文：<span style="color:#E74C3C">あそこに</span><span style="color:#3498DB">公園があります。</span> 中文：那邊（較遠處）有公園。
- 日文：<span style="color:#E74C3C">ここらへんで</span><span style="color:#3498DB">待ち合わせましょう。</span> 中文：就約在這一帶碰面吧。（ここら＝ここ的口語鬆散版，指「這一帶」）
- 日文：<span style="color:#2ECC71">こちらへ</span><span style="color:#3498DB">どうぞ。</span> 中文：請往這邊走。（禮貌，服務業常用）
- 日文：<span style="color:#2ECC71">そちらは</span><span style="color:#3498DB">どうですか。</span> 中文：您那邊還好嗎？
- 日文：<span style="color:#E74C3C">あちらに</span><span style="color:#3498DB">郵便局があります。</span> 中文：那邊有郵局。
- 日文：<span style="color:#2ECC71">これは</span><span style="color:#3498DB">私のペンです。</span> 中文：這是我的筆。
- 日文：<span style="color:#E67E22">それを</span><span style="color:#3498DB">見てください。</span> 中文：請看那個。
- 日文：<span style="color:#2ECC71">あれが</span><span style="color:#3498DB">私たちの学校です。</span> 中文：那個（遠處）是我們的學校。
- 日文：<span style="color:#2ECC71">トイレは</span><span style="color:#3498DB">どこですか。</span> 中文：廁所在哪裡？
- 日文：<span style="color:#2ECC71">どれが</span><span style="color:#3498DB">好きですか。</span> 中文：你喜歡哪一個？（三個以上選一）
- 日文：<span style="color:#E67E22">映画と演劇と</span><span style="color:#3498DB">どちらが好きですか。</span> 中文：電影和舞台劇，你比較喜歡哪一個？（二選一）

## ⚠️ 易混淆辨析

| 對照組 | 差異重點 |
| :--- | :--- |
| **こっち vs こちら** | 意思相同（這邊），但 **こっち 為口語隨便**，こちら **較禮貌／正式**，服務業、對客人、書信用語一律用 こちら／そちら／あちら／どちら。 |
| **どれ vs どちら** | **どれ＝三個（含）以上選一**；**どちら＝兩個之中選一**，且 どちら 同時也是禮貌版的「どこ・だれ」（如「どちら様ですか」＝您是哪位）。 |
| **それ vs あれ** | それ＝物品在**聽者附近**（中稱）；あれ＝物品**離雙方都遠**，或雙方都知道、記憶中共有的事物（遠稱）。判斷基準是「聽者的距離」，不是說話者自己的距離。 |
| **これから vs ここから** | 兩者字面都是「從…起」，但 **これから＝時間起點**（從現在起）；**ここから＝空間起點**（從這個地點起）。莫因為長得像而混用。 |

## 🎯 FSI Pattern Drill

> [!tip] Base Sentence
> これは私（わたし）のペンです。
> （這是我的筆。）

### 1️⃣ Substitution Drill（替換練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| それ／傘（かさ） | **それは**私（わたし）の傘（かさ）です。 | 那是我的傘。 |
| あれ／車（くるま） | **あれが**私（わたし）たちの車（くるま）です。 | 那（遠處）是我們的車。 |
| どれ／かばん | **どれが**あなたのかばんですか。 | 哪一個是你的包包？ |
| この／本（ほん） | **この本（ほん）は**面白（おもしろ）いです。 | 這本書很有趣。 |

### 2️⃣ Transformation Drill（轉換練習）

| Transformation | Expected Response | Translation（中文翻譯） |
|---|---|---|
| → 場所代名詞 | **ここは**私（わたし）の部屋（へや）です。 | 這裡是我的房間。 |
| → 禮貌方向 | **こちらへ**どうぞ。 | 請往這邊走。 |
| → 疑問（二選一） | コーヒーと紅茶（こうちゃ）と**どちらが**好（す）きですか。 | 咖啡和紅茶，你比較喜歡哪一個？ |
| → 疑問（三選一以上） | この中（なか）で**どれが**一番（いちばん）安（やす）いですか。 | 這當中哪一個最便宜？ |

### 3️⃣ Response Drill（應答練習）

| Prompt | Expected Response | Translation（中文翻譯） |
|---|---|---|
| これはあなたの傘（かさ）ですか。 | はい、それは私（わたし）の傘（かさ）です。 | 是的，那是我的傘。 |
| トイレはどこですか。 | あちらです。 | 在那邊。 |
| 映画（えいが）と演劇（えんげき）とどちらが好（す）きですか。 | 映画（えいが）のほうが好（す）きです。 | 我比較喜歡電影。 |

### 4️⃣ Expansion Drill（擴展練習）

| Step | Sentence | Translation（中文翻譯） |
|---|---|---|
| 核心 | ペンです。 | 是筆。 |
| +指示詞 | **これは**ペンです。 | 這是筆。 |
| +所有格 | これは**私（わたし）の**ペンです。 | 這是我的筆。 |
| +語氣助詞 | これは私（わたし）のペンです**よ**。 | 這是我的筆喔。 |
